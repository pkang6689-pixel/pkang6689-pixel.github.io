Lesson 7.8: Technology in Business Finance - Summary

Key Concepts: Technology in Business Finance

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is 'cloud accounting'?

Cloud accounting refers to accounting software hosted on remote servers (the 'cloud') that can be accessed from any internet-connected device, enabling real-time collaboration and automatic data backup.

Using Excel's PMT function, a business calculates a $200,000 loan at 6% annual rate for 15 years. The monthly payment is approximately $1,688. What is the total interest paid?

Total payments = $1,688 x 180 months = $303,840. Total interest = $303,840 - $200,000 = $103,840.

A business pays $65/month for QuickBooks, $45/month for payroll service, and $30/month for POS software. What is the total annual cost of financial technology?

Monthly total = $65 + $45 + $30 = $140. Annual cost = $140 x 12 = $1,680. These tools are highly cost-effective compared to hiring a full-time bookkeeper.

A business creates a financial model in Excel projecting 3 scenarios: Base case ($80K profit), Optimistic ($120K profit, 25% probability), and Pessimistic ($30K profit, 35% probability). Base case probability is 40%. What is the expected profit?

Expected profit = ($120K x 0.25) + ($80K x 0.40) + ($30K x 0.35) = $30,000 + $32,000 + $10,500 = $72,500.

Real-World Applications

Example: Cloud accounting advantages include:

Cloud-based accounting provides remote access, automatic data syncing with bank accounts, real-time collaboration with bookkeepers/accountants, and automatic data backups.

Example: What does 'Goal Seek' do in Excel?

Goal Seek is an Excel what-if analysis tool that works backward: you specify a target result (e.g., a specific profit level), and it calculates what input value (e.g., price or volume) is needed to achieve it.

Example: A company uses Excel's Goal Seek to find what price is needed to achieve $50,000 profit. Fixed costs are $80,000, variable cost is $15/unit, and projected volume is 5,000 units. What price does Goal Seek return?

Required total CM = Fixed costs + Target profit = $80,000 + $50,000 = $130,000. CM per unit = $130,000 / 5,000 = $26. Price = Variable cost + CM = $15 + $26 = $41 per unit.

Example: An online store receives payments via Stripe. In January, it processes $45,000 in sales across 600 transactions. What are the total processing fees for January?

Percentage fee = $45,000 x 2.9% = $1,305. Per-transaction fee = 600 x $0.30 = $180. Total = $1,305 + $180 = $1,485.

Example: A business uses automated invoicing and reduces its average collection time from 45 days to 20 days. With $30,000 in monthly revenue, how much additional cash is available at any given time?

Daily revenue = $30,000 / 30 = $1,000. Days saved = 45 - 20 = 25 days. Additional cash freed up = 25 x $1,000 = $25,000 that is no longer tied up in outstanding receivables.

Example: A business evaluates two payment processors: Processor A charges 2.9% + $0.30 per transaction. Processor B charges a flat 2.5% with no per-transaction fee. The business averages 500 transactions/month at $80 each. Which is cheaper?

Processor A: ($80 x 0.029 + $0.30) x 500 = ($2.32 + $0.30) x 500 = $2.62 x 500 = $1,310/month. Processor B: $80 x 0.025 x 500 = $2.00 x 500 = $1,000/month. Processor B saves $310/month ($3,720/year).
