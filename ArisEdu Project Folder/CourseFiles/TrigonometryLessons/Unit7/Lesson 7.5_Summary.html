Lesson 7.5: Applications in Surveying - Summary

Key Concepts: Navigation & Bearing Applications

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'bearing' in navigation?

A bearing is an angular direction measured clockwise from north (three-digit system) or from N/S toward E/W (quadrant system).

What is 'true north'?

True north points toward the geographic North Pole, as opposed to magnetic north.

What is a 'heading' in aviation?

The heading is the compass direction the aircraft is pointed toward.

What is 'course' vs. 'heading'?

Wind or current can cause the actual path (course) to differ from the direction the craft faces (heading).

What is a 'nautical mile'?

A nautical mile is approximately 1.852 km and corresponds to one minute of latitude.

What is a 'knot'?

One knot = one nautical mile per hour, a standard speed unit in maritime and aviation navigation.

What is a 'triangulation' method in navigation?

Triangulation uses angle measurements from known points to determine an unknown position using triangle geometry.

What is the 'quadrant bearing' system?

The quadrant system specifies direction using N or S as the base and rotating toward E or W by a given angle.

What is 'dead reckoning'?

Dead reckoning calculates position by advancing a known position using course, speed, and elapsed time.

A search plane flies from base on bearing 025° for 80 km, then bearing 115° for 60 km. Using the Law of Cosines, what is the approximate straight-line distance back to base?

The angle between the legs at the turning point = 115° − 25° = 90°. d = √(80² + 60²) = √(10000) = 100 km.
