Lesson 6.1: Linear Trig Equations - Summary

Key Concepts: Solving Linear Trig Equations

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'reference angle'?

A reference angle is the positive acute angle between the terminal side of the angle and the nearest part of the x-axis.

What is the 'general solution' of a trigonometric equation?

The general solution expresses every possible solution by adding full-period multiples (like + 2nπ) where n is any integer.

What is meant by the 'period' of a trigonometric function?

The period is the length of one complete cycle of the function before it begins repeating.

What is a 'linear' trigonometric equation?

A linear trig equation has the trigonometric function raised only to the first power, such as 2 sin θ + 1 = 0.

What is a 'coterminal angle'?

Coterminal angles differ by multiples of 2π (or 360°) and share the same terminal side on the unit circle.

What is the 'unit circle'?

The unit circle has radius 1 and center at the origin; coordinates on it give cosine and sine values for each angle.

A sound technician measures that the displacement of a speaker cone follows d(t) = 0.3 sin(440πt). She needs the cone to be at displacement d = 0.15 mm. What is the first positive time this occurs?

Set 0.3 sin(440πt) = 0.15, so sin(440πt) = 1/2. The first solution is 440πt = π/6, giving t = 1/2640 s.

A pendulum's angular displacement is θ(t) = 0.2 sin(3πt). An engineer needs to know when the pendulum first passes through θ = 0.1 radians. What is the time?

Set 0.2 sin(3πt) = 0.1, so sin(3πt) = 1/2. The first positive solution is 3πt = π/6, so t = 1/18 s.

A surveyor sights a tower and records the equation tan α = 2.5, where α is the angle of elevation. She then moves to a point where tan β = 1. What is the angle β?

tan β = 1 gives β = 45° (or π/4 radians), since tangent equals 1 at that angle.

A physics student launches a projectile. The horizontal range equation simplifies to sin α = 0.6, where α is the launch angle between 0° and 90°. What is α to the nearest degree?

α = arcsin(0.6) ≈ 36.87° ≈ 37°. Since 0 < α < 90°, this is the only solution in the given range.
