Lesson 6.7: Applications in AP Prep - Summary

Key Concepts: AP Exam Preparation

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is an 'identity' in trigonometry?

An identity holds for every permissible value of the variable, unlike a conditional equation.

What is the 'sum-to-product' identity?

Sum-to-product converts a sum of sines (or cosines) into a product, which is useful for factoring.

What is the 'product-to-sum' identity?

Product-to-sum converts a product of trig functions into a sum or difference.

What is an AP 'free response' strategy for trig equations?

AP free-response requires clear, justified work showing each step of the solution process.

An AP problem states: 'A particle moves along the x-axis with position x(t) = 3sin(2t) + 4cos(2t). Find the amplitude of oscillation.' What is the answer?

Amplitude = √(3² + 4²) = √25 = 5, using the formula for combining sine and cosine with the same frequency.

An AP multiple-choice question asks: 'How many solutions does sin(3x) = 1/2 have in [0, 2π)?' What is the answer?

3x ∈ [0, 6π). sin(3x) = 1/2 has 2 solutions per 2π-period × 3 periods = 6 solutions.

Real-World Applications

Example: What does 'extraneous solution' mean on an AP exam?

Extraneous solutions arise from non-reversible operations like squaring both sides, and must be checked against the original equation.

Example: On a practice AP exam, a student encounters: 'Find all solutions of 2sin²x − cos x − 1 = 0 in [0, 2π).' Using sin²x = 1 − cos²x, how many solutions exist?

Substituting gives 2cos²x + cosx − 1 = 0 → (2cosx − 1)(cosx + 1) = 0. cosx = 1/2 gives x = π/3, 5π/3; cosx = −1 gives x = π. Total: 3 solutions.

Example: An AP multiple-choice question asks: 'How many solutions does sin(3x) = 1/2 have in [0, 2π)?' What is the answer?

3x ∈ [0, 6π). sin(3x) = 1/2 has 2 solutions per 2π-period × 3 periods = 6 solutions.

Example: A student must solve sin x cos x = 1/4 on an AP exam. A key first step is to recognize this equals:

Using sin(2x) = 2sinx cosx, we get sinx cosx = (1/2)sin(2x) = 1/4, so sin(2x) = 1/2.
