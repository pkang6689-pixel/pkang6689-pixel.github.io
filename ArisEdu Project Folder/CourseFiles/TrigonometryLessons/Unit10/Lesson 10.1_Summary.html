Lesson 10.1: Trigonometric Proofs - Summary

Key Concepts: Trigonometric Proofs & Verification

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

Verify: tan(x) + cot(x) = sec(x)csc(x). What is the first step?

Convert to sines and cosines: sin/cos + cos/sin = (sin^2 + cos^2)/(sin*cos) = 1/(sin*cos) = sec*csc. This verifies the identity.

What is sin(x) * csc(x)?

sin(x) * csc(x) = sin(x) * (1/sin(x)) = 1. A function times its reciprocal always equals 1 (when defined).

What is a trigonometric identity?

A trigonometric identity is an equation that holds true for every value of the variable(s) where both sides are defined. Examples include sin^2(x) + cos^2(x) = 1 and tan(x) = sin(x)/cos(x).

What is a cofunction identity?

Cofunction identities relate a trig function to its cofunction applied to the complementary angle: sin(pi/2 - x) = cos(x), tan(pi/2 - x) = cot(x), sec(pi/2 - x) = csc(x), and their reverses.

What is the first recommended strategy when verifying a trig identity?

The standard approach is to start with the more complicated side and use known identities, algebraic manipulation, and simplification to transform it into the simpler side. You should NOT treat it as an equation and perform operations on both sides simultaneously.

What is the double angle formula for sin(2x)?

The double angle formula for sine is sin(2x) = 2sin(x)cos(x). This follows from the angle addition formula sin(x+x) = sin(x)cos(x) + cos(x)sin(x) = 2sin(x)cos(x).

What is the sum-to-product identity for sin(A) + sin(B)?

sin(A) + sin(B) = 2sin((A+B)/2)cos((A-B)/2). Sum-to-product identities convert sums of trig functions into products, often useful in proofs and simplifications.

What is the half-angle formula for sin(x/2)?

sin(x/2) = +-sqrt((1 - cos(x))/2), where the sign depends on the quadrant of x/2. This is derived from the double angle formula cos(x) = 1 - 2sin^2(x/2).

A student must prove: csc(x) - sin(x) = cos(x)cot(x). Working from the left side, what is the result?

LHS = 1/sin(x) - sin(x) = (1 - sin^2(x))/sin(x) = cos^2(x)/sin(x) = cos(x) * cos(x)/sin(x) = cos(x)cot(x) = RHS.

Real-World Applications

Example: A teacher asks students to disprove: sin(A + B) = sin(A) + sin(B). Which counterexample works?

Let A = B = pi/2: LHS = sin(pi) = 0, but RHS = sin(pi/2) + sin(pi/2) = 1 + 1 = 2. Since 0 ≠ 2, the statement is false. This demonstrates that sine is NOT a linear function.
