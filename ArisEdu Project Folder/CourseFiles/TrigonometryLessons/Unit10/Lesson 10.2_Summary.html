Lesson 10.2: Trigonometric Substitutions (integration prep) - Summary

Key Concepts: Trigonometric Substitution

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If x = 5tan(theta), what is sqrt(25 + x^2) in terms of theta?

25 + x^2 = 25 + 25tan^2(theta) = 25(1 + tan^2(theta)) = 25sec^2(theta). So sqrt(25 + x^2) = 5sec(theta).

Using x = 4sec(theta), what is sqrt(x^2 - 16)?

x^2 - 16 = 16sec^2(theta) - 16 = 16(sec^2(theta) - 1) = 16tan^2(theta). sqrt = 4|tan(theta)| = 4tan(theta) for appropriate theta range.

What is trigonometric substitution?

Trigonometric substitution replaces a variable (like x) with a trig expression (like a sin(theta)) to eliminate square roots in integrals. The Pythagorean identities transform the radical into a simple trig expression.

What is the reference triangle in the context of trig substitution?

A reference triangle is a right triangle constructed from the substitution (e.g., if x = a sin(theta), the triangle has opposite = x, hypotenuse = a). It allows converting the theta-based answer back to the original variable x.

What is back-substitution in trig substitution?

After integrating in terms of theta, back-substitution converts the result back to x. Using theta = arcsin(x/a) and the reference triangle, every trig function of theta can be expressed in terms of x.

What is completing the square, and why is it related to trig substitution?

Completing the square converts expressions like x^2 + 6x + 5 into (x+3)^2 - 4, which has the form u^2 - a^2 suitable for trig substitution. It is an essential preparatory step for many trig substitution problems.

What is the integral of sec(theta)?

The integral of sec(theta) = ln|sec(theta) + tan(theta)| + C. This result frequently appears when using trig substitution with x = a tan(theta).

The area of a circle of radius r is found by evaluating 4 * integral from 0 to r of sqrt(r^2 - x^2) dx. Using x = r sin(theta), what is the resulting integral?

sqrt(r^2 - x^2) = r cos(theta), dx = r cos(theta) d(theta). When x=0, theta=0; when x=r, theta=pi/2. Integral = 4 * integral of r cos(theta) * r cos(theta) d(theta) = 4r^2 * integral of cos^2(theta) d(theta) from 0 to pi/2.
