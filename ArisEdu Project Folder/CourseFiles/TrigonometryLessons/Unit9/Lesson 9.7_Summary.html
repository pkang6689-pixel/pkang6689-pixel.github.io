Lesson 9.7: Applications in AP Calculus - Summary

Key Concepts: AP Calculus Connections

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

The velocity vector is the derivative of position: v(t) = r'(t) = = <2t, 3>.

What is the speed of a particle at t = 2 if v(t) = <3, 4t>?

At t = 2: v(2) = <3, 8>. Speed = |v(2)| = sqrt(9 + 64) = sqrt(73) ≈ 8.54.

v(t) = integral of a(t) dt + v(0) = <0t + 10, -9.8t + 20> = <10, 20 - 9.8t>. The velocity is found by integrating acceleration and applying the initial condition.

What is a vector-valued function?

A vector-valued function maps a real number t to a vector r(t) = . The parameter t often represents time, and the output traces a curve in the plane or space as t varies.

What is the arc length formula for a vector-valued function r(t)?

The arc length is L = integral from a to b of |r'(t)| dt = integral of sqrt((dx/dt)^2 + (dy/dt)^2) dt. This integrates the speed along the curve to find the total distance traveled.

What is the gradient vector of a function f(x, y)?

The gradient grad(f) = is a vector that points in the direction of the steepest increase of f at a given point. Its magnitude gives the rate of that steepest increase.

What is the directional derivative?

The directional derivative D_u(f) = grad(f) . u gives the rate of change of f in the direction of unit vector u. It uses the dot product to project the gradient onto the desired direction.

What is the unit tangent vector T(t)?

The unit tangent vector T(t) = r'(t)/|r'(t)| is the normalized velocity vector. It points in the direction of motion and has magnitude 1, separating direction from speed.

What is curvature of a curve?

Curvature kappa measures how sharply a curve bends at a point. It equals |dT/ds|, the rate at which the unit tangent vector changes with respect to arc length. A straight line has zero curvature; a small circle has high curvature.

What is the difference between distance traveled and displacement for a particle?

Distance traveled = integral of |v(t)| dt (always positive, total path length). Displacement = r(b) - r(a) (vector from start to end). Distance >= |displacement|, with equality only if the path is a straight line.

What is the relationship between integration and vector-valued functions?

Integration of a vector-valued function is performed component by component. This allows finding position from velocity (or velocity from acceleration) by integrating each component separately and applying initial conditions.

What is a parametric curve?

A parametric curve expresses both x and y as separate functions of a parameter t: (x(t), y(t)). This allows representation of curves that cannot be written as y = f(x), like circles, loops, and paths that double back.

A particle moves along r(t) = <2cos(t), 2sin(t)> representing circular motion. What is the acceleration vector at t = pi/2, and what direction does it point?

v(t) = <-2sin(t), 2cos(t)>. a(t) = <-2cos(t), -2sin(t)>. At t = pi/2: a = <-2cos(pi/2), -2sin(pi/2)> = <0, -2>. This points from the position (0, 2) toward the center (0, 0), confirming centripetal acceleration.

A satellite's position is r(t) = <5cos(2t), 5sin(2t)> km, where t is in hours. What is the satellite's speed?

v(t) = <-10sin(2t), 10cos(2t)>. Speed = |v(t)| = sqrt(100sin^2(2t) + 100cos^2(2t)) = sqrt(100) = 10 km/h. The speed is constant for uniform circular motion.

A hiker climbs a hill whose elevation is h(x,y) = 1000 - x^2 - y^2 meters. At position (10, 20), what is the slope in the direction of <3, 4>?

grad(h) = <-2x, -2y> = <-20, -40> at (10, 20). Unit direction: <3/5, 4/5>. Directional derivative = <-20, -40> . <3/5, 4/5> = -60/5 + (-160)/5 = -220/5 = -44 m per unit distance.

Real-World Applications

Example: An AP Calculus exam problem asks: a particle moves so that its position at time t >= 0 is r(t) = <3t - t^3/3, 3t^2/2>. Find all times t > 0 when the particle is moving purely in the vertical direction.

v(t) = <3 - t^2, 3t>. Moving purely vertically means the x-component of velocity is 0: 3 - t^2 = 0, so t = sqrt(3). At this time, v = <0, 3sqrt(3)>, confirming purely vertical motion.
