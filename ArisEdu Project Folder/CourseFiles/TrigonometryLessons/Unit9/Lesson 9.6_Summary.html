Lesson 9.6: Case Studies in Computer Graphics - Summary

Key Concepts: Computer Graphics Applications

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

In 2D computer graphics, what is the result of rotating the point (1, 0) by 90° counterclockwise?

Using the rotation formula: x' = cos(90°)(1) - sin(90°)(0) = 0 and y' = sin(90°)(1) + cos(90°)(0) = 1. The point moves to (0, 1).

A 2D rotation matrix for angle theta is [[cos(theta), -sin(theta)], [sin(theta), cos(theta)]]. What is the rotation matrix for theta = 60°?

cos(60°) = 1/2 and sin(60°) = sqrt(3)/2. The rotation matrix is [[1/2, -sqrt(3)/2], [sqrt(3)/2, 1/2]].

A sprite at position (3, 4) is translated by the vector <-1, 2>. What is its new position?

Translation adds the vector to the position: (3 + (-1), 4 + 2) = (2, 6). Translation is the simplest transformation, just shifting position.

A game character moves at velocity <5, 3> pixels per frame. After 10 frames, what is the character's displacement?

Displacement = velocity * time = 10 * <5, 3> = <50, 30> pixels. Scalar multiplication scales each component by the number of frames.

A 3D model's surface normal at a point is n = <0, 1, 0>. A light source is in direction L = <1, 1, 0>. What is the cosine of the angle between them (used for diffuse lighting)?

cos(theta) = (n . L)/(|n||L|) = (0+1+0)/(1*sqrt(2)) = 1/sqrt(2) ≈ 0.707. This value is used in Lambert's cosine law for diffuse shading.

In ray tracing, the reflection of an incoming ray d off a surface with normal n is given by r = d - 2(d . n)n. If d = <1, -1> and n = <0, 1>, what is r?

d . n = 1(0) + (-1)(1) = -1. r = <1, -1> - 2(-1)<0, 1> = <1, -1> + <0, 2> = <1, 1>. The ray bounces upward at the same horizontal direction.

The dot product of vectors <2, 3> and <4, -1> determines whether the angle between two game objects is acute or obtuse. What is the dot product and the angle type?

Dot product = 2(4) + 3(-1) = 8 - 3 = 5. Since the dot product is positive, cos(theta) > 0, meaning the angle is acute (less than 90°).

A camera at position C = (10, 5) looks at target T = (3, 1). What is the direction vector from camera to target?

Direction = T - C = (3-10, 1-5) = <-7, -4>. This vector points from the camera toward the target and would be normalized for use as the viewing direction.

What is a transformation in computer graphics?

A transformation in computer graphics is a mathematical operation (often represented by a matrix) that modifies geometric objects. Common transformations include translation, rotation, scaling, and reflection.

What is a surface normal in 3D graphics?

A surface normal is a unit vector perpendicular to a surface at a given point. Normals are essential for lighting (determining brightness based on the angle between light and surface) and for collision detection.

What is a translation in the context of 2D graphics?

A translation moves every point of an object by adding the same displacement vector . The shape, size, and orientation of the object remain unchanged. P' = P + .

What is Lambert's cosine law in 3D rendering?

Lambert's cosine law states that the apparent brightness (diffuse reflection) of a surface is proportional to cos(theta), where theta is the angle between the surface normal and the direction to the light source. This is computed using the dot product: n . L.

What is a rotation matrix?

A rotation matrix rotates all points around the origin by angle theta. In 2D: R = [[cos(theta), -sin(theta)], [sin(theta), cos(theta)]]. It preserves distances (isometry) and the determinant is always 1.

What is the purpose of normalizing a vector in graphics?

Normalizing a vector divides it by its magnitude to produce a unit vector (magnitude 1) pointing in the same direction. This is essential in lighting, physics, and direction calculations where only direction matters.

What is a homogeneous coordinate system in graphics?

Homogeneous coordinates represent a 2D point (x, y) as (x, y, 1). This allows translations, rotations, and scaling to all be represented as 3x3 matrix multiplications, enabling them to be combined (composed) efficiently.

What is a viewing (or look-at) vector in 3D graphics?

The viewing (or look-at) vector goes from the camera position to the target point the camera is looking at. It is typically normalized and used to construct the camera's coordinate system for rendering.

What is back-face culling in 3D rendering?

Back-face culling skips rendering polygons facing away from the camera. If the dot product of the surface normal and the vector toward the camera is negative, the polygon faces away and is not rendered, improving performance.

What is linear interpolation (lerp) between two vectors?

Linear interpolation smoothly blends between two vectors A and B: lerp(A, B, t) = (1-t)A + tB. At t=0, the result is A; at t=1, the result is B; at t=0.5, it is the midpoint. Used extensively for animation, shading, and transitions.

A game developer needs to rotate a spaceship sprite 45° counterclockwise. The ship's tip is at (10, 0). What is the new position of the tip?

x' = 10cos(45°) - 0sin(45°) = 10(sqrt(2)/2) = 5sqrt(2) and y' = 10sin(45°) + 0cos(45°) = 5sqrt(2). New position: (5sqrt(2), 5sqrt(2)) ≈ (7.07, 7.07).

A 3D renderer computes lighting for a surface with normal n = <0, 0, 1> and light direction L = <0, 1, 1> (toward the light). What is the diffuse intensity factor (n . L_normalized)?

L_normalized = <0, 1, 1>/sqrt(2) = <0, 1/sqrt(2), 1/sqrt(2)>. Intensity = n . L_normalized = 0 + 0 + 1/sqrt(2) = 1/sqrt(2) ≈ 0.707, meaning the surface receives about 70.7% of full brightness.

A pinball game simulates a ball bouncing off a wall with normal n = <1, 0> (pointing right). The ball's velocity is v = <-3, 4>. Using the reflection formula r = v - 2(v . n)n, what is the reflected velocity?

v . n = (-3)(1) + 4(0) = -3. r = <-3, 4> - 2(-3)<1, 0> = <-3, 4> + <6, 0> = <3, 4>. The horizontal component reverses while the vertical component stays the same.

A physics engine computes the collision normal between two circles centered at C1 = (2, 3) and C2 = (6, 6). What is the unit collision normal pointing from C1 to C2?

Direction = C2 - C1 = <4, 3>. Magnitude = sqrt(16+9) = 5. Unit normal = <4/5, 3/5>. This normalized vector is used to compute the collision response.
