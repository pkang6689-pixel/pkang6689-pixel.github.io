Lesson 2.6: Symmetry & Periodicity - Summary

Key Concepts: Symmetry & Periodicity

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A signal repeats every 2π seconds: f(t) = sin(t). A technician checks the signal at time t + 4π. What is f(t + 4π)?

sin has period 2π. 4π = 2 × 2π, so sin(t + 4π) = sin(t).

A rotating wheel completes one cycle every T seconds. The position is modeled by sin(2πt/T). What is sin(2π(t + T)/T)?

sin(2π(t+T)/T) = sin(2πt/T + 2π) = sin(2πt/T), by periodicity.

A tuning fork produces a tone modeled by f(t) = sin(880πt). What is the smallest positive T such that f(t + T) = f(t)?

T = 2π/(880π) = 1/440 s.

An oceanographer records sin(π/6) = 0.5. Using the odd property of sine, what is sin(−π/6)?

sin(−π/6) = −sin(π/6) = −0.5.
