Lesson 2.2: Reciprocal Functions (csc, sec, cot) - Summary

Key Concepts: Reciprocal Functions (csc, sec, cot)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A building's shadow indicates the sun's elevation angle θ, where sin θ = 3/5. A technician needs csc θ to complete a calculation. What is csc θ?

csc θ = 1/sin θ = 1/(3/5) = 5/3.

A security camera has a field of view described by angle θ where cos θ = 5/13. What is sec θ for this viewing angle?

sec θ = 1/cos θ = 1/(5/13) = 13/5.

An engineer measures that tan θ = 4/3 for a slope. What is cot θ for the same slope?

cot θ = 1/tan θ = 1/(4/3) = 3/4.

A physics student finds that for a projectile launched at angle θ, csc θ = 2. What is sin θ?

sin θ = 1/csc θ = 1/2, so θ = 30°.

A pilot needs sec θ for a navigation formula. If the aircraft climbs at angle θ where cos θ = √3/2, what is sec θ?

sec θ = 1/cos θ = 1/(√3/2) = 2/√3 = 2√3/3.

A satellite dish is tilted at angle θ. Measurements show sec θ = √2. What is the tilt angle in degrees?

sec θ = √2 → cos θ = 1/√2 = √2/2, so θ = 45°.

An architect finds that sec θ = 5/4 for a roof angle. What is tan θ for this roof?

sec θ = 5/4 → cos θ = 4/5 → sin θ = 3/5. tan θ = sin/cos = (3/5)/(4/5) = 3/4.
