Lesson 10.3: Applications in Optimization - Summary

Key Concepts: Applications in Optimization

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the maximum value of f(x) = -2x^2 + 8x + 1?

Vertex at x = -8/(2(-2)) = 2. f(2) = -2(4) + 8(2) + 1 = -8 + 16 + 1 = 9. Since a < 0, the parabola opens downward and 9 is the maximum.

What is the minimum value of f(x) = x^2 + 4x + 7?

Vertex at x = -4/2 = -2. f(-2) = 4 - 8 + 7 = 3. Since a = 1 > 0, the parabola opens upward and the minimum value is 3.

What is the maximum product of two numbers whose sum is 30?

P = x(30 - x) = 30x - x^2. Maximum at x = 15, so P = 15 * 15 = 225.

What is a 'critical point' of a function?

A critical point is a value in the domain of f where f'(x) = 0 or f'(x) does not exist. These are the candidates for local extrema.

What is a 'constraint' in an optimization problem?

A constraint is a condition (such as a fixed perimeter, budget, or material amount) that restricts which values the variables can take in an optimization problem.

What is the 'objective function' in an optimization problem?

The objective function is the quantity you seek to optimize (maximize or minimize). For example, if maximizing the area of a garden, the area formula is the objective function.

What is an 'inflection point'?

An inflection point is where a function changes concavity, switching from concave up to concave down or vice versa. At such points, f''(x) = 0 or is undefined.

A company finds its profit function is P(x) = -x^2 + 120x - 2000 dollars, where x is units produced. How many units maximize profit, and what is the maximum profit?

P'(x) = -2x + 120 = 0 gives x = 60. P(60) = -3600 + 7200 - 2000 = 1600. Since the leading coefficient is negative, x = 60 gives the maximum profit of $1600.

A ball is thrown upward from ground level with an initial velocity of 48 ft/s. Its height is h(t) = 48t - 16t^2. What is the maximum height reached?

h'(t) = 48 - 32t = 0 gives t = 1.5 s. h(1.5) = 48(1.5) - 16(2.25) = 72 - 36 = 36 ft.

Real-World Applications

Example: A store sells 200 shirts per month at $25 each. For every $1 price increase, sales drop by 5 shirts. What price maximizes monthly revenue?

Let x = number of $1 increases. Revenue R = (25 + x)(200 - 5x) = 5000 + 75x - 5x^2. R'(x) = 75 - 10x = 0 gives x = 7.5. Optimal price = 25 + 7.5 = $32.50.
