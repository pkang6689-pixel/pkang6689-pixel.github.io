Lesson 10.7: Capstone Project: Modeling with Functions - Summary

Key Concepts: Capstone Project: Modeling with Functions

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'mathematical model'?

A mathematical model uses functions, equations, or systems to represent and predict real-world phenomena. Models simplify reality to capture the most important relationships.

What is a 'residual' in regression analysis?

A residual is the difference between an observed data value and the value predicted by the model: residual = observed - predicted. Small residuals indicate a good fit.

What is the meaning of 'parameter' in a mathematical model?

A parameter is a constant in a model (like a, b in y = ab^x) whose specific value is determined by fitting the model to observed data. Changing parameters adjusts the model's shape.

What is 'regression' in the context of data analysis?

Regression is a statistical technique that determines the parameters of a function that best fits a set of data, typically by minimizing the sum of squared residuals.

What is the meaning of 'domain restriction' when applying a model to real-world data?

When modeling real situations, the mathematical domain must be restricted to values that are physically meaningful. For example, time and quantity cannot be negative in most models.

A town's population was 5000 in 2010 and 8000 in 2020. Assuming exponential growth P(t) = 5000*b^t where t is years since 2010, what is the predicted population in 2030?

At t = 10: 8000 = 5000*b^10, so b^10 = 1.6. At t = 20: P = 5000*b^20 = 5000*(b^10)^2 = 5000*(1.6)^2 = 5000*2.56 = 12800.

A scientist collects the following data for a decaying substance: (0, 100), (2, 82), (4, 67), (6, 55), (8, 45). She fits an exponential model y = 100*e^(-kt). Using the point (2, 82), what is the approximate value of k?

82 = 100*e^(-2k), so e^(-2k) = 0.82. Taking ln: -2k = ln(0.82) = -0.1985. Thus k = 0.0993, approximately 0.099.

A car's value depreciates: Year 0: $30,000; Year 1: $25,500; Year 2: $21,675; Year 3: $18,424. What type of model best fits this data, and what is the decay factor?

Checking ratios: 25500/30000 = 0.85, 21675/25500 = 0.85, 18424/21675 = 0.85. Constant ratio confirms exponential decay V(t) = 30000(0.85)^t with decay factor 0.85 (15% loss per year).

A biologist tracks a bacterial colony that grows logistically. The data shows: initial population 100, rapid growth phase reaching 500 after 6 hours, and leveling off near 1000. Using P(t) = 1000/(1 + 9e^(-0.5t)), what is the predicted population at t = 4 hours?

P(4) = 1000/(1 + 9e^(-2)) = 1000/(1 + 9(0.1353)) = 1000/(1 + 1.218) = 1000/2.218 = approximately 451 bacteria.

An economist models housing prices over 10 years. The data shows: prices rose linearly from $200K to $350K in years 0-5, then dropped linearly from $350K to $280K in years 5-10. What is the appropriate piecewise model?

First segment: slope = (350-200)/5 = 30, so P = 30t + 200. Second segment: slope = (280-350)/5 = -14, and at t=5, P=350, so 350 = -14(5) + b gives b = 420. Thus P = -14t + 420 for 5 < t <= 10.

A pharmaceutical researcher tests drug absorption: data shows rapid initial increase fitting a square root model y = 50*sqrt(t) for the first 4 hours. What is the predicted absorption at t = 9 hours if the model continues to hold?

y(9) = 50*sqrt(9) = 50 * 3 = 150 units. Note: this is extrapolation beyond the observed data range (0-4 hours), so the prediction should be used cautiously.

An environmental agency monitors carbon dioxide levels (ppm) over 12 months: the data appears sinusoidal with a mean of 415 ppm, amplitude of 5 ppm, and a period of 12 months (peaking in May, month 5). Using T(m) = 5*cos((2*pi/12)(m - 5)) + 415, what is the predicted CO2 level in September (month 9)?

T(9) = 5*cos((2*pi/12)(9 - 5)) + 415 = 5*cos(2*pi/3) + 415 = 5(-0.5) + 415 = -2.5 + 415 = 412.5 ppm.

Real-World Applications

Example: Which function type is best for modeling data that grows by a constant percentage each period?

Constant percentage growth (constant ratio between successive values) is modeled by an exponential function f(x) = ab^x, where b is the growth factor.

Example: Monthly average temperatures in a city vary between 30 F (January) and 80 F (July). Using a sinusoidal model T(m) = A*sin(B(m - C)) + D where m is the month number, what are A and D?

Amplitude A = (max - min)/2 = (80 - 30)/2 = 25. Vertical shift D = (max + min)/2 = (80 + 30)/2 = 55. So the temperature oscillates 25 degrees above and below 55 F.
