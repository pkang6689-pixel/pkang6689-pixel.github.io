Lesson 10.4: Area Under Curves (intro to integration) - Summary

Key Concepts: Area Under Curves (intro to integration)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the area under f(x) = 4 from x = 0 to x = 5?

The area under a constant function f(x) = 4 from x = 0 to x = 5 is a rectangle with width 5 and height 4, so A = 4 * 5 = 20.

What is the area under f(x) = x from x = 0 to x = 6?

The region is a triangle with base 6 and height 6. Area = (1/2)(6)(6) = 18.

What is the area under f(x) = 2x from x = 0 to x = 3?

The region is a triangle with base 3 and height f(3) = 6. Area = (1/2)(3)(6) = 9.

What is the exact area under f(x) = x^2 from x = 0 to x = 3?

The antiderivative of x^2 is x^3/3. Evaluating from 0 to 3: (3^3)/3 - 0 = 27/3 = 9.

Using the trapezoidal rule with n = 2 on f(x) = x^2 from x = 0 to x = 2, what is the estimate?

h = (2-0)/2 = 1. Trap = (h/2)[f(0) + 2f(1) + f(2)] = (1/2)[0 + 2(1) + 4] = (1/2)(6) = 3.

What is the area of the region bounded by f(x) = 3 and the x-axis from x = 2 to x = 7?

The region is a rectangle with width (7 - 2) = 5 and height 3. Area = 3 * 5 = 15.

What is a 'Riemann sum'?

A Riemann sum approximates the area under a curve by dividing the region into rectangles, computing each rectangle's area, and summing them all.

What is the 'definite integral'?

The definite integral of f(x) from a to b is defined as the limit of Riemann sums as the number of subintervals n approaches infinity and the width of each subinterval approaches zero.

What is an 'antiderivative' of a function f(x)?

An antiderivative (or indefinite integral) of f(x) is a function F(x) such that F'(x) = f(x). For example, x^3/3 is an antiderivative of x^2.

What is the 'trapezoidal rule'?

The trapezoidal rule approximates the area under a curve by connecting consecutive points with straight lines, forming trapezoids. It typically gives a better estimate than left or right Riemann sums.

What is the meaning of the integral symbol with limits: the integral from a to b of f(x) dx?

The notation represents the definite integral, which computes the net signed area between the curve y = f(x) and the x-axis over the interval [a, b].

A company's marginal revenue function is MR(x) = 50 - 0.4x dollars per unit. What is the total revenue earned from selling the first 100 units?

TR = integral from 0 to 100 of (50 - 0.4x) dx = [50x - 0.2x^2] from 0 to 100 = 5000 - 2000 = $3000.

A meteorologist models rainfall rate as r(t) = 0.5t inches per hour for t in [0, 4]. Using left-endpoint Riemann sum with 4 rectangles of width 1, what is the estimated total rainfall?

Left endpoints: t = 0, 1, 2, 3. Heights: r(0) = 0, r(1) = 0.5, r(2) = 1.0, r(3) = 1.5. Sum = 1(0 + 0.5 + 1.0 + 1.5) = 3 inches.
