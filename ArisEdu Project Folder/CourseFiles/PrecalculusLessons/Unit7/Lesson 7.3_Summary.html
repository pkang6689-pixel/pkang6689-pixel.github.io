Lesson 7.3: Double-Angle & Half-Angle Formulas - Summary

Key Concepts: Double-Angle & Half-Angle Formulas

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is tan 2θ if tan θ = 1/2?

tan 2θ = 2tanθ/(1−tan²θ) = 2(1/2)/(1−1/4) = 1/(3/4) = 4/3.

What is cos 2θ if cos θ = −3/5 (Q II)?

cos 2θ = 2cos²θ − 1 = 2(9/25) − 1 = 18/25 − 25/25 = −7/25.

If sin θ = 5/13 and θ is in Q I, what is sin 2θ?

cos θ = 12/13. sin 2θ = 2(5/13)(12/13) = 120/169.

What is cos(θ/2) if cos θ = 0 and θ/2 is in Q I?

cos(θ/2) = √((1+cosθ)/2) = √((1+0)/2) = √(1/2) = √2/2.

What is the 'double-angle formula' for sine?

The double-angle formula for sine states sin 2θ = 2 sin θ cos θ.

What is a 'half-angle formula'?

Half-angle formulas like sin(θ/2) = ±√((1−cosθ)/2) express functions of half an angle using the full angle.

What is the 'power-reducing formula' for sin²θ?

The power-reducing formula rewrites sin²θ without a square: sin²θ = (1 − cos 2θ)/2.

What is the double-angle formula for tangent?

tan 2θ = 2tanθ/(1−tan²θ), derived from the tangent sum formula with A = B = θ.

What is the half-angle formula for tangent?

The half-angle tangent formula has two equivalent forms: sinθ/(1+cosθ) and (1−cosθ)/sinθ.

A laser beam reflects off a surface, doubling its angle from the normal. If sin θ = 3/5, what is sin 2θ?

cos θ = 4/5. sin 2θ = 2(3/5)(4/5) = 24/25.

An engineer needs cos(π/8). Using the half-angle formula with θ = π/4, what is cos(π/8)?

cos(π/8) = √((1+cos(π/4))/2) = √((1+√2/2)/2) = √((2+√2)/4).

A computer graphics rotation involves cos 2α where cos α = 0.6. What is cos 2α?

cos 2α = 2cos²α − 1 = 2(0.36) − 1 = 0.72 − 1 = −0.28.

A physics model uses tan(θ/2) where cos θ = 4/5 and θ is in Q I. What is tan(θ/2)?

tan(θ/2) = sinθ/(1+cosθ) = (3/5)/(1+4/5) = (3/5)/(9/5) = 1/3.

A signal processor rewrites 4sin(x)cos(x) using a double-angle identity. What is the result?

4sinxcosx = 2(2sinxcosx) = 2sin 2x.

An architect needs sin²(22.5°) for a design. Using the power-reducing formula with 2(22.5°) = 45°, what is the value?

sin²(22.5°) = (1−cos45°)/2 = (1−√2/2)/2 = (2−√2)/4.
