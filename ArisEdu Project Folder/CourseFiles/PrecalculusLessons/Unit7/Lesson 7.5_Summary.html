Lesson 7.5: Solving Trigonometric Equations - Summary

Key Concepts: Solving Trigonometric Equations

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'trigonometric equation'?

Unlike an identity, a trig equation holds only for specific angle values that must be found.

What is a 'general solution' of a trig equation?

The general solution includes all solutions by adding integer multiples of the period to the principal solutions.

What is a 'quadratic-type' trig equation?

Quadratic-type equations treat sinθ (or cosθ) as a variable u, factor or use the quadratic formula, then solve for θ.

What is the 'principal solution' of a trig equation?

Principal solutions are the base solutions in [0, 2π); the general solution adds multiples of the period.

A finance model involves solving cos(2πt) = −1 for t ≥ 0. What is the smallest positive solution?

cos(2πt) = −1 when 2πt = π → t = 1/2.

Real-World Applications

Example: What strategy should you use when a trig equation involves multiple trig functions?

Using identities (like sin²θ = 1−cos²θ) to express everything in one function allows standard solving methods.
