Lesson 7.8: AP Calculus Prep - Summary

Key Concepts: AP Calculus Prep

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the exact value of sin(π/6)?

From the unit circle, sin(π/6) = sin 30° = 1/2.

What is the value of lim(x→0) [sin x / x]?

The special trigonometric limit lim(x→0) sin x / x = 1, which is fundamental in calculus.

What is cos(π/4)?

From the unit circle, cos(π/4) = cos 45° = √2/2.

What is the derivative of sin x?

The derivative of sin x is cos x, a key result derived using the limit definition and lim(h→0) sin h / h = 1.

What is tan(π/3)?

tan(π/3) = sin(π/3)/cos(π/3) = (√3/2)/(1/2) = √3.

What is the derivative of cos x?

The derivative of cos x is −sin x, which uses both lim(h→0) sin h / h = 1 and lim(h→0) (cos h − 1)/h = 0.

What is the exact value of cos(π/6)?

From the unit circle, cos(π/6) = cos 30° = √3/2.

What is the Squeeze Theorem?

The Squeeze Theorem (or Sandwich Theorem) states that if g(x) ≤ f(x) ≤ h(x) near x = a and both outer limits equal L, then lim f(x) = L as well.

What is 'continuity at a point' in calculus?

The three conditions for continuity at x = a are: f(a) is defined, lim(x→a) f(x) exists, and lim(x→a) f(x) = f(a).

What is the 'limit definition of the derivative'?

The derivative is defined as f'(x) = lim(h→0) [f(x+h) − f(x)] / h, representing the instantaneous rate of change.

What is a 'removable discontinuity'?

A removable discontinuity (hole) occurs where the limit exists but either f(a) is undefined or f(a) does not equal the limit. It can be 'removed' by redefining f(a).

What is 'L'Hôpital's Rule' used for?

L'Hôpital's Rule states that if lim f(x)/g(x) gives 0/0 or ∞/∞, then it equals lim f'(x)/g'(x), provided this latter limit exists.

What is a 'secant line' in the context of calculus?

A secant line passes through two points on a curve. Its slope equals the average rate of change, and as the two points merge, it becomes the tangent line.

A ball is thrown upward with height h(t) = −16t² + 48t + 5 feet. Using the limit definition, what is the instantaneous velocity at t = 1 second?

h'(t) = −32t + 48. At t = 1, h'(1) = −32(1) + 48 = 16 ft/s.

An engineer models the displacement of a vibrating beam as s(t) = 3 sin(2t). What is s'(t), the velocity function?

Using the chain rule, s'(t) = 3 · cos(2t) · 2 = 6 cos(2t).

A population is modeled by P(t) = 500/(1 + 9e^(−0.2t)). What is the value of lim(t→∞) P(t), the carrying capacity?

As t → ∞, e^(−0.2t) → 0, so P(t) → 500/(1 + 0) = 500. The carrying capacity is 500.

A student needs to evaluate lim(x→0) [tan x / x]. Using the fact that tan x = sin x / cos x and the known limit lim(x→0) sin x / x = 1, what is the answer?

tan x / x = (sin x / x) · (1/cos x). As x → 0, sin x/x → 1 and 1/cos x → 1, so the limit is 1.

A rocket's altitude is given by h(t) = t³ − 6t² + 12t (in km). What is the average rate of change of altitude from t = 0 to t = 3 seconds?

h(3) = 27 − 54 + 36 = 9, h(0) = 0. Average rate = (9 − 0)/(3 − 0) = 3 km/s.

A drug concentration in the bloodstream is modeled by C(t) = 5t/(t² + 4) mg/L. What is lim(t→∞) C(t)?

The degree of the denominator (2) exceeds the degree of the numerator (1), so lim(t→∞) C(t) = 0. The drug eventually clears from the bloodstream.

On an AP exam, a student encounters f(x) = (x² − 9)/(x − 3). What is lim(x→3) f(x)?

Factor: (x² − 9)/(x − 3) = (x − 3)(x + 3)/(x − 3) = x + 3 for x ≠ 3. So lim(x→3) = 3 + 3 = 6.

A physics student models the angle of a pendulum as θ(t) = 0.5 cos(4πt). What is the angular velocity θ'(t)?

θ'(t) = 0.5 · (−sin(4πt)) · 4π = −2π sin(4πt).

A manufacturing company's cost function is C(x) = 1000 + 50x − 0.1x². Using the limit definition, what is the marginal cost C'(x) at x = 100 units?

C'(x) = 50 − 0.2x. At x = 100, C'(100) = 50 − 0.2(100) = 50 − 20 = $30.

Real-World Applications

Example: What does 'average rate of change' mean for a function?

The average rate of change over [a, b] is [f(b) − f(a)]/(b − a), which is the slope of the secant line connecting the two endpoints.

Example: A rocket's altitude is given by h(t) = t³ − 6t² + 12t (in km). What is the average rate of change of altitude from t = 0 to t = 3 seconds?

h(3) = 27 − 54 + 36 = 9, h(0) = 0. Average rate = (9 − 0)/(3 − 0) = 3 km/s.

Example: On an AP exam, a student encounters f(x) = (x² − 9)/(x − 3). What is lim(x→3) f(x)?

Factor: (x² − 9)/(x − 3) = (x − 3)(x + 3)/(x − 3) = x + 3 for x ≠ 3. So lim(x→3) = 3 + 3 = 6.
