Lesson 7.2: Sum & Difference Formulas - Summary

Key Concepts: Sum & Difference Formulas

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is cos(A + B) equal to?

cos(A+B) = cosAcosB − sinAsinB. Note the minus sign for cosine sum.

What is sin(A − B)?

sin(A−B) = sinAcosB − cosAsinB.

If sin A = 3/5 (Q I) and cos B = 5/13 (Q I), what is sin(A + B)?

cos A = 4/5, sin B = 12/13. sin(A+B) = (3/5)(5/13) + (4/5)(12/13) = 15/65 + 48/65 = 63/65. Wait: 3·5=15, 4·12=48. 15+48=63. So 63/65. Let me reconsider options... The answer is 63/65.

What is tan(A − B)?

tan(A−B) = (tanA − tanB)/(1 + tanAtanB). Note the plus sign in the denominator for difference.

What is the 'sum formula for sine'?

The sine sum formula: sin(A+B) = sinAcosB + cosAsinB.

What is the 'difference formula for cosine'?

cos(A−B) = cosAcosB + sinAsinB. Note the plus sign for cosine difference.

What is the key sign difference between cos(A + B) and cos(A − B)?

cos(A+B) = cosAcosB − sinAsinB and cos(A−B) = cosAcosB + sinAsinB. The sign between terms is opposite to the operation.

What is the sign pattern in the sine sum vs. sine difference formulas?

sin(A+B) = sinAcosB + cosAsinB, sin(A−B) = sinAcosB − cosAsinB. The sign matches the operation.

What is the 'sum formula for tangent' used for?

The tangent sum formula allows computation of tan(A+B) from tanA and tanB.

Two sound waves with phases θ₁ and θ₂ are combined. The resultant involves sin(θ₁ + θ₂). If θ₁ = π/6 and θ₂ = π/4, what is the exact value of sin(θ₁ + θ₂)?

sin(5π/12) = sin(π/6+π/4) = sin(π/6)cos(π/4)+cos(π/6)sin(π/4) = (1/2)(√2/2)+(√3/2)(√2/2) = (√2+√6)/4.

An engineer needs cos(105°) for a structural calculation. Using 105° = 60° + 45°, what is the exact value?

cos(60°+45°) = cos60cos45 − sin60sin45 = (1/2)(√2/2) − (√3/2)(√2/2) = (√2−√6)/4.

A satellite dish is oriented at angle (α − β) where α and β are known. If tan α = 2 and tan β = 1/3, what is tan(α − β)?

tan(α−β) = (2−1/3)/(1+2·1/3) = (5/3)/(5/3) = 1.

A physics experiment requires sin(π/4 − π/6). What is the exact value?

sin(π/4−π/6) = sin(π/4)cos(π/6)−cos(π/4)sin(π/6) = (√2/2)(√3/2)−(√2/2)(1/2) = (√6−√2)/4.

A surveyor needs to compute sin(A + B) where sin A = 4/5, cos A = 3/5 (Q I) and sin B = 5/13, cos B = 12/13 (Q I). What is sin(A + B)?

sin(A+B) = sinAcosB + cosAsinB = (4/5)(12/13) + (3/5)(5/13) = 48/65 + 15/65 = 63/65.

A robotics rotation involves computing cos(30° − 45°). What is the exact value?

cos(−15°) = cos(15°) = cos(45°−30°) = cos45cos30+sin45sin30 = (√6+√2)/4.

A navigation system computes the angle between two headings using tan(α − β). If α = 135° (tan α = −1) and β = 120° (tan β = −√3), what is tan(α − β)?

tan(135°−120°) = tan15° = (−1−(−√3))/(1+(−1)(−√3)) = (√3−1)/(1+√3). Rationalizing: (√3−1)²/((1+√3)(√3−1)) = (4−2√3)/2 = 2−√3.

A physicist simplifies sin(x + π/2) in a wave equation. What is the simplified form?

sin(x+π/2) = sinxcos(π/2) + cosxsin(π/2) = sinx(0) + cosx(1) = cosx.
