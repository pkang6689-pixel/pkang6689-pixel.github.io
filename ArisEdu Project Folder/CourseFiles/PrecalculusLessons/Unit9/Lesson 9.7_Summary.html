Key Concepts: Piecewise Functions & Limits

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'piecewise function'?

A piecewise function uses different formulas on different sub-intervals, joined at boundary points. It may or may not be continuous.

What is a 'Heaviside step function'?

The Heaviside step function jumps from 0 to 1 at x = 0. It models on/off switches in engineering.

What is a 'spline'?

A spline is a piecewise polynomial where not only the function but also its derivatives match at boundary points, ensuring smooth connections.

What is a 'removable discontinuity' in a piecewise function?

A removable discontinuity can be 'repaired' by redefining the function value to equal the limit at that point.
