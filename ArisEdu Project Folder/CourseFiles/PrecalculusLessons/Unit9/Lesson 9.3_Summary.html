Lesson 9.3: One-Sided Limits - Summary

Key Concepts: One-Sided Limits

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is lim(x→0⁺) 1/x?

As x approaches 0 from the right (positive values), 1/x grows without bound toward +∞.

What is lim(x→0⁻) 1/x?

As x approaches 0 from the left (negative values), 1/x decreases without bound toward −∞.

If f(x) = {x+1, x<2; 5, x≥2}, what is lim(x→2⁻) f(x)?

For x < 2, f(x) = x + 1. As x → 2⁻: 2 + 1 = 3.

Using the same function from Q4, what is lim(x→2⁺) f(x)?

For x ≥ 2, f(x) = 5. As x → 2⁺: f(x) = 5.

What is lim(x→3⁺) √(x − 3)?

As x → 3⁺, x − 3 → 0⁺, so √(x − 3) → 0.

What is lim(x→3⁻) √(x − 3)?

For x < 3, x − 3 is negative, so √(x − 3) is not real. The left-hand limit does not exist in the real numbers.

If f(x) = |x − 1|/(x − 1), what is lim(x→1⁺) f(x)?

For x > 1, |x−1| = x−1, so f(x) = (x−1)/(x−1) = 1.

What is lim(x→1⁻) |x − 1|/(x − 1)?

For x < 1, |x−1| = −(x−1), so f(x) = −(x−1)/(x−1) = −1.

What is a 'one-sided limit'?

A one-sided limit examines the behavior of f(x) as x approaches a from one side only: left (x → a⁻) or right (x → a⁺).

What is a 'jump discontinuity'?

A jump discontinuity occurs when lim(x→a⁻) f(x) and lim(x→a⁺) f(x) both exist but are different values.

What is a 'left-hand limit'?

The left-hand limit, written lim(x→a⁻) f(x), is the value f(x) approaches when x comes from the left (x < a).

What is a 'right-hand limit'?

The right-hand limit, written lim(x→a⁺) f(x), is the value f(x) approaches when x comes from the right (x > a).

What is the 'greatest integer function' (floor function)?

The floor function ⌊x⌋ rounds down to the nearest integer. For example, ⌊3.7⌋ = 3 and ⌊−1.2⌋ = −2.

A parking garage charges $5 for the first hour and $3 for each additional hour (or part thereof). The cost function has jump discontinuities. What is lim(t→2⁻) C(t)?

For t just under 2 hours, the charge is $5 + $3 = $8 (first hour plus one additional hour).

A tax bracket system charges 10% on income up to $10,000 and 20% on income above $10,000. The marginal rate function r(x) = {0.10 if x ≤ 10000; 0.20 if x > 10000}. What is lim(x→10000⁺) r(x)?

For x just above $10,000, the rate is 0.20. So lim(x→10000⁺) r(x) = 0.20.

A dose of medication is modeled by D(t) = {100 if t < 6; 200 if t ≥ 6} mg, where a second dose is given at t = 6 hours. What is lim(t→6⁻) D(t)?

Just before the second dose at t = 6, D(t) = 100 mg.

A shipping company charges by weight: $5 for 0 < w ≤ 1 lb, $8 for 1 < w ≤ 3 lb, and $12 for 3 < w ≤ 5 lb. What is lim(w→1⁺) C(w)?

For w just above 1 lb, the cost switches to $8.

An electrical circuit switches on at t = 0. The current is I(t) = {0, t < 0; 5, t ≥ 0} amps. What is lim(t→0⁻) I(t)?

Before the switch at t = 0, I(t) = 0. So lim(t→0⁻) I(t) = 0.

A utility company charges $0.10/kWh for the first 500 kWh and $0.15/kWh after that. At exactly 500 kWh, what is lim(x→500⁺) of the marginal rate?

For usage just above 500 kWh, the marginal rate is $0.15/kWh.

A roller coaster's speed at the bottom of a loop is modeled by v(t) = {8t if t < 5; 40 if t = 5; −8(t−10) if t > 5}. What is lim(t→5⁻) v(t) and lim(t→5⁺) v(t)?

lim(t→5⁻) = 8(5) = 40. lim(t→5⁺) = −8(5−10) = −8(−5) = 40. Both one-sided limits equal 40.

Real-World Applications

Example: A parking garage charges $5 for the first hour and $3 for each additional hour (or part thereof). The cost function has jump discontinuities. What is lim(t→2⁻) C(t)?

For t just under 2 hours, the charge is $5 + $3 = $8 (first hour plus one additional hour).
