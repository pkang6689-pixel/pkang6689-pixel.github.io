Lesson 9.8: AP Calculus Prep - Summary

Key Concepts: AP Calculus Prep

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the horizontal asymptote of f(x) = (5x − 1)/(2x + 3)?

Same degree → ratio of leading coefficients: 5/2.

Using the limit definition, what is the derivative of f(x) = x³ at x = 1?

f'(x) = 3x². At x = 1: 3(1)² = 3.

What is the average rate of change of f(x) = x³ from x = 1 to x = 3?

[f(3) − f(1)]/(3 − 1) = (27 − 1)/2 = 26/2 = 13.

What is the 'Intermediate Value Theorem'?

The IVT guarantees that a continuous function on [a,b] attains every value between f(a) and f(b).

What is a 'limit at infinity'?

A limit at infinity describes the end behavior of f(x) as x grows without bound in the positive or negative direction.

What is the 'Squeeze Theorem'?

The Squeeze Theorem traps f(x) between two functions with the same limit, forcing f to have that limit as well.

What is the 'limit definition of the derivative'?

The derivative is defined as f'(x) = lim(h→0)[f(x+h)−f(x)]/h, representing the instantaneous rate of change.

What is 'end behavior'?

End behavior describes how a function behaves as x goes to positive or negative infinity.

What is a 'tangent line approximation'?

The tangent line approximation (linearization) uses L(x) = f(a) + f'(a)(x−a) to estimate f(x) near x = a.

On an AP exam, a student must evaluate lim(x→0) [1/x − 1/(x+x²)]. What is the result?

1/x − 1/(x(1+x)) = [1+x−1]/(x(1+x)) = x/(x(1+x)) = 1/(1+x). As x→0: 1/1 = 1.

How to Calculate

Example: Find k such that f(x) = {kx², x≤1; 3x+1, x>1} is continuous at x = 1.

Solution: k(1)² = 3(1) + 1 → k = 4.

Real-World Applications

Example: What is the average rate of change of f(x) = x³ from x = 1 to x = 3?

[f(3) − f(1)]/(3 − 1) = (27 − 1)/2 = 26/2 = 13.

Example: What does 'average rate of change' mean?

The average rate of change over [a,b] is Δy/Δx = [f(b)−f(a)]/(b−a), the slope of the secant line.

Example: An AP exam question asks: if f is continuous on [0, 5], f(0) = −3, and f(5) = 7, must there be a c in (0, 5) where f(c) = 0?

Since 0 is between f(0) = −3 and f(5) = 7 and f is continuous, the IVT guarantees some c with f(c) = 0.

Example: On an AP exam, a student must evaluate lim(x→0) [1/x − 1/(x+x²)]. What is the result?

1/x − 1/(x(1+x)) = [1+x−1]/(x(1+x)) = x/(x(1+x)) = 1/(1+x). As x→0: 1/1 = 1.
