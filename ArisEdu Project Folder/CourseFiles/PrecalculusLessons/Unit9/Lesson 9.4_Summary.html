Lesson 9.4: Continuity & Types of Discontinuities - Summary

Key Concepts: Continuity & Types of Discontinuities

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'removable discontinuity'?

A removable discontinuity is a 'hole' where the limit exists but f(a) either is undefined or does not match the limit. It can be removed by redefining f(a).

What is a 'jump discontinuity'?

A jump discontinuity occurs when lim(x→a⁻) and lim(x→a⁺) both exist but are different, causing a 'jump' in the graph.

What is an 'infinite discontinuity'?

An infinite discontinuity occurs when f(x) → ±∞ as x → a from one or both sides, typically at a vertical asymptote.

What is an 'oscillating discontinuity'?

An oscillating discontinuity occurs when f(x) oscillates infinitely near x = a (like sin(1/x) as x → 0), preventing the limit from existing.

How to Calculate

Example: A velocity function is v(t) = {3t², t<1; at+b, t≥1}. Find a and b so that v is continuous and has a continuous first derivative at t = 1.

Solution: Continuity: 3(1)² = a(1)+b → 3 = a+b. Smooth: v'(1⁻) = 6t|₁ = 6 = a. So a = 6, b = 3 − 6 = −3.
