Lesson 9.1: Concept of Limits (graphical, numerical approaches) - Summary

Key Concepts: Concept of Limits (graphical, numerical approaches)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If f(x) = 5 for all x, what is lim(x→a) f(x)?

The limit of a constant function is the constant itself: lim(x→a) 5 = 5.

What is lim(x→0) (x + 3)?

For a polynomial, simply substitute: lim(x→0)(x + 3) = 0 + 3 = 3.

From a graph, if f(x) approaches 2 from both sides as x → 1, but f(1) = 5, what is lim(x→1) f(x)?

The limit depends on what f(x) approaches, not the actual value f(1). Since f(x) → 2 from both sides, the limit is 2.

What is lim(x→4) √x?

lim(x→4) √x = √4 = 2 by direct substitution.

What is lim(x→∞) 1/x?

As x grows without bound, 1/x approaches 0.

What is lim(x→0) |x|/x?

From the right, |x|/x = 1; from the left, |x|/x = −1. Since the one-sided limits differ, the limit does not exist.

What is a 'limit' in mathematics?

A limit is the value that f(x) approaches as x gets closer and closer to a specific point, regardless of whether f(x) actually equals that value at the point.

What is 'direct substitution' in limit evaluation?

Direct substitution means evaluating f(a) directly. If the result is a real number (no 0/0), the limit equals f(a).

What is a 'graphical approach' to finding limits?

The graphical approach examines the graph to see what y-value the curve approaches from both sides as x approaches the target value.

What is a 'numerical approach' to finding limits?

The numerical approach creates a table of values with x getting closer to the target (from both sides) and observes what f(x) is approaching.

A biologist models a population as P(t) = 1000t/(t + 5). What is lim(t→∞) P(t), the carrying capacity?

Divide by t: P(t) = 1000/(1 + 5/t). As t → ∞, 5/t → 0, so P → 1000.

A falling object has velocity v(t) = (4.9t² − 4.9)/(t − 1) m/s near t = 1. Using a table approach, what is lim(t→1) v(t)?

4.9(t² − 1)/(t − 1) = 4.9(t + 1). As t → 1: 4.9(2) = 9.8 m/s.

An economics model gives revenue R(x) = (3x² − 12)/(x − 2) thousand dollars for x units (in thousands). What is lim(x→2) R(x)?

Factor: 3(x² − 4)/(x − 2) = 3(x+2). As x → 2: 3(4) = 12 thousand dollars.

A traffic flow model gives vehicles per minute as f(t) = (t² − 9)/(t − 3) near rush hour at t = 3. What is lim(t→3) f(t)?

(t² − 9)/(t − 3) = (t+3) for t ≠ 3. As t → 3: t + 3 = 6.

An engineer studies the stress σ(x) = (x³ − 8)/(x − 2) kPa at critical point x = 2. What is lim(x→2) σ(x)?

(x³ − 8)/(x − 2) = x² + 2x + 4 by factoring the difference of cubes. At x = 2: 4 + 4 + 4 = 12 kPa.
