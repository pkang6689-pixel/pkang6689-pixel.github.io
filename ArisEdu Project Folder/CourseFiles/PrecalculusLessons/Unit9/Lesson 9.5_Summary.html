Lesson 9.5: Infinite Limits & Asymptotic Behavior - Summary

Key Concepts: Infinite Limits & Asymptotic Behavior

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is lim(x→0⁺) 1/x²?

As x → 0⁺, x² → 0⁺, so 1/x² → +∞.

What is the horizontal asymptote of f(x) = (2x + 1)/(x − 3)?

The degrees are equal, so the horizontal asymptote is y = leading coefficient ratio = 2/1 = 2.

What is the vertical asymptote of f(x) = 1/(x − 5)?

The denominator is zero at x = 5, so there is a vertical asymptote at x = 5.

What is lim(x→∞) (3x² + x)/(x² + 1)?

Divide by x²: (3 + 1/x)/(1 + 1/x²) → 3/1 = 3.

What is lim(x→∞) (x + 2)/(x³ − 1)?

Degree of numerator (1) < degree of denominator (3), so the limit is 0.

What is lim(x→∞) (x³ − 2x)/(x² + 4)?

Degree of numerator (3) > degree of denominator (2), so the limit is ∞.

What is lim(x→2⁺) 1/(x − 2)?

As x → 2⁺, (x − 2) → 0⁺, so 1/(x − 2) → +∞.

What is the horizontal asymptote of f(x) = 5/(x² + 1)?

Degree of numerator (0) < degree of denominator (2), so lim(x→±∞) = 0. The horizontal asymptote is y = 0.

What is an 'infinite limit'?

An infinite limit means f(x) → +∞ or −∞ as x → a, indicating the function grows without bound near that point.

What is a 'vertical asymptote'?

A vertical asymptote is a vertical line x = a where the function approaches ±∞ from one or both sides.

What is a 'horizontal asymptote'?

A horizontal asymptote y = L is a horizontal line that the graph approaches as x goes to ±∞: lim(x→±∞) f(x) = L.

What is a 'limit at infinity'?

A limit at infinity examines the end behavior of f(x) as x grows positively or negatively without bound.

What is an 'oblique (slant) asymptote'?

An oblique asymptote occurs when the numerator's degree is exactly one more than the denominator's. Polynomial long division reveals y = mx + b.

A population model is P(t) = 5000t/(t + 50). What is the long-term population (carrying capacity)?

lim(t→∞) 5000t/(t+50) = lim 5000/(1+50/t) = 5000.

A chemical reaction follows C(t) = 200/(1 + 4e^(−0.5t)). What is the equilibrium concentration as t → ∞?

As t → ∞, e^(−0.5t) → 0, so C → 200/(1+0) = 200.

An internet bandwidth model gives speed as S(n) = 100n/(n + 10) Mbps where n is the number of servers. What is the maximum theoretical speed?

lim(n→∞) 100n/(n+10) = 100. Even with unlimited servers, the speed approaches but never exceeds 100 Mbps.

An engineer graphs f(x) = (x² − 1)/(x² + 1). What is the horizontal asymptote, and does the function ever cross it?

lim(x→±∞) (x²−1)/(x²+1) = 1. Setting f(x) = 1 gives x²−1 = x²+1, or −1 = 1, which is impossible. It never reaches y = 1.

A logistic growth model for a fish population is P(t) = 10000/(1 + 99e^(−0.3t)). What is P(0) and what does P approach as t → ∞?

P(0) = 10000/(1+99) = 10000/100 = 100. As t → ∞, e^(−0.3t) → 0, so P → 10000/1 = 10,000.

Real-World Applications

Example: An economist models cost per unit as C(x) = (500 + 3x)/x. What does the average cost approach as production becomes very large?

C(x) = 500/x + 3. As x → ∞, 500/x → 0, so C(x) → $3.

Example: A drug dosage model gives concentration C(t) = 50t·e^(−t). As t → ∞, what concentration does the drug approach?

e^(−t) decays faster than t grows, so 50t·e^(−t) → 0 as t → ∞.
