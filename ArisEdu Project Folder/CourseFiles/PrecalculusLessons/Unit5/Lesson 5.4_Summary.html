Lesson 5.4: Change of Base Formula - Summary

Key Concepts: Change of Base Formula

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the change of base formula for log_b(x)?

The change of base formula is log_b(x) = log_a(x) / log_a(b), where a can be any valid logarithm base (commonly 10 or e).

What is the relationship between log_b(a) and log_a(b)?

The reciprocal property states that log_b(a) = 1/log_a(b). This follows directly from applying the change of base formula.

What is log₁/₂(4)?

(1/2)⁻² = 2² = 4, so log₁/₂(4) = −2. Bases between 0 and 1 yield negative logs for arguments greater than 1.

A computer scientist needs to determine how many bits are required to represent 1000 distinct values. This requires evaluating log₂(1000). What is the approximate value?

log₂(1000) = ln(1000)/ln(2) ≈ 6.9078/0.6931 ≈ 9.97. So 10 bits are needed to represent 1000 values.

A musician is studying the frequency ratio between musical notes. She needs to evaluate log₁₂(2) to find how many semitones are in an octave doubling. What is the approximate value?

log₁₂(2) = ln(2)/ln(12) ≈ 0.6931/2.4849 ≈ 0.279.

An engineer needs to solve 5^x = 200 for a thermal model. Using the change of base formula, what is the approximate value of x?

x = log₅(200) = ln(200)/ln(5) ≈ 5.2983/1.6094 ≈ 3.29.

A chemistry student knows that log₂(x) = 4 and needs to find log₄(x). Using the relationship between bases 2 and 4, what is log₄(x)?

If log₂(x) = 4, then x = 2⁴ = 16. Now log₄(16) = log₄(4²) = 2.

A biologist models bacterial growth using base-3 logarithms. She needs log₃(50) for her calculations. What is the approximate value?

log₃(50) = ln(50)/ln(3) ≈ 3.9120/1.0986 ≈ 3.56.

A math student is asked to find log₅(x) if log₅(x) = 2.5. What is x?

If log₅(x) = 2.5, then x = 5^2.5 = 5² · 5^(0.5) = 25√5 ≈ 25(2.236) ≈ 55.9.

A data analyst needs to convert log₂(N) to log₁₀(N) for a report. If log₂(N) = 20, what is log₁₀(N)?

If log₂(N) = 20, then N = 2²⁰. So log₁₀(N) = log₁₀(2²⁰) = 20 · log₁₀(2) ≈ 20(0.30103) ≈ 6.02.

A pharmacologist uses the equation t = log₀.₅(0.1) · T_half to find when a drug drops to 10% of its dose. What is log₀.₅(0.1)?

log₀.₅(0.1) = ln(0.1)/ln(0.5) = (−2.3026)/(−0.6931) ≈ 3.322. Note: the result is positive because both numerator and denominator are negative.
