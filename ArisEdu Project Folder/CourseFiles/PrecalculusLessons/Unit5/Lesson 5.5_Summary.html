Lesson 5.5: Solving Exponential & Logarithmic Equations - Summary

Key Concepts: Solving Exponential & Logarithmic Equations

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is an 'extraneous solution' in logarithmic equations?

An extraneous solution arises from algebraic manipulation but makes a logarithmic argument negative or zero, violating the domain. It must be rejected.

What is the one-to-one property of logarithms?

The one-to-one property states that if two logarithms with the same base are equal, their arguments must be equal: log_b(x) = log_b(y) implies x = y.

What is the standard first step when solving an exponential equation like 5ˣ = 200?

To isolate x in the exponent, take the logarithm (ln or log) of both sides: x ln(5) = ln(200), then solve for x.

What is the standard approach to solving log(x) + log(x − 3) = 1?

First combine: log(x(x − 3)) = 1. Then convert to exponential form: x(x − 3) = 10¹. Solve the quadratic and check for extraneous solutions.

A biologist models a cell culture with N(t) = 50 · 2^(t/3). She needs to find when the culture reaches 6400 cells. What is t?

6400 = 50 · 2^(t/3) gives 2^(t/3) = 128 = 2⁷, so t/3 = 7, thus t = 21.

A chemist dissolves a compound and measures the equation log(x) + log(x − 3) = 1 for the concentration x (in moles). What is x?

Combine: log(x(x − 3)) = 1, so x² − 3x = 10, giving x² − 3x − 10 = 0, which factors as (x − 5)(x + 2) = 0. Since x = −2 makes log(x) undefined, x = 5.

A physicist needs to solve 5ˣ⁺¹ = 5³ˣ⁻² for a symmetry analysis. What is x?

Same base: equate exponents. x + 1 = 3x − 2 gives 3 = 2x, so x = 3/2.

A financial planner needs to solve e^(0.06t) = 3 to find when an investment triples. What is the approximate value of t?

0.06t = ln(3) ≈ 1.0986, so t = 1.0986/0.06 ≈ 18.3 years.

A population ecologist encounters log₂(x) + log₂(x + 2) = 3 while modeling species interaction. What is x?

Combine: log₂(x(x + 2)) = 3, so x² + 2x = 8, giving x² + 2x − 8 = 0, factoring as (x + 4)(x − 2) = 0. Since x = −4 is invalid (log argument negative), x = 2.

A technician needs to solve 7^(2x+1) = 300 for an amplifier circuit calibration. What is the approximate value of x?

Take ln: (2x + 1) ln(7) = ln(300). So 2x + 1 = ln(300)/ln(7) ≈ 5.7038/1.9459 ≈ 2.932. Then 2x = 1.932, giving x ≈ 0.966, which rounds to about 0.97.

A quality control analyst solves ln(2x + 1) = 0 to determine the baseline measurement. What is x?

ln(2x + 1) = 0 means 2x + 1 = e⁰ = 1, so 2x = 0 and x = 0.

Real-World Applications

Example: A medical researcher solves ln(x²) = 4 for a dosage calculation. What are the solutions?

ln(x²) = 2 ln|x| = 4, so ln|x| = 2, giving |x| = e². Therefore x = e² or x = −e².
