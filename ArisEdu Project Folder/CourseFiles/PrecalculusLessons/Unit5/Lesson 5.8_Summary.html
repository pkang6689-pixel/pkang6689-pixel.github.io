Lesson 5.8: Logistic Growth Models - Summary

Key Concepts: Logistic Growth Models

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

For P(t) = 1000/(1 + 9e^(−0.5t)), what is the carrying capacity?

In the form P(t) = L/(1 + be^(−kt)), L is the numerator. Here L = 1000.

P(t) = 500/(1 + 24e^(−0.3t)). What is P(0)?

P(0) = 500/(1 + 24) = 500/25 = 20.

For P(t) = 500/(1 + 24e^(−0.3t)), what is the inflection point population?

The inflection point occurs at L/2 = 500/2 = 250.

If L = 5000 and P(0) = 500, what is the value of b in P(t) = L/(1 + be^(−kt))?

P(0) = L/(1 + b), so 500 = 5000/(1 + b), giving 1 + b = 10, thus b = 9.

What is the difference between logistic growth and exponential growth?

Exponential growth models unlimited increase (Peᵏᵗ), while logistic growth (L/(1 + be⁻ᵏᵗ)) includes a maximum (L) that constrains the population.

A fish hatchery stocks a lake with 200 fish. The lake can support 4000 fish. A biologist uses the logistic model. What is b?

P(0) = L/(1 + b), so 200 = 4000/(1 + b), giving 1 + b = 20, thus b = 19.

Real-World Applications

Example: A wildlife manager models a deer population with P(t) = 800/(1 + 3e^(−0.2t)). What population level triggers the fastest growth?

The fastest growth occurs at the inflection point: P = L/2 = 800/2 = 400 deer.

Example: A pharmaceutical company models drug adoption as P(t) = 100,000/(1 + 999e^(−0.05t)). What percentage of the carrying capacity is the initial adoption?

P(0) = 100,000/(1 + 999) = 100,000/1000 = 100. As a percentage of L: 100/100,000 = 0.1%.
