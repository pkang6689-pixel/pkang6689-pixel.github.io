Lesson 3.8: Applications of Polynomial Models - Summary

Key Concepts: Applications of Polynomial Models

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A ball follows h(t) = −4.9t² + 20t + 1.5. What is the initial height?

The initial height is h(0) = 1.5 m (the constant term).

What is 'marginal cost'?

Marginal cost is the derivative C'(x), representing the additional cost to produce one more unit at production level x.

A civil engineer designs a road curve following y(x) = 0.001x³ − 0.06x² + x for 0 ≤ x ≤ 40 meters. What is the road height at x = 0?

y(0) = 0 + 0 + 0 = 0 meters.

Real-World Applications

Example: What does 'average cost' mean?

Average cost per unit is the total cost divided by the number of units: C̄(x) = C(x)/x.

Example: A pharmaceutical company finds that the effectiveness of a drug follows E(d) = −d³ + 12d² − 36d for dosage d mg. To find the dosage that produces zero effectiveness, they need to solve E(d) = 0. Factoring gives:

E(d) = −d(d² − 12d + 36) = −d(d − 6)². Zero effectiveness at d = 0 and d = 6.

Example: A package delivery company's daily cost is C(x) = 0.01x³ − 0.6x² + 15x + 200 for x packages. The cost of delivering 0 packages is $200. What does this $200 represent?

C(0) = $200 is the fixed cost, representing expenses the company incurs regardless of how many packages are delivered.
