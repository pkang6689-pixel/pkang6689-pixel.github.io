Lesson 6.8: Coterminal & Reference Angles - Summary

Key Concepts: Coterminal & Reference Angles

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the reference angle for 150°?

150° is in Q II. Reference angle = 180° − 150° = 30°.

What is the reference angle for 225°?

225° is in Q III. Reference angle = 225° − 180° = 45°.

What is the reference angle for 315°?

315° is in Q IV. Reference angle = 360° − 315° = 45°.

What is the reference angle for 5π/3?

5π/3 is in Q IV. Reference angle = 2π − 5π/3 = π/3.

What is the reference angle for 2π/3?

2π/3 is in Q II. Reference angle = π − 2π/3 = π/3.

What is a 'reference angle'?

A reference angle is always positive, acute (between 0° and 90°), and measured from the terminal side to the nearest x-axis.

What is the reference angle formula for an angle in Quadrant II?

In Q II (90° < θ < 180°), the reference angle = 180° − θ.

What is the reference angle formula for an angle in Quadrant III?

In Q III (180° < θ < 270°), the reference angle = θ − 180°.

What is the reference angle formula for an angle in Quadrant IV?

In Q IV (270° < θ < 360°), the reference angle = 360° − θ.

What is the range of possible values for a reference angle?

A reference angle is always a positive acute angle, so it is between 0° and 90° (exclusive).

A wind turbine blade passes through 750° of rotation. What is the coterminal angle in [0°, 360°), and how many full rotations did it make?

750° − 2(360°) = 750° − 720° = 30°. Two full rotations completed.

A robot arm rotates to −400°. What is the equivalent positive angle in standard position?

−400° + 360° = −40°. Still negative, so −40° + 360° = 320°.

A compass heading of 200° must be analyzed. What is its reference angle for trig calculations?

200° is in Q III. Reference angle = 200° − 180° = 20°.

A GPS system records a bearing of 330°. What is the reference angle, and in which quadrant does this bearing fall?

330° is in Q IV. Reference angle = 360° − 330° = 30°.

A Ferris wheel seat starts at 0° and rotates to 500°. What coterminal angle in [0°, 360°) describes its position, and what is the reference angle?

500° − 360° = 140° (Q II). Reference angle = 180° − 140° = 40°.

A surveyor needs sin(240°). Using the reference angle approach, what is the value?

240° is in Q III, reference angle = 60°. sin is negative in Q III: sin(240°) = −sin(60°) = −√3/2.

A clock's second hand reaches 1080° of total rotation from the 12 o'clock position. What is the equivalent position?

1080° = 3 × 360°, so the hand is back at 0° (12 o'clock).

An engineer calculates cos(7π/4) using the reference angle. What is the reference angle and the value?

7π/4 is in Q IV, reference angle = 2π − 7π/4 = π/4. cos is positive in Q IV, so cos(7π/4) = √2/2.
