Lesson 6.6: Inverse Trigonometric Functions - Summary

Key Concepts: Inverse Trigonometric Functions

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is arcsin(1/2)?

sin(π/6) = 1/2, and π/6 is in [−π/2, π/2], so arcsin(1/2) = π/6.

What is arccos(0)?

cos(π/2) = 0 and π/2 is in [0, π], so arccos(0) = π/2.

What is arctan(1)?

tan(π/4) = 1, and π/4 is in (−π/2, π/2), so arctan(1) = π/4.

What is arcsin(−√2/2)?

sin(−π/4) = −√2/2, and −π/4 is in [−π/2, π/2], so arcsin(−√2/2) = −π/4.

What is arccos(−1)?

cos(π) = −1 and π is in [0, π], so arccos(−1) = π.

What is arctan(−√3)?

tan(−π/3) = −√3, and −π/3 is in (−π/2, π/2), so arctan(−√3) = −π/3.

What is arccos(√3/2)?

cos(π/6) = √3/2, and π/6 is in [0, π], so arccos(√3/2) = π/6.

What is arcsin(1)?

sin(π/2) = 1, and π/2 is in [−π/2, π/2], so arcsin(1) = π/2.

What is an 'inverse trigonometric function'?

Inverse trig functions 'undo' trig functions: given a ratio, they return the angle.

What is the range of arcsin(x)?

arcsin returns angles in [−π/2, π/2] (Q I and Q IV on the right side of the unit circle).

What is the range of arccos(x)?

arccos returns angles in [0, π] (Q I and Q II, the top of the unit circle).

What is the domain of arcsin(x)?

Since sine only outputs values in [−1, 1], arcsin is only defined for inputs in [−1, 1].

What is the range of arctan(x)?

arctan returns angles in the open interval (−π/2, π/2) -- it approaches but never reaches ±π/2.

What is the domain of arctan(x)?

Since tangent can output any real number, arctan accepts all real numbers as input.

A GPS receiver determines that the angle of elevation to a satellite is arctan(4/3). What is the sine of this angle?

If θ = arctan(4/3), then opposite = 4, adjacent = 3, hypotenuse = 5. sin θ = 4/5.

A road sign says a hill has a grade of 100% (rise equals run). Using inverse trig, what is the angle of incline?

100% grade means rise/run = 1. θ = arctan(1) = 45°.

A spotlight on the ground is 20 m from a building. It illuminates a point 20√3 m up the wall. What is the angle of elevation?

tan θ = 20√3/20 = √3, so θ = arctan(√3) = 60°.

A pilot's instrument shows the aircraft's glide slope has sin θ = 0.05. What is the glide angle in radians (approximately)?

For small angles, arcsin(x) ≈ x. So arcsin(0.05) ≈ 0.05 radians (about 2.87°).

A photographer wants to capture a building that subtends an angle of arctan(3) at the camera lens. What is cos(arctan(3))?

If θ = arctan(3), then opposite = 3, adjacent = 1, hypotenuse = √10. cos θ = 1/√10.

A zip-line cable drops from a height of 50 m to 20 m over a horizontal distance of 30√3 m. What is the angle of descent using inverse trig?

Vertical drop = 50 − 20 = 30 m. tan θ = 30/(30√3) = 1/√3 = √3/3. θ = arctan(√3/3) = 30°.

A construction manager needs the angle of a roof that rises 12 ft over a 12√3 ft horizontal span. What is this angle?

tan θ = 12/(12√3) = 1/√3. θ = arctan(1/√3) = 30°.
