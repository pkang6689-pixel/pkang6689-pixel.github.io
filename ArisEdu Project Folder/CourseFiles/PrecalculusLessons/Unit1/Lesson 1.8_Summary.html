Lesson 1.8: Inverse Functions & Composition - Summary

Key Concepts: Inverse Functions & Composition

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

The Celsius-to-Fahrenheit formula is F(C) = (9/5)C + 32. What is the inverse that converts Fahrenheit to Celsius?

Solve F = (9/5)C + 32 for C: F − 32 = (9/5)C, so C = (5/9)(F − 32).

Population P(t) = 200 + 50t and g(x) = 2x. What is (P ∘ g)(3)?

(P ∘ g)(3) = P(g(3)) = P(6) = 200 + 50(6) = 500.

A store applies 20% discount D(p) = 0.8p then 8% tax T(p) = 1.08p. What is T(D(100))?

D(100) = 0.8(100) = 80. T(80) = 1.08(80) = $86.40.

An encryption function is f(x) = 2x + 7. What is the decryption (inverse) function?

y = 2x + 7 -> y − 7 = 2x -> x = (y − 7)/2. So f⁻¹(x) = (x − 7)/2.

Real-World Applications

Example: Annual salary from hourly wage: A(w) = 2080w. What hourly wage gives a $52,000 salary?

Inverse: w = A/2080 = 52000/2080 = $25.00/hr.

Example: Shipping costs C(w) = 3w + 5 dollars for weight w pounds. A customer paid $26. What was the package weight?

Inverse: w = (C − 5)/3 = (26 − 5)/3 = 21/3 = 7 pounds.
