Lesson 1.5: Function Notation & Evaluation - Summary

Key Concepts: Function Notation & Evaluation

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A biologist models bacteria as P(t) = 500(2)^t, where t is hours. What is P(3)?

P(3) = 500(2)³ = 500(8) = 4,000 bacteria.

A ball's height is h(t) = −16t² + 48t + 5 feet. What is h(2)?

h(2) = −16(4) + 48(2) + 5 = −64 + 96 + 5 = 37 feet.

An online subscription costs C(n) = 15 + 5(n − 1) dollars for n users. What is C(6)?

C(6) = 15 + 5(6 − 1) = 15 + 5(5) = 15 + 25 = $40.

A car depreciates as V(t) = 25000(0.85)^t dollars. What is V(2)?

V(2) = 25000(0.85)² = 25000(0.7225) = $18,062.50.

A pizza costs C(p) = 8p + 3 dollars, and delivery adds D(c) = c + 5. What is D(C(4))?

C(4) = 8(4) + 3 = 35. Then D(35) = 35 + 5 = $40.

Temperature at altitude: T(h) = 68 − 3.5h °F, where h is thousands of feet. What is T(4)?

T(4) = 68 − 3.5(4) = 68 − 14 = 54°F.

An investment grows as A(t) = 1000(1.05)^t. What is A(3) − A(1)?

A(3) = 1000(1.157625) = $1,157.63. A(1) = 1000(1.05) = $1,050.00. Difference: $107.63.
