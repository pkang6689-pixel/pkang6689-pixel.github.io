Lesson 1.4: Domain & Range (interval notation) - Summary

Key Concepts: Domain & Range (interval notation)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A drone can fly at altitudes between 0 and 400 feet inclusive. What is the range of the altitude function in interval notation?

Since the drone can fly at exactly 0 feet (ground) and exactly 400 feet, both endpoints are included, giving [0, 400].

A water tank holds V(t) = 500 - 10t gallons, where t is hours. If volume cannot be negative, what is the domain?

Time starts at t = 0, and 500 - 10t >= 0 gives t <= 50. Both endpoints are achievable, so the domain is [0, 50].

A store's profit is P(x) = √(x − 100) thousand dollars, where x is items sold. What is the domain?

For the square root to be defined, x − 100 ≥ 0, so x ≥ 100. Domain: [100, ∞).

A patient's medication concentration is C(t) = 20t/(t² + 4) mg/L, with maximum 5 mg/L at t = 2. For t ≥ 0, what is the range?

At t = 0, C(0) = 0. Maximum is 5 mg/L at t = 2. The concentration is always non-negative, so the range is [0, 5].

An engineer models signal strength as S(d) = 100/d² watts, where d is distance in meters. If the receiver works between 1 and 50 meters, what is the domain?

The receiver operates from 1 to 50 meters inclusive, so the practical domain is [1, 50].

A ball is thrown upward with height h(t) = −16t² + 64t feet. What is the range of the height function?

The ball starts at h = 0, reaches max at t = 2: h(2) = −16(4) + 64(2) = 64 feet, then returns to 0. Range is [0, 64].

A pool temperature is modeled by T(t) = 60 + 15sin(t). What is the range of this function?

Since sin(t) ranges from −1 to 1, T(t) ranges from 60 − 15 = 45 to 60 + 15 = 75. Range: [45, 75].

A rental company charges C(m) = 30√(m − 1) + 50 dollars for m months. What is the domain if the minimum rental is 1 month?

For the square root, m − 1 ≥ 0 means m ≥ 1. The minimum rental is 1 month (included), so domain is [1, ∞).

A factory's efficiency is E(w) = w/(w + 10) where w is number of workers. For w > 0, efficiency approaches but never reaches 1. What is the range?

For w > 0, E(w) is always positive (never 0) and always less than 1 (approaches 1 as w → ∞ but never reaches it). Range: (0, 1).
