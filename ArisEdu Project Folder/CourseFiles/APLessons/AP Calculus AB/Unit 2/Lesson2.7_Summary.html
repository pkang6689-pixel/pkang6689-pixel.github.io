AP Calculus AB - Unit 2 - Lesson 2.7 - Summary

Key Concepts

Exponential and Logarithmic Derivatives extend differentiation to transcendental functions. The natural exponential has the unique property d/dx[eˣ] = eˣ. For general exponentials: d/dx[aˣ] = aˣ ln a. For logarithms: d/dx[ln x] = 1/x and d/dx[logₐ x] = 1/(x ln a). Combined with the chain rule: d/dx[e^(g(x))] = e^(g(x)) · g'(x) and d/dx[ln(g(x))] = g'(x)/g(x). These derivatives are essential for modeling growth, decay, and natural phenomena.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • d/dx[eˣ] = eˣ: The exponential function is its own derivative; the only function (up to constant multiples) with this property
  • d/dx[aˣ] = aˣ ln a: For any positive base a: derived from aˣ = e^(x ln a) and the chain rule; d/dx[2ˣ] = 2ˣ ln 2
  • d/dx[ln x] = 1/x: The derivative of the natural log; note that d/dx[ln|x|] = 1/x for x ≠ 0
  • d/dx[logₐ x] = 1/(x ln a): Change of base: logₐ x = ln x / ln a, so d/dx[logₐ x] = 1/(x ln a)
  • Chain rule combinations: d/dx[e^(f(x))] = e^(f(x)) · f'(x); d/dx[ln(f(x))] = f'(x)/f(x)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Radioactive half-life: N(t) = N₀·(1/2)^(t/τ), and N'(t) = N₀·(1/2)^(t/τ)·(-ln2/τ) gives the decay rate at any time—critical for nuclear medicine dosing
  • Population modeling: P(t) = P₀e^(rt), so P'(t) = rP₀e^(rt) = rP(t), showing exponential growth rate is proportional to population size
  • pH calculations: pH = -log₁₀[H⁺], and the derivative d(pH)/d[H⁺] = -1/([H⁺] ln 10) shows pH sensitivity to concentration changes
