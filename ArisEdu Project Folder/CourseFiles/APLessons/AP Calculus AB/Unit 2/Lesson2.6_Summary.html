AP Calculus AB - Unit 2 - Lesson 2.6 - Summary

Key Concepts

Trigonometric Derivatives form a closed set of six interconnected derivatives. The fundamental pair: d/dx[sin x] = cos x and d/dx[cos x] = -sin x (derived from the limit definition using lim(h→0) sin(h)/h = 1). From these, using quotient rules: d/dx[tan x] = sec²x, d/dx[cot x] = -csc²x, d/dx[sec x] = sec x tan x, d/dx[csc x] = -csc x cot x. Note the pattern: 'co' functions have negative derivatives.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • d/dx[sin x] = cos x: Derived from the limit definition using sin(x+h) = sin x cos h + cos x sin h and special limits
  • d/dx[cos x] = -sin x: Derived similarly; the negative sign reflects that cosine decreases when sine is positive
  • d/dx[tan x] = sec²x: Derived from d/dx[sin x / cos x] using the quotient rule; always positive, so tan x has no horizontal tangents
  • d/dx[sec x] = sec x tan x: Derived from d/dx[1/cos x] using the chain rule; undefined where cos x = 0
  • Co-function pattern: Derivatives of 'co' functions (cos, cot, csc) all have negative signs compared to their counterparts (sin, tan, sec)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Simple harmonic motion: position x(t) = A sin(ωt) gives velocity v(t) = Aω cos(ωt) and acceleration a(t) = -Aω² sin(ωt), showing oscillatory behavior
  • AC electrical circuits: voltage V(t) = V₀ sin(ωt) has derivative V'(t) = V₀ω cos(ωt), the rate of voltage change used in capacitor/inductor analysis
  • Wave mechanics: the derivative of a sine wave gives its phase-shifted cosine companion, fundamental to understanding interference and diffraction patterns
