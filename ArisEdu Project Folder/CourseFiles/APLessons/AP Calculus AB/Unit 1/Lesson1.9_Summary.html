AP Calculus AB - Unit 1 - Lesson 1.9 - Summary

Key Concepts

One-to-One Functions and Inverses are functions where each output corresponds to exactly one input: f(a) = f(b) implies a = b. The horizontal line test determines if a function is one-to-one. One-to-one functions have inverse functions f⁻¹ where f⁻¹(f(x)) = x. The graph of f⁻¹ is the reflection of f across y = x. The derivative of an inverse function is (f⁻¹)'(a) = 1/f'(f⁻¹(a)), derived from the chain rule applied to f(f⁻¹(x)) = x.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • One-to-one function: A function where each output value corresponds to exactly one input value; passes the horizontal line test
  • Inverse function: f⁻¹ reverses f: if f(a) = b, then f⁻¹(b) = a; domain of f⁻¹ = range of f, and vice versa
  • Horizontal line test: A function is one-to-one if and only if every horizontal line intersects its graph at most once
  • Inverse function derivative: (f⁻¹)'(a) = 1/f'(f⁻¹(a)); the derivative of the inverse at a point is the reciprocal of the derivative of f at the corresponding point
  • Restricting domain: Non-one-to-one functions can be made invertible by restricting the domain (e.g., sin x restricted to [-π/2, π/2] gives arcsin)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Cryptography: encryption functions must be one-to-one (so each plaintext maps to a unique ciphertext) and have computable inverses for authorized decryption
  • Temperature conversion: F = (9/5)C + 32 and its inverse C = (5/9)(F - 32) are one-to-one functions used worldwide for unit conversion
  • Inverse kinematics in robotics: computing joint angles from desired end-effector position requires inverting the forward kinematic equations
