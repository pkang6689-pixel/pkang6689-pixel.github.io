AP Calculus AB - Unit 5 - Lesson 5.9 - Summary

Key Concepts

Extreme Value Theorem guarantees that every continuous function on a closed, bounded interval [a,b] achieves both its absolute maximum and absolute minimum values. This theorem provides the theoretical foundation for optimization: it assures us that optimal values exist before we search for them. The theorem fails if the interval is open (extrema may not be attained), the function is not continuous (could have gaps), or the domain is unbounded.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Extreme Value Theorem: If f is continuous on [a,b], then f attains an absolute maximum value M and an absolute minimum value m on [a,b]
  • Closed interval requirement: The interval must include both endpoints; on open intervals (a,b), extrema may be approached but not attained
  • Continuity requirement: Discontinuous functions can 'jump over' potential extrema; continuity ensures no gaps in the range
  • Existence vs. location: EVT guarantees extrema exist but does not tell us where; we use critical point analysis and endpoint evaluation to locate them
  • Applications to optimization: Before solving an optimization problem, EVT confirms that a maximum/minimum exists (if the domain is a closed interval)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Structural load analysis: EVT guarantees the existence of a maximum stress point on a continuous beam, ensuring engineers can find and design for worst-case loads
  • Temperature extremes: maximum and minimum daily temperatures are guaranteed to exist (continuous function on closed time interval), justifying weather forecasting
  • Resource allocation: EVT confirms that an optimal investment allocation exists over a bounded feasible region, justifying the search for a maximum return
