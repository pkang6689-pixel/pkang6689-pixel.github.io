AP Calculus AB - Unit 5 - Lesson 5.6 - Summary

Key Concepts

Second Derivative Test provides a quick classification of critical points using concavity. If f'(c) = 0 and f''(c) > 0, then c is a local minimum (the graph is concave up, like a valley). If f'(c) = 0 and f''(c) < 0, then c is a local maximum (concave down, like a hilltop). If f''(c) = 0, the test is inconclusive—use the first derivative test instead. The second derivative test is efficient for polynomial and transcendental functions where f'' is easy to evaluate.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Second Derivative Test: If f'(c) = 0: f''(c) > 0 → local min; f''(c) < 0 → local max; f''(c) = 0 → inconclusive
  • Concavity at critical points: Local min: concave up valley (f'' > 0); local max: concave down hill (f'' < 0)
  • When the test fails: When f''(c) = 0, the critical point could be a max, min, or inflection point (e.g., f(x) = x⁴ has a min at x = 0 but f''(0) = 0)
  • Efficiency of the test: Faster than the first derivative test when f'' is easy to compute; avoids analyzing sign changes in a neighborhood
  • Comparison with first derivative test: First derivative test: always works but requires sign analysis on an interval; second derivative test: simpler but can be inconclusive

Real-World Applications and Examples

This topic has significant real-world importance:

  • Profit maximization: after finding the critical point of P(x), checking P''(x) < 0 confirms it's a maximum profit level, not a minimum
  • Structural engineering: the second derivative of a beam's deflection function indicates whether a critical point is a maximum deflection (sag) or minimum
  • Trajectory optimization: confirming that a critical point of a fuel consumption function represents minimum fuel use (f'' > 0) rather than maximum
