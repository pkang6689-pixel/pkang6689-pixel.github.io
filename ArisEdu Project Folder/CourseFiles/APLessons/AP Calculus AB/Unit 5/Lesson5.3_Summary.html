AP Calculus AB - Unit 5 - Lesson 5.3 - Summary

Key Concepts

Increasing and Decreasing Functions uses the first derivative to determine where a function rises and falls. If f'(x) > 0 on an interval, then f is increasing on that interval; if f'(x) < 0, f is decreasing. Sign charts of f'(x) reveal the intervals of increase/decrease. The First Derivative Test uses these sign changes at critical points to classify local extrema: f' changes + to - → local max; f' changes - to + → local min.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Increasing function: f'(x) > 0 on an interval means f is strictly increasing: if a < b then f(a) < f(b)
  • Decreasing function: f'(x) < 0 on an interval means f is strictly decreasing: if a < b then f(a) > f(b)
  • Sign chart of f': A number line showing where f'(x) is positive (+) or negative (-), with critical points marked as boundaries
  • First Derivative Test: At a critical point c: f' changes + to -: local max; f' changes - to +: local min; no sign change: neither
  • Monotonic function: A function that is either entirely increasing or entirely decreasing on its domain; has an inverse function

Real-World Applications and Examples

This topic has significant real-world importance:

  • Stock price analysis: periods where the derivative (price change rate) is positive = bull market (increasing), negative = bear market (decreasing)
  • Marathon pace strategy: analyzing a runner's velocity function to identify periods of acceleration (increasing velocity) and deceleration (decreasing velocity)
  • Climate data: identifying periods of warming (increasing temperature trend) vs. cooling using derivative analysis of temperature time series
