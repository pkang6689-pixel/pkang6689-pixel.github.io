AP Calculus AB - Unit 4 - Lesson 4.6 - Summary

Key Concepts

Linearization and Approximation uses derivatives for estimating function values and analyzing errors. The linear approximation (tangent line approximation) at x = a: L(x) = f(a) + f'(a)(x - a). Differentials: if y = f(x), then dy = f'(x)dx approximates the actual change Δy. The error in linear approximation is related to f''(x) and (x-a)². Newton's method uses tangent lines iteratively to find roots: xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ).

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Linear approximation: L(x) = f(a) + f'(a)(x - a); the tangent line at x = a used to estimate f(x) for x near a
  • Differentials: dy = f'(x)dx; approximates Δy = f(x + dx) - f(x) when dx is small; useful for error propagation
  • Newton's method: xₙ₊₁ = xₙ - f(xₙ)/f'(xₙ); iteratively finds roots of f(x) = 0 using tangent lines; converges quadratically near roots
  • Error in approximation: The error |f(x) - L(x)| ≈ ½|f''(c)|(x-a)² for some c between a and x; smaller when f'' is small or x is close to a
  • Propagation of error: If y = f(x) and x has error Δx, then the error in y is approximately |f'(x)|·Δx

Real-World Applications and Examples

This topic has significant real-world importance:

  • Approximating √4.02 ≈ √4 + (1/2√4)(0.02) = 2 + 0.005 = 2.005; actual value is 2.00499..., showing excellent accuracy for nearby values
  • Engineering error analysis: propagation of measurement uncertainty through formulas using differentials to estimate how input errors affect computed results
  • Newton's method in GPS: iteratively solving nonlinear equations to determine position from satellite distance measurements
