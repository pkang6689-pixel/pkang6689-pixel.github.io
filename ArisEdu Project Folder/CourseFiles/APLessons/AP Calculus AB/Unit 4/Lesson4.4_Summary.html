AP Calculus AB - Unit 4 - Lesson 4.4 - Summary

Key Concepts

Optimization Problems use derivatives to find maximum and minimum values of functions in real-world contexts. The method: (1) identify the quantity to optimize, (2) express it as a function of one variable using constraints, (3) find critical points where f'(x) = 0, (4) verify maxima/minima using the first or second derivative test, (5) check endpoints if on a closed interval. Optimization applies to maximizing profit, minimizing cost, finding optimal dimensions, and determining the most efficient design.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Global vs. local extrema: Global: the absolute maximum/minimum on the entire domain; local: max/min in a neighborhood; global extrema include endpoint values
  • Critical points: Points where f'(x) = 0 or f'(x) is undefined; candidates for local extrema
  • First derivative test: If f' changes from + to - at c: local maximum; from - to +: local minimum; no sign change: neither
  • Second derivative test: If f'(c) = 0 and f''(c) < 0: local maximum; f''(c) > 0: local minimum; f''(c) = 0: test is inconclusive
  • Optimization strategy: Define objective function, express in one variable using constraints, find critical points, test for max/min, interpret the answer

Real-World Applications and Examples

This topic has significant real-world importance:

  • Minimizing packaging material: finding the dimensions of a box with minimum surface area for a given volume (cuboid optimization)
  • Maximizing crop yield: determining the optimal planting density that balances per-plant yield against total plant count
  • Minimizing travel time: finding the optimal route combining different speed zones (e.g., running on sand vs. swimming) using calculus of optimization
