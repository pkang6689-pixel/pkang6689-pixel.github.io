AP Calculus AB - Unit 6 - Lesson 6.10 - Summary

Key Concepts

Antiderivatives: Trigonometric and Exponential Forms covers integration of transcendental functions. Key formulas: ∫sin x dx = -cos x + C, ∫cos x dx = sin x + C, ∫sec²x dx = tan x + C, ∫csc²x dx = -cot x + C, ∫sec x tan x dx = sec x + C, ∫csc x cot x dx = -csc x + C. Exponential: ∫eˣdx = eˣ + C, ∫aˣdx = aˣ/ln a + C. Combined with u-substitution, these handle a wide range of AP Calculus integrals.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Trigonometric antiderivatives: ∫sin x dx = -cos x + C; ∫cos x dx = sin x + C; ∫sec²x dx = tan x + C; each reverses the corresponding derivative
  • Reciprocal trig antiderivatives: ∫csc²x dx = -cot x + C; ∫sec x tan x dx = sec x + C; ∫csc x cot x dx = -csc x + C
  • Exponential antiderivatives: ∫eˣdx = eˣ + C; ∫e^(kx)dx = (1/k)e^(kx) + C; ∫aˣdx = aˣ/(ln a) + C
  • Logarithmic antiderivative: ∫(1/x)dx = ln|x| + C; the antiderivative of x⁻¹ is NOT the power rule (which would give x⁰/0)
  • Combining with u-substitution: ∫sin(3x)dx: u = 3x, du = 3dx → (1/3)∫sin u du = -(1/3)cos(3x) + C

Real-World Applications and Examples

This topic has significant real-world importance:

  • Modeling oscillatory motion: integrating v(t) = A cos(ωt) gives position s(t) = (A/ω)sin(ωt) + C for springs, pendulums, and vibrating systems
  • Electrical engineering: integrating sinusoidal voltage V(t) = V₀sin(ωt) across a capacitor to find charge: Q = ∫V dt = -(V₀/ω)cos(ωt) + C
  • Radioactive decay: integrating the decay rate dN/dt = -λN gives N(t) = N₀e^(-λt), the fundamental decay equation
