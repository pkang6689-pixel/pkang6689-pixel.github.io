AP Calculus AB - Unit 6 - Lesson 6.8 - Summary

Key Concepts

Average Value of a Function extends the concept of arithmetic mean to continuous functions. The average value of f on [a,b] is f_avg = (1/(b-a))∫ₐᵇ f(x)dx. The Mean Value Theorem for Integrals guarantees there exists a c in [a,b] where f(c) = f_avg—the function actually achieves its average value at some point. Average value has the same units as f(x) and represents the constant function that would produce the same integral (same total area) over [a,b].

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Average value formula: f_avg = (1/(b-a))∫ₐᵇ f(x)dx; the integral divided by the interval length
  • Mean Value Theorem for Integrals: If f is continuous on [a,b], there exists c ∈ [a,b] where f(c) = (1/(b-a))∫ₐᵇ f(x)dx
  • Geometric interpretation: The average value is the height of the rectangle with the same base (b-a) and the same area as the region under f
  • Units: Average value has the same units as f(x); the (1/(b-a)) factor divides out the length units
  • Average rate of change vs. average value: Average rate of change of f: [f(b)-f(a)]/(b-a); average value of f': same thing (by FTC)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Average temperature: f_avg = (1/24)∫₀²⁴ T(t)dt gives the average temperature over a 24-hour period, used for energy consumption estimates
  • Average voltage of AC power: V_avg = (1/T)∫₀ᵀ V(t)dt; for sinusoidal AC, the average over a full cycle is zero, motivating the use of RMS voltage
  • Average drug concentration: (1/T)∫₀ᵀ C(t)dt determines the mean effective concentration in the bloodstream over a dosing interval
