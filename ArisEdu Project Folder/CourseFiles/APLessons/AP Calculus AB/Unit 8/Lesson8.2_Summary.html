AP Calculus AB - Unit 8 - Lesson 8.2 - Summary

Key Concepts

Area Between Curves computes the area enclosed between two functions. If f(x) ≥ g(x) on [a,b], then Area = ∫ₐᵇ [f(x) - g(x)]dx. When the curves cross, split the integral at intersection points and use absolute values. For horizontal slicing: if right curve minus left curve, Area = ∫_c^d [right(y) - left(y)]dy, integrating with respect to y. The choice of vertical vs. horizontal slicing depends on which gives simpler integrands.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Area between curves (vertical): Area = ∫ₐᵇ [top(x) - bottom(x)]dx; always subtract lower from upper to get positive area
  • Area between curves (horizontal): Area = ∫_c^d [right(y) - left(y)]dy; integrate with respect to y when functions are expressed as x = f(y)
  • Finding intersection points: Set f(x) = g(x) and solve; these x-values define the limits of integration
  • Multiple intersection points: Split into separate integrals between consecutive intersections, taking absolute value of each
  • Choosing vertical vs. horizontal: Use whichever method avoids splitting into multiple integrals; if curves are easier to express as x = f(y), slice horizontally

Real-World Applications and Examples

This topic has significant real-world importance:

  • Land surveying: computing the area of irregularly shaped plots bounded by curves (roads, rivers) using integration
  • Economics: consumer surplus = area between the demand curve and market price line = ∫(demand function - price)dx
  • Engineering: computing cross-sectional area of unusually shaped beams or channels for structural and flow analysis
