AP Calculus AB - Unit 3 - Lesson 3.2 - Summary

Key Concepts

Inverse Function Differentiation uses the relationship between a function and its inverse to find derivatives. If y = f⁻¹(x), then f(y) = x, so f'(y)·dy/dx = 1, giving dy/dx = 1/f'(y) = 1/f'(f⁻¹(x)). This yields the derivatives of inverse trigonometric functions: d/dx[arcsin x] = 1/√(1-x²), d/dx[arccos x] = -1/√(1-x²), d/dx[arctan x] = 1/(1+x²). These are important for integration and solving applied problems.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Inverse function theorem: If f is differentiable and f'(x) ≠ 0, then f⁻¹ is differentiable and (f⁻¹)'(a) = 1/f'(f⁻¹(a))
  • d/dx[arcsin x] = 1/√(1-x²): Derived from sin(y) = x → cos(y)·dy/dx = 1 → dy/dx = 1/cos(y) = 1/√(1-x²)
  • d/dx[arctan x] = 1/(1+x²): Derived from tan(y) = x → sec²(y)·dy/dx = 1 → dy/dx = 1/sec²(y) = 1/(1+x²)
  • d/dx[arccos x] = -1/√(1-x²): Negative of the arcsin derivative; arccos is a decreasing function
  • Chain rule with inverses: d/dx[arctan(g(x))] = g'(x)/(1+[g(x)]²); combine inverse derivatives with the chain rule

Real-World Applications and Examples

This topic has significant real-world importance:

  • Navigation: calculating bearing angles using arctan(Δy/Δx) and differentiating to find rate of angle change for tracking systems
  • Control systems engineering: inverse tangent functions model phase shifts in frequency response, with derivatives indicating sensitivity to frequency changes
  • Probability: the normal distribution PDF involves e^(-x²/2), whose antiderivative relates to the error function (erf), constructed from inverse function theory
