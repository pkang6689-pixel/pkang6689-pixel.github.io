AP Calculus AB - Unit 3 - Lesson 3.4 - Summary

Key Concepts

Higher-Order Derivatives and Concavity extends differentiation beyond the first derivative. The second derivative f''(x) measures the rate of change of f'(x) and determines concavity: f''(x) > 0 means concave up (cup-shaped), f''(x) < 0 means concave down (cap-shaped). Inflection points occur where concavity changes (f''(x) = 0 or undefined, with sign change). In physics, if s(t) is position, s'(t) is velocity, and s''(t) is acceleration—the 'jerk' is s'''(t).

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Second derivative: f''(x) = d²y/dx²; the derivative of the first derivative; measures how the slope itself is changing
  • Concave up: f''(x) > 0; the graph curves upward like a cup; the slope is increasing
  • Concave down: f''(x) < 0; the graph curves downward like a cap; the slope is decreasing
  • Inflection point: A point where concavity changes; f''(x) = 0 or undefined AND the sign of f''(x) changes
  • Higher-order derivatives: f'''(x), f⁴(x), etc.; in physics: position → velocity → acceleration → jerk → snap → crackle → pop

Real-World Applications and Examples

This topic has significant real-world importance:

  • Acceleration as the second derivative of position: a car speeding up has positive s''(t), slowing down has negative s''(t), and constant speed has s''(t) = 0
  • Economics: the second derivative of profit with respect to quantity determines whether a critical point is a maximum (f'' < 0) or minimum (f'' > 0)
  • Roller coaster design: inflection points (where concavity changes) create the thrilling transitions between upward and downward curves that riders feel
