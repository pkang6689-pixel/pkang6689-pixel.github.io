AP Calculus AB - Unit 3 - Lesson 3.3 - Summary

Key Concepts

Derivatives of Exponential and Logarithmic Functions in depth covers differentiation of functions involving variable bases and exponents. Logarithmic differentiation handles functions of the form y = [f(x)]^g(x) by taking ln of both sides: ln y = g(x)·ln(f(x)), then differentiating implicitly. This technique is essential for functions like xˣ, (sin x)^x, or x^(1/x) where both base and exponent are variable.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Logarithmic differentiation: Take ln of both sides, differentiate implicitly, solve for dy/dx; essential when both base and exponent are variable
  • d/dx[f(x)^g(x)]: y = f^g → ln y = g·ln f → (1/y)(dy/dx) = g'·ln f + g·f'/f → dy/dx = f^g · [g'·ln f + g·f'/f]
  • d/dx[xˣ]: Using logarithmic differentiation: ln y = x ln x → dy/dx = xˣ(ln x + 1)
  • d/dx[e^(f(x))]: = e^(f(x)) · f'(x); the most common chain rule application with exponentials
  • Simplification with logarithms: Taking ln before differentiating can simplify products, quotients, and powers: ln(abc/de) = ln a + ln b + ln c - ln d - ln e

Real-World Applications and Examples

This topic has significant real-world importance:

  • Compound interest: computing d/dn[(1 + r/n)^(nt)] using logarithmic differentiation to analyze how compounding frequency affects growth
  • Entropy in information theory: H = -Σ pᵢ log₂(pᵢ) involves logarithmic functions whose derivatives measure information sensitivity
  • Modeling the spread of viral content online: growth functions with variable exponents like n(t)^(f(t)) require logarithmic differentiation for rate analysis
