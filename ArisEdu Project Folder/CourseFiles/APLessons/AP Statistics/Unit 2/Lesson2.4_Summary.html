AP Statistics - Unit 2 - Lesson 2.4 - Summary

Key Concepts

Transformations and Power Functions address non-linear relationships by applying mathematical transformations (logarithmic, power, exponential) to achieve linearity. When a residual plot from a linear fit shows a clear curve, transforming x, y, or both can create a linear relationship suitable for least-squares regression. Common transformations include log(y) vs. x for exponential relationships and log(y) vs. log(x) for power relationships, with back-transformation needed to interpret predictions in original units.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Logarithmic transformation: Taking the logarithm of variable values to linearize exponential or power relationships for regression analysis
  • Exponential model: A model of the form y = ab^x, which becomes linear when expressed as log(y) = log(a) + x·log(b)
  • Power model: A model of the form y = ax^b, which becomes linear when expressed as log(y) = log(a) + b·log(x)
  • Linearization: The process of transforming non-linear data so that a linear regression model is appropriate
  • Back-transformation: Converting predictions from transformed scale back to the original scale (e.g., exponentiating a log-predicted value)

Real-World Applications and Examples

This topic has significant real-world importance:

  • Modeling population growth of bacteria over time using exponential transformation, then predicting colony size at future time points
  • Analyzing the power relationship between body mass and metabolic rate in animals using log-log transformation
  • Linearizing the relationship between time and radioactive decay to estimate half-lives using transformed regression
