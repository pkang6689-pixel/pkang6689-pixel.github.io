AP Statistics - Unit 2 - Lesson 2.2 - Summary

Key Concepts

Distributions describe the pattern of values a variable takes and how frequently each value occurs. Key distributions include the uniform distribution (all outcomes equally likely), binomial distribution (counts of successes in fixed trials), and normal distribution (bell-shaped continuous outcomes). Choosing the correct probability distribution model for a given situation is fundamental to making valid statistical inferences.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Probability distribution: A function that assigns probabilities to every possible value of a random variable, with all probabilities summing to 1
  • Uniform distribution: A distribution where every outcome in the sample space has equal probability
  • Expected value E(X): The long-run average value of a random variable, calculated as the weighted mean of all possible values: E(X) = Σ[x·P(x)]
  • Standard deviation of a random variable: The square root of variance, measuring the typical deviation of a random variable from its expected value: σ = √Var(X)
  • Cumulative distribution function (CDF): F(x) = P(X ≤ x); gives the probability that a random variable takes a value less than or equal to x

Real-World Applications and Examples

This topic has significant real-world importance:

  • Quality inspection using the binomial distribution to calculate the probability of finding defective items in a batch sample
  • Modeling customer arrival times at a store using appropriate probability distributions to optimize staffing schedules
  • Financial risk modeling using probability distributions to estimate potential portfolio losses (Value at Risk calculations)
