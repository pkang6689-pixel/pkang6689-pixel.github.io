AP Statistics - Unit 1 - Lesson 1.1 - Summary

Key Concepts

Data Collection and Types of Variables forms the foundation of statistical analysis by distinguishing between categorical variables (which assign labels or categories) and quantitative variables (which take numerical measurements). Proper identification of variable types determines which statistical methods, graphs, and summary measures are appropriate. Data collection methods—including census, survey, and experiment—each have distinct strengths and limitations that affect the validity and generalizability of conclusions.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Categorical variable: A variable that places individuals into groups or categories (e.g., gender, political party, blood type) and is summarized using counts or proportions
  • Quantitative variable: A variable that takes numerical values for which arithmetic operations (mean, standard deviation) are meaningful; can be discrete (counts) or continuous (measurements)
  • Observational unit: The individual person, animal, object, or entity on which data are collected in a study
  • Explanatory variable: The variable suspected of influencing or predicting the response variable in a study
  • Response variable: The outcome variable being measured or observed; the variable we want to explain or predict

Real-World Applications and Examples

This topic has significant real-world importance:

  • Classifying patient data in clinical trials as categorical (treatment group) versus quantitative (blood pressure reading) to select appropriate analysis methods
  • Designing customer satisfaction surveys that properly distinguish between ordinal scales (satisfaction rating 1-5) and nominal categories (product purchased)
  • Identifying confounding variables in public health studies, such as separating the effect of diet from exercise on heart disease rates
