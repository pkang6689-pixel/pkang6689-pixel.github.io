AP Statistics - Unit 9 - Lesson 9.1 - Summary

Key Concepts

Linear Regression and Inference extends exploratory regression to formal inference about the population regression line β₀ + β₁x. The sample regression line b₀ + b₁x estimates the population parameters, with residuals assumed to be independent and normally distributed with constant variance (LINE conditions: Linear, Independent, Normal, Equal variance). Inference requires checking these conditions via residual plots and normal probability plots before constructing confidence intervals or significance tests for the slope β₁.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Population regression model: y = β₀ + β₁x + ε, where ε ~ N(0, σ); the true linear relationship between x and y in the population
  • LINE conditions: Linear relationship, Independent observations, Normal residuals, and Equal (constant) variance of residuals—must be verified before inference
  • Standard error of the slope: SE(b₁) = s/√(Σ(x - x̄)²), where s is the regression standard error; measures the precision of the slope estimate
  • Regression standard error (s): s = √(Σresidual²/(n-2)); estimates the standard deviation of the residuals (σ) with n-2 degrees of freedom
  • Residual analysis: Plotting residuals vs. x and making a normal probability plot of residuals to verify LINE conditions before performing inference

Real-World Applications and Examples

This topic has significant real-world importance:

  • Testing whether temperature is a significant predictor of ice cream sales using regression inference on slope β₁ with a t-test
  • Economists checking LINE conditions before publishing a regression model of how education level predicts income
  • Environmental scientists using residual plots to verify that a linear model is appropriate for predicting water quality from upstream land use data
