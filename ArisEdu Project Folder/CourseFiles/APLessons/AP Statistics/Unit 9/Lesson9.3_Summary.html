AP Statistics - Unit 9 - Lesson 9.3 - Summary

Key Concepts

Significance Tests for Slopes test H₀: β₁ = 0 (no linear relationship) using t = b₁/SE(b₁) with df = n-2. Rejecting H₀ means there is statistically significant evidence that the explanatory variable x is a useful linear predictor of y. This test is equivalent to testing whether the correlation coefficient ρ equals zero. The F-test from ANOVA output provides the same conclusion for simple linear regression (F = t²).

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • t-test for slope: Tests H₀: β₁ = 0 using t = b₁/SE(b₁) with df = n - 2; determines whether x is a significant linear predictor of y
  • Null hypothesis for slope: H₀: β₁ = 0 states that there is no linear relationship between x and y in the population
  • Connection to correlation: Testing β₁ = 0 is equivalent to testing ρ = 0; both assess whether a linear relationship exists
  • ANOVA F-test: In simple regression, F = t²; the F-test from the ANOVA table produces the same p-value as the t-test for the slope
  • Practical significance: A statistically significant slope may have little practical importance if the effect is very small; always consider R² and the magnitude of b₁

Real-World Applications and Examples

This topic has significant real-world importance:

  • Testing whether advertising spending significantly predicts sales revenue (H₀: β₁ = 0) to justify continued marketing budget allocation
  • Climate scientists testing whether year is a significant predictor of global temperature anomaly (testing for a statistically significant warming trend)
  • Educational researchers testing whether hours of tutoring significantly predict improvement in test scores, while noting that R² = 0.12 suggests other factors dominate
