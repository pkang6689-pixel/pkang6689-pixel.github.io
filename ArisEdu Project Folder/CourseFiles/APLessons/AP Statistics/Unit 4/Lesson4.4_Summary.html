AP Statistics - Unit 4 - Lesson 4.4 - Summary

Key Concepts

Discrete Random Variables take countable values (often integers), with each value having an associated probability. The probability distribution table lists all possible values and their probabilities (which must sum to 1), enabling calculation of the mean μ = E(X) = Σ[x·P(x)] and variance σ² = Σ[(x-μ)²·P(x)]. Linear transformations Y = a + bX have predictable effects: μY = a + bμX and σY = |b|σX.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Discrete random variable: A variable that takes a finite or countably infinite number of specific values, each with a definite probability
  • Probability distribution (discrete): A table, formula, or graph giving every possible value of a discrete random variable and its associated probability
  • Expected value E(X): The mean of a random variable: μ = Σ[x·P(x)]; represents the long-run average outcome over many repetitions
  • Variance of a discrete random variable: σ² = Σ[(x - μ)²·P(x)]; measures the expected squared deviation from the mean
  • Linear transformation rules: For Y = a + bX: E(Y) = a + b·E(X) and Var(Y) = b²·Var(X); adding a constant shifts the mean but does not change spread

Real-World Applications and Examples

This topic has significant real-world importance:

  • Insurance companies calculating expected claim payouts by multiplying each possible claim amount by its probability to set profitable premium rates
  • Game theory analysis determining expected winnings in lottery games by computing E(X) = Σ[prize × P(prize)] and comparing to ticket cost
  • Business decision analysis using expected value to choose between investment options with different probability distributions of returns
