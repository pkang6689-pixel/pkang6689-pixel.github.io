AP Statistics - Unit 7 - Lesson 7.3 - Summary

Key Concepts

Significance Tests for Means follow the four-step hypothesis testing process: (1) state H₀: μ = μ₀ and Hₐ, (2) verify conditions (random, 10%, Normal/Large Sample), (3) compute t = (x̄ - μ₀)/(s/√n) and find the p-value from the t-distribution with df = n-1, (4) conclude in context. The relationship between confidence intervals and significance tests is direct: a value μ₀ falls outside a C% confidence interval if and only if a two-sided test at significance level α = 1-C rejects H₀: μ = μ₀.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Significance test for μ: A formal procedure using t = (x̄ - μ₀)/(s/√n) to decide whether sample evidence contradicts H₀: μ = μ₀
  • Type I error: Rejecting H₀ when it is actually true; probability equals α, the significance level
  • Type II error: Failing to reject H₀ when Hₐ is actually true; probability denoted β
  • Power: The probability of correctly rejecting H₀ when Hₐ is true; Power = 1 - β; increased by larger n, larger effect size, or larger α
  • Effect size: The magnitude of the difference between the true parameter and the null value; determines the practical importance of a statistically significant result

Real-World Applications and Examples

This topic has significant real-world importance:

  • Environmental agency testing whether mean mercury concentration in fish exceeds the 0.5 ppm safety limit (H₀: μ = 0.5, Hₐ: μ > 0.5) to issue consumption advisories
  • Auto manufacturer testing whether a new engine design achieves the claimed 35 mpg fuel efficiency based on a sample of test drives
  • Pharmaceutical company conducting power analysis to determine sample size needed to detect a clinically meaningful 5-point drop in blood pressure
