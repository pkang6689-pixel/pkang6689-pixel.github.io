Lesson 8.4: Multiple Regression (introductory) - Summary

Key Concepts: Multiple Regression (introductory)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is multiple regression?

Multiple regression extends simple regression by allowing more than one predictor to explain variation in a single response.

What is multicollinearity?

Multicollinearity means predictors share redundant information, making their individual coefficients unstable.

What is adjusted R²?

Adjusted R² accounts for the number of predictors, only increasing when a new variable provides a genuine improvement.

What is a dummy variable?

Dummy variables code categories numerically (e.g., male = 0, female = 1) so they can enter a regression equation.

What is stepwise regression?

Stepwise methods automate variable selection, though they have limitations such as dependence on the order of variable entry.

A hospital uses multiple regression to predict length of stay (y, days) from patient age (x₁), severity score (x₂), and number of procedures (x₃): ŷ = 1.5 + 0.03x₁ + 0.8x₂ + 1.2x₃. For a 60-year-old patient with severity score 5 and 2 procedures, what is the predicted stay?

ŷ = 1.5 + 0.03(60) + 0.8(5) + 1.2(2) = 1.5 + 1.8 + 4.0 + 2.4 = 9.7 days.

A researcher predicting college GPA includes both SAT Math score and SAT Total score as predictors. The VIF for SAT Math is 18.5. What is the problem and what should she consider?

VIF > 10 signals severe multicollinearity. Since SAT Total includes SAT Math, these predictors are redundant and one should be removed.

A health study uses height (x₁, inches) and weight (x₂, lbs) to predict blood pressure. These predictors are correlated (r = 0.75). What is the practical consequence for the model?

When predictors are correlated, their shared information makes individual coefficients harder to estimate precisely.

An agricultural researcher models crop yield using rainfall (x₁), fertilizer (x₂), and sunlight hours (x₃). The overall F-test has p-value < 0.001, but the individual p-values for x₁ and x₃ are both above 0.05. What is the best interpretation?

A significant F-test with individually insignificant predictors often points to multicollinearity — the predictors together explain variation, but their shared information prevents any single one from standing out.

Real-World Applications

Example: What is multiple regression?

Multiple regression extends simple regression by allowing more than one predictor to explain variation in a single response.

Example: What does the F-test assess in multiple regression?

The F-test compares the model with predictors to a model with no predictors (intercept only) to determine overall significance.

Example: A hospital uses multiple regression to predict length of stay (y, days) from patient age (x₁), severity score (x₂), and number of procedures (x₃): ŷ = 1.5 + 0.03x₁ + 0.8x₂ + 1.2x₃. For a 60-year-old patient with severity score 5 and 2 procedures, what is the predicted stay?

ŷ = 1.5 + 0.03(60) + 0.8(5) + 1.2(2) = 1.5 + 1.8 + 4.0 + 2.4 = 9.7 days.

Example: A real estate model predicts house price (y, in $1,000s) using square footage (x₁), number of bedrooms (x₂), and a dummy variable for pool (x₃ = 1 if pool, 0 if not): ŷ = 50 + 0.1x₁ + 10x₂ + 25x₃. How much more is a house with a pool predicted to sell for, all else equal?

The coefficient on the pool dummy variable is 25, meaning a pool adds a predicted $25,000 to the house price, holding square footage and bedrooms constant.

Example: A researcher predicting college GPA includes both SAT Math score and SAT Total score as predictors. The VIF for SAT Math is 18.5. What is the problem and what should she consider?

VIF > 10 signals severe multicollinearity. Since SAT Total includes SAT Math, these predictors are redundant and one should be removed.

Example: An insurance company models claim amount (y) using age (x₁), driving experience (x₂), and an interaction term (x₁ × x₂). The interaction coefficient is negative. What does this mean?

A negative interaction term means the combined effect of age and experience reduces claims more than each factor alone would suggest; experience moderates the effect of age.

Example: A school district predicts student test scores using class size (x₁), teacher experience in years (x₂), and free lunch percentage (x₃). Adding x₃ raises R² from 0.58 to 0.59 but adjusted R² drops from 0.56 to 0.55. Should x₃ be kept?

When adjusted R² decreases despite R² increasing, the new variable adds more noise than signal and the simpler model is preferred.

Example: A retailer adds 'day of the week' as a predictor in a sales model. Day has 7 categories. How many dummy variables are needed?

For a categorical variable with k categories, k − 1 dummy variables are needed; one category serves as the reference level.
