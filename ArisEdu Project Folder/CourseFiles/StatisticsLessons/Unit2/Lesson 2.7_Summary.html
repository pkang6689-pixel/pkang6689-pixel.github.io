Lesson 2.7: Choosing Appropriate Graphs - Summary

Key Concepts: Choosing Appropriate Graphs

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a 'time series plot'?

A time series plot uses a line graph with time on the x-axis and the measured variable on the y-axis to show trends over time.

What is a 'stacked bar graph'?

A stacked bar graph divides each bar into colored segments representing sub-categories, showing both total and composition.

What is a 'dot plot'?

A dot plot uses dots stacked above a number line, with one dot per observation, showing both individual values and the distribution.

What is a 'segmented bar graph' (100% stacked bar)?

A 100% stacked bar graph makes all bars the same height and shows the percentage breakdown within each bar, facilitating proportion comparisons.

What is a 'pictograph'?

A pictograph uses repeated symbols (like small icons) where each symbol represents a certain number of items, making data visually intuitive.
