Lesson 4.2: Counting Principles (permutations, combinations) - Summary

Key Concepts: Counting Principles (permutations, combinations)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a factorial?

n! = n × (n−1) × (n−2) × ... × 2 × 1. For example, 4! = 24.

What is a permutation?

A permutation is an arrangement where order matters. ABC is different from CBA.

What is a combination?

A combination counts selections without regard to order. Choosing {A, B, C} is the same selection as {C, A, B}.

What is a 'permutation with repetition'?

When some items are identical, we divide by the factorials of the repeat counts. Example: 'MISSISSIPPI' has 11!/(4!4!2!) arrangements.

What is a 'circular permutation'?

Circular permutations fix one object to eliminate rotational duplicates. n objects in a circle = (n−1)! arrangements.

What is the 'addition principle' in counting?

The addition principle is used when choosing between mutually exclusive alternatives: total ways = sum of ways for each alternative.
