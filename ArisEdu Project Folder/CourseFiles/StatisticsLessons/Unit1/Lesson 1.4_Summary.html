Lesson 1.4: Types of Data (qualitative vs. quantitative) - Summary

Key Concepts: Types of Data (qualitative vs. quantitative)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is qualitative data?

Qualitative (categorical) data describes qualities or categories, such as gender, color, or brand, rather than numerical amounts.

What is quantitative data?

Quantitative data represents numerical values that can be counted or measured, such as height, weight, or number of items.

What is a histogram used for?

A histogram groups quantitative data into bins (intervals) and uses adjacent bars to display the frequency distribution.

What is the key difference between discrete and continuous data?

Discrete values are countable with gaps (e.g., 0, 1, 2), while continuous values can be any number within an interval (e.g., 3.14159).

What is a bar chart best used for?

Bar charts use separated bars to show the frequency or proportion of different categories, making them ideal for categorical data.

Real-World Applications

Example: Which is an example of quantitative data?

Height is a measurable numerical value, making it quantitative data.

Example: Eye color is an example of:

Eye color describes a category, not a numerical measurement, so it is qualitative.

Example: Which of the following is NOT an example of qualitative data?

Number of pets is a count, making it a quantitative (discrete) value rather than qualitative.

Example: A researcher studying phone usage records each participant's phone brand (Apple, Samsung, Google) and daily screen time in hours (3.2, 5.1, 4.7). The appropriate graph for the phone brand data would be a _____ and for screen time a _____.

Phone brand is categorical, best shown with a bar chart, while screen time is continuous quantitative data, best shown with a histogram.

Example: A city collects zip code data from residents (e.g., 90210, 10001, 30301). A new analyst calculates the 'average zip code' as 43,504. Why is this meaningless?

Although zip codes are written as numbers, they are qualitative labels for geographic areas, so computing an average has no meaningful interpretation.

Example: A school cafeteria tracks both the number of meals served each day (150, 203, 178) and the type of entree offered (pizza, tacos, salad). A statistician wants to find the average number of meals per day and the most popular entree. Which analysis is correct?

Meal counts are discrete quantitative data suitable for averaging, while entree types are categorical data best summarized by the mode (most frequent category).
