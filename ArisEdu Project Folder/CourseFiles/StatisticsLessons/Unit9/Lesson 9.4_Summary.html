Lesson 9.4: Chi-Square Tests - Summary

Key Concepts: Chi-Square Tests

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a goodness-of-fit test?

A goodness-of-fit test compares observed category counts to the counts expected under a hypothesized distribution to see if the data 'fit' the model.

What is a test of independence?

A test of independence uses a contingency table to examine whether there is an association between two categorical variables from a single sample.

What is a test of homogeneity?

A homogeneity test compares the distribution of a categorical variable across two or more populations to see if they share the same proportions.

What is the difference between independence and homogeneity tests?

Both use the same chi-square formula and same df calculation. The difference is conceptual: independence examines two variables from one sample; homogeneity compares one variable's distribution across separate populations.

A geneticist crosses two heterozygous pea plants and observes 315 round-yellow, 108 round-green, 101 wrinkled-yellow, and 32 wrinkled-green offspring. The expected Mendelian ratio is 9:3:3:1. With 556 total offspring, what is the expected count for round-yellow?

The 9:3:3:1 ratio means 9/16 should be round-yellow. Expected count = 556 x (9/16) = 556 x 0.5625 = 312.75.

A hospital compares patient satisfaction (satisfied, neutral, dissatisfied) across three departments (surgery, ER, pediatrics). The row totals are Surgery: 200, ER: 150, Pediatrics: 100. The column totals are Satisfied: 250, Neutral: 120, Dissatisfied: 80. What is the expected count for the Surgery-Satisfied cell?

Grand total = 200 + 150 + 100 = 450. Expected = (Row total x Column total) / Grand total = (200 x 250) / 450 = 50,000 / 450 = 111.1.

A market researcher surveys 400 consumers about brand preference (A, B, C, D). She expects equal preference across all four brands. She observes: A = 120, B = 95, C = 90, D = 95. The chi-square value is 4.75. With df = 3 and a critical value of 7.815 at alpha = 0.05, what is the conclusion?

Chi-square = 4.75 < critical value 7.815, so we fail to reject H-null. Expected count for each brand = 400/4 = 100 (all >= 5). The observed differences are not statistically significant.

A school district examines whether school lunch choice (hot meal, sandwich, salad) is independent of grade level (elementary, middle, high). The chi-square test yields chi-square = 15.8 with df = 4. The critical value at alpha = 0.01 is 13.28. What is the conclusion?

Chi-square = 15.8 > critical value 13.28 at alpha = 0.01, so we reject H-null. Lunch choice and grade level are significantly associated even at the 1% significance level.

A die is rolled 120 times with results: 1 (25 times), 2 (17 times), 3 (19 times), 4 (22 times), 5 (18 times), 6 (19 times). If the die is fair, each expected count is 20. What is the chi-square contribution from face 1?

Chi-square contribution from face 1 = (O - E)^2 / E = (25 - 20)^2 / 20 = 25/20 = 1.25.
