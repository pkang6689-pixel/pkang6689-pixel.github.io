Lesson 9.2: Expected Value - Summary

Key Concepts: Expected Value

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the expected value of a random variable?

The expected value (or mean) is the probability-weighted average of all possible outcomes: E(X) = sum of x_i times P(x_i).

What is the formula for E(X) when X is a continuous random variable?

For continuous random variables, the summation in the discrete formula is replaced by integration: E(X) = integral of x times f(x) dx over all possible values.

What is the relationship between expected value and variance?

Variance quantifies how much a random variable typically deviates from its expected value. It is defined as the expected squared deviation from the mean.

What is the 'law of the unconscious statistician' (LOTUS)?

LOTUS states that E[g(X)] = sum of g(x_i)·P(x_i) for discrete X, allowing you to compute the expected value of a function of X directly from X's distribution.

A lottery ticket costs $5. There is a 1/500 chance of winning $1,000 and a 1/100 chance of winning $50. What is the expected profit per ticket?

E(winnings) = (1/500)(1000) + (1/100)(50) = 2.00 + 0.50 = $2.50. E(profit) = $2.50 - $5.00 = -$2.50 per ticket.

An insurance company charges $600 per year for a home insurance policy. Data shows there is a 2% chance of a $15,000 claim and a 0.5% chance of a $50,000 claim in any given year (assume only one claim can occur). What is the company's expected profit per policy?

E(claims) = 0.02(15,000) + 0.005(50,000) = 300 + 250 = $550. E(profit) = $600 - $550 = $50 per policy.

A game costs $3 to play. You roll two fair dice and win $1 for every dot showing (sum of both dice). What is your expected profit per game?

E(sum of two dice) = 3.5 + 3.5 = 7. You win $7 on average. E(profit) = $7 - $3 = $4 per game.

A salesperson earns a $200 base salary per day plus a $50 commission for each sale. If the expected number of sales per day is 4, what is the expected daily income?

Let X = number of sales. Income = 200 + 50X. E(Income) = 200 + 50·E(X) = 200 + 50(4) = 200 + 200 = $400.

A factory produces widgets with the following defect distribution: 0 defects (probability 0.7), 1 defect (probability 0.2), and 2 defects (probability 0.1). Each defect costs $30 to repair. What is the expected repair cost per widget?

E(defects) = 0(0.7) + 1(0.2) + 2(0.1) = 0 + 0.2 + 0.2 = 0.4 defects per widget. E(repair cost) = 0.4 × $30 = $12.00.

A student's grade in a course depends on three exams. Exam 1 (weight 30%) has E(score) = 85, Exam 2 (weight 30%) has E(score) = 78, and the Final (weight 40%) has E(score) = 90. What is the expected weighted average?

E(grade) = 0.30(85) + 0.30(78) + 0.40(90) = 25.5 + 23.4 + 36.0 = 84.9.

A ride-share driver earns $12 per trip. On any given hour, the number of trips completed follows this distribution: 0 trips (10%), 1 trip (30%), 2 trips (40%), 3 trips (20%). What is the driver's expected hourly earnings?

E(trips) = 0(0.10) + 1(0.30) + 2(0.40) + 3(0.20) = 0 + 0.30 + 0.80 + 0.60 = 1.70 trips. E(earnings) = 1.70 × $12 = $20.40.

A venture capitalist invests $100,000 in a startup. There is a 70% chance the startup fails (returning $0), a 20% chance it returns $200,000, and a 10% chance it returns $1,000,000. What is the expected return on the investment?

E(return) = 0.70(0) + 0.20(200,000) + 0.10(1,000,000) = 0 + 40,000 + 100,000 = $140,000. The expected profit is $140,000 - $100,000 = $40,000.

A bakery makes a special cake each day. It costs $20 to make. If it sells (probability 0.8), the revenue is $55. If it does not sell, the baker salvages $5 worth of ingredients. What is the expected daily profit from this cake?

E(revenue) = 0.8($55) + 0.2($5) = $44 + $1 = $45. E(profit) = $45 - $20 = $25.00.

Real-World Applications

Example: Multiplying X by a constant a changes E(X) to:

E(aX) = a · E(X). Multiplying a random variable by a constant scales the expected value by that constant.
