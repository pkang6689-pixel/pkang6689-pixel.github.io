Lesson 5.4: Normal Distribution - Summary

Key Concepts: Normal Distribution

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the Empirical Rule (68-95-99.7 Rule)?

The Empirical Rule provides approximate percentages for data within 1, 2, and 3 standard deviations of the mean in a normal distribution.

What is a z-score?

A z-score standardizes a value by expressing how many standard deviations it lies above or below the mean.

What is a 'z-table'?

A z-table provides the area under the standard normal curve to the left of any given z-score.

What is the 'inverse normal' or 'finding a value from a percentile'?

The inverse normal process works backward from a probability to find the corresponding value or z-score.

The time to complete an exam is normally distributed with mu = 50 minutes and sigma = 8 minutes. What is the probability a student finishes between 42 and 58 minutes?

42 = 50 - 8 and 58 = 50 + 8, so this range is mu +/- 1 sigma. By the Empirical Rule, about 68%.

Real-World Applications

Example: If mu = 100, sigma = 15, the z-score for X = 130 is:

Z = (130 - 100)/15 = 30/15 = 2.

Example: What is a z-score?

A z-score standardizes a value by expressing how many standard deviations it lies above or below the mean.

Example: What does a negative z-score indicate?

A negative z-score means the value lies below the mean by that many standard deviations.

Example: IQ scores are normally distributed with mu = 100 and sigma = 15. What percentage of people have an IQ above 130?

Z = (130 - 100)/15 = 2.0. P(Z > 2) = 0.0228, which is about 2.3%.

Example: A machine fills bottles with a mean of 500 mL and a standard deviation of 5 mL. Bottles with less than 490 mL are rejected. What percentage of bottles are rejected?

Z = (490 - 500)/5 = -2.0. P(Z < -2) = 0.0228, so about 2.3% of bottles are rejected.

Example: A student scores in the 84th percentile on a standardized test. Approximately what z-score does this correspond to?

P(Z < 1.0) = 0.8413, which is approximately the 84th percentile.

Example: SAT scores are normally distributed with mu = 1060 and sigma = 195. What score separates the top 10% of students?

The top 10% starts at the 90th percentile. Z = 1.28. X = 1060 + 1.28(195) = 1060 + 249.6 = about 1310.

Example: A tire company guarantees its tires for 40,000 miles. Tire lifespan is normally distributed with mu = 45,000 miles and sigma = 3,000 miles. What percentage of tires will fail before the guarantee expires?

Z = (40,000 - 45,000)/3,000 = -5,000/3,000 = -1.67. P(Z < -1.67) = 0.0475, so about 4.8%.

Example: The time to complete an exam is normally distributed with mu = 50 minutes and sigma = 8 minutes. What is the probability a student finishes between 42 and 58 minutes?

42 = 50 - 8 and 58 = 50 + 8, so this range is mu +/- 1 sigma. By the Empirical Rule, about 68%.

Example: A company's delivery times are normally distributed with mu = 4 days and sigma = 0.5 days. They promise delivery within 5 days. What percentage of deliveries arrive on time?

Z = (5 - 4)/0.5 = 2.0. P(Z < 2) = 0.9772, so about 97.7% arrive within 5 days.

Example: A grading system assigns an A to the top 10% of test scores. If scores are normally distributed with mu = 72 and sigma = 9, what minimum score earns an A?

The 90th percentile has z = 1.28. X = 72 + 1.28(9) = 72 + 11.52 = 83.52, approximately 83.5.

Example: Birth weights of full-term babies are normally distributed with mu = 7.5 lbs and sigma = 1.2 lbs. A hospital classifies babies under 5.5 lbs as low birth weight. What percentage of full-term babies are classified as low birth weight?

Z = (5.5 - 7.5)/1.2 = -2.0/1.2 = -1.67. P(Z < -1.67) = 0.0475, so about 4.8%.
