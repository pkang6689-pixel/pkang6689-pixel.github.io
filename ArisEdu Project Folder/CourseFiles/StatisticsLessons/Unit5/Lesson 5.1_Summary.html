Lesson 5.1: Discrete Random Variables - Summary

Key Concepts: Discrete Random Variables

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is a random variable?

A random variable assigns a numerical value to each outcome of a random experiment.

What is a probability distribution?

A probability distribution pairs every possible value of a random variable with its probability.

What is the shortcut formula for variance?

The shortcut formula is Var(X) = E(X^2) - [E(X)]^2, which is often easier to compute than the definition.

What is the 'mode' of a discrete probability distribution?

The mode is the most probable value -- the one with the largest P(X = x).

What is a 'probability histogram'?

A probability histogram uses bars to display P(X = x) for each possible value of a discrete random variable.

A bakery sells cupcakes in boxes. Let X be the number of broken cupcakes per box. The distribution is: P(0) = 0.60, P(1) = 0.25, P(2) = 0.10, P(3) = 0.05. What is the expected number of broken cupcakes per box?

E(X) = 0(0.60) + 1(0.25) + 2(0.10) + 3(0.05) = 0 + 0.25 + 0.20 + 0.15 = 0.60.

A game costs $5 to play. You roll a fair die: if you roll a 6, you win $20; otherwise you win nothing. What is the expected profit per game?

E(profit) = (1/6)(20) + (5/6)(0) - 5 = 3.33 - 5 = -$1.67. On average you lose about $1.67 per game.

A call center tracks X, the number of calls in a 5-minute window. If E(X) = 3.2 and Var(X) = 2.56, what is the standard deviation?

Standard deviation = sqrt(Var(X)) = sqrt(2.56) = 1.6 calls.

An insurance company pays $10,000 for a claim. The probability of a claim is 0.02. If each policy costs $300, what is the company's expected profit per policy?

Expected payout = 0.02 x $10,000 = $200. Expected profit = $300 - $200 = $100 per policy.

A teacher records the number of absences per class: P(0) = 0.4, P(1) = 0.3, P(2) = 0.2, P(3) = 0.1. If the teacher has 5 classes per day, what is the expected total number of absences per day?

E(X per class) = 0(0.4) + 1(0.3) + 2(0.2) + 3(0.1) = 1.0. For 5 classes: 5 x 1.0 = 5.0 absences per day.

A shipping company knows that X, the number of damaged packages per shipment, has E(X) = 1.2 and E(X^2) = 2.64. What is Var(X)?

Var(X) = E(X^2) - [E(X)]^2 = 2.64 - (1.2)^2 = 2.64 - 1.44 = 1.20.

A raffle sells 200 tickets at $2 each. One ticket wins a $100 prize, and two tickets each win a $25 prize. What is the expected net gain for a single ticket buyer?

E(winnings) = (1/200)(100) + (2/200)(25) + (197/200)(0) = 0.50 + 0.25 + 0 = 0.75. Net gain = 0.75 - 2.00 = -$1.25.

A factory worker earns a base pay of $15/hour plus $3 per widget produced. If the expected number of widgets per hour is 8, what is the expected hourly earnings?

Earnings = 15 + 3X. E(Earnings) = 15 + 3(8) = 15 + 24 = $39 per hour, using E(aX + b) = aE(X) + b.

A small store tracks the number of returns per day. The distribution is: P(0) = 0.50, P(1) = 0.30, P(2) = 0.15, P(3) = 0.05. Is this a valid distribution, and what is the expected number of returns?

Sum = 0.50 + 0.30 + 0.15 + 0.05 = 1.00, so it is valid. E(X) = 0(0.50) + 1(0.30) + 2(0.15) + 3(0.05) = 0.75.

A hospital monitors the number of emergency room visits per hour overnight. E(X) = 2.5. Each visit costs the hospital an average of $800 in resources. In a 10-hour overnight shift, what is the expected total cost?

Expected visits in 10 hours = 10 x 2.5 = 25. Expected cost = 25 x $800 = $20,000.

Real-World Applications

E(X) = 0(0.60) + 1(0.25) + 2(0.10) + 3(0.05) = 0 + 0.25 + 0.20 + 0.15 = 0.60.
