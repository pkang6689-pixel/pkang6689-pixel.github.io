Lesson 4.4: Singular vs. Non-Singular Matrices - Summary

Key Concepts: Singular vs. Non-Singular Matrices

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If matrix A is non-singular, what is the rank of an n x n matrix A?

A non-singular n x n matrix has full rank n, meaning all its rows and columns are linearly independent.

What is a singular matrix?

A singular matrix is a square matrix with determinant zero, which means no inverse exists for it.

What is a non-singular matrix?

A non-singular (invertible) matrix is a square matrix whose determinant is nonzero, guaranteeing the existence of a unique inverse.

What is the rank of a matrix?

Rank measures the dimension of the column space (or row space), counting the maximum number of linearly independent rows or columns.

What is linear dependence among rows of a matrix?

Rows are linearly dependent when at least one row can be written as a combination of the others, reducing the effective rank of the matrix.

What is the null space of a matrix?

The null space (kernel) consists of all vectors x such that Ax = 0. For a singular matrix this space is nontrivial.

What is full rank for a square matrix?

An n x n matrix has full rank when its rank equals n, meaning all rows and columns are linearly independent.

What is a rank-deficient matrix?

A rank-deficient matrix has rank smaller than its theoretical maximum, indicating linear dependence among its rows or columns.

What is the determinant test for invertibility?

The determinant test states that a square matrix is invertible (non-singular) if and only if its determinant is not equal to zero.

What is the column space of a matrix?

The column space is the span of all column vectors and represents every vector b for which Ax = b has a solution.

A game developer uses a 3x3 matrix to transform objects. She notices that after transformation, all objects appear flattened into a line. What is the rank of her matrix?

Objects collapsing into a line means the image of the transformation is one-dimensional, so the matrix has rank 1 and is singular.

Real-World Applications

Example: An economics student multiplies two non-singular matrices A and B to form C = AB. Her professor asks: is C singular or non-singular?

The product of two non-singular matrices is non-singular because det(AB) = det(A)*det(B), and the product of two nonzero numbers is nonzero.
