Lesson 4.3: Adjoint Method - Summary

Key Concepts: Adjoint Method

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is adj(I) for the nxn identity matrix?

Since the cofactor of each entry of I gives back I's structure, adj(I) = I.

If det(A) = 3 and adj(A) = [[6,-3],[0,3]], what is A^(-1)?

A^(-1) = adj(A)/det(A) = (1/3)*[[6,-3],[0,3]] = [[2,-1],[0,1]].

What is the computational complexity of the adjoint method for an nxn inverse?

Computing n^2 cofactors, each requiring an (n-1)x(n-1) determinant, gives O(n^2 * n^2) = O(n^4) using cofactor expansion, or O(n^2 * (n-1)^3) using elimination for each.

If A = [[2,1],[5,3]], what is adj(A)?

For 2x2: swap diagonal (2,3)->(3,2), negate off-diagonal (1,5)->(-1,-5). adj(A) = [[3,-1],[-5,2]].

For a singular matrix (det=0), what is adj(A)?

A*adj(A) = det(A)*I = 0*I = 0. The adjugate itself may have nonzero entries, but the product with A is always zero.

What is the 'adjugate' of a matrix?

Adjugate is the modern mathematical term for the classical adjoint, to avoid confusion with the Hermitian adjoint.

What is det(adj(A)) for an nxn matrix A?

det(adj(A)) = det(A)^(n-1). This follows from A*adj(A) = det(A)*I and taking determinants of both sides.

What is the adjugate of a diagonal matrix diag(a,b,c)?

For a diagonal matrix, the cofactors give products of the other diagonal entries: adj = diag(bc, ac, ab).

What is the use of the adjugate in differential equations?

In systems of ODEs dx/dt = Ax, the adjugate of (sI-A) appears in the Laplace transform solution and transfer functions.

A cryptographer needs the inverse of [[3,5],[2,7]] modulo 26. det = 11, and 11^(-1) mod 26 = 19. Using adj(A) = [[7,-5],[-2,3]], what is A^(-1) mod 26?

A^(-1) = 19*[[7,-5],[-2,3]] mod 26 = [[133,-95],[-38,57]] mod 26 = [[3,9],[14,5]].

A control engineer has a 2x2 transfer function matrix. She uses the adjugate to find the inverse symbolically: if A = [[s+1, 2],[3, s+4]], what is adj(A)?

For 2x2: swap diagonal entries and negate off-diagonal: adj = [[s+4, -2],[-3, s+1]].

A mathematician proves that for any nxn matrix A, A*adj(A) = det(A)*I. If A is 3x3 with det(A)=5, what is A*adj(A)?

A*adj(A) = det(A)*I = 5*I = diag(5,5,5).

A chemical engineer uses the formula adj(A)/det(A) for A = [[4,7],[2,6]]. det = 24-14 = 10. adj = [[6,-7],[-2,4]]. What is A^(-1)?

A^(-1) = (1/10)*[[6,-7],[-2,4]] = [[0.6,-0.7],[-0.2,0.4]].
