Lesson 5.7: Applications in Computer Graphics - Summary

Key Concepts: Applications in Computer Graphics

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the result of applying the matrix [[1, 0], [0, -1]] to a point (x, y)?

[[1, 0], [0, -1]] * (x, y) = (x, -y). This is a reflection across the x-axis. The x-coordinate is unchanged, while the y-coordinate is negated.

What is the determinant of a 2D rotation matrix?

det([[cos(theta), -sin(theta)], [sin(theta), cos(theta)]]) = cos^2(theta) + sin^2(theta) = 1. The determinant of a rotation matrix is always 1, reflecting the fact that rotations preserve area and orientation.

What is an 'affine transformation' in computer graphics?

An affine transformation is a linear transformation followed by a translation: T(x) = Ax + b. It preserves collinearity (points on a line stay on a line), parallelism, and ratios of distances. It includes rotations, reflections, scaling, shears, and translations.

What is a 'projection matrix' in computer graphics?

A projection matrix in graphics transforms 3D coordinates into 2D screen coordinates. Perspective projection creates the illusion of depth (distant objects appear smaller), while orthographic projection preserves parallel lines and relative sizes.

What is a 'model-view matrix'?

The model-view matrix is the product of the view matrix (camera transformation) and the model matrix (object positioning). It transforms vertices from model (local) coordinates to eye (camera) coordinates in a single matrix multiplication step.

What is a 'normal vector' used for in 3D graphics?

A normal vector is a vector perpendicular to a surface at a given point. In graphics, normals are essential for lighting: the angle between the normal and the light direction determines how brightly the surface is illuminated (Lambertian shading: brightness proportional to cos(angle)).

What is 'perspective division' in the graphics pipeline?

Perspective division is the step where homogeneous coordinates (x, y, z, w) are divided by w to produce normalized device coordinates (x/w, y/w, z/w). This creates the perspective effect where distant objects (larger w) appear smaller on screen.

What is a 'viewport transformation'?

The viewport transformation is the final step in the graphics pipeline that maps normalized device coordinates (typically in the range [-1, 1]) to window/pixel coordinates on the actual screen. It is a scaling and translation that depends on the window size and position.

What is the purpose of a 'clipping algorithm' in the graphics pipeline?

Clipping removes parts of geometry that fall outside the camera's viewing volume (defined by near/far planes and the field of view). This prevents rendering invisible objects and avoids artifacts from projecting points behind the camera.

A UI designer creates a button that must first be scaled by 2, then rotated 45 degrees, then translated to position (100, 50). Using homogeneous coordinates, what is the combined transformation matrix M?

Transformations are applied right-to-left in matrix multiplication. The order of application is: first S (scale), then R (rotate), then T (translate). So M = T * R * S, where S is closest to the point vector. This correctly scales first, rotates second, and translates last.

Real-World Applications

Example: To apply rotation followed by translation to a 2D object, using homogeneous coordinates, you multiply the transformation matrices in which order?

In matrix multiplication, transformations are applied right to left. To first rotate then translate, you compute T * R * point, where R is applied first (closest to the point) and T is applied second. The combined matrix is T * R.

Example: A VR developer needs to reflect a virtual room across the yz-plane to create a mirror image. Which transformation matrix achieves this in 3D?

Reflecting across the yz-plane negates the x-coordinate while keeping y and z unchanged. The matrix [[-1,0,0],[0,1,0],[0,0,1]] maps (x,y,z) to (-x,y,z). Its determinant is -1, confirming it is a reflection (orientation-reversing).
