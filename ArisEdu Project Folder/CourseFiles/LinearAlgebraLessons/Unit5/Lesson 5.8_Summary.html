Lesson 5.8: Applications in Data Science - Summary

Key Concepts: Applications in Data Science

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the covariance matrix of a centered data matrix X?

For a centered data matrix X (columns have mean zero) with n samples as rows, the sample covariance matrix is S = (1/(n-1)) * X^T X. It is a p x p symmetric positive semi-definite matrix whose diagonal entries are the variances of each feature.

What is 'Principal Component Analysis' (PCA)?

PCA is a technique that transforms data into a new coordinate system where the axes (principal components) are orthogonal directions ordered by decreasing variance. It is used for dimensionality reduction, visualization, and noise reduction by keeping only the top components.

What is 'Singular Value Decomposition' (SVD)?

The SVD decomposes any m x n matrix A into U * Sigma * V^T, where U (m x m) and V (n x n) are orthogonal matrices and Sigma (m x n) is a diagonal matrix of non-negative singular values. It exists for every matrix and is fundamental in data science.

What is 'low-rank approximation' of a matrix?

Low-rank approximation replaces a matrix A with a rank-k matrix A_k (where k < rank(A)) that is as close to A as possible. Using SVD, A_k is obtained by keeping only the k largest singular values. The Eckart-Young theorem proves this is optimal in the Frobenius and spectral norms.

What is the 'least squares projection' in linear algebra?

The least squares projection finds the vector in Col(A) closest to b. This projected vector is p = A * (A^T A)^(-1) * A^T * b, and the projection matrix is P = A(A^T A)^(-1)A^T. The residual b - p is orthogonal to Col(A).

What is 'matrix factorization' in the context of recommender systems?

Matrix factorization in recommender systems decomposes the sparse user-item rating matrix R into R approximately equal to U * V^T, where U contains user latent factors and V contains item latent factors. Missing ratings are predicted by computing the corresponding entries of U * V^T.

What is 'latent semantic analysis' (LSA)?

LSA applies SVD to a term-document matrix (where entry (i,j) represents the frequency of term i in document j). The low-rank approximation reveals latent semantic structure, allowing documents with similar meanings (but different words) to be identified as related.

What is the 'pseudoinverse' (Moore-Penrose inverse) of a matrix?

The pseudoinverse A^+ generalizes the inverse to any matrix. For Ax = b, x = A^+ * b gives the least squares solution with minimum norm. Using SVD, if A = U * Sigma * V^T, then A^+ = V * Sigma^+ * U^T, where Sigma^+ inverts the nonzero singular values.

A data scientist has a dataset with 1000 samples and 50 features. She runs PCA and finds that the first 5 eigenvalues account for 95% of the total variance. She decides to project the data onto 5 principal components. What is the shape of the transformed dataset?

Projecting 1000 samples onto 5 principal components produces a 1000 x 5 matrix. Each sample is now represented by 5 coordinates (its projections onto the 5 principal component directions) instead of the original 50 features.

A machine learning engineer trains a linear regression model with 20 features but only 15 training samples. He finds that X^T X is singular. What is the issue and what can he do?

When p > n (more features than samples), rank(X) <= n < p, so X^T X (p x p) is singular and the normal equation has infinitely many solutions. Ridge regression adds a regularization term (X^T X + lambda * I)b = X^T y, making the matrix invertible and selecting a unique, stable solution.

A retail company wants to predict sales (y) using 8 economic indicators (features). They fit y = Xb using least squares and get a coefficient vector b. They then discover that two features are nearly perfectly correlated (correlation 0.99). What problem does this cause and what is the linear algebra explanation?

Near-perfect correlation between two features means their column vectors are nearly linearly dependent, making X^T X nearly singular (one eigenvalue is near zero). The condition number is large, causing the least squares solution to be numerically unstable. Small data perturbations lead to wildly different coefficient estimates. Solutions include dropping one feature, using PCA, or applying ridge regression.

Real-World Applications

Example: An image compression algorithm uses SVD on a 1000x1000 grayscale image matrix. Using a rank-50 approximation, how many values need to be stored instead of the original 1,000,000?

A rank-50 SVD stores U_50 (1000 x 50 = 50,000 values), 50 singular values, and V_50 (1000 x 50 = 50,000 values), totaling 100,050 values. This is about 10% of the original 1,000,000 entries, achieving significant compression.

Example: A natural language processing researcher represents each word as a 300-dimensional vector (word embedding). She wants to find the word most similar to 'king' - 'man' + 'woman'. What linear algebra operation is she performing?

She computes the vector v = v_king - v_man + v_woman (a linear combination in R^300) and then searches for the word whose embedding is closest to v (nearest neighbor by cosine similarity or Euclidean distance). The answer is typically 'queen', demonstrating that word embeddings capture semantic relationships through linear structure.
