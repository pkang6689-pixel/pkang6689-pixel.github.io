Lesson 5.6: Null Space & Column Space - Summary

Key Concepts: Null Space & Column Space

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

For A = [[1, 2], [3, 6]], what is the null space of A?

Row reducing: [[1,2],[3,6]] -> [[1,2],[0,0]]. The equation x1 + 2x2 = 0 gives x1 = -2x2. With x2 = t free, x = t(-2, 1). So Nul(A) = span{(-2, 1)}, a line through the origin.

If A is a 3x5 matrix with rank 2, what is the dimension of Nul(A)?

By the rank-nullity theorem, rank(A) + dim(Nul(A)) = n (number of columns). So dim(Nul(A)) = 5 - 2 = 3. The null space is a 3-dimensional subspace of R^5.

If A is a 4x4 invertible matrix, what is the null space of A?

An invertible matrix has full rank (rank = 4 for a 4x4 matrix). By rank-nullity, dim(Nul(A)) = 4 - 4 = 0. The only vector in the null space is the zero vector, meaning Ax = 0 has only the trivial solution.

What is the column space of the 2x2 identity matrix I?

The columns of the 2x2 identity matrix are (1,0) and (0,1), which are the standard basis vectors for R^2. Their span is all of R^2, so Col(I) = R^2.

What is the dimension of the column space called?

The rank of a matrix is the dimension of its column space (equivalently, the dimension of its row space). It equals the number of pivot columns in the row echelon form of the matrix.

What is the 'null space' (or kernel) of a matrix?

The null space (also called the kernel) of a matrix A is the set of all vectors x in R^n that satisfy the homogeneous equation Ax = 0. It is always a subspace of R^n and contains at least the zero vector.

What is the 'column space' of a matrix?

The column space of A, denoted Col(A), is the span of the columns of A. Equivalently, it is the set of all vectors b such that Ax = b has a solution, or the range (image) of the linear transformation x -> Ax.

What is the 'rank-nullity theorem'?

The rank-nullity theorem states that for an m x n matrix A, rank(A) + nullity(A) = n, where nullity is dim(Nul(A)). This means the number of pivot columns plus the number of free variables equals the total number of columns.

What is the 'nullity' of a matrix?

The nullity of a matrix A is the dimension of its null space: nullity(A) = dim(Nul(A)). It equals the number of free variables in the solution to Ax = 0, which is the number of non-pivot columns.

What is the 'row space' of a matrix?

The row space of A is the span of the rows of A. It equals the column space of A^T (the transpose). The dimension of the row space equals the rank of A, the same as the dimension of the column space.

What is the 'left null space' of a matrix A?

The left null space of A is Nul(A^T) = {y : A^T y = 0}, equivalently {y : y^T A = 0}. It is called 'left' because y acts on A from the left. It is a subspace of R^m for an m x n matrix A.

What is the 'range' of a linear transformation T(x) = Ax?

The range (or image) of T(x) = Ax is the set of all output vectors {Ax : x in R^n}. This is exactly the column space of A, because Ax is a linear combination of the columns of A with coefficients from x.

What is a 'pivot column' and how does it relate to the column space?

A pivot column is a column of the original matrix A that corresponds to a column containing a leading 1 (pivot) in the row echelon form. The pivot columns of the original matrix A (not the echelon form) form a basis for Col(A).

An electrical engineer has a circuit with 5 nodes and 8 branches. The incidence matrix A is 5x8. She computes that rank(A) = 4. What is the dimension of the null space of A, and what does it represent?

By rank-nullity: dim(Nul(A)) = 8 - 4 = 4. In circuit theory, the null space of the incidence matrix corresponds to the independent loop (cycle) currents. There are 4 independent loops in this circuit (consistent with 8 - 5 + 1 = 4 by Euler's formula).

A communications engineer designs a 3x5 encoding matrix A that maps 5-bit messages to 3-symbol codewords. What is the minimum dimension of Nul(A)?

By rank-nullity, dim(Nul(A)) = 5 - rank(A). The maximum rank of a 3x5 matrix is 3 (limited by the number of rows). Therefore dim(Nul(A)) >= 5 - 3 = 2. At least 2 different messages will encode to the same codeword, which is a design concern.

Real-World Applications

Example: A manufacturing plant produces 3 products using 4 raw materials. The production matrix A (4x3) maps production levels to material usage. If the null space of A has dimension 1, what does this mean for the plant?

If Nul(A) has dimension 1, there is a one-parameter family of production level vectors x such that Ax = 0. This means there exists a nontrivial combination of production levels that uses zero net materials, implying one product's material profile is a combination of the others, making it redundant.
