Lesson 8.7: Applications in Machine Learning - Summary

Key Concepts: Applications in Machine Learning

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

The kernel trick in SVMs implicitly computes inner products in a higher-dimensional space. What is the polynomial kernel of degree 2 for vectors x and y in R^2?

The polynomial kernel of degree 2 is K(x,y) = (x . y + c)^2 (commonly c = 1). This computes the inner product in a feature space that includes all monomials up to degree 2 without explicitly constructing the high-dimensional features.

What is a 'feature vector' in machine learning?

A feature vector is a point in a high-dimensional space where each coordinate represents a measurable attribute of the data. Machine learning algorithms operate on these vector representations.

What is 'regularization' in the context of linear models?

Regularization adds a penalty like lambda * ||w||^2 (L2/Ridge) or lambda * ||w||_1 (L1/Lasso) to the loss function. This discourages large weights, reducing overfitting and improving generalization.

What is the 'Hessian matrix' in optimization for machine learning?

The Hessian H has entries H_ij = d^2L/dw_i dw_j. Its eigenvalues indicate curvature: positive eigenvalues mean the function curves upward (local minimum direction), negative means downward (local maximum direction), and mixed indicates a saddle point.

What is a 'latent factor' in matrix factorization for recommendation systems?

Latent factors are hidden dimensions discovered by matrix factorization. For movies, latent factors might correspond to genres, era, mood, or other abstract qualities that explain rating patterns.

What is 'spectral clustering' in machine learning?

Spectral clustering constructs a graph Laplacian from the data similarity matrix, then uses its smallest eigenvectors to embed data in a low-dimensional space where clusters are easily separable.

What is the 'Moore-Penrose pseudoinverse' used for in machine learning?

The Moore-Penrose pseudoinverse A^+ generalizes matrix inversion to non-square and singular matrices. It gives the minimum-norm least-squares solution x = A^+ b and can be computed via SVD.

A data scientist fits a linear regression model y = w1*x1 + w2*x2 + b to 100 data points. The design matrix A is 100x3 (including a column of 1s for the bias). What is the dimension of A^T A?

A is 100x3, so A^T is 3x100, and A^T A is 3x3. This small system is computationally easy to solve, giving the least-squares weights.

A recommendation engine uses matrix factorization to predict missing ratings. If user i's embedding is u_i = (2, 1) and item j's embedding is v_j = (3, 4), what is the predicted rating?

The predicted rating is u_i . v_j = 2*3 + 1*4 = 6 + 4 = 10. The dot product of user and item embeddings estimates how much the user would like the item.

A natural language processing engineer computes cosine similarity between document vectors d1 = (3, 4) and d2 = (4, 3). What is the cosine similarity?

cos(theta) = (d1 . d2) / (||d1|| ||d2||) = (12 + 12) / (5 * 5) = 24/25 = 0.96. The documents are very similar in direction.

A speech recognition system uses an orthogonal transformation to convert mel-spectrogram features. If the transformation matrix Q is 40x40 orthogonal, what is the computational benefit of using Q^T instead of Q^(-1)?

For orthogonal Q, Q^(-1) = Q^T. Computing a transpose is O(n^2) (just rearranging entries) while computing a general inverse is O(n^3). For a 40x40 matrix, this means 1,600 vs ~64,000 operations.

Real-World Applications

Example: In a convolutional neural network, a convolution operation can be expressed as multiplication by a Toeplitz matrix. This matrix has what special structure?

A Toeplitz matrix has constant values along each descending diagonal, reflecting the translation-invariant nature of convolution. The kernel weights appear shifted along the diagonals.

Example: A natural language processing engineer computes cosine similarity between document vectors d1 = (3, 4) and d2 = (4, 3). What is the cosine similarity?

cos(theta) = (d1 . d2) / (||d1|| ||d2||) = (12 + 12) / (5 * 5) = 24/25 = 0.96. The documents are very similar in direction.

Example: A computer vision engineer uses the Singular Value Decomposition (SVD) to compress a 1000x1000 image by keeping only the top 50 singular values. How many values need to be stored?

A rank-50 SVD stores: 50 left singular vectors (1000 entries each) + 50 right singular vectors (1000 entries each) + 50 singular values = 50*1000 + 50*1000 + 50 = 100,050. This is about 10% of the original 1,000,000 entries.

Example: A reinforcement learning agent uses a neural network with weight matrices W1 (100x50) and W2 (50x10). The total number of trainable weight parameters (excluding biases) is:

W1 has 100 * 50 = 5,000 parameters and W2 has 50 * 10 = 500 parameters, for a total of 5,500 trainable weight parameters.
