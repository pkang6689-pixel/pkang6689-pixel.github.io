Lesson 10.3: Applications in Computer Science (networks, algorithms) - Summary

Key Concepts: Applications in Computer Science (networks, algorithms)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A webpage links to 3 other pages and no pages link back to it. In the Google PageRank model, what is this page's contribution to each linked page per iteration?

In PageRank, a page distributes its rank equally among all outgoing links. With 3 outgoing links, each linked page receives 1/3 of the page's current rank.

Given the adjacency matrix A = [[0,1,0],[0,0,1],[1,0,0]], what is the trace of A?

The trace is the sum of diagonal entries: 0 + 0 + 0 = 0. A trace of 0 means no self-loops exist in the graph.

What is the computational complexity of standard matrix multiplication for two n x n matrices?

Standard matrix multiplication of two n x n matrices requires three nested loops, each iterating n times, giving O(n^3) time complexity.

What is an adjacency matrix in the context of computer science?

An adjacency matrix is a square matrix of size n x n (where n is the number of nodes) with entry (i,j) equal to 1 if there is an edge from node i to node j, and 0 otherwise.

What is PageRank in the context of linear algebra?

PageRank computes the importance of web pages by finding the dominant eigenvector (eigenvalue 1) of a column-stochastic transition matrix derived from the web's hyperlink structure.

What is a stochastic matrix?

A stochastic matrix has non-negative entries where each column (column-stochastic) or each row (row-stochastic) sums to 1, making it suitable for representing transition probabilities in Markov chains.

What is a 'steady-state vector' in the context of Markov chains?

The steady-state vector is the eigenvector of the transition matrix corresponding to eigenvalue 1, normalized so its entries sum to 1. It represents the long-run probability distribution of the Markov chain.

What is the 'power method' in numerical linear algebra?

The power method starts with a random vector and repeatedly multiplies it by the matrix, normalizing at each step. The vector converges to the eigenvector corresponding to the largest-magnitude eigenvalue.

A web consists of 3 pages. Page 1 links to pages 2 and 3, page 2 links to page 3, and page 3 links to page 1. The column-stochastic transition matrix is M = [[0, 0, 1],[1/2, 0, 0],[1/2, 1, 0]]. What is M times the vector [1/3, 1/3, 1/3]^T?

M*[1/3,1/3,1/3]^T: Row 1: 0*(1/3)+0*(1/3)+1*(1/3)=1/3. Row 2: (1/2)*(1/3)+0*(1/3)+0*(1/3)=1/6. Row 3: (1/2)*(1/3)+1*(1/3)+0*(1/3)=1/6+1/3=1/2. Result: [1/3, 1/6, 1/2]^T.

A data scientist uses the graph Laplacian L = D - A for a 4-node graph where D = diag(2,3,2,1) and A is the adjacency matrix. What is the sum of each row of L?

In the Laplacian L = D - A, row i has the degree d_i on the diagonal and -1 for each neighbor. The row sum is d_i - (sum of adjacencies in row i) = d_i - d_i = 0. Every row of a graph Laplacian sums to 0.

A computer scientist models internet traffic between 3 servers using a Markov chain with transition matrix T = [[0.5, 0.25, 0],[0.5, 0.5, 0.5],[0, 0.25, 0.5]]. What is the probability that a request currently at server 2 will be at server 1 after one step?

In the column-stochastic transition matrix, T[i][j] gives the probability of moving from state j to state i. T[1][2] = 0.25, so there is a 25% probability of transitioning from server 2 to server 1.

A game developer models a random walk on a 4-node cycle graph (1-2-3-4-1). At each step, the walker moves to an adjacent node with equal probability. The transition matrix has entries T[i][j] = 1/2 if nodes i and j are adjacent, 0 otherwise. Starting at node 1, what is the probability of being at node 3 after exactly 2 steps?

From node 1, after 1 step the walker is at node 2 or node 4 (each with prob 1/2). From node 2, prob of reaching node 3 is 1/2. From node 4, prob of reaching node 3 is 1/2. Total: (1/2)(1/2) + (1/2)(1/2) = 1/4 + 1/4 = 1/2.

Example: A web consists of 3 pages. Page 1 links to pages 2 and 3, page 2 links to page 3, and page 3 links to page 1. The column-stochastic transition matrix is M = [[0, 0, 1],[1/2, 0, 0],[1/2, 1, 0]]. What is M times the vector [1/3, 1/3, 1/3]^T?

Example: A search engine uses the power method to compute PageRank. After many iterations, the algorithm converges to the vector [0.4, 0.35, 0.25]. A new page (page 4) is added that links only to page 1. The engineer asks: before recomputing, which page currently has the highest rank?

The converged PageRank vector [0.4, 0.35, 0.25] shows page 1 has the highest rank at 0.4. The newly added page 4 has not yet been incorporated into the computation.

Example: A software team stores a 1000x1000 sparse matrix with only 5000 non-zero entries. How much memory is saved by using sparse storage (storing only non-zero entries with row, column, value) compared to dense storage, assuming each value takes 8 bytes and each index takes 4 bytes?

Dense storage: 1000*1000*8 = 8,000,000 bytes = ~7.63 MB. Sparse storage (COO): 5000*(8+4+4) = 80,000 bytes = ~0.076 MB. Savings: ~8,000,000 - 80,000 = 7,920,000 bytes, approximately 7.92 MB.

Example: A cybersecurity analyst examines a network of 5 computers. The graph Laplacian has eigenvalues 0, 1, 2, 4, 5. The number of connected components in the network is determined by the multiplicity of the zero eigenvalue. How many connected components does this network have?

The number of connected components equals the multiplicity of the eigenvalue 0 in the graph Laplacian. Here, 0 appears exactly once, so the network has 1 connected component, meaning all computers are reachable from each other.
