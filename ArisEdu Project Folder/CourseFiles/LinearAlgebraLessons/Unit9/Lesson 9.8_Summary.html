Lesson 9.8: Applications in Statistics - Summary

Key Concepts: Applications in Statistics

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

The variance of a linear combination a^T X is Var(a^T X) = a^T Sigma a. For Sigma = [[4, 2], [2, 3]] and a = (1, -1), what is Var(a^T X)?

a^T Sigma a = [1,-1] [[4,2],[2,3]] [1,-1]^T = [1,-1] [4-2, 2-3]^T = [1,-1] [2, -1]^T = 2 + 1 = 3.

What is the 'covariance matrix' in statistics?

The covariance matrix Sigma has entry Sigma_ij = Cov(X_i, X_j) = E[(X_i - mu_i)(X_j - mu_j)]. Diagonal entries are variances, off-diagonal entries measure linear co-movement between variable pairs.

What is 'multivariate normal distribution'?

The multivariate normal N(mu, Sigma) is a joint distribution where any linear combination a^T X is univariate normal. It is completely determined by its mean vector mu and covariance matrix Sigma.

What is 'principal component regression' (PCR)?

PCR first applies PCA to X to get orthogonal principal components, then regresses y on the top k components. This eliminates multicollinearity (components are orthogonal) and removes noise in the low-variance directions.

What is the 'precision matrix' (inverse covariance)?

The precision matrix Omega = Sigma^{-1} directly encodes the conditional independence structure. If Omega_ij = 0, then X_i and X_j are conditionally independent given all other variables. This is central to graphical models.

What is 'Hotelling's T-squared test'?

Hotelling's T^2 tests whether a multivariate mean vector equals a hypothesized value. It uses the sample covariance matrix S to account for correlations, and follows an F-distribution under the null hypothesis.

What is 'MANOVA' (Multivariate Analysis of Variance)?

MANOVA generalizes ANOVA to multiple dependent variables. It compares mean vectors using the ratio of between-group to within-group scatter matrices (B and W). Test statistics involve eigenvalues of W^{-1}B.

What is the 'graphical lasso'?

The graphical lasso estimates the precision matrix Omega by maximizing the penalized log-likelihood: max [log det(Omega) - trace(S*Omega) - lambda * ||Omega||_1]. The L1 penalty encourages sparsity, revealing the conditional independence graph.

What is 'discriminant analysis' using linear algebra?

Linear discriminant analysis finds directions w that maximize the ratio w^T S_B w / w^T S_W w, where S_B is the between-class scatter matrix and S_W is the within-class scatter. This is a generalized eigenvalue problem.

A social scientist uses factor analysis on 20 survey questions and finds that 3 factors explain 80% of the variance. The loading matrix Lambda is 20x3. If the factor scores F = (1.5, -0.5, 0.8) and the first row of Lambda is (0.8, 0.1, 0.3), what is the model's prediction for variable 1?

X_1 = Lambda_1 . F = 0.8(1.5) + 0.1(-0.5) + 0.3(0.8) = 1.20 - 0.05 + 0.24 = 1.39. This is the dot product of the first row of the loading matrix with the factor score vector.

A marketing analyst uses canonical correlation analysis between 5 customer demographic variables (X) and 3 purchase behavior variables (Y). What is the maximum number of canonical correlations?

The number of canonical correlations equals min(p, q) = min(5, 3) = 3. Each canonical correlation is the maximum correlation between a linear combination of X and a linear combination of Y, subject to orthogonality constraints.

How to Calculate

Example: The Mahalanobis distance from a point x to a distribution with mean mu and covariance Sigma is d = sqrt((x-mu)^T Sigma^{-1} (x-mu)). Why is this preferred over Euclidean distance for outlier detection?

Solution: Mahalanobis distance normalizes by the covariance structure, so a point far from the mean in a low-variance direction is considered more unusual than one equally far in a high-variance direction. It transforms elliptical contours to spherical ones.

Real-World Applications

Example: In canonical correlation analysis (CCA), the goal is to find linear combinations of two sets of variables that are maximally correlated. This is solved as a generalized eigenvalue problem involving which matrices?

CCA solves Sigma_xy Sigma_yy^{-1} Sigma_yx a = rho^2 Sigma_xx a, a generalized eigenvalue problem. The canonical correlations are the square roots of these eigenvalues, and the vectors a, b give the optimal linear combinations.

Example: What does 'leverage' mean in regression, in terms of the hat matrix?

The leverage h_ii = e_i^T H e_i measures how much observation i influences its own predicted value. High leverage points (h_ii close to 1) are far from the center of the predictor space and can unduly affect the regression.

Example: What does 'shrinkage estimator' mean for covariance matrices?

When n < p (more variables than samples), the sample covariance is singular. Shrinkage estimators blend the sample covariance S with a well-conditioned target T (like alpha*I): Sigma_hat = alpha*S + (1-alpha)*T.
