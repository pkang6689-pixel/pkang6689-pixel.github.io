Lesson 9.2: Jordan Canonical Form - Summary

Key Concepts: Jordan Canonical Form

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the Jordan canonical form used for?

The Jordan canonical form is the 'best possible' near-diagonal form for matrices that are not diagonalizable. It uses Jordan blocks with eigenvalues on the diagonal and 1s on the superdiagonal.

What is the Jordan form of the matrix A = [[5, 1], [0, 5]]?

A is already in Jordan canonical form -- it is a single 2x2 Jordan block with eigenvalue 5. The eigenvalue 5 has algebraic multiplicity 2 but geometric multiplicity 1.

For the Jordan block J = [[3, 1], [0, 3]], what is J^2?

J^2 = [[3,1],[0,3]]^2 = [[9, 3+3],[0, 9]] = [[9, 6],[0, 9]]. For a 2x2 Jordan block, J^n = [[lambda^n, n*lambda^{n-1}], [0, lambda^n]].

What is a 'Jordan block'?

A Jordan block J_k(lambda) is a k x k matrix with eigenvalue lambda on every diagonal entry and 1 on each superdiagonal entry. All other entries are 0. It is the building block of the Jordan canonical form.

What is a 'generalized eigenvector'?

A generalized eigenvector of rank k satisfies (A - lambda*I)^k v = 0 but (A - lambda*I)^{k-1} v != 0. Ordinary eigenvectors are generalized eigenvectors of rank 1. These form the columns of the change-of-basis matrix P.

What is a 'defective matrix'?

A defective matrix does not have a full set of linearly independent eigenvectors (geometric multiplicity < algebraic multiplicity for some eigenvalue). Its Jordan form contains at least one block of size >= 2.

What is the 'nilpotent part' of a Jordan block?

Any Jordan block J = lambda*I + N decomposes into a scalar part lambda*I and a nilpotent part N (with 1s on the superdiagonal). N^k = 0 where k is the block size, making it nilpotent.

What is the 'Jordan chain' for a generalized eigenvector?

A Jordan chain is a sequence where v_1 is an eigenvector and each subsequent v_j is a generalized eigenvector satisfying (A - lambda*I)v_j = v_{j-1}. The chain length equals the Jordan block size.

What is the 'minimal polynomial' of a matrix?

The minimal polynomial m(x) is the monic polynomial of lowest degree satisfying m(A) = 0. Its roots are exactly the distinct eigenvalues of A, and the power of each root equals the size of the largest Jordan block for that eigenvalue.

What is the 'index of nilpotency' of a nilpotent matrix N?

The index of nilpotency k is the smallest power for which N^k = 0. For the nilpotent part of a Jordan block of size m, the index is m. For a general nilpotent matrix, it equals the size of the largest Jordan block.

A physicist computes A^{100} for a 2x2 matrix with Jordan form J = [[2, 1], [0, 2]]. Using J^n = [[2^n, n*2^{n-1}], [0, 2^n]], what is the (1,2) entry of J^{100}?

For a 2x2 Jordan block with eigenvalue lambda, J^n has (1,2) entry = n * lambda^{n-1}. With lambda = 2 and n = 100: 100 * 2^{99}.

A differential equations student needs to compute e^{Jt} for the Jordan block J = [[3, 1], [0, 3]]. Using the formula for a 2x2 Jordan block, what is e^{Jt}?

For J = lambda*I + N, e^{Jt} = e^{lambda*t}(I + tN). With lambda = 3: e^{Jt} = e^{3t} * [[1, t], [0, 1]] = [[e^{3t}, t*e^{3t}], [0, e^{3t}]]. The t factor in the (1,2) entry comes from the non-diagonalizable part.

A numerical analyst encounters a matrix where the eigenvalue 5 has algebraic multiplicity 4 and she finds dim(ker(A-5I)) = 2, dim(ker(A-5I)^2) = 3, dim(ker(A-5I)^3) = 4. What is the Jordan structure for eigenvalue 5?

The jumps in kernel dimensions are: 2, 1, 1 (from k=1,2,3). The number of blocks of size >= j equals dim(ker(A-5I)^j) - dim(ker(A-5I)^{j-1}). Blocks >= 1: 2, blocks >= 2: 1, blocks >= 3: 1. So one block of size 3 and one of size 1.

A robotics engineer models a system with matrix A = [[0, 1, 0], [0, 0, 1], [0, 0, 0]]. This is a single Jordan block with eigenvalue 0 and size 3. What is A^3?

A is the nilpotent part of a 3x3 Jordan block with eigenvalue 0. A^2 = [[0,0,1],[0,0,0],[0,0,0]], and A^3 = 0 (the zero matrix). The nilpotency index equals the block size.

A mathematician studying matrix functions needs to compute f(A) where A has Jordan form J. If f(x) = x^2 + 1 and J = [[2, 1], [0, 2]], what is f(J)?

f(J) = J^2 + I. J^2 = [[4, 4], [0, 4]]. So f(J) = [[4,4],[0,4]] + [[1,0],[0,1]] = [[5, 4], [0, 5]]. Alternatively, for a 2x2 Jordan block: f(J) = [[f(lambda), f'(lambda)], [0, f(lambda)]] = [[5, 4], [0, 5]] since f'(2) = 2*2 = 4.

A stability analyst examines a 4x4 system where A has Jordan form diag(J_2(1), J_2(-1)) -- two 2x2 Jordan blocks with eigenvalues 1 and -1. For large n, what is the behavior of A^n?

For eigenvalue 1: J_2(1)^n = [[1, n],[0, 1]], which grows linearly in n. For eigenvalue -1: J_2(-1)^n = [[(-1)^n, n(-1)^{n-1}],[0, (-1)^n]], which oscillates with linearly growing amplitude. Overall, A^n grows like O(n).

Real-World Applications

Example: A 3x3 matrix has eigenvalue 2 with algebraic multiplicity 3 and geometric multiplicity 1. How many Jordan blocks does it have for eigenvalue 2?

The number of Jordan blocks for an eigenvalue equals its geometric multiplicity (dimension of eigenspace). With geometric multiplicity 1, there is one Jordan block. Since the algebraic multiplicity is 3, it must be a single 3x3 block.

Example: A 4x4 matrix has eigenvalue 1 with algebraic multiplicity 4 and geometric multiplicity 2. Which Jordan form structure is possible?

Geometric multiplicity 2 means 2 Jordan blocks. These must have sizes summing to 4 (the algebraic multiplicity). Options: 3+1 or 2+2. Both are valid Jordan structures; the answer 'Two 2x2 Jordan blocks' is one possibility. Note: 3+1 is also possible.

Example: What does 'algebraic multiplicity' of an eigenvalue mean?

The algebraic multiplicity is the multiplicity of lambda as a root of det(A - lambda*I) = 0. It determines the total size (sum of dimensions) of all Jordan blocks associated with that eigenvalue.

Example: What does 'geometric multiplicity' of an eigenvalue mean?

The geometric multiplicity is dim(ker(A - lambda*I)), the number of linearly independent eigenvectors for eigenvalue lambda. It equals the number of Jordan blocks for that eigenvalue and satisfies 1 <= geo. mult. <= alg. mult.

Example: A control engineer analyzes a system with state matrix A that has eigenvalue 0 with algebraic multiplicity 3 and geometric multiplicity 2. She needs to determine the Jordan form. Which structure does A have?

With geometric multiplicity 2, there are 2 Jordan blocks. Their sizes must sum to 3 (algebraic multiplicity). The only partition of 3 into 2 parts is 2 + 1, so there is one 2x2 and one 1x1 Jordan block.

Example: A numerical analyst encounters a matrix where the eigenvalue 5 has algebraic multiplicity 4 and she finds dim(ker(A-5I)) = 2, dim(ker(A-5I)^2) = 3, dim(ker(A-5I)^3) = 4. What is the Jordan structure for eigenvalue 5?

The jumps in kernel dimensions are: 2, 1, 1 (from k=1,2,3). The number of blocks of size >= j equals dim(ker(A-5I)^j) - dim(ker(A-5I)^{j-1}). Blocks >= 1: 2, blocks >= 2: 1, blocks >= 3: 1. So one block of size 3 and one of size 1.

Example: A stability analyst examines a 4x4 system where A has Jordan form diag(J_2(1), J_2(-1)) -- two 2x2 Jordan blocks with eigenvalues 1 and -1. For large n, what is the behavior of A^n?

For eigenvalue 1: J_2(1)^n = [[1, n],[0, 1]], which grows linearly in n. For eigenvalue -1: J_2(-1)^n = [[(-1)^n, n(-1)^{n-1}],[0, (-1)^n]], which oscillates with linearly growing amplitude. Overall, A^n grows like O(n).

Example: An applied mathematician solves a system of ODEs dx/dt = Ax where A has Jordan form [[2, 1, 0], [0, 2, 0], [0, 0, 3]]. The general solution for the first two components involves e^{2t} multiplied by:

The 2x2 Jordan block with eigenvalue 2 contributes solutions e^{2t} and t*e^{2t}. So the first component is (c1 + c2*t)e^{2t}. The polynomial factor's degree is one less than the block size.
