Lesson 9.4: Applications in Machine Learning - Summary

Key Concepts: Applications in Machine Learning

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

In batch processing of neural networks, inputs are stacked as rows of a matrix X (batch x features). The forward pass computes Y = XW^T + b. If X is 32x784 and W is 256x784, what is the dimension of Y?

X is 32x784 and W^T is 784x256, so XW^T is 32x256. Each of the 32 samples gets mapped from 784 features to 256 hidden units.

What is 'gradient descent' from a linear algebra perspective?

Gradient descent uses the gradient vector (direction of steepest ascent) to iteratively update parameters: w_{t+1} = w_t - alpha * grad(L). The gradient is a vector in parameter space computed using matrix calculus.

What is the 'vanishing gradient problem' in terms of matrix multiplication?

In deep networks, the gradient flows backward through products of weight matrices. If the singular values of these matrices are < 1, the gradient magnitude shrinks exponentially with depth, preventing early layers from learning.

What is the 'kernel trick' in machine learning?

The kernel trick replaces explicit computation of phi(x)^T phi(y) in a high-dimensional feature space with a kernel function K(x,y) that gives the same result. This enables nonlinear methods using only inner products.

What is 'batch normalization' from a linear algebra standpoint?

Batch normalization subtracts the mean and divides by the standard deviation of each feature across the mini-batch. This is a per-feature affine transformation that stabilizes the distribution of layer inputs during training.

What is 'transfer learning' in terms of the weight matrices of a neural network?

Transfer learning uses weight matrices learned on a large dataset (like ImageNet) as starting parameters for a new task. The learned linear transformations capture useful features that transfer across related tasks.

What is 'spectral graph theory' and its role in machine learning?

Spectral graph theory analyzes graph properties through the eigenvalues/eigenvectors of associated matrices. In ML, it enables spectral clustering, graph partitioning, and graph neural network design.

What is 'Fisher's Linear Discriminant' (LDA)?

LDA finds the projection direction w that maximizes the ratio of between-class scatter to within-class scatter: max (w^T S_B w) / (w^T S_W w). This is a generalized eigenvalue problem: S_B w = lambda S_W w.

What is 'tensor decomposition' in machine learning?

Tensor decomposition (CP, Tucker, etc.) extends matrix factorization to 3D+ arrays. It is used for multi-way data analysis, knowledge graph completion, and compressing neural network weight tensors.

A computer vision engineer applies SVD-based compression to a convolutional layer with weight tensor of shape (256, 128, 3, 3). She reshapes it to a 256 x 1152 matrix and applies rank-32 SVD. What is the compression ratio?

Original: 256 * 1152 = 294,912 params. Rank-32 SVD stores U (256 x 32) + S (32) + V^T (32 x 1152) = 8,192 + 32 + 36,864 = 45,088. Ratio: 294,912 / 45,088 ~ 6.5:1.

A natural language processing researcher measures word similarity using cosine similarity of embedding vectors. If word A has embedding (1, 2, 2) and word B has embedding (2, 1, 2), what is their cosine similarity?

cos(theta) = (A . B) / (||A|| ||B||) = (2 + 2 + 4) / (sqrt(9) * sqrt(9)) = 8 / 9. The words are quite similar in the embedding space.

A graph neural network uses the adjacency matrix A of a social network with 1000 nodes. The message passing layer computes H' = sigma(AHW). If H is 1000x64 and W is 64x32, what is the dimension of H'?

A (1000x1000) * H (1000x64) = 1000x64, then * W (64x32) = 1000x32. After activation sigma, H' is 1000x32. Each node's 64-dim feature is aggregated with neighbors' features and projected to 32 dimensions.

A reinforcement learning researcher uses a neural network to approximate a Q-function. The weight update involves the outer product gradient dL/dW = delta * x^T where delta is the error vector and x is the input. What is the rank of this gradient matrix?

The outer product delta * x^T of two vectors always produces a rank-1 matrix (every row is a scalar multiple of x^T). In stochastic gradient descent, each sample contributes a rank-1 update to the weights.

A machine learning engineer implements self-attention in a Transformer. The attention output for a single head is Attention(Q,K,V) = softmax(QK^T / sqrt(d_k)) V. If Q, K, V are all 10x64, what is the dimension of QK^T?

Q (10x64) * K^T (64x10) = 10x10 attention score matrix. Each entry (i,j) represents how much token i attends to token j, computed as the dot product of their query and key vectors.

Real-World Applications

Example: A deep learning researcher trains a language model with embedding dimension 768. Each token is mapped to a vector in R^768. If the vocabulary has 50,000 tokens, how many parameters are in the embedding matrix?

The embedding matrix E is 50,000 x 768, containing 50,000 * 768 = 38,400,000 parameters. Each row is a 768-dimensional vector representing one vocabulary token.

Example: A researcher applies LoRA (Low-Rank Adaptation) to fine-tune a large language model. Instead of updating a weight matrix W directly, she learns W + BA where B is d x r and A is r x d with r << d. If d = 4096 and r = 8, how many trainable parameters does LoRA use compared to full fine-tuning of W?

Full W: 4096 * 4096 = 16,777,216 params. LoRA: B (4096x8) + A (8x4096) = 32,768 + 32,768 = 65,536 params. Ratio: 16,777,216 / 65,536 = 256. LoRA uses 256x fewer trainable parameters.
