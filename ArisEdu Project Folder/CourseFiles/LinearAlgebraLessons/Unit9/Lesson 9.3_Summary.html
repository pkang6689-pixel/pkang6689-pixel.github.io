Lesson 9.3: Singular Value Decomposition (SVD) - Summary

Key Concepts: Singular Value Decomposition (SVD)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the relationship between the singular values and the 2-norm (operator norm) of a matrix?

The operator 2-norm (spectral norm) of a matrix equals its largest singular value: ||A||_2 = sigma_1 = max(||Ax|| / ||x||). This represents the maximum stretching factor of A.

If a matrix A has singular values 5, 3, and 1, what is the Frobenius norm ||A||_F?

||A||_F = sqrt(sum of sigma_i^2) = sqrt(25 + 9 + 1) = sqrt(35). The Frobenius norm equals the square root of the sum of squared singular values.

What is a 'singular value' of a matrix?

Singular values are the non-negative square roots of the eigenvalues of A^T A. They measure the amount of stretching that A applies along each of its principal directions (right singular vectors).

What is the 'pseudoinverse' (Moore-Penrose inverse) A^+ of a matrix?

The pseudoinverse is computed via SVD: replace each nonzero sigma_i with 1/sigma_i in S, then A^+ = V S^+ U^T. It gives the minimum-norm least-squares solution for any linear system.

What is the 'truncated SVD'?

The truncated (or thin) SVD keeps only the k largest singular values and their corresponding left and right singular vectors: A_k = U_k S_k V_k^T. This is the optimal rank-k approximation.

What is the 'compact SVD' (also called thin SVD)?

The compact SVD uses only the r columns of U and V corresponding to nonzero singular values. For an m x n matrix of rank r, U_r is m x r, S_r is r x r, and V_r is n x r.

What is the 'nuclear norm' (trace norm) of a matrix?

The nuclear norm is the sum of singular values. It is the tightest convex relaxation of the rank function and is used in matrix completion and low-rank optimization problems.

What is the geometric interpretation of the SVD?

The SVD decomposes any linear transformation into three steps: rotate/reflect input (V^T), scale along the coordinate axes (S), then rotate/reflect output (U). A unit sphere maps to an ellipsoid whose semi-axes are the singular values.

What is 'randomized SVD'?

Randomized SVD uses random projections to find an approximate low-rank basis, then computes the SVD on the smaller projected matrix. It is much faster than full SVD for large matrices where only the top k singular values are needed.

A numerical analyst finds that a 4x4 matrix A has singular values 100, 50, 0.01, and 0.005. What is the condition number, and what does it imply?

kappa = sigma_max / sigma_min = 100 / 0.005 = 20,000. This large condition number means the system is ill-conditioned -- small perturbations in the data can cause large changes in the solution.

Real-World Applications

Example: A data scientist compresses a 1000x800 image matrix using rank-20 SVD approximation. How many values must be stored for the compressed representation?

Store 20 left singular vectors (1000 entries each = 20,000) + 20 right singular vectors (800 entries each = 16,000) + 20 singular values = 36,020. Compare to 800,000 for the full matrix -- about 22:1 compression.

Example: A computer vision researcher applies SVD to a 100x100 grayscale image. The image has singular values sigma_1 = 200, sigma_2 = 50, sigma_3 = 10, and all others less than 1. Using rank-3 approximation, what fraction of the Frobenius norm is captured?

||A_3||_F^2 / ||A||_F^2 = (200^2 + 50^2 + 10^2) / (200^2 + 50^2 + 10^2 + remaining) = 42,600 / (42,600 + small) ~ 99.99%. The top 3 singular values dominate the image information.

Example: A natural language processing engineer computes SVD of a word-context matrix to create word embeddings. She keeps the top 300 dimensions. The resulting word vectors capture semantic similarity because:

SVD extracts latent semantic dimensions from word co-occurrence patterns. Words appearing in similar contexts project similarly onto these dimensions, creating embeddings where cosine similarity correlates with semantic relatedness.
