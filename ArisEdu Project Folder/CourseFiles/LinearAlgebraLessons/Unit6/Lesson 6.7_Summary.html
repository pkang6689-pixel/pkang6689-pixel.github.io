Lesson 6.7: Applications in Physics - Summary

Key Concepts: Applications in Physics

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If a force vector F = (3, 4) N is applied to an object, what is the magnitude of F?

||F|| = sqrt(3^2 + 4^2) = sqrt(9 + 16) = sqrt(25) = 5 N. The magnitude is computed using the Euclidean norm, which is preserved under rotations.

If a 2D stress tensor is given by S = [[10, 3],[3, 5]] MPa, what is the stress on a surface with normal direction (1, 0)?

The stress vector on a surface with normal n is t = Sn. So t = [[10,3],[3,5]] * (1,0) = (10, 3) MPa. The first column of S gives the stress on surfaces perpendicular to the x-axis.

In optics, a thin lens system can be described by a 2x2 matrix acting on a ray vector (height, angle). What is this matrix called?

The ray transfer matrix (or ABCD matrix) is a 2x2 matrix that describes how optical elements transform a light ray's height and angle. Cascading multiple elements is done by multiplying their matrices.

What is a 'tensor' in physics?

A tensor is a mathematical object that transforms according to specific rules under coordinate changes. Scalars are rank-0 tensors, vectors are rank-1, and matrices like the stress tensor are rank-2. They are fundamental to physics.

What is 'principal axis' in the context of the inertia tensor?

A principal axis is an eigenvector direction of the inertia tensor. Rotating about a principal axis produces angular momentum L = I*omega that is parallel to the rotation axis, simplifying the dynamics.

What is a 'unitary matrix' in quantum mechanics?

A unitary matrix satisfies U*U^dagger = U^dagger*U = I, where U^dagger is the conjugate transpose. It preserves the inner product in complex vector spaces, ensuring probability conservation in quantum mechanics.

What is the 'stress tensor' in continuum mechanics?

The stress tensor sigma is a symmetric 3x3 matrix. Given a surface with unit normal n, the traction (stress vector) on that surface is t = sigma*n. Its eigenvalues are the principal stresses.

What is the 'rotation group' SO(3) in physics?

SO(3) is the special orthogonal group of 3x3 rotation matrices. Each element represents a rotation in 3D space. It is fundamental to describing the symmetries of physical systems and conservation laws.

An optics engineer designs a two-lens system. Lens 1 (focal length f1 = 10 cm) has ray transfer matrix M1 = [[1,0],[-0.1, 1]] and a free space distance d = 5 cm has matrix D = [[1, 5],[0, 1]]. What is the system matrix for light passing through Lens 1 then propagating distance d?

M = D * M1 = [[1,5],[0,1]] * [[1,0],[-0.1,1]] = [[1+5*(-0.1), 0+5*1],[0+1*(-0.1), 0+1*1]] = [[0.5, 5],[-0.1, 1]]. The ray transfer matrices multiply in reverse order of the physical sequence.

A quantum computing researcher uses the Hadamard gate matrix H = (1/sqrt(2))[[1,1],[1,-1]] on a qubit in state |0> = (1,0). What is the resulting state?

H|0> = (1/sqrt(2))[[1,1],[1,-1]] * (1,0) = (1/sqrt(2))(1, 1). This is an equal superposition of |0> and |1>, meaning the qubit has a 50% chance of being measured as 0 or 1.

A mechanical engineer has a rigid body with inertia tensor I = [[4, 0, 0],[0, 2, 0],[0, 0, 3]] kg*m^2. If the body spins with angular velocity w = (1, 2, 3) rad/s, what is the angular momentum L = I*w?

L = I*w = [[4,0,0],[0,2,0],[0,0,3]] * (1,2,3) = (4*1, 2*2, 3*3) = (4, 4, 9) kg*m^2/s. Since I is diagonal (principal axes), each component of L is simply the corresponding moment of inertia times the angular velocity component.
