Lesson 6.8: Applications in Engineering - Summary

Key Concepts: Applications in Engineering

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

A structural engineer solves Kd = F where K is the stiffness matrix, d is displacement, and F is the force vector. If K = [[4, -1],[-1, 4]] and F = (3, 6), what is d?

det(K) = 16-1 = 15. K^(-1) = (1/15)[[4,1],[1,4]]. d = K^(-1)*F = (1/15)(4*3+1*6, 1*3+4*6) = (1/15)(18, 27) = (1.2, 1.8). The displacements are 1.2 and 1.8 units respectively.

A 3-node electrical circuit has the conductance matrix G = [[3,-1,-1],[-1,3,-1],[-1,-1,3]]. What is the sum of each row of G?

Row 1 sum: 3 + (-1) + (-1) = 1. Each row sums to 1, which reflects the total conductance seen from each node to ground. This is related to Kirchhoff's current law at each node.

If a system has transfer function matrix H(s) and input U(s), what is the output Y(s)?

In the frequency domain, the output Y(s) equals the product of the transfer function matrix H(s) and the input U(s). This is a direct application of linear transformation -- H(s) linearly maps inputs to outputs.

What is a 'state-space model' in control engineering?

A state-space model represents a dynamic system with matrices: x' = Ax + Bu describes state evolution and y = Cx + Du describes the output. It compactly captures multi-input, multi-output system behavior using linear algebra.

What is a 'stiffness matrix' in structural engineering?

The stiffness matrix K linearly maps displacement vectors d to force vectors F via F = Kd. It encodes how the structure resists deformation, and its properties (positive definiteness, symmetry) reflect physical stability.

What is the 'transfer function' in engineering?

The transfer function H(s) = C(sI - A)^(-1)B + D relates the Laplace-transformed output to input. It captures the system's frequency response and is a rational function whose poles are the eigenvalues of A.

What is the 'finite element method' (FEM)?

FEM discretizes a continuous domain into finite elements, constructs local stiffness matrices for each element, assembles them into a global matrix equation, and solves the resulting linear system to approximate the solution.

What is a 'Markov chain' in engineering and probability?

A Markov chain is a system that transitions between states probabilistically, where the probability of the next state depends only on the current state (memoryless property). The transition probabilities form a matrix whose columns sum to 1.

What is a 'graph Laplacian' matrix?

The graph Laplacian L = D - A has diagonal entries equal to node degrees and off-diagonal entries equal to the negative of adjacency values. Its eigenvalues reveal graph connectivity, and it is used in network analysis and spectral clustering.

What is the 'condition number' of a matrix?

The condition number kappa(A) = sigma_max/sigma_min measures numerical sensitivity. A large condition number means small input changes can cause large output changes, making the system ill-conditioned for numerical computation.

A robotics engineer has a 2-link planar robot arm. The Jacobian matrix J relates joint velocities to end-effector velocity. If J = [[2, 1],[0, 3]] and joint velocities are (1, 2) rad/s, what is the end-effector velocity?

v = J * q_dot = [[2,1],[0,3]] * (1,2) = (2*1+1*2, 0*1+3*2) = (4, 6). The Jacobian linearly maps joint velocities to the end-effector's Cartesian velocity.

A telecommunications engineer designs a network routing system where traffic at 3 nodes is redistributed each hour by matrix P = [[0.5, 0.3, 0.2],[0.3, 0.4, 0.3],[0.2, 0.3, 0.5]]. Starting with traffic vector t = (100, 200, 300), what is the total traffic after one redistribution step?

Each column of P sums to 1 (it is column-stochastic), so total traffic is conserved: sum(P*t) = (1,1,1)*P*t = (1,1,1)*t = 100+200+300 = 600. Redistribution moves traffic between nodes without creating or destroying it.

Real-World Applications

Example: In signal processing, the Discrete Fourier Transform (DFT) can be expressed as a matrix multiplication F*x. What kind of matrix is F?

The DFT matrix F has entries F[j,k] = omega^(jk) where omega = e^(-2*pi*i/N). After normalization by 1/sqrt(N), it is unitary (F*F^dagger = I), meaning the transform preserves energy.

Example: In image processing, a convolution operation can be represented as multiplication by what type of matrix?

Convolution with a fixed kernel can be expressed as multiplication by a Toeplitz matrix (or circulant matrix for circular convolution). Each row is a shifted copy of the kernel, making the operation a matrix-vector product.

Example: An electrical engineer has a circuit with node voltages satisfying Gv = I, where G = [[5, -2],[-2, 5]] Siemens and I = (3, 6) Amperes. What are the node voltages?

det(G) = 25-4 = 21. G^(-1) = (1/21)[[5,2],[2,5]]. v = (1/21)(5*3+2*6, 2*3+5*6) = (1/21)(27, 36) = (9/7, 12/7) approximately (1.286, 1.714) volts.

Example: A web search engine models page importance using the PageRank algorithm, which uses a transition matrix M. After many iterations, the ranking converges to the eigenvector of M corresponding to which eigenvalue?

PageRank converges to the eigenvector of M with eigenvalue 1 (the Perron-Frobenius eigenvector). Since M is a column-stochastic matrix, 1 is its largest eigenvalue, and repeated multiplication drives any initial vector toward this steady-state distribution.
