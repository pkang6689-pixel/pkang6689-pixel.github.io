Lesson 6.3: Matrix Representation of Transformations - Summary

Key Concepts: Matrix Representation of Transformations

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If T: R^2 -> R^2 is a linear transformation with T(e1) = (3,1) and T(e2) = (2,4), what is the matrix representation of T?

The matrix of a linear transformation is formed by placing the images of the standard basis vectors as columns. T(e1)=(3,1) is column 1 and T(e2)=(2,4) is column 2, giving [[3,2],[1,4]].

What is the matrix representation of the identity transformation on R^3?

The identity transformation T(v)=v maps each standard basis vector to itself: T(e1)=e1, T(e2)=e2, T(e3)=e3. Placing these as columns gives the 3x3 identity matrix.

If A is a 3x2 matrix representing T: R^2 -> R^3, and v = (1,2), what is the dimension of T(v)?

A 3x2 matrix multiplied by a 2x1 vector produces a 3x1 vector. So T(v) = Av is a vector in R^3 with 3 components.

If A = [[2,1],[3,4]] and v = (1,1), what is Av?

Av = [[2,1],[3,4]] * [1,1]^T = [2(1)+1(1), 3(1)+4(1)] = [3, 7]. Each row of A is dotted with the vector v.

If A = [[1,2,3],[4,5,6]], what is A times the vector (1,0,0)?

Multiplying A by e1 = (1,0,0) simply extracts the first column of A: (1,4). This confirms that the columns of a transformation matrix are the images of the standard basis vectors.

If T: R^2 -> R^2 has matrix A = [[0,-1],[1,0]], what is T(3,4)?

T(3,4) = [[0,-1],[1,0]] * [3,4]^T = [0(3)+(-1)(4), 1(3)+0(4)] = [-4, 3]. This matrix represents a 90-degree counterclockwise rotation.

What is a 'standard matrix' of a linear transformation T: R^n -> R^m?

The standard matrix of T is the unique matrix whose columns are the images of the standard basis vectors under T. It allows computation of T(x) as the matrix-vector product Ax for any input x.

What is a 'change of basis matrix'?

A change of basis matrix P converts coordinate vectors expressed in one basis B to their representation in another basis B'. Its columns are the old basis vectors expressed in the new basis coordinates.

What is a 'coordinate vector' of v with respect to basis B = {b1, b2, ..., bn}?

The coordinate vector [v]_B = (c1, c2, ..., cn) contains the unique scalars needed to express v as a linear combination of the basis vectors: v = c1*b1 + c2*b2 + ... + cn*bn.

What is a 'transition matrix' from basis B to basis B'?

The transition matrix from B to B' has columns that are the basis vectors of B written in the coordinate system of B'. It converts any coordinate vector from B-coordinates to B'-coordinates.

What is a 'diagonal matrix representation' of a transformation?

A diagonal matrix representation has nonzero entries only on the main diagonal. When achieved, it means the transformation simply scales each basis vector independently, making computation and analysis especially simple.

An engineer uses two sensors to measure a system's state. Sensor 1 gives 2x + y and sensor 2 gives x - 3y, where (x,y) is the true state. She arranges this as T(x,y) = (2x+y, x-3y). If the sensors read (7, -5), what is the transformation matrix she should invert to recover the true state?

The transformation matrix has rows corresponding to each sensor's linear combination: row 1 is [2,1] and row 2 is [1,-3], giving A = [[2,1],[1,-3]]. To recover (x,y) from the reading (7,-5), she solves A * [x,y]^T = [7,-5]^T.

A physicist studies a quantum system with two basis states |a> and |b>. A colleague's lab uses a different basis |a'> and |b'>. The change-of-basis matrix is P = [[1,1],[1,-1]]. If the transformation matrix in the first basis is A = [[3,0],[0,1]], what is the matrix in the second basis?

The transformation in the new basis is P^(-1)AP. P^(-1) = (1/2)[[1,1],[1,-1]], so P^(-1)AP = (1/2)[[1,1],[1,-1]][[3,0],[0,1]][[1,1],[1,-1]] = (1/2)[[3,1],[3,-1]][[1,1],[1,-1]] = (1/2)[[4,2],[2,4]] = [[2,1],[1,2]].

A GPS navigation system converts latitude-longitude to flat map coordinates using T. The transformation maps (1,0) to (111,0) and (0,1) to (0,85) (km per degree). What is the transformation matrix, and what map coordinates correspond to a position shift of (0.5, 2) degrees?

The matrix columns are the images of standard basis vectors: A = [[111,0],[0,85]]. Then A*(0.5, 2) = (111*0.5 + 0*2, 0*0.5 + 85*2) = (55.5, 170) km.
