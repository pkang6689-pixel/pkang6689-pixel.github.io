Lesson 1.2: Matrix Notation & Dimensions - Summary

Key Concepts: Matrix Notation & Dimensions

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

In the matrix B = [[3, 0, -1], [2, 7, 4], [1, -5, 8]], what is the entry b_{2,3}?

The entry b_{2,3} is in row 2, column 3. Row 2 is [2, 7, 4], so the third entry is 4.

What is the size of the matrix [[7]]?

A matrix containing a single entry arranged in one row and one column has dimensions 1 x 1. While it contains just one number, it is still formally a matrix.

What is the 'main diagonal' of a square matrix?

The main diagonal of a square matrix consists of entries a_{i,i} where the row index and column index are equal, running from the top-left to the bottom-right.

What is a 'diagonal matrix'?

A diagonal matrix is a square matrix in which every entry not on the main diagonal is zero. Only the entries a_{i,i} may be nonzero.

What is an 'upper triangular matrix'?

An upper triangular matrix is a square matrix in which all entries below the main diagonal are zero. Entries on and above the diagonal may be nonzero.

What is a 'lower triangular matrix'?

A lower triangular matrix is a square matrix where every entry above the main diagonal is zero. Only the entries on and below the diagonal may be nonzero.

What is a 'symmetric matrix'?

A symmetric matrix is a square matrix A where A = A^T, meaning a_{i,j} = a_{j,i} for all i and j. The matrix is a mirror image across its main diagonal.

Real-World Applications

Example: A retail chain tracks monthly sales across 10 stores and 12 months. The manager wants to look up Store 7's sales in March (month 3). Which entry should she access?

With stores as rows and months as columns, Store 7 is row 7 and March is column 3. The correct entry is a_{7,3}.

Example: A teacher creates a grade matrix for 30 students across 4 exams. She realizes she needs to add a 5th exam column. What will the new matrix dimensions be?

The original matrix is 30 x 4 (30 students, 4 exams). Adding one more exam column changes it to 30 x 5 while keeping the same 30 rows.

Example: A data analyst receives a matrix that is 1000 x 1. Her manager asks if this could represent a dataset of customer ages. How should she interpret this matrix?

A 1000 x 1 matrix is a column vector with 1000 entries. Each entry could represent one customer's age, making it a valid representation of 1000 customer ages.
