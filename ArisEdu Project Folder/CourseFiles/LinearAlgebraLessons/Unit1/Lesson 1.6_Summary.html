Lesson 1.6: Transpose of a Matrix - Summary

Key Concepts: Transpose of a Matrix

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is the transpose of [[1, 2, 3], [4, 5, 6]]?

The transpose swaps rows and columns. Row 1 [1,2,3] becomes column 1, and row 2 [4,5,6] becomes column 2, giving [[1,4],[2,5],[3,6]].

What is (A^T)^T?

Transposing a matrix twice returns the original matrix: (A^T)^T = A. The two transpositions cancel each other out.

What is the transpose of the column vector [[3], [7], [1]]?

The transpose of a 3 x 1 column vector is a 1 x 3 row vector: [[3],[7],[1]]^T = [3, 7, 1].

What is (A + B)^T?

The transpose of a sum is the sum of the transposes: (A + B)^T = A^T + B^T. This is a fundamental property of the transpose operation.

What is (AB)^T?

The transpose of a product reverses the order: (AB)^T = B^T * A^T. This is known as the 'reversal rule' or 'shoe-sock property' for transposes.

What is (cA)^T where c is a scalar?

Scalars pass through the transpose: (cA)^T = c * A^T. Since transposing only rearranges positions of entries, the scalar multiplier is unaffected.

If A = [[1, 3], [0, 2]], what is entry (2,1) of A^T?

Entry (i,j) of A^T equals entry (j,i) of A. So (A^T)_{2,1} = A_{1,2} = 3.

What is the 'transpose' of a matrix?

The transpose of a matrix A, denoted A^T, is formed by turning rows into columns and columns into rows: (A^T)_{i,j} = A_{j,i}.

What is a 'symmetric matrix'?

A symmetric matrix satisfies A = A^T, meaning a_{i,j} = a_{j,i} for all i and j. The matrix mirrors across its main diagonal.

What is a 'skew-symmetric' (or antisymmetric) matrix?

A skew-symmetric matrix satisfies A^T = -A, meaning a_{i,j} = -a_{j,i}. This implies all diagonal entries must be zero.

What is an 'orthogonal matrix'?

An orthogonal matrix Q satisfies Q^T * Q = Q * Q^T = I, meaning the transpose equals the inverse. Its columns (and rows) form an orthonormal set.

What is the relationship between the transpose and the trace of a matrix?

The trace of a matrix equals the trace of its transpose: tr(A^T) = tr(A). This is because the diagonal entries are unchanged by transposition.

A physicist models an angular velocity using a 3 x 3 skew-symmetric matrix W. An assistant asks why all diagonal entries of W are zero. What is the correct mathematical explanation?

For a skew-symmetric matrix W^T = -W. The diagonal entries satisfy w_{i,i} = -w_{i,i}, so 2w_{i,i} = 0, which means w_{i,i} = 0.

How to Calculate

Example: A sports league has a results matrix W where w_{i,j} = 1 if team i beat team j, and w_{i,j} = 0 otherwise. A statistician computes W + W^T and finds some entries equal 1. What does an entry of 1 in W + W^T at position (i,j) mean?

Solution: Entry (i,j) in W + W^T is w_{i,j} + w_{j,i}. If this equals 1, then exactly one of w_{i,j} and w_{j,i} is 1, meaning exactly one team beat the other.

Example: A social network represents directed friendships in matrix F, where f_{i,j} = 1 means person i follows person j. The network wants to find mutual follows (where both follow each other). Which matrix operation helps identify mutual follows?

Solution: The element-wise product of F and F^T gives 1 at position (i,j) only when both f_{i,j} = 1 and f_{j,i} = 1, identifying mutual follows.
