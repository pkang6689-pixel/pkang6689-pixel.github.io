Lesson 1.3: Vector Operations (addition, scalar multiplication) - Summary

Key Concepts: Vector Operations (addition, scalar multiplication)

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

What is (2, 5) + (3, -1)?

Vector addition is done component-wise: (2+3, 5+(-1)) = (5, 4).

What is 3 * (4, -2)?

Scalar multiplication multiplies each component by the scalar: 3*(4, -2) = (3*4, 3*(-2)) = (12, -6).

If u = (1, 0, -3) and v = (2, 4, 1), what is u + v?

Adding component-wise: (1+2, 0+4, -3+1) = (3, 4, -2).

What is -2 * (5, -3, 1)?

Multiplying each component by -2: (-2*5, -2*(-3), -2*1) = (-10, 6, -2).

If u = (3, 7) and v = (3, 7), what is u - v?

Subtracting identical vectors: (3-3, 7-7) = (0, 0). Any vector minus itself gives the zero vector.

What is 0 * (8, -4, 2)?

Multiplying any vector by the scalar 0 yields the zero vector: 0*(8, -4, 2) = (0, 0, 0).

Vector addition is commutative, meaning u + v = v + u. What is (1, 2) + (3, 4)?

Adding component-wise: (1+3, 2+4) = (4, 6). The commutative property guarantees (3, 4) + (1, 2) gives the same result.

If v = (6, -2, 4), what is (1/2) * v?

Multiplying each component by 1/2: (6/2, -2/2, 4/2) = (3, -1, 2).

What is (4, -1, 3) - (2, -1, 5)?

Subtracting component-wise: (4-2, -1-(-1), 3-5) = (2, 0, -2).

What is 'vector addition'?

Vector addition takes two vectors of the same dimension and produces a new vector by adding their corresponding components: (a1+b1, a2+b2, ..., an+bn).

What is the 'commutative property' of vector addition?

The commutative property states that the order of addition does not matter: u + v = v + u for any two vectors of the same dimension.

What is a 'linear combination' of vectors?

A linear combination is an expression c1*v1 + c2*v2 + ... + ck*vk, where each vi is a vector and each ci is a scalar. It combines scaling and addition.

What is the 'additive inverse' of a vector v?

The additive inverse of v is -v (equivalently, (-1)*v), obtained by negating each component. Adding v and -v produces the zero vector: v + (-v) = 0.

A delivery truck starts at position (2, 3) on a city grid and then drives along the displacement vector (5, -2). What is the truck's final position?

The final position is found by adding the starting position and displacement: (2, 3) + (5, -2) = (2+5, 3+(-2)) = (7, 1).

Two forces act on an object: F1 = (10, 6) Newtons and F2 = (-4, 8) Newtons. What is the net (resultant) force on the object?

The net force is the vector sum of all forces: F1 + F2 = (10+(-4), 6+8) = (6, 14) Newtons.

An airplane's velocity relative to the air is v_air = (500, 0) km/h (due east), and the wind velocity is v_wind = (-50, 30) km/h. What is the airplane's ground velocity (v_air + v_wind)?

Ground velocity = air velocity + wind velocity = (500+(-50), 0+30) = (450, 30) km/h. The headwind slows the eastward speed, and the wind adds a northward component.

A painter mixes two color vectors representing RGB values: color1 = (200, 50, 100) and color2 = (100, 200, 50). The painter wants a blend that is 0.5*color1 + 0.5*color2. What is the result?

Computing the linear combination: 0.5*(200,50,100) + 0.5*(100,200,50) = (100,25,50) + (50,100,25) = (150, 125, 75).

A robot receives two movement commands: first move by (3, -2, 1) meters, then move by (-1, 4, -1) meters. Due to a software update, all movements are doubled. What is the robot's total displacement after both doubled movements?

Each movement is doubled: 2*(3,-2,1) = (6,-4,2) and 2*(-1,4,-1) = (-2,8,-2). Total displacement: (6,-4,2) + (-2,8,-2) = (4, 4, 0).

Real-World Applications

Example: What does 'scalar multiplication' of a vector mean?

Scalar multiplication takes a scalar c and a vector v, and produces a new vector by multiplying every component of v by c. This scales the vector's magnitude.

Example: What does it mean for a scalar multiple cv to 'reverse direction' relative to v?

When a vector is multiplied by a negative scalar, all components change sign, which geometrically reverses the direction of the vector while scaling its magnitude.

Example: A clothing store sells shirts, pants, and jackets. Monday's sales = (40, 25, 10) and Tuesday's sales = (35, 30, 15). The manager wants to find the combined two-day total. What operation should she use?

To find the combined total, add the two sales vectors: (40, 25, 10) + (35, 30, 15) = (75, 55, 25). Vector addition combines corresponding components.

Example: A company's monthly revenue is r = (12000, 8000, 15000) dollars across three product lines. If each product line experiences a 10% increase, the new revenue can be expressed as what scalar multiple of r?

A 10% increase means the new value is 110% of the original, or 1.10 times the original. So the new revenue is 1.10 * r = (13200, 8800, 16500).
