Lesson 3.2: Properties of Determinants - Summary

Key Concepts: Properties of Determinants

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If a matrix has two identical rows, what is its determinant?

If two rows are identical, swapping them gives the same matrix but changes the sign: det(A) = -det(A), which implies 2*det(A) = 0, so det(A) = 0.

What is det(AB) for square matrices A and B?

A fundamental property of determinants is that det(AB) = det(A) * det(B) for any two square matrices of the same size.

What is the relationship between det(A) and det(A^T)?

The determinant of a matrix equals the determinant of its transpose: det(A^T) = det(A). This means all row properties of determinants also apply to columns.

If A is an n x n matrix and k is a scalar, what is det(kA)?

Multiplying the entire matrix by k multiplies each of the n rows by k, so det(kA) = k^n * det(A).

If det(A) = 5, what is det(A^(-1))?

Since A * A^(-1) = I and det(AB) = det(A)*det(B), we get det(A)*det(A^(-1)) = det(I) = 1, so det(A^(-1)) = 1/det(A) = 1/5.

A 3x3 matrix A has det(A) = 4. What is det(2A)?

det(2A) = 2^3 * det(A) = 8 * 4 = 32, since A is 3x3.

If a matrix has a row of all zeros, what is its determinant?

A row of all zeros means the rows are linearly dependent (the zero row is a trivial linear combination), so the determinant is 0.

What is an 'elementary matrix' in the context of determinants?

An elementary matrix results from applying a single elementary row operation to the identity matrix. Its determinant depends on the operation: -1 for a swap, k for scaling, 1 for adding a multiple.

A physicist has a 3x3 matrix with det(A) = 6. She scales the first row by 2 and swaps rows 2 and 3. What is the new determinant?

Scaling row 1 by 2 multiplies det by 2: 6*2 = 12. Then swapping rows 2 and 3 changes the sign: 12*(-1) = -12.

An engineer has matrices A and B with det(A) = 3 and det(B) = -2. She computes the product C = AB. A colleague asks for det(C). What is it?

det(C) = det(AB) = det(A)*det(B) = 3*(-2) = -6.

A computer graphics programmer applies a 3x3 scaling matrix S (scaling x by 2, y by 3, z by 4) to a 3D object with volume 10 cubic units. What is the volume of the transformed object?

S = diag(2,3,4), so det(S) = 2*3*4 = 24. The volume scales by |det(S)| = 24, so new volume = 24 * 10 = 240 cubic units.

A student computes det(A) = 7 for a 4x4 matrix. She then computes det(3A). What is her answer?

det(3A) = 3^4 * det(A) = 81 * 7 = 567, since A is 4x4.

A cryptographer performs 5 row swaps while reducing a matrix to upper triangular form with diagonal entries 1, 2, 3. What is the determinant of the original matrix?

The product of the diagonal entries of the triangular form is 1*2*3 = 6. With 5 row swaps, the sign changes 5 times: (-1)^5 * 6 = -6.

A data scientist has a matrix A with det(A) = 10. She computes A^3. What is det(A^3)?

det(A^3) = (det A)^3 = 10^3 = 1000.

A structural engineer verifies that matrix A has det(A) = -4. She needs det(A^(-1)). What is it?

det(A^(-1)) = 1/det(A) = 1/(-4) = -1/4.

An economist has a 3x3 matrix where row 3 = row 1 + 2*(row 2). Without computing entries, what is the determinant?

Since row 3 is a linear combination of rows 1 and 2, the rows are linearly dependent. Therefore det = 0 regardless of the actual entries.

A teacher asks: if det(A) = 2 and det(B) = 3, what is det(A^T B^(-1))? What should the student answer?

det(A^T B^(-1)) = det(A^T) * det(B^(-1)) = det(A) * 1/det(B) = 2 * (1/3) = 2/3.

A quality control team has matrices A (3x3, det=5) and B (3x3, det=-2). They form the block diagonal matrix M = diag(A, B), a 6x6 matrix. What is det(M)?

For a block diagonal matrix, det(M) = det(A) * det(B) = 5 * (-2) = -10.

Real-World Applications

Example: If one row of a matrix is multiplied by a scalar k, what happens to the determinant?

Multiplying a single row by k multiplies the determinant by k. Note: if you multiply the entire nxn matrix by k, the determinant is multiplied by k^n.

Example: If adding a multiple of one row to another row is performed, what happens to the determinant?

Adding a scalar multiple of one row to a different row does not change the determinant. This is the most common row operation in Gaussian elimination.

Example: What does the 'multiplicative property' of determinants state?

The multiplicative property states det(AB) = det(A) * det(B). Note that while the third option is also true, it is just a special case of this general property.
