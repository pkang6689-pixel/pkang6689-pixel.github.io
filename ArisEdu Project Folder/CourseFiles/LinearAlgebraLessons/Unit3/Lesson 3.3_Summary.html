Lesson 3.3: Cofactor Expansion - Summary

Key Concepts: Cofactor Expansion

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

Using cofactor expansion along row 1, what is det([[2,1,3],[0,4,5],[0,0,6]])?

This is upper triangular, so det = 2*4*6 = 48. Via cofactor expansion: 2*det([[4,5],[0,6]]) - 1*det([[0,5],[0,6]]) + 3*det([[0,4],[0,0]]) = 2(24) - 1(0) + 3(0) = 48.

What is the cofactor C_{11} of entry a_{11} = 3 in the matrix [[3,1],[2,5]]?

C_{11} = (-1)^(1+1) * M_{11} = (+1) * 5 = 5. The minor M_{11} is the determinant of the submatrix obtained by removing row 1 and column 1, which is [[5]], so M_{11} = 5.

What is the sign factor (-1)^(i+j) for entry a_{2,3}?

(-1)^(2+3) = (-1)^5 = -1. The checkerboard pattern gives a minus sign at position (2,3).

What is det([[5,0,0],[0,3,0],[0,0,7]]) using cofactor expansion?

Expanding along row 1: 5*det([[3,0],[0,7]]) - 0 + 0 = 5*(21) = 105. Alternatively, for a diagonal matrix, det = product of diagonal = 5*3*7 = 105.

What is 'cofactor expansion' (also called Laplace expansion)?

Cofactor expansion computes det(A) by summing products of entries and their cofactors along any single row or column: det(A) = sum of a_{ij} * C_{ij}.

What is a 'submatrix' in the context of cofactor expansion?

In cofactor expansion, the submatrix for entry a_{ij} is the (n-1)x(n-1) matrix obtained by deleting row i and column j.

What is the 'recursive nature' of cofactor expansion?

Cofactor expansion reduces an nxn determinant to a sum of (n-1)x(n-1) determinants. Each of these can be expanded further, recursing down to 2x2 or 1x1 base cases.

What is the 'adjugate' (classical adjoint) matrix?

The adjugate adj(A) is the transpose of the cofactor matrix. It satisfies A * adj(A) = det(A) * I, which gives the formula A^(-1) = adj(A)/det(A).

What is the computational complexity of computing an nxn determinant by cofactor expansion?

Naive cofactor expansion has O(n!) complexity because each nxn determinant spawns n determinants of size (n-1), leading to n! total operations. This is why Gaussian elimination (O(n^3)) is preferred for large n.

What is an 'alien cofactor' expansion?

If you multiply entries from row i by cofactors of row k (where i != k), the sum is always zero. This is used to prove the formula A * adj(A) = det(A) * I.

What is the connection between cofactors and the inverse of a matrix?

The inverse formula A^(-1) = (1/det(A)) * adj(A) uses cofactors directly. The (i,j) entry of A^(-1) is C_{ji}/det(A).

A structural engineer needs det of a 3x3 stiffness matrix [[10, -3, 0], [-3, 10, -3], [0, -3, 10]]. She expands along row 1. What is the first term (from entry a_{11})?

First term: a_{11} * C_{11} = 10 * det([[10,-3],[-3,10]]) = 10 * (100 - 9) = 10 * 91 = 910.

A physicist needs det([[2,-1,0,0],[-1,2,-1,0],[0,-1,2,-1],[0,0,-1,2]]). She expands along row 1 and gets the first two terms as 2*det(3x3) and 1*det(3x3). After full computation, what is the determinant?

This is a tridiagonal matrix with all 2s on diagonal and -1s on off-diagonals. The determinants follow the pattern D_n = 2*D_{n-1} - D_{n-2}. D_1=2, D_2=3, D_3=4, D_4=5.

A network engineer has a 3x3 adjacency-related matrix [[0, 1, 1], [1, 0, 1], [1, 1, 0]]. He expands along row 1 to find det. What is the result?

det = 0*C_{11} - 1*det([[1,1],[1,0]]) + 1*det([[1,0],[1,1]]) = 0 - 1*(0-1) + 1*(1-0) = 0 + 1 + 1 = 2.

An economist uses cofactor expansion to verify det of [[3,0,2],[1,4,0],[0,5,1]]. Expanding along column 1 (entries 3, 1, 0), what is the result?

3*det([[4,0],[5,1]]) - 1*det([[0,2],[5,1]]) + 0 = 3*(4-0) - 1*(0-10) = 12 + 10 = 22. The determinant is 22.

A data analyst needs the adjugate of a 2x2 matrix [[a,b],[c,d]]. Using cofactors, what is adj(A)?

The cofactor matrix is [[d,-c],[-b,a]]. The adjugate is its transpose: [[d,-b],[-c,a]]. Then A^(-1) = adj(A)/(ad-bc).
