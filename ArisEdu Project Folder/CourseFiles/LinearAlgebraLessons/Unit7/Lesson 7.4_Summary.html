Lesson 7.4: Diagonalization - Summary

Key Concepts: Diagonalization

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If A = PDP^(-1), what is A^5?

A^5 = (PDP^(-1))^5 = PD(P^(-1)P)D(P^(-1)P)...(P^(-1)P)DP^(-1) = PD^5 P^(-1). The inner P^(-1)P pairs all cancel, leaving only the outermost P and P^(-1).

What is D^k for a diagonal matrix D = diag(2, 3)?

For a diagonal matrix, D^k = diag(d1^k, d2^k, ...). Each diagonal entry is raised to the k-th power independently.

Diagonalize A = [[4, 1], [0, 3]]. What is D?

The eigenvalues are 4 and 3 (diagonal entries of this upper triangular matrix). So D = diag(4, 3).

If A = PDP^(-1) with D = diag(1, -1), what is A^100?

A^100 = PD^100 P^(-1) = P diag(1^100, (-1)^100) P^(-1) = P diag(1, 1) P^(-1) = P I P^(-1) = I. Any diagonalizable involution raised to an even power is the identity.

If A is diagonalizable with eigenvalues 2 and 5, what is det(A)?

det(A) = det(PDP^(-1)) = det(D) = 2 * 5 = 10. The determinant is always the product of the eigenvalues.

What is the Spectral Theorem for real symmetric matrices?

The Spectral Theorem states that every real symmetric matrix has real eigenvalues and orthogonal eigenvectors, so A = QDQ^T where Q is an orthogonal matrix.

What is a 'defective matrix' in the context of diagonalization?

A defective matrix has at least one eigenvalue whose geometric multiplicity is less than its algebraic multiplicity, resulting in fewer than n independent eigenvectors.

What is meant by 'matrix similarity'?

Matrices A and B are similar if B = P^(-1)AP for some invertible P. Similar matrices share the same eigenvalues, trace, determinant, and characteristic polynomial.

What is the role of the matrix P in the diagonalization A = PDP^(-1)?

In A = PDP^(-1), P is the modal matrix whose columns are the eigenvectors of A. The i-th column of P is the eigenvector corresponding to the i-th diagonal entry of D.

What is a 'matrix exponential' and how does diagonalization help compute it?

The matrix exponential e^A = I + A + A^2/2! + ... If A = PDP^(-1), then e^A = P e^D P^(-1) where e^D = diag(e^(d1), e^(d2), ...), making computation straightforward.

What is a 'diagonal matrix' in linear algebra?

A diagonal matrix has nonzero entries only on the main diagonal (d_ij = 0 when i != j). It is the simplest type of matrix for computation.

A population biologist models a species with growth matrix A = [[0.5, 0.6], [0.4, 0.9]]. To predict the population after 20 generations efficiently, she diagonalizes A. If eigenvalues are 0.2 and 1.2, what is the dominant term of A^20?

A^20 = P diag(0.2^20, 1.2^20) P^(-1). Since 0.2^20 is essentially 0 and 1.2^20 is approximately 38.34, the long-term behavior is dominated by the eigenvalue 1.2 and its eigenvector.

A financial engineer uses diagonalization to compute the matrix exponential e^(At) for portfolio risk modeling with A = [[-1, 2], [0, -3]]. What is e^(0*A)?

e^(0) = e^0 = I for any matrix. This is the initial condition: at time t = 0, the state transition matrix is the identity.

A climate modeler diagonalizes the heat transfer matrix A = [[0.6, 0.3], [0.4, 0.7]] to predict long-term temperature distribution. The eigenvalues are 1 and 0.3. What is the steady-state temperature ratio between the two regions?

The steady state is the eigenvector for lambda=1. A - I = [[-0.4, 0.3], [0.4, -0.3]], giving 0.4x1 = 0.3x2, so x1/x2 = 3/4. The steady-state distribution is (3, 4) normalized to (3/7, 4/7).

Real-World Applications

Example: An image processing engineer needs to apply a sharpening filter A repeatedly. She diagonalizes A and finds eigenvalues 1.5 and 0.8. After 10 applications, which eigenvalue dominates?

1.5^10 approximately equals 57.67 while 0.8^10 approximately equals 0.107. The component along the eigenvector for 1.5 grows exponentially, dominating after multiple applications.
