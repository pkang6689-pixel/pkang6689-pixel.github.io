Lesson 7.3: Finding Eigenvectors - Summary

Key Concepts: Finding Eigenvectors

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

For the identity matrix I, what is the eigenspace for lambda = 1?

I - 1*I = 0 (the zero matrix). The null space of the zero matrix is all of R^n, so every nonzero vector is an eigenvector of I with eigenvalue 1.

For A = [[1, 0, 0], [0, 2, 0], [0, 0, 3]], what is the eigenvector corresponding to lambda = 2?

For diagonal matrices, the eigenvector for eigenvalue d_i is the standard basis vector e_i. Since lambda = 2 is the second diagonal entry, the eigenvector is e2 = (0, 1, 0).

What is an eigenvector of a matrix?

An eigenvector is a nonzero vector v satisfying Av = lambda v. It is a direction that the matrix scales without changing direction (or reverses for negative lambda).

What is the eigenspace of a matrix for a given eigenvalue?

The eigenspace for eigenvalue lambda is the null space of (A - lambda I). It consists of all eigenvectors for that eigenvalue together with the zero vector, forming a subspace.

What is a 'generalized eigenvector' of a matrix A?

A generalized eigenvector satisfies (A - lambda I)^k v = 0 for some k >= 1. Regular eigenvectors correspond to k = 1. Generalized eigenvectors are needed when A is not diagonalizable.

What is the 'null space' of (A - lambda I) in the context of eigenvectors?

The null space of (A - lambda I) is exactly the eigenspace for eigenvalue lambda. It contains all vectors v with (A - lambda I)v = 0, i.e., Av = lambda v.

What is meant by 'left eigenvector' versus 'right eigenvector'?

A right eigenvector v satisfies Av = lambda v (column vector on the right of A). A left eigenvector w satisfies w^T A = lambda w^T (row vector on the left of A). Left eigenvectors are eigenvectors of A^T.

What is an 'invariant subspace' with respect to a matrix A?

An invariant subspace W under A satisfies A(W) subset of W. Every eigenspace is an invariant subspace, since if v is in the eigenspace, Av = lambda v is also in the eigenspace.

A Google-style PageRank algorithm uses the matrix M = [[0, 1/2], [1, 1/2]]. The dominant eigenvector (for lambda = 1) gives the page rankings. What is this eigenvector (unnormalized)?

M - I = [[-1, 1/2], [1, -1/2]]. Row reducing gives [[1, -1/2], [0, 0]], so x1 = x2/2. Taking x2 = 2 gives (1, 2). Page 2 has twice the rank of page 1.

An ecologist models a population with the Leslie matrix L = [[0, 6], [0.5, 0]]. The dominant eigenvalue is sqrt(3). What is the corresponding eigenvector (stable age distribution)?

L - sqrt(3)I = [[-sqrt(3), 6], [0.5, -sqrt(3)]]. From row 1: -sqrt(3)*x1 + 6*x2 = 0, so x1 = 6x2/sqrt(3) = 2sqrt(3)*x2. The eigenvector is (2sqrt(3), 1).

A mechanical engineer analyzes a coupled spring system with matrix K = [[3, -1, 0], [-1, 2, -1], [0, -1, 3]]. One eigenvalue is 3. What is an eigenvector for this eigenvalue?

K - 3I = [[0,-1,0],[-1,-1,-1],[0,-1,0]]. From rows 1 and 3: x2 = 0. From row 2: -x1-x3 = 0, so x1 = -x3. The eigenvector is (1, 0, -1), representing outer masses moving opposite while center stays still.
