Lesson 7.5: Applications in Differential Equations - Summary

Key Concepts: Applications in Differential Equations

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

If a 2x2 system has eigenvalues 2 and -1, what is the behavior near the origin?

One positive eigenvalue (2) and one negative eigenvalue (-1) produce a saddle point. Solutions grow along one eigenvector direction and decay along the other.

For the system x' = Ax with purely imaginary eigenvalues +/- 3i, what is the behavior?

Purely imaginary eigenvalues (a = 0) produce a center with closed orbits. Solutions oscillate with period 2*pi/3 and constant amplitude.

What is the matrix exponential e^(At) used for in differential equations?

The solution to the linear ODE system x' = Ax with x(0) = x0 is x(t) = e^(At) x0. The matrix exponential serves as the fundamental matrix of the system.

What is a 'phase portrait' in the context of linear ODE systems?

A phase portrait shows the trajectories of solutions in the x1-x2 plane (state space). The shape of trajectories depends on the eigenvalues: nodes, saddles, spirals, or centers.

What is the 'fundamental matrix' of a linear system x' = Ax?

The fundamental matrix Phi(t) has columns that are n linearly independent solutions. The general solution is x(t) = Phi(t) c for a constant vector c. The matrix exponential e^(At) is a fundamental matrix.

What is a 'node' in the classification of equilibrium points?

A node has two real eigenvalues of the same sign. If both are negative, it is a stable node (sink); if both are positive, it is an unstable node (source).

What is the 'half-life' of a decaying mode in a linear system?

For a decaying exponential e^(lambda t) with lambda < 0, the half-life is t_half = ln(2)/|lambda|. This is the time for the mode amplitude to decrease by 50%.

What is a 'saddle point' in the phase plane?

A saddle point has one positive and one negative eigenvalue. Trajectories approach along the eigenvector of the negative eigenvalue and depart along that of the positive eigenvalue.

What is meant by 'variation of parameters' for a nonhomogeneous system x' = Ax + g(t)?

Variation of parameters writes x_p(t) = Phi(t) integral of Phi^(-1)(s) g(s) ds, using the fundamental matrix Phi(t) to find a particular solution to the nonhomogeneous system.

What is the 'state-space representation' of a differential equation?

State-space form converts a single nth-order ODE into a system of n first-order ODEs by introducing state variables x1 = y, x2 = y', etc., giving x' = Ax + Bu.

What is meant by the 'time constant' of a linear system mode?

The time constant tau = 1/|lambda| characterizes how quickly a mode decays (or grows). After time tau, the mode has decayed to e^(-1) or about 36.8% of its initial value.

A mechanical engineer models a damped spring-mass system with the equation mx'' + cx' + kx = 0 where m=1, c=5, k=6. Using eigenvalues, what is the response type?

The characteristic equation lambda^2 + 5lambda + 6 = (lambda+2)(lambda+3) = 0 gives eigenvalues -2 and -3. Two distinct negative reals = overdamped. The discriminant c^2 - 4mk = 25 - 24 = 1 > 0.

A control engineer designs a feedback system with x' = [[-2, 1], [-1, -2]]x. The eigenvalues are -2 +/- i. What is the frequency of oscillation?

For eigenvalues a +/- bi = -2 +/- i, the oscillation frequency is b = 1 rad/s. The real part a = -2 determines the decay rate. The solution oscillates at frequency 1 while decaying with time constant 1/2.

A robotics engineer linearizes a nonlinear system around an equilibrium and gets x' = [[0, 1], [-9, -6]]x. The eigenvalues are -3 (repeated). What is the nature of this equilibrium?

lambda = -3 repeated. A - (-3)I = [[3,1],[-9,-3]] has rank 1, so geometric multiplicity is 1 (defective). This gives a degenerate stable node: the system is critically damped with solutions decaying as (c1 + c2 t)e^(-3t).
