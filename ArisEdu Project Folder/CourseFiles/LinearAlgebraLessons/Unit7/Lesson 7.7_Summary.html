Lesson 7.7: Applications in Computer Science - Summary

Key Concepts: Applications in Computer Science

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

In a Markov chain with transition matrix P, what is the steady-state vector?

The steady-state vector pi satisfies P*pi = pi, making it the eigenvector for eigenvalue 1. It represents the long-run probability distribution.

What is the algebraic connectivity (Fiedler value) of a graph?

The algebraic connectivity (Fiedler value) is the second smallest eigenvalue of the graph Laplacian. It measures how well connected the graph is; a larger value means more connected.

What is 'spectral graph theory'?

Spectral graph theory analyzes graph properties using the spectrum (eigenvalues) of matrices like the adjacency matrix or Laplacian. It connects linear algebra to combinatorial properties.

What is the 'graph Laplacian' matrix?

The Laplacian L = D - A is a positive semidefinite matrix where D is diagonal with vertex degrees. It encodes the graph structure and its eigenvalues reveal connectivity.

What is the 'damping factor' in Google's PageRank and how does it affect eigenvalues?

The damping factor d blends the link matrix with a uniform matrix: M' = dM + (1-d)(1/n)ee^T. This guarantees all secondary eigenvalues have |lambda| <= d < 1, ensuring unique convergence.

What is 'convergence rate' of an iterative method in terms of eigenvalues?

The error after k iterations decays approximately as rho(B)^k, where rho(B) is the spectral radius of the iteration matrix. Smaller rho means faster convergence.

What is 'eigenface' in computer vision?

Eigenfaces are eigenvectors of the covariance matrix computed from a set of face images. They form a basis for face space, allowing efficient face recognition by projection.

What is the relationship between the adjacency matrix eigenvalues and graph properties?

The spectral radius of the adjacency matrix provides bounds on the chromatic number, independence number, and clique number. The spectrum encodes much structural information about the graph.

What is 'spectral embedding' in dimensionality reduction?

Spectral embedding maps data points to coordinates given by eigenvectors of a similarity or Laplacian matrix. Methods like Laplacian Eigenmaps and spectral clustering use this approach.

What is 'matrix factorization' in recommender systems?

Matrix factorization approximates the sparse rating matrix R as a product of low-rank matrices (R ~ UV^T), where U represents user factors and V represents item factors. This is closely related to eigendecomposition.

Example: A social media company uses PageRank on a network of 4 accounts with transition matrix M = [[0, 1/2, 0, 0], [1/3, 0, 0, 1/2], [1/3, 0, 0, 1/2], [1/3, 1/2, 1, 0]]. The dominant eigenvector gives the influence ranking. Which property of M ensures convergence?

M is column-stochastic (each column sums to 1), guaranteeing eigenvalue 1. With the damping factor ensuring primitivity, the power method converges to the unique PageRank vector.

Example: A network security analyst examines the adjacency matrix of a computer network with 5 nodes. The Laplacian has eigenvalues 0, 1, 2, 4, 5. How many connected components does the network have?

The number of connected components equals the multiplicity of eigenvalue 0 in the Laplacian. Since 0 appears exactly once, the network is fully connected (1 component).

Example: A search engine engineer implements the power method to compute PageRank. The second-largest eigenvalue magnitude is 0.85 (the damping factor). How many iterations are needed to reduce the error by a factor of 1000?

Error decays as 0.85^k. For 0.85^k = 0.001: k = log(0.001)/log(0.85) = -3/(-0.0706) = 42.5. About 42 iterations are needed for three decimal places of accuracy.

Example: A streaming service uses collaborative filtering with a user-movie matrix R (1000 users x 5000 movies). To find latent factors, they compute the top 50 eigenvalues of R^T R. What percentage of storage is saved compared to storing R?

Original: 1000 * 5000 = 5,000,000. With rank-50 approximation: 1000*50 + 50 + 50*5000 = 50,000 + 50 + 250,000 = 300,050. Savings: (5M - 300K)/5M = 94%. This is the power of low-rank approximation.
