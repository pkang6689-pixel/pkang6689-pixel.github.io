Lesson 2.1: Solving Systems by Substitution & Elimination - Summary

Key Concepts: Solving Systems by Substitution & Elimination

Overview

This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.

Key Definitions

Solve the system: x + y = 5 and x - y = 1. What is x?

Adding the equations: 2x = 6, so x = 3. Substituting back: 3 + y = 5, so y = 2.

Using substitution, if y = 2x and x + y = 9, what is x?

Substitute y = 2x into x + y = 9: x + 2x = 9, so 3x = 9, thus x = 3.

Solve by elimination: 2x + 3y = 12 and 2x + y = 8. What is y?

Subtract the second from the first: (2x+3y)-(2x+y) = 12-8, so 2y = 4, thus y = 2.

Solve: 3x - y = 7 and x + y = 5. What is y?

Add the equations: 4x = 12, so x = 3. Then from x + y = 5: y = 5 - 3 = 2.

Solve by substitution: x = 3y - 1 and 2x + y = 12. What is y?

Substitute x = 3y - 1 into 2x + y = 12: 2(3y-1) + y = 12, so 7y - 2 = 12, 7y = 14, y = 2.

Solve: 5x + 2y = 20 and 3x + 2y = 12. What is x?

Subtract: (5x+2y)-(3x+2y) = 20-12, so 2x = 8, x = 4.

What is the 'substitution method' for solving systems of equations?

The substitution method involves isolating one variable in one equation and replacing it in the other, reducing the system to one equation in one unknown.

What is the 'elimination method' (also called the addition method)?

The elimination method strategically adds or subtracts equations (after multiplying by appropriate constants) so that one variable cancels out.

What is a 'system of linear equations'?

A system of linear equations is a set of two or more equations, each linear in the same variables, that must all be satisfied simultaneously.

What is a 'free variable' in the context of solving systems?

A free variable is not determined by the system and can be assigned any value. Other variables are then expressed in terms of free variables.

A phone plan charges a monthly fee plus a per-minute rate. 100 minutes costs $35 and 200 minutes costs $45. What is the per-minute rate?

f + 100r = 35 and f + 200r = 45. Subtracting: 100r = 10, so r = $0.10 per minute.

A boat travels 30 km upstream in 3 hours and 30 km downstream in 2 hours. What is the boat's speed in still water?

Upstream: b - c = 10. Downstream: b + c = 15. Adding: 2b = 25, so b = 12.5 km/h.

Real-World Applications

Example: To eliminate x from 3x + 2y = 10 and x + 4y = 12, what should you multiply the second equation by before subtracting?

Multiply the second equation by 3 to get 3x + 12y = 36. Then subtract the first to eliminate x: 10y = 26.
