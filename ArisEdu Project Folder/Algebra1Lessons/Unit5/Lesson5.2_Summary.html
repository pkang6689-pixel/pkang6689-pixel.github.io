Lesson 5.2: Exponential Decay

Key Concepts: Exponential Decay

An exponential decay function has the form f(x) = a · bˣ where a > 0 and 0 < b < 1.

The decay factor b = 1 − r, where r is the decay rate.

The graph approaches the x-axis but never touches it — the x-axis is a horizontal asymptote.

Real-world examples: radioactive decay, depreciation of car value, medicine leaving the body.
