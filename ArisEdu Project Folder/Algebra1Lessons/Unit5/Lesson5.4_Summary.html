Lesson 5.4: Geometric Sequences

Key Concepts: Geometric Sequences

A geometric sequence has a common ratio (r) between consecutive terms.

The explicit formula is: a(n) = a(1) · r^(n−1).

If |r| > 1, the sequence grows; if 0 < |r| < 1, the sequence shrinks.

Example: 3, 6, 12, 24, ... has r = 2 and a(n) = 3 · 2^(n−1).
