Lesson 5.10: Exponential Regression

Key Concepts: Exponential Regression

Exponential regression uses technology to find the best-fit exponential equation y = a · bˣ for a data set.

Use a graphing calculator or software: enter data, run ExpReg to get a and b.

The correlation coefficient r (or r²) indicates how well the model fits. Closer to ±1 (or 1) means a better fit.

Use the regression equation to make predictions, but be cautious with extrapolation.
