Lesson 7.2: Solving by Factoring — Trinomials

Key Concepts: Solving by Factoring — Trinomials

Factor x² + bx + c: find two numbers that multiply to c and add to b.

Example: x² + 5x + 6 = (x + 2)(x + 3) because 2 × 3 = 6 and 2 + 3 = 5.

For ax² + bx + c (a ≠ 1), use the AC method: find two numbers that multiply to ac and add to b, then factor by grouping.

After factoring, set each factor equal to 0 to solve.
