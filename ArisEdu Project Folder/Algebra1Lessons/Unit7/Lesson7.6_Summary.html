Lesson 7.6: The Discriminant

Key Concepts: The Discriminant

The discriminant is the expression under the square root in the quadratic formula: D = b² − 4ac.

If D > 0: two distinct real solutions. If D = 0: one repeated real solution. If D < 0: no real solutions (two complex solutions).

The discriminant tells you how many times the parabola crosses the x-axis.

Example: For x² + 4x + 5 = 0, D = 16 − 20 = −4 < 0 → no real solutions.
