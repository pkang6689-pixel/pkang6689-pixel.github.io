Lesson 7.4: Completing the Square

Key Concepts: Completing the Square

Completing the square rewrites ax² + bx + c in vertex form a(x − h)² + k.

Steps: (1) Move c to the other side. (2) Take half of b, square it, add to both sides. (3) Factor the perfect square. (4) Solve.

Example: x² + 6x = 7 → x² + 6x + 9 = 16 → (x+3)² = 16 → x + 3 = ±4 → x = 1 or x = −7.

If a ≠ 1, divide the entire equation by a first.
