Key Concepts: Residuals & Residual Plots

A residual is the difference between an observed value and the predicted value: residual = observed âˆ’ predicted.

Positive residuals mean the point is above the line of best fit; negative residuals mean it is below.

A residual plot graphs residuals on the y-axis versus x-values on the x-axis.

If the residual plot shows a random scatter with no pattern, the linear model is a good fit. If there is a pattern (curve), a non-linear model may be better.
