Lesson 9.1: Simplifying Rational Expressions

Key Concepts: Simplifying Rational Expressions

A rational expression is a fraction where the numerator and/or denominator are polynomials.

To simplify, factor both the numerator and denominator, then cancel common factors.

The expression is undefined where the denominator equals zero. State excluded values.

Example: (x² − 4)/(x + 2) = (x+2)(x−2)/(x+2) = x − 2, where x ≠ −2.
