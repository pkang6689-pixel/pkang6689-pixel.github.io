Lesson 4.3: Domain & Range

Key Concepts: Domain & Range

The domain is the set of all possible input values (x-values) for a function.

The range is the set of all possible output values (y-values).

Restrictions on domain: cannot divide by zero, cannot take the square root of a negative (in real numbers).

Domain and range can be expressed in set-builder notation {x | condition} or interval notation (a, b).
