Lesson 4.4: Evaluating & Interpreting Functions

Key Concepts: Evaluating & Interpreting Functions

To evaluate a function, substitute the given input value and simplify.

For real-world functions, interpret the input and output in context (what they represent).

Interpreting f(a) = b means: when the input is a, the output is b.

Example: If C(t) = 50t + 200 represents cost over time, C(3) = 350 means the cost at 3 hours is $350.
