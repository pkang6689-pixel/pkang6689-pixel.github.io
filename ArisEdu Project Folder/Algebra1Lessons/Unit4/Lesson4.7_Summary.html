Lesson 4.7: Piecewise & Step Functions

Key Concepts: Piecewise & Step Functions

A piecewise function is defined by different rules for different parts of its domain.

Each piece has its own equation and a specified interval. Use open/closed circles at boundaries to show inclusion.

A step function is a piecewise function where each piece is a constant (horizontal segment). The greatest integer function ⌊x⌋ is a common example.

Real-world examples include tax brackets, shipping rates, and phone plans with tiered pricing.
