Lesson 10.4: Solving Radical Equations

Key Concepts: Solving Radical Equations

To solve a radical equation, isolate the radical, then raise both sides to the appropriate power.

For square roots: square both sides. For cube roots: cube both sides.

Always check for extraneous solutions — squaring can introduce false solutions.

Example: √(x + 3) = 5 → x + 3 = 25 → x = 22. Check: √25 = 5 ✓
