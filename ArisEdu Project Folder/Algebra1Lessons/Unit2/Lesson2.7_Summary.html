Lesson 2.7: Absolute Value Equations & Inequalities

Key Concepts: Absolute Value Equations & Inequalities

The absolute value of a number is its distance from zero: |x| ≥ 0 always.

To solve |x| = a (where a ≥ 0): set up two equations: x = a or x = −a.

To solve |x| < a: write −a < x < a (AND compound inequality).

To solve |x| > a: write x < −a or x > a (OR compound inequality).
