Lesson 2.1: One-Step & Two-Step Equations

Key Concepts: One-Step & Two-Step Equations

A one-step equation requires a single operation to solve. Use inverse operations: addition ↔ subtraction, multiplication ↔ division.

Examples: x + 5 = 12 → x = 7; 3x = 18 → x = 6.

A two-step equation requires two operations. Undo addition/subtraction first, then undo multiplication/division.

Example: 2x + 3 = 11 → 2x = 8 → x = 4. Always check by substituting back.
