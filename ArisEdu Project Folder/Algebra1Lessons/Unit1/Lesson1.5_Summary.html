Lesson 1.5: Box Plots & Dot Plots

Key Concepts: Box Plots & Dot Plots

A box plot (box-and-whisker plot) displays the five-number summary: minimum, Q1, median, Q3, and maximum.

The box shows the IQR (middle 50%), and the whiskers extend to the min and max (excluding outliers).

A dot plot places a dot above a number line for each data value. It shows the shape and individual values clearly.

Outliers in a box plot are typically shown as individual points beyond 1.5 × IQR from Q1 or Q3.
