Lesson 6.1: Key Features of Parabolas

Key Concepts: Key Features of Parabolas

A quadratic function has the form y = ax² + bx + c. Its graph is a parabola.

If a > 0, the parabola opens upward (minimum). If a < 0, it opens downward (maximum).

The vertex is the highest or lowest point. The axis of symmetry is x = −b/(2a).

The y-intercept is c (set x = 0). The x-intercepts (zeros/roots) are where y = 0.
