AP Calculus AB - Unit 4 - Lesson 4.5 - Summary

Key Concepts

Motion Problems apply derivatives and integrals to analyze the motion of objects along a line. Position s(t), velocity v(t) = s'(t), and acceleration a(t) = v'(t) = s''(t) form the derivative chain. Speed is |v(t)|. An object is speeding up when v and a have the same sign, slowing down when they have opposite signs. Total distance traveled is ∫|v(t)|dt, which differs from displacement ∫v(t)dt = s(b) - s(a) when the object changes direction.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Position, velocity, acceleration: s(t) → v(t) = s'(t) → a(t) = v'(t) = s''(t); each is the derivative of the previous
  • Speed vs. velocity: Speed = |v(t)| (always non-negative); velocity v(t) includes direction (sign): positive = moving right/up, negative = moving left/down
  • Speeding up vs. slowing down: Speeding up: v(t) and a(t) have the same sign; slowing down: they have opposite signs
  • Displacement vs. distance: Displacement = ∫ₐᵇ v(t)dt = s(b) - s(a) (net change); distance = ∫ₐᵇ |v(t)|dt (total path length)
  • Change of direction: An object changes direction when v(t) = 0 and v changes sign; these points divide the time interval for distance calculations

Real-World Applications and Examples

This topic has significant real-world importance:

  • Analyzing car acceleration: v(t) = 60(1-e^(-0.5t)) mph models a car from rest; s'(t) gives instantaneous speed and s''(t) shows the acceleration decreasing over time
  • Elevator motion control: position, velocity, and acceleration constraints ensure smooth rides—jerk (a'(t)) must be limited for passenger comfort
  • Projectile motion: h(t) = -16t² + v₀t + h₀ gives height, h'(t) = -32t + v₀ gives velocity, and h''(t) = -32 ft/s² is the constant gravitational acceleration
