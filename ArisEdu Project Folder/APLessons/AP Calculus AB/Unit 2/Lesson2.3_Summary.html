AP Calculus AB - Unit 2 - Lesson 2.3 - Summary

Key Concepts

Power Rule and Basic Derivatives extends differentiation to polynomial and simple algebraic functions. The power rule d/dx[xⁿ] = nxⁿ⁻¹ works for all real exponents, including negative integers (d/dx[x⁻²] = -2x⁻³) and rational exponents (d/dx[√x] = d/dx[x^(1/2)] = (1/2)x^(-1/2)). Combined with constant multiple and sum rules, any polynomial can be differentiated term by term. Additional basic derivatives: d/dx[eˣ] = eˣ, d/dx[ln x] = 1/x.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Power rule for all exponents: d/dx[xⁿ] = nxⁿ⁻¹ for any real n; includes x⁻¹ = 1/x, x^(1/2) = √x, x^(2/3), etc.
  • Negative exponent derivatives: d/dx[x⁻ⁿ] = -nx⁻ⁿ⁻¹; equivalently d/dx[1/xⁿ] = -n/x^(n+1)
  • Fractional exponent derivatives: d/dx[x^(p/q)] = (p/q)x^(p/q - 1); applies to roots: d/dx[∛x] = d/dx[x^(1/3)] = (1/3)x^(-2/3)
  • Exponential function: d/dx[eˣ] = eˣ; the only function that is its own derivative (up to a constant multiple)
  • Natural logarithm: d/dx[ln x] = 1/x for x > 0; extends to d/dx[ln|x|] = 1/x for x ≠ 0

Real-World Applications and Examples

This topic has significant real-world importance:

  • Radioactive decay: if N(t) = N₀e^(-λt), then N'(t) = -λN₀e^(-λt), showing the decay rate is proportional to the current amount—a hallmark of exponential decay
  • Population growth modeling: d/dt[P(t)] where P involves power functions helps ecologists predict changes in wildlife populations
  • Fluid dynamics: the velocity profile in a pipe follows a power law, and its derivative gives the shear rate at any radial position
