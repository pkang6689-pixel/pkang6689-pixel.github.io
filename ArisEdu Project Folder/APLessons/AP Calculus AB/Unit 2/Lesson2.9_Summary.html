AP Calculus AB - Unit 2 - Lesson 2.9 - Summary

Key Concepts

Implicit Differentiation finds dy/dx when y is not explicitly solved as a function of x. For equations like x² + y² = 25, differentiate both sides with respect to x, treating y as a function of x (applying the chain rule: d/dx[y²] = 2y·dy/dx). Then solve algebraically for dy/dx. This technique is essential for curves that fail the vertical line test, conic sections, and relations where solving for y is impractical or impossible.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Implicit function: A relation where y is defined implicitly by an equation F(x,y) = 0 rather than explicitly as y = f(x)
  • Implicit differentiation process: Differentiate both sides with respect to x, apply chain rule to y terms (multiply by dy/dx), then solve for dy/dx
  • d/dx[yⁿ] = nyⁿ⁻¹ · dy/dx: Chain rule applied to powers of y during implicit differentiation
  • Second derivative implicitly: Differentiate dy/dx again with respect to x (implicitly), then substitute the expression for dy/dx to simplify
  • Implicit curves: Equations like x² + y² = r² (circle), x²/a² + y²/b² = 1 (ellipse), xy = c (hyperbola) define implicit curves

Real-World Applications and Examples

This topic has significant real-world importance:

  • Finding the slope of a circle x² + y² = 25 at any point: dy/dx = -x/y, giving the tangent line direction for navigation and engineering applications
  • Economic equilibrium: supply and demand curves defined implicitly by S(p,q) = 0 and D(p,q) = 0; implicit differentiation finds how quantity responds to price changes
  • Planetary orbits: elliptical orbits x²/a² + y²/b² = 1 analyzed with implicit differentiation to find orbital velocity direction at each point
