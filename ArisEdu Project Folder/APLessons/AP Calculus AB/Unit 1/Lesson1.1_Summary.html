AP Calculus AB - Unit 1 - Lesson 1.1 - Summary

Key Concepts

Limits and Continuity are the foundation of calculus. A limit describes the value a function approaches as the input approaches a specific value: lim(x→c) f(x) = L means f(x) gets arbitrarily close to L as x approaches c. A function is continuous at x = c if: (1) f(c) exists, (2) lim(x→c) f(x) exists, and (3) lim(x→c) f(x) = f(c). The Intermediate Value Theorem states that a continuous function on [a,b] takes every value between f(a) and f(b).

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Limit: The value a function approaches as the input approaches a particular value: lim(x→c) f(x) = L; the function need not equal L at x = c
  • One-sided limits: Left-hand limit: lim(x→c⁻) f(x); right-hand limit: lim(x→c⁺) f(x); the two-sided limit exists only if both one-sided limits are equal
  • Continuity: A function f is continuous at c if f(c) is defined, lim(x→c) f(x) exists, and lim(x→c) f(x) = f(c)
  • Intermediate Value Theorem: If f is continuous on [a,b] and k is between f(a) and f(b), then there exists at least one c in (a,b) where f(c) = k
  • Removable discontinuity: A point where the limit exists but either f(c) is undefined or f(c) ≠ lim(x→c) f(x); can be 'fixed' by redefining f(c)

Real-World Applications and Examples

This topic has significant real-world importance:

  • IVT guarantees a root exists: if f(a) < 0 and f(b) > 0 for continuous f, there must be a zero between a and b—used in numerical root-finding algorithms
  • GPS navigation continuously tracking position: mathematically modeled as a continuous function of time, allowing interpolation between data points
  • Temperature modeling: since temperature changes continuously, the IVT guarantees that between any two measured temperatures, every intermediate value was achieved
