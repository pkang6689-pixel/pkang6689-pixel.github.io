AP Calculus AB - Unit 6 - Lesson 6.1 - Summary

Key Concepts

Antiderivatives and Indefinite Integrals introduce integration as the reverse of differentiation. An antiderivative of f(x) is a function F(x) such that F'(x) = f(x). The indefinite integral ∫f(x)dx = F(x) + C, where C is the constant of integration (since derivatives of constants are zero). Basic antiderivative rules: ∫xⁿdx = xⁿ⁺¹/(n+1) + C (n ≠ -1), ∫eˣdx = eˣ + C, ∫(1/x)dx = ln|x| + C, ∫cos x dx = sin x + C, ∫sin x dx = -cos x + C.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Antiderivative: A function F(x) whose derivative is f(x): F'(x) = f(x); the reverse operation of differentiation
  • Indefinite integral: ∫f(x)dx = F(x) + C; represents the family of all antiderivatives, differing by a constant C
  • Constant of integration: The + C in every indefinite integral accounts for the fact that F(x) + C has the same derivative for any constant C
  • Power rule for integration: ∫xⁿdx = xⁿ⁺¹/(n+1) + C for n ≠ -1; reverses the power rule for derivatives
  • Initial value problem: Finding a specific antiderivative by using a condition like F(0) = 3 to determine the value of C

Real-World Applications and Examples

This topic has significant real-world importance:

  • Physics: integrating acceleration a(t) gives velocity v(t) = ∫a(t)dt + v₀, and integrating velocity gives position s(t) = ∫v(t)dt + s₀
  • Population modeling: if the growth rate dP/dt = f(t) is known, integrating gives the population P(t) = ∫f(t)dt + P₀
  • Engineering: integrating a force-displacement curve to find work done W = ∫F(x)dx, or integrating current to find total charge Q = ∫I(t)dt
