AP Calculus AB - Unit 6 - Lesson 6.7 - Summary

Key Concepts

Definite Integral Properties provide algebraic rules for working with integrals. Key properties: ∫ₐᵇ [f ± g]dx = ∫ₐᵇ f dx ± ∫ₐᵇ g dx (linearity); ∫ₐᵇ cf dx = c∫ₐᵇ f dx (constant multiple); ∫ₐᵇ f dx + ∫ᵇᶜ f dx = ∫ₐᶜ f dx (additivity over intervals); ∫ₐᵃ f dx = 0; ∫ₐᵇ f dx = -∫ᵇₐ f dx (reversing limits). If f(x) ≥ 0 on [a,b], then ∫ₐᵇ f dx ≥ 0 (non-negativity).

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Linearity: ∫ₐᵇ [αf(x) + βg(x)]dx = α∫ₐᵇ f(x)dx + β∫ₐᵇ g(x)dx; integrals distribute over addition and factor out constants
  • Interval additivity: ∫ₐᵇ f dx + ∫ᵇᶜ f dx = ∫ₐᶜ f dx; splitting or combining integration intervals
  • Reversing limits: ∫ₐᵇ f dx = -∫ᵇₐ f dx; switching the limits of integration negates the integral
  • Zero-width integral: ∫ₐᵃ f dx = 0; integrating over an interval of width zero gives zero
  • Comparison property: If f(x) ≥ g(x) on [a,b], then ∫ₐᵇ f dx ≥ ∫ₐᵇ g dx

Real-World Applications and Examples

This topic has significant real-world importance:

  • Decomposing complex problems: splitting ∫₀¹⁰ f dx into ∫₀⁵ f dx + ∫₅¹⁰ f dx when the function has different behavior on each interval
  • Physics: work done by a varying force over a complex path can be split into segments using interval additivity
  • Financial modeling: total revenue over a year can be split into quarterly integrals for seasonal analysis
