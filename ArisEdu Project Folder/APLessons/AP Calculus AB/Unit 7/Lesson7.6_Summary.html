AP Calculus AB - Unit 7 - Lesson 7.6 - Summary

Key Concepts

Logistic Differential Equations: Analysis examines the logistic model in greater depth. The logistic equation dP/dt = rP(1-P/K) can be rewritten as dP/dt = (r/K)P(K-P), showing the growth rate depends on the product of P and (K-P). This is zero at P = 0 and P = K (equilibria), and maximum at P = K/2. Phase line analysis reveals stability: P = 0 is unstable (arrows point away), P = K is stable (arrows point toward). Harvesting models modify the equation to dP/dt = rP(1-P/K) - H.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Phase line analysis: A one-dimensional tool showing equilibria and the direction of change; arrows point toward stable equilibria, away from unstable ones
  • Stable equilibrium: P = K: if the population is perturbed slightly above or below K, it returns to K
  • Unstable equilibrium: P = 0: any nonzero population grows away from zero (in the standard logistic model)
  • Maximum growth rate: dP/dt is maximized when P = K/2; found by setting d²P/dt² = 0 or differentiating dP/dt with respect to P
  • Harvesting model: dP/dt = rP(1-P/K) - H; constant harvesting H changes equilibria and can cause extinction if H is too large

Real-World Applications and Examples

This topic has significant real-world importance:

  • Fisheries management: setting harvest rate H below maximum sustainable yield (at P = K/2) ensures the fish population remains stable
  • Epidemiology: phase line analysis of disease spread models determines whether an outbreak will grow or die out based on the basic reproduction number R₀
  • Antibiotic dosing: logistic-type models of bacterial growth with 'harvesting' (antibiotic killing) help determine effective dosing schedules
