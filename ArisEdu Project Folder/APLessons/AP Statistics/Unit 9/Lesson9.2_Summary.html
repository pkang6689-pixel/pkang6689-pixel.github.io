AP Statistics - Unit 9 - Lesson 9.2 - Summary

Key Concepts

Confidence Intervals for Slopes estimate the true population slope β₁ using b₁ ± t*·SE(b₁) with df = n-2. The interval provides a range of plausible values for the rate of change in y per unit change in x. If the interval does not contain 0, we have evidence that x and y are linearly related. Technology output typically provides b₁, SE(b₁), t, and p-value directly, so interpreting computer regression output is an essential skill.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • CI for β₁: b₁ ± t*·SE(b₁) with df = n - 2; a range of plausible values for the true population slope
  • Interpretation of slope CI: We are C% confident that for each one-unit increase in x, the mean response y changes by between [lower, upper] units
  • Degrees of freedom (regression): df = n - 2 for simple linear regression; accounts for estimating both the intercept and slope
  • Zero in the interval: If 0 is in the CI for β₁, we cannot conclude that x and y have a significant linear relationship at that confidence level
  • Reading regression output: Standard computer output provides coefficients (b₀, b₁), standard errors, t-statistics, p-values, and R² for interpretation

Real-World Applications and Examples

This topic has significant real-world importance:

  • A real estate analyst constructing a 95% CI for the slope of price vs. square footage: 'Each additional square foot increases price by $150 to $210'
  • Medical researchers estimating the rate at which blood pressure changes per year of age, with a confidence interval quantifying the uncertainty in the estimate
  • Reading Minitab/R/Excel regression output to identify the slope estimate, standard error, and confidence interval for a research report
