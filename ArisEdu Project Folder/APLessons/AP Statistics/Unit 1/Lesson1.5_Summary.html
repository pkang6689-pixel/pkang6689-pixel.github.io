AP Statistics - Unit 1 - Lesson 1.5 - Summary

Key Concepts

Standardization and Z-Scores transform raw data values into a common scale by expressing each observation as the number of standard deviations it falls above or below the mean: z = (x - μ)/σ. Z-scores allow meaningful comparison of values from different distributions with different units, centers, and spreads. Looking up z-scores in the standard normal table (or using technology) converts them to percentiles and probabilities, enabling precise statements about relative standing.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Z-score: A standardized value calculated as z = (x - μ)/σ indicating how many standard deviations an observation lies from the mean
  • Percentile: The percentage of observations in a distribution that fall at or below a given value
  • Standard normal table (z-table): A table giving cumulative probabilities (areas to the left) for z-scores in the standard normal distribution
  • Standardization: The process of converting raw scores to z-scores, making different distributions comparable on a common scale
  • Relative standing: An observation's position within a distribution, typically expressed as a z-score or percentile rank

Real-World Applications and Examples

This topic has significant real-world importance:

  • Comparing student performance across different exams by converting raw scores to z-scores (e.g., scoring 1.5σ above the mean on both math and English)
  • Medical diagnostics using z-scores to classify lab results as normal or abnormal based on population reference ranges
  • College admissions comparing applicants who took different standardized tests (SAT vs. ACT) by converting to common z-score scale
