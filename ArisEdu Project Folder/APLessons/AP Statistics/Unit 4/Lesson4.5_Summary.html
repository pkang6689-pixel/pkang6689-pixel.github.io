AP Statistics - Unit 4 - Lesson 4.5 - Summary

Key Concepts

Binomial and Geometric Distributions model two important types of count data. The binomial distribution counts the number of successes in a fixed number of independent trials with constant success probability: P(X=k) = C(n,k)p^k(1-p)^(n-k), with mean np and standard deviation √(np(1-p)). The geometric distribution counts the number of trials until the first success: P(X=k) = (1-p)^(k-1)·p, with mean 1/p.

Key Concepts and Vocabulary

Essential concepts in this lesson include:

  • Binomial distribution: Models the count of successes in n independent trials, each with success probability p; requires BINS: Binary outcomes, Independent trials, fixed Number, Same probability
  • Binomial probability formula: P(X = k) = C(n,k)·p^k·(1-p)^(n-k), where C(n,k) = n!/(k!(n-k)!) counts the number of ways to choose k successes
  • Geometric distribution: Models the number of trials until the first success in independent trials with constant probability p; P(X = k) = (1-p)^(k-1)·p
  • Mean of binomial: μ = np; the expected number of successes in n trials
  • Mean of geometric: μ = 1/p; the expected number of trials needed to achieve the first success

Real-World Applications and Examples

This topic has significant real-world importance:

  • Quality control calculating the probability of finding 0, 1, 2, or more defective items in a batch of 50 using the binomial distribution with known defect rate
  • Sales probability: computing the expected number of calls before making a first sale (geometric distribution with p = probability of closing a deal)
  • Genetics: calculating the probability that exactly 3 out of 5 offspring display a recessive phenotype using binomial probability with p = 0.25
