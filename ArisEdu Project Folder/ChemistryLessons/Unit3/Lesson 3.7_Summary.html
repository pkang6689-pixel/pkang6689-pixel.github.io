Lesson 3.7: Element Synthesis Summary

Key Concepts: Element Synthesis

Core Definitions

  • Element Synthesis: The creation of new elements by bombarding target nuclei with high-speed particles.
  • Particle Accelerator: A machine that speeds up charged particles to collide with target atoms.
  • Transuranium Elements: Elements with atomic numbers greater than 92 (uranium); all are synthetic.
  • Nuclear Fusion: Combining two lighter nuclei to form a heavier one; releases enormous energy.
  • Synthetic Elements: Elements not found naturally; created in labs using nuclear reactions.
  • Super-heavy Elements: Elements with very high atomic numbers (e.g., oganesson, element 118); extremely unstable.
