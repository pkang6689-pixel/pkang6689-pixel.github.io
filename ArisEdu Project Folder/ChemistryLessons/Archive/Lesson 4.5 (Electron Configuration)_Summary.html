Lesson 3.7: Electron Configuration Summary

Key Concepts: Electron Configuration

Core Definitions

  • Configuration for Carbon (6): 1s2 2s2 2p2
  • Configuration for Neon (10): 1s2 2s2 2p6
  • Aufbau Principle states: Electrons fill lowest energy levels first
  • Hund's Rule states: Electrons fill orbitals singly before pairing
  • Pauli Exclusion Principle: Orbitals hold max 2 electrons with opposite spins
