Lesson 10.3: Naming Acids

Key Concepts: Naming Acids

Core Definitions

  • Naming Binary Acids: hydro- + element root + -ic acid. Example: HBr → hydrobromic acid.
  • Naming Oxyacids (-ate): Drop "-ate," add "-ic acid." Example: H₂SO₄ (sulfate → sulfuric acid).
  • Naming Oxyacids (-ite): Drop "-ite," add "-ous acid." Example: H₂SO₃ (sulfite → sulfurous acid).
  • Writing Formulas: Identify the anion, determine its charge, balance with H⁺ ions.
