Lesson 2.4: Metric System Summary

Key Concepts: The Metric System

Core Definitions

  • Metric System: A decimal-based system of measurement used in science; based on powers of 10.
  • Base Units: Meter (length), gram (mass), liter (volume), second (time).
  • Kilo- (k): Prefix meaning 1,000 times the base unit (1 km = 1000 m).
  • Centi- (c): Prefix meaning 1/100 of the base unit (1 cm = 0.01 m).
  • Milli- (m): Prefix meaning 1/1000 of the base unit (1 mm = 0.001 m).
  • Micro- (μ): Prefix meaning 1/1,000,000 of the base unit.
