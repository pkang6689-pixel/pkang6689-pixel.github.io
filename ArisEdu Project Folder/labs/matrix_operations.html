Matrix Algebra

A 2×2 matrix can represent a linear transformation of the plane: rotation, scaling, reflection, shearing. The determinant tells how areas scale (det=0 means the transformation collapses space). Matrix multiplication composes transformations. The inverse A⁻¹ undoes A's transformation (exists only when det ≠ 0). The canvas shows a unit square transformed by each matrix.
