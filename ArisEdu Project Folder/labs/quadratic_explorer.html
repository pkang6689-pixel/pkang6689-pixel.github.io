Quadratic Functions

The vertex form y = a(x−h)²+k directly shows the vertex at (h, k). When a > 0 the parabola opens up; a < 0 opens down. |a| controls width — larger |a| means narrower. The discriminant Δ = b²−4ac tells you: Δ > 0 → 2 real roots, Δ = 0 → 1 root (tangent to x-axis), Δ < 0 → no real roots. The focus and directrix give the parabola its reflective property.
