Standing Wave Patterns

A standing wave forms when two waves of equal frequency travel in opposite directions and interfere. Nodes (zero displacement) and antinodes (max displacement) form at fixed positions. For a string (both ends fixed): λₙ = 2L/n, fₙ = n·f₁. For an open pipe: same pattern. For a closed pipe (one end closed): only odd harmonics exist — λₙ = 4L/n, n = 1,3,5…
