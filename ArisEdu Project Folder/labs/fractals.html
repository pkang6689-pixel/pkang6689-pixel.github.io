Fractals

Fractals are infinitely complex patterns that are self-similar across different scales. The Mandelbrot set tests whether the iteration z → z² + c stays bounded — boundary points create the famous shape. The Sierpinski triangle recursively removes central triangles. The Koch snowflake recursively adds triangles to each edge, creating infinite perimeter with finite area. Barnsley's fern uses an iterated function system (IFS).
