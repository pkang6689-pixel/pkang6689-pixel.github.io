Lesson 1.3: Systems of Linear Equations

Key Concepts: Systems of Linear Equations

What is a System? A system of equations consists of two or more equations with the same variables. A solution must satisfy all equations simultaneously.

Graphical Solutions: Graph each line and find intersection point(s). One intersection = unique solution. Parallel lines = no solution. Same line = infinitely many solutions.

Types of Systems: Consistent and independent: exactly one solution. Consistent and dependent: infinitely many solutions. Inconsistent: no solution.

Algebraic Methods: Prepare to learn substitution and elimination methods in the next lessons. Both methods transform the system to isolate variables and find the solution.
