Key Concepts: Review of Functions & Notation

Functions are relations that assign exactly one output to each input. A function f(x) can be evaluated by substituting a value for x and simplifying.

Function Notation: f(x) represents the output of function f when the input is x. This replaces the older y = notation and makes it clear which variable is the input.

Domain and Range: The domain is the set of all possible input values, while the range is the set of all possible output values. Restrictions occur when denominators are zero or when values are under even-root radicals.
