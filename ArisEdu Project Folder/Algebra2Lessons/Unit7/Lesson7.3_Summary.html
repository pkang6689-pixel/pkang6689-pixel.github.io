Key Concepts: Probability Basics & Events

Probability Definition: P(event) = (favorable outcomes)/(total outcomes) for equally likely outcomes. Ranges from 0 (impossible) to 1 (certain). 0% to 100%. Complement rule: P(not A) = 1 - P(A).

Sample Space & Events: Sample space is all possible outcomes. Event is subset of sample space. Experiment can produce random outcomes. Example: die roll has sample space {1,2,3,4,5,6}.

Independent Events: Events A and B are independent if P(A and B) = P(A) 路 P(B). One event doesn't affect the other. Examples: coin flips, die rolls.

Dependent Events: Events A and B are dependent if P(A then B) = P(A) 路 P(B|A) where P(B|A) is conditional probability of B given A occurred. Drawing without replacement creates dependency.
