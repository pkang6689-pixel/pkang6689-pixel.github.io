Lesson 7.5: Normal Distributions

Key Concepts: Normal Distributions

Normal Distribution: Bell curve, symmetric around mean μ. Characterized by mean and standard deviation σ. Z-score: (X - μ)/σ standardizes to N(0,1). Empirical rule: 68% within 1σ, 95% within 2σ, 99.7% within 3σ.

Standard Normal Distribution: Mean = 0, standard deviation = 1. Denoted Z ~ N(0,1). Z-table gives cumulative probability P(Z ≤ z). Any normal random variable can be converted to standard form: Z = (X - μ)/σ.

Finding Probabilities: P(X ≤ a) requires standardizing to Z, then using table. P(a < X < b) = P(X < b) - P(X < a). P(X > c) = 1 - P(X ≤ c).

Inverse Problem: Given probability, find the value. Example: Find c such that P(X < c) = 0.90. Use Z-table in reverse to find z, then solve X = μ + z·σ.
