Lesson 9.3: Vectors & Vector Operations

Key Concepts: Vectors & Vector Operations

Vector Basics: Vector has magnitude and direction: v = ⟨a, b⟩ = ai + bj. Magnitude |v| = √(a² + b²). Unit vector: u = v/|v|. Zero vector 0 = ⟨0, 0⟩. Position vector from origin to point (a, b).

Vector Operations: Addition: ⟨a₁, b₁⟩ + ⟨a₂, b₂⟩ = ⟨a₁+a₂, b₁+b₂⟩ (parallelogram law). Scalar multiplication: c⟨a, b⟩ = ⟨ca, cb⟩. Dot product: u·v = |u||v|cos θ where θ is angle between vectors. Also u·v = a₁a₂ + b₁b₂.

Angle Between Vectors: cos θ = (u·v)/(|u||v|). Orthogonal (perpendicular) vectors: u·v = 0. Parallel vectors: one is scalar multiple of other. Component of u along v: (u·v)/|v|.

Vector Equations & Applications: Line: r(t) = r₀ + tv (parametric form). Projectile motion: r(t) = (vcos α)t i + ((vsin α)t - ½gt²) j. Work: W = F·d. Physics problems model forces and motion as vectors.
