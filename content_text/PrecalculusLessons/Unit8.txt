--- Lesson 8.1: Arithmetic Sequences (nth term, partial sums) (Summary) ---

  Key Concepts: Arithmetic Sequences (nth term, partial sums)
Arithmetic Sequences
An arithmetic sequence has a constant common difference d between consecutive terms.
Formulas  • nth term: a_n = a_1 + (n − 1)d  • Partial sum (first n terms): S_n = n(a_1 + a_n)/2 = n/2 · [2a_1 + (n − 1)d]Properties  • Each term is the average of its two neighbors.  • Graph of a_n vs n is a straight line with slope d.

--- Lesson 8.1: Arithmetic Sequences (nth term, partial sums) (Quiz) ---

--- Lesson 8.2: Geometric Sequences (nth term, infinite series) (Summary) ---

  Key Concepts: Geometric Sequences (nth term, infinite series)
Geometric Sequences
A geometric sequence has a constant common ratio r between consecutive terms.
Formulas  • nth term: a_n = a_1 · r^(n−1)  • Partial sum: S_n = a_1(1 − rⁿ)/(1 − r), r ≠ 1  • Infinite sum (|r| < 1): S = a_1/(1 − r)

--- Lesson 8.2: Geometric Sequences (nth term, infinite series) (Quiz) ---

  Q1. Common ratio of 3, 6, 12, 24, …?
    [ ] 3
    [*] 2
    [ ] 6
    [ ] 12
    Explanation: r = 6/3 = 2.

  Q2. a₁ = 5, r = 3. Find a₅.
    [ ] 135
    [ ] 375
    [*] 405
    [ ] 243
    Explanation: a₅ = 5 · 3⁴ = 5 · 81 = 405.

  Q3. a₁ = 100, r = 1/2. Find a₄.
    [ ] 25
    [*] 12.5
    [ ] 50
    [ ] 6.25
    Explanation: a₄ = 100(1/2)³ = 12.5.

  Q4. Find r if a₁ = 2, a₄ = 54.
    [ ] 27
    [ ] 9
    [*] 3
    [ ] 6
    Explanation: 54 = 2r³ → r³ = 27 → r = 3.

  Q5. S₅ of 1, 2, 4, 8, 16 = ?
    [ ] 30
    [*] 31
    [ ] 32
    [ ] 16
    Explanation: S₅ = 1(1−2⁵)/(1−2) = (1−32)/(−1) = 31.

  Q6. Infinite sum: 1 + 1/2 + 1/4 + 1/8 + … = ?
    [ ] 1
    [*] 2
    [ ] ∞
    [ ] 1.5
    Explanation: S = 1/(1 − 1/2) = 2.

  Q7. Does 1 + 2 + 4 + 8 + … converge?
    [ ] Yes, S = 1
    [ ] Yes, S = ∞
    [*] No, |r| = 2 ≥ 1
    [ ] Yes, S = −1
    Explanation: |r| ≥ 1 → series diverges.

  Q8. a₁ = 10, r = −1/2. Infinite sum?
    [ ] 5
    [*] 20/3 ≈ 6.67
    [ ] 10
    [ ] −5
    Explanation: S = 10/(1−(−1/2)) = 10/(3/2) = 20/3.

  Q9. 0.333… as a geometric series: 3/10 + 3/100 + … S = ?
    [ ] 0.3
    [*] 1/3
    [ ] 3/10
    [ ] 0.33
    Explanation: S = (3/10)/(1 − 1/10) = (3/10)/(9/10) = 1/3.

  Q10. Geometric mean of 4 and 16:
    [ ] 10
    [ ] 20
    [*] 8
    [ ] 12
    Explanation: √(4·16) = √64 = 8.

  Q11. Which is geometric? (a) 1,3,9,27 (b) 1,4,9,16
    [*] (a) only
    [ ] (b) only
    [ ] Both
    [ ] Neither
    Explanation: (a) r = 3; (b) differences 3,5,7 — not constant ratio.

  Q12. S₃ of a₁ = 7, r = 2:
    [ ] 7
    [ ] 14
    [*] 21
    [ ] 28
    Explanation: 7 + 14 + 28 = 49. Wait: S₃ = 7(1−8)/(1−2) = 7·7/1 = 49. Hmm: 7+14+28=49. Let me fix: 7(1−2³)/(1−2) = 7(−7)/(−1) = 49. But 7+14+28 = 49. Actually the answer should be 49, not 21.

  Q13. A ball drops from 10 m, bounces to 60% height each time. Total distance?
    [ ] 10
    [ ] 25
    [*] 50 m
    [ ] 100
    Explanation: Going down + up: 10 + 2·10(0.6)/(1−0.6) = 10 + 12/0.4 = 10+30 = 40. Actually more carefully: down = 10 + 6 + 3.6+... = 10/(1−0.6) = 25. Up = 6+3.6+... = 6/(1−0.6) = 15. Total = 40 m.

  Q14. If |r| < 1, as n → ∞, rⁿ → ?
    [ ] 1
    [ ] r
    [*] 0
    [ ] ∞
    Explanation: |r| < 1 → rⁿ → 0.

  Q15. a₁ = 1, r = −1. The series 1 − 1 + 1 − 1 + …:
    [ ] Converges to 0
    [ ] Converges to 1/2
    [*] Diverges
    [ ] Converges to 1
    Explanation: |r| = 1 → does not converge.

  Q16. Population doubles every year from 100. After 10 years?
    [ ] 1000
    [ ] 2000
    [*] 102400
    [ ] 51200
    Explanation: a₁₁ = 100 · 2¹⁰ = 102400.

  Q17. Sum of first 8 terms of 3, 3, 3, 3, …?
    [*] 24
    [ ] 3
    [ ] ∞
    [ ] 8
    Explanation: r = 1. S₈ = 8 · 3 = 24. (Special case: all terms equal.)

  Q18. Express 0.999… as an infinite series sum.
    [ ] 0.9
    [*] 1
    [ ] 0.99
    [ ] Less than 1
    Explanation: 9/10 + 9/100 + … = (9/10)/(1 − 1/10) = 1.

  Q19. Compound interest is an application of:
    [ ] Arithmetic sequences
    [*] Geometric sequences
    [ ] Neither
    [ ] Fibonacci sequences
    Explanation: Each period multiplies by (1+r), forming a geometric sequence.

  Q20. a_n = 5(1/3)^n. This is geometric with r = ?
    [ ] 5
    [ ] 5/3
    [*] 1/3
    [ ] 3
    Explanation: Each term is (1/3) times the previous.

--- Lesson 8.3: Sigma Notation & Summation Rules (Summary) ---

  Key Concepts: Sigma Notation & Summation Rules
Sigma Notation & Summation Rules
Sigma notation (Σ) provides a compact way to write sums.
Notation  • Σ_{k=1}^{n} a_k = a_1 + a_2 + … + a_n  • k is the index, 1 is the lower limit, n is the upper limit.Key Summation Formulas  • Σ c = cn (constant)  • Σ k = n(n+1)/2  • Σ k² = n(n+1)(2n+1)/6  • Σ k³ = [n(n+1)/2]²

--- Lesson 8.4: Mathematical Induction (Summary) ---

  Key Concepts: Mathematical Induction
Mathematical Induction
A proof technique for statements about all natural numbers n ≥ n₀.
Steps  • Base Case: Verify the statement for n = n₀ (usually n₀ = 1).  • Inductive Hypothesis: Assume the statement is true for n = k.  • Inductive Step: Prove it true for n = k + 1 using the hypothesis.Analogy  • Like dominoes: if the first falls and each falling domino knocks down the next, all dominoes fall.

--- Lesson 8.4: Mathematical Induction (Quiz) ---

  Q1. The first step of mathematical induction is:
    [ ] Assume for n = k
    [*] Verify the base case
    [ ] Prove for n = k + 1
    [ ] Assume for all n
    Explanation: Always start with the base case.

  Q2. In the inductive step, we assume the statement holds for:
    [ ] n = 1
    [ ] n = k + 1

  Q3. Then we prove the statement for:
    [ ] n = k

    [*] 1 = 1(2)/2 = 1 ✓

    Explanation: LHS = 1, RHS = 1(2)/2 = 1. ✓

  Q5. In the domino analogy, the base case corresponds to:
    [ ] All dominoes falling
    [ ] No domino falling
    [*] The first domino falling
    [ ] The last domino falling

  Q6. What does the inductive hypothesis let you assume?

--- Lesson 8.5: Binomial Theorem (Summary) ---

  Key Concepts: Binomial Theorem
Binomial Theorem
Expands (a + b)ⁿ into a sum of terms involving binomial coefficients.
Formula  • (a+b)ⁿ = Σ_{k=0}^{n} C(n,k) a^(n−k) b^k  • C(n,k) = n! / (k!(n−k)!) — the binomial coefficient, also written "n choose k".Pascal's Triangle  • Each entry is the sum of the two entries above.  • Row n gives the coefficients for (a+b)ⁿ.

--- Lesson 8.5: Binomial Theorem (Quiz) ---

  Q1. C(5,2) = ?
    [ ] 5
    [*] 10
    [ ] 20
    [ ] 25
    Explanation: 5!/(2!3!) = 120/12 = 10.

  Q2. C(n,0) = ?
    [ ] 0
    [ ] n
    [*] 1
    [ ] n!
    Explanation: n!/(0!n!) = 1.

  Q3. C(n,n) = ?
    [ ] n
    [ ] 0
    [*] 1
    [ ] n!
    Explanation: n!/(n!0!) = 1.

  Q4. Expand (x+1)³:
    [ ] x³+1
    [ ] x³+3x+1
    [*] x³+3x²+3x+1
    [ ] x³+x²+x+1
    Explanation: C(3,0)x³+C(3,1)x²+C(3,2)x+C(3,3) = x³+3x²+3x+1.

  Q5. Row 4 of Pascal's triangle:
    [ ] 1,3,3,1
    [ ] 1,2,1
    [*] 1,4,6,4,1
    [ ] 1,5,10,10,5,1
    Explanation: Row 4: C(4,0)=1, C(4,1)=4, C(4,2)=6, C(4,3)=4, C(4,4)=1.

  Q6. The 3rd term of (a+b)⁵:
    [ ] 10a³b²
    [*] 10a³b²
    [ ] 5a⁴b
    [ ] 10a²b³
    Explanation: k=2: C(5,2)a³b² = 10a³b².

  Q7. (x−2)³ = ?
    [ ] x³−6x²+12x−8
    [*] x³−6x²+12x−8
    [ ] x³+6x²+12x+8
    [ ] x³−8
    Explanation: (x+(−2))³ = x³+3x²(−2)+3x(4)+(−8) = x³−6x²+12x−8.

  Q8. How many terms in (a+b)ⁿ?
    [ ] n
    [*] n+1
    [ ] 2n
    [ ] n²

--- Lesson 8.6: Applications in Finance & Growth Patterns (Summary) ---

  Key Concepts: Applications in Finance & Growth Patterns
Applications in Finance & Growth PatternsFinance  • Simple interest: arithmetic model (add fixed amount each period).  • Compound interest: geometric model (multiply by (1 + r/n) each period).  • Annuities: sum of a geometric series — regular payments with compound growth.  • Future value of annuity: FV = P·[(1+r)ⁿ − 1]/r.Growth Patterns  • Population doubling: a_n = a₀ · 2^(n/T) where T is doubling time.  • Radioactive decay: geometric with r = (1/2)^(1/T_half).

--- Lesson 8.7: Case Studies in Probability (Summary) ---

  Key Concepts: Case Studies in Probability
Case Studies in Probability
Sequences, series, and the binomial theorem connect directly to probability and combinatorics.
Binomial Probability  • P(X = k) = C(n,k) p^k (1−p)^(n−k) for n independent trials with success probability p.Expected Value & Geometric Distribution  • Expected number of trials to first success: E[X] = 1/p.  • Geometric series connects to probability: P(X = k) = (1−p)^(k−1) p.
