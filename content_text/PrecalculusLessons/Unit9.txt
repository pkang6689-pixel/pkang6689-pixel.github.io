--- Lesson 9.1: Concept of Limits (graphical, numerical approaches) (Summary) ---

  Key Concepts: Concept of Limits (graphical, numerical approaches)
Concept of Limits
The limit of f(x) as x approaches c, written lim_{x→c} f(x) = L, means f(x) gets arbitrarily close to L as x gets close to c.
Approaches  • Graphical: Read the y-value the graph approaches as x → c.  • Numerical: Make a table of x-values approaching c from both sides.  • The limit may exist even if f(c) is undefined (e.g., holes).When Limits Don't Exist  • Left-hand and right-hand limits disagree.  • Function oscillates without settling.  • Function increases/decreases without bound.

--- Lesson 9.2: Evaluating Limits Algebraically (Summary) ---

  Key Concepts: Evaluating Limits Algebraically
Evaluating Limits AlgebraicallyTechniques  • Direct substitution: Try plugging in c first.  • Factoring: If 0/0, factor and cancel common factors.  • Rationalizing: Multiply by conjugate for radical expressions.  • Common factor: Simplify complex fractions.Limit Laws  • Sum: lim(f+g) = lim f + lim g  • Product: lim(f·g) = lim f · lim g  • Quotient: lim(f/g) = lim f / lim g (if lim g ≠ 0)

  Q1. lim_{x→3} (x² + 2x) = ?
    [ ] 9
    [ ] 15
    [*] 15
    [ ] 12
    Explanation: Direct: 9 + 6 = 15.

  Q2. lim_{x→1} (x³ − 1)/(x − 1) = ?
    [ ] 0
    [ ] 1
    [*] 3
    [ ] Undefined

  Q3. lim_{x→4} (x − 4)/(√x − 2) = ?
    [ ] 0
    [ ] 2
    [*] 4
    [ ] Undefined
    Explanation: Rationalize: multiply by (√x+2)/(√x+2). (x−4)/((√x−2)(√x+2)) · ... = (√x+2)(x−4)/(x−4) = √x+2. At x=4: 4.

  Q4. lim_{x→0} (√(x+9) − 3)/x = ?
    [ ] 0
    [ ] 3
    [*] 1/6
    [ ] Undefined

--- Lesson 9.3: One-Sided Limits (Summary) ---

  Key Concepts: One-Sided Limits
One-Sided Limits
One-sided limits consider the behavior of f(x) as x approaches c from one direction only.
Notation  • lim_{x→c⁻} f(x): approach from the left (x < c).  • lim_{x→c⁺} f(x): approach from the right (x > c).Key Principle  • lim_{x→c} f(x) exists ⟺ lim_{x→c⁻} f(x) = lim_{x→c⁺} f(x).

--- Lesson 9.4: Continuity & Types of Discontinuities (Summary) ---

  Key Concepts: Continuity & Types of Discontinuities
Continuity & DiscontinuitiesContinuity at a Point  • f is continuous at c if: (1) f(c) is defined, (2) lim_{x→c} f(x) exists, (3) lim_{x→c} f(x) = f(c).Types of Discontinuities  • Removable (hole): limit exists but f(c) is missing or different.  • Jump: left and right limits both exist but differ.  • Infinite: function approaches ±∞ (vertical asymptote).  • Oscillating: function oscillates without settling (e.g., sin(1/x) at 0).

--- Lesson 9.5: Infinite Limits & Asymptotic Behavior (Summary) ---

  Key Concepts: Infinite Limits & Asymptotic Behavior
Infinite Limits & Asymptotic BehaviorInfinite Limits  • lim_{x→c} f(x) = ∞ means f(x) grows without bound near c.  • Corresponds to a vertical asymptote at x = c.Limits at Infinity  • lim_{x→∞} f(x) = L means f(x) → L as x grows large.  • Corresponds to a horizontal asymptote y = L.End Behavior of Rationals  • degree(num) < degree(denom): HA at y = 0.  • degree(num) = degree(denom): HA at y = ratio of leading coefficients.  • degree(num) > degree(denom): no HA (oblique or grows without bound).

--- Lesson 9.6: Applications: Instantaneous Rate of Change (Summary) ---

  Key Concepts: Applications: Instantaneous Rate of Change
Instantaneous Rate of Change
The instantaneous rate of change of f at x = a is the limit of the average rate of change over shrinking intervals.
Formula  • Instantaneous rate = lim_{h→0} [f(a+h) − f(a)] / h  • This equals the derivative f'(a) — the slope of the tangent line at x = a.Average vs Instantaneous  • Average rate = Δy/Δx = [f(b)−f(a)]/(b−a) (secant line).  • Instantaneous rate = limit of average rate as interval shrinks to zero (tangent line).

--- Lesson 9.7: Piecewise Functions & Limits (Summary) ---

  Key Concepts: Piecewise Functions & Limits
Piecewise Functions & Limits
A piecewise function is defined by different formulas on different intervals.
Evaluating Limits  • At a boundary point c between pieces, compute both one-sided limits.  • lim_{x→c⁻}: use the formula for x < c.  • lim_{x→c⁺}: use the formula for x > c (or x ≥ c).  • Limit exists only if both one-sided limits agree.Making Piecewise Functions Continuous  • Choose parameters so that the pieces 'meet' at boundary points.

--- Lesson 9.8: AP Calculus Prep (Summary) ---

  Key Concepts: AP Calculus Prep
AP Calculus Prep: Limits & Continuity
Limits and continuity form the foundation of AP Calculus AB/BC. Mastery of these precalculus skills is essential.
AP-Essential Skills  • Evaluate limits (direct substitution, factoring, rationalizing, squeeze theorem).  • Analyze one-sided limits and determine when the two-sided limit exists.  • Classify discontinuities (removable, jump, infinite).  • Apply the Intermediate Value Theorem.  • Know that continuity implies integrability and is needed for the Mean Value Theorem.  • Understand lim sin x/x = 1 and lim (1−cos x)/x = 0.

--- Lesson 9.8: AP Calculus Prep (Quiz) ---

  Q1. lim_{x→2} (x²−4)/(x−2) = ?
    [ ] 0
    [ ] 2
    [*] 4
    [ ] Undefined
    Explanation: Factor: (x+2)(x−2)/(x−2) → x+2 → 4.

  Q2. lim_{x→0} sin(5x)/(3x) = ?
    [ ] 0
    [ ] 1
    [ ] 3/5
    [*] 5/3
    Explanation: (5/3)·[sin(5x)/(5x)] → 5/3.

  Q3. lim_{x→∞} (2x²+3)/(5x²−1) = ?
    [ ] ∞
    [ ] 0
    [*] 2/5
    [ ] 5/2
    Explanation: Same degree: 2/5.

  Q4. f(x) = (x−1)/(x²−1) at x=1:
    [ ] Infinite
    [*] Removable (limit = 1/2)
    [ ] Jump
    [ ] Continuous
    Explanation: = 1/((x+1)) for x≠1. Limit = 1/2.

  Q5. lim_{x→0⁺} x ln x = ?
    [ ] −∞
    [ ] ∞
    [*] 0
    [ ] 1
    Explanation: L'Hôpital or analysis: x ln x → 0.

  Q6. lim_{x→0} (1 − cos x)/x² = ?
    [ ] 0
    [ ] 1
    [*] 1/2
    [ ] ∞
    Explanation: Standard limit (often shown via Taylor: cos x ≈ 1 − x²/2).

  Q7. IVT: f(x) = x³ − 4x + 1. f(0) = 1, f(1) = −2. Conclusion?
    [ ] No root
    [*] Root exists in (0, 1)
    [ ] Root at x = 0.5
    [ ] f is not continuous
    Explanation: f continuous, f(0)>0, f(1)<0 → zero in (0,1) by IVT.

  Q8. lim_{x→3} (√(x+1) − 2)/(x − 3) = ?
    [ ] 0
    [ ] 2
    [*] 1/4
    [ ] Undefined
    Explanation: Rationalize: multiply by (√(x+1)+2)/(√(x+1)+2). = (x+1−4)/((x−3)(√(x+1)+2)) = 1/(√(x+1)+2). At 3: 1/4.

  Q9. lim_{x→∞} sin x:
    [ ] 1
    [ ] 0
    [*] Does not exist (oscillates)
    [ ] ∞
    Explanation: sin x oscillates between −1 and 1 forever.

  Q10. Classify: f(x) = tan x at x = π/2:
    [ ] Removable
    [ ] Jump
    [*] Infinite
    [ ] Continuous
    Explanation: tan x → ±∞ at π/2.

  Q11. If lim_{x→c} f(x) = f(c), then f is ___ at c.
    [ ] Differentiable
    [*] Continuous
    [ ] Bounded
    [ ] Integrable
    Explanation: Definition of continuity at a point.

  Q12. The squeeze theorem requires:
    [ ] Only f between g and h
    [*] g ≤ f ≤ h AND lim g = lim h = L
    [ ] f = g = h
    [ ] lim f = lim g
    Explanation: Both bounding functions must have the same limit.

  Q13. lim_{x→4} (x − 4)/(√x − 2) = ?
    [ ] 0
    [ ] 2
    [*] 4
    [ ] Undefined
    Explanation: Rationalize: (x−4)(√x+2)/((√x−2)(√x+2)) = (x−4)(√x+2)/(x−4) = √x+2. At 4: 4.

  Q14. For AP Calc, which limit is fundamental?
    [ ] lim x/sin x = π
    [*] lim sin x/x = 1 as x→0
    [ ] lim sin x = 0 as x→∞
    [ ] lim cos x/x = 1

  Q15. A function continuous on [a,b] must have:
    [ ] A maximum only
    [ ] A minimum only

  Q16. lim_{x→0} (e^x − 1)/x = ?
    [ ] 0
    [*] 1
    [ ] e
    [ ] ∞
    Explanation: Fundamental limit. (e^x = 1 + x + x²/2 + … → (e^x−1)/x → 1.)

    [*] Continuous on [a,b] and differentiable on (a,b)

  Q18. lim_{h→0} [(3+h)² − 9]/h = ?

    [*] Derivatives and integrals are both defined as limits
    [ ] They're easy
