--- Lesson 1.1: Review of Algebra (factoring, radicals, rational expressions) (Summary) ---

  Key Concepts: Review of Algebra (factoring, radicals, rational expressions)
Review of Algebra
A solid algebra foundation is essential for Precalculus.
Factoring  • GCF: Always factor out the greatest common factor first.  • Difference of squares: a² − b² = (a + b)(a − b).  • Perfect square trinomials: a² ± 2ab + b² = (a ± b)².  • Trinomial factoring: ax² + bx + c — find factors of ac that add to b.  • Grouping: For four-term polynomials, group pairs and factor each group.Radicals  • √(ab) = √a · √b; √(a/b) = √a/√b.  • Rationalizing: Multiply by the conjugate to eliminate radicals from denominators.Rational Expressions  • Factor numerator and denominator, cancel common factors.  • Add/subtract: Find LCD, rewrite fractions, combine.  • Multiply/divide: Factor, cancel, and simplify.

--- Lesson 1.1: Review of Algebra (factoring, radicals, rational expressions) (Quiz) ---

  Q1. Factor: x² − 9 =
    [ ] (x − 3)²
    [ ] (x + 9)(x − 1)
    [*] (x + 3)(x − 3)
    [ ] (x − 9)(x + 1)
    Explanation: Difference of squares: x² − 9 = x² − 3² = (x+3)(x−3).

  Q2. Factor: 2x² + 6x =
    [ ] 2x(x + 6)
    [*] 2x(x + 3)
    [ ] x(2x + 6)
    [ ] 2(x² + 3x)
    Explanation: GCF = 2x; 2x² + 6x = 2x(x + 3).

  Q3. Factor: x² + 5x + 6 =
    [ ] (x + 1)(x + 6)
    [*] (x + 2)(x + 3)
    [ ] (x + 5)(x + 1)
    [ ] (x − 2)(x − 3)
    Explanation: Find factors of 6 that add to 5: 2 and 3.

  Q4. Simplify: √50 =
    [ ] 5√10
    [ ] 25√2
    [*] 5√2
    [ ] √50
    Explanation: √50 = √(25·2) = 5√2.

  Q5. Rationalize: 1/√3 =
    [ ] 1/3
    [*] √3/3
    [ ] 3/√3
    [ ] √3
    Explanation: Multiply by √3/√3: 1/√3 = √3/3.

  Q6. Simplify: (x² − 4)/(x + 2) =
    [ ] x + 2
    [ ] x − 4
    [*] x − 2
    [ ] (x−4)/(x+2)
    Explanation: Factor: (x+2)(x−2)/(x+2) = x − 2 (x ≠ −2).

  Q7. Factor: x² − 2x + 1 =
    [ ] (x + 1)²
    [*] (x − 1)²
    [ ] (x − 1)(x + 1)
    [ ] x(x − 2) + 1
    Explanation: Perfect square trinomial: x² − 2x + 1 = (x − 1)².

  Q8. The LCD of 1/(x+1) and 1/(x−1) is:
    [ ] x² + 1
    [ ] x
    [*] (x+1)(x−1)
    [ ] x + 1
    Explanation: LCD is the product of the distinct denominators.

  Q9. Simplify: (3/x) + (2/x) =
    [ ] 6/x²
    [ ] 5/x²
    [*] 5/x
    [ ] 6/x
    Explanation: Same denominator: 3/x + 2/x = 5/x.

  Q10. Factor: 6x² + 11x + 3 =
    [*] (2x + 3)(3x + 1)
    [ ] (6x + 1)(x + 3)
    [ ] (3x + 3)(2x + 1)
    [ ] (2x + 1)(3x + 3)
    Explanation: 6·3 = 18; factors of 18 that add to 11: 9 and 2. Split and group.

  Q11. √a · √b equals:
    [ ] √(a − b)
    [ ] √(a + b)
    [*] √(ab)
    [ ] a√b
    Explanation: The product rule for radicals: √a · √b = √(ab).

  Q12. Factor: x³ + 8 =
    [ ] (x + 2)³
    [*] (x + 2)(x² − 2x + 4)
    [ ] (x − 2)(x² + 2x + 4)
    [ ] (x + 8)(x² − 1)
    Explanation: Sum of cubes: a³ + b³ = (a + b)(a² − ab + b²); 8 = 2³.

  Q13. Factor by grouping: x³ + 2x² + 3x + 6 =
    [ ] (x² + 2)(x + 3)
    [*] (x² + 3)(x + 2)
    [ ] (x + 6)(x² + 1)
    [ ] x(x² + 2x + 3) + 6
    Explanation: Group: x²(x+2) + 3(x+2) = (x²+3)(x+2).

  Q14. Simplify: (x²−1)/(x²+x) =
    [ ] x − 1
    [*] (x−1)/x
    [ ] (x+1)/x
    [ ] 1/(x+1)
    Explanation: (x+1)(x−1) / [x(x+1)] = (x−1)/x.

  Q15. Divide: (2/x) ÷ (4/x²) =
    [ ] 8/x³
    [ ] 1/2
    [*] x/2
    [ ] 2x
    Explanation: Invert and multiply: (2/x) · (x²/4) = 2x²/(4x) = x/2.

  Q16. The conjugate of (3 + √5) is:
    [ ] 3 − 5
    [ ] −3 + √5
    [*] (3 − √5)
    [ ] 3 + √5
    Explanation: The conjugate changes the sign of the radical term.

  Q17. √(a/b) equals:
    [ ] a/√b
    [ ] √a + √b
    [*] √a / √b
    [ ] a/b
    Explanation: The quotient rule for radicals.

  Q18. Factor: 4x² − 25 =
    [ ] (4x − 25)(x + 1)
    [*] (2x + 5)(2x − 5)
    [ ] (2x − 5)²
    [ ] (4x + 5)(x − 5)
    Explanation: Difference of squares: (2x)² − 5² = (2x+5)(2x−5).

  Q19. To add 1/(x+2) + 3/(x−1), the LCD is:
    [ ] x + 2
    [*] (x+2)(x−1)
    [ ] x − 1
    [ ] x² + x − 2
    Explanation: LCD = (x+2)(x−1). Note: x² + x − 2 = (x+2)(x−1) would also be correct as expanded form.

  Q20. A rational expression is undefined when:
    [ ] The numerator is 0
    [*] The denominator is 0
    [ ] Both are 0
    [ ] Neither is 0
    Explanation: Division by zero is undefined.

--- Lesson 1.2: Order of Operations & Exponents (Summary) ---

  Key Concepts: Order of Operations & Exponents
Order of Operations & Exponents
PEMDAS: Parentheses → Exponents → Multiplication/Division (left→right) → Addition/Subtraction (left→right).
Exponent Rules  • am · an = am+n  • am/an = am−n  • (am)n = amn  • a⁰ = 1 (a ≠ 0)  • a−n = 1/an  • (ab)n = anbn  • am/n = ⁿ√(am) = (ⁿ√a)m
These rules are used constantly throughout Precalculus and Calculus.

--- Lesson 1.2: Order of Operations & Exponents (Quiz) ---

  Q1. Evaluate: 2 + 3 × 4 =
    [ ] 20
    [*] 14
    [ ] 24
    [ ] 9
    Explanation: Multiply first: 3×4 = 12, then 2+12 = 14.

  Q2. Simplify: x³ · x⁵ =
    [ ] x¹⁵
    [*] x⁸
    [ ] x²
    [ ] 2x⁸
    Explanation: Product rule: add exponents: 3+5 = 8.

  Q3. Simplify: x⁷/x³ =
    [ ] x²¹
    [ ] x³
    [*] x⁴
    [ ] x/3
    Explanation: Quotient rule: 7−3 = 4.

  Q4. Simplify: (x²)⁴ =
    [ ] x⁶
    [*] x⁸
    [ ] 4x²
    [ ] x²⁴
    Explanation: Power of a power: 2×4 = 8.

  Q5. Evaluate: 5⁰ =
    [ ] 0
    [ ] 5
    [*] 1
    [ ] Undefined
    Explanation: Any nonzero number to the zero power equals 1.

  Q6. Simplify: x⁻³ =
    [ ] −x³
    [ ] x³
    [*] 1/x³
    [ ] −3x
    Explanation: Negative exponent: x⁻³ = 1/x³.

  Q7. Simplify: (2x)³ =
    [ ] 2x³
    [ ] 6x³
    [*] 8x³
    [ ] 2x⁹
    Explanation: (2x)³ = 2³ · x³ = 8x³.

  Q8. Evaluate: 8^(2/3) =
    [*] 4
    [ ] 8
    [ ] 2
    [ ] 16
    Explanation: 8^(2/3) = (∛8)² = 2² = 4.

  Q9. Simplify: 27^(1/3) =
    [ ] 9
    [*] 3
    [ ] 27
    [ ] 1
    Explanation: 27^(1/3) = ∛27 = 3.

  Q10. Evaluate: 3 × (4 + 2)² =
    [ ] 108
    [ ] 54
    [*] 108
    [ ] 36
    Explanation: Parentheses: 4+2=6; exponent: 6²=36; multiply: 3×36=108.

  Q11. Simplify: (x⁴y²)³ =
    [ ] x⁷y⁵
    [*] x¹²y⁶
    [ ] 3x⁴y²
    [ ] x¹²y²
    Explanation: Distribute the exponent: x^(4·3)y^(2·3) = x¹²y⁶.

  Q12. Simplify: x⁻² · x⁵ =
    [ ] x⁻¹⁰
    [*] x³
    [ ] x⁻³
    [ ] x⁷
    Explanation: Add exponents: −2+5 = 3.

  Q13. Evaluate: 16^(3/4) =
    [ ] 4
    [ ] 12
    [*] 8
    [ ] 64
    Explanation: 16^(3/4) = (⁴√16)³ = 2³ = 8.

  Q14. Simplify: (x/y)⁻² =
    [ ] x²/y²
    [ ] −x²/y²
    [*] y²/x²
    [ ] xy²
    Explanation: Negative exponent flips: (y/x)² = y²/x².

  Q15. Evaluate: 2³ + 2² =
    [ ] 2⁵
    [ ] 32
    [*] 12
    [ ] 10
    Explanation: 2³ = 8, 2² = 4, total = 12. Exponents don't add when bases are added.

  Q16. Simplify: (3a²b)² =
    [ ] 6a⁴b²
    [*] 9a⁴b²
    [ ] 3a⁴b²
    [ ] 9a²b²
    Explanation: 3² · a^(2·2) · b² = 9a⁴b².

  Q17. Evaluate: (−2)⁴ =
    [ ] −16
    [*] 16
    [ ] −8
    [ ] 8
    Explanation: Even exponent makes the result positive: (−2)⁴ = 16.

  Q18. Evaluate: −2⁴ =
    [*] −16
    [ ] 16
    [ ] −8
    [ ] 8
    Explanation: Exponent applies only to 2: −(2⁴) = −16.

  Q19. Simplify: (x^(1/2))⁶ =
    [ ] x^(1/12)
    [ ] x⁶
    [*] x³
    [ ] 6√x
    Explanation: (x^(1/2))⁶ = x^(6/2) = x³.

  Q20. 0⁰ is:
    [ ] 0
    [ ] 1
    [*] Typically considered indeterminate (though sometimes defined as 1 by convention)
    [ ] Undefined
    Explanation: 0⁰ is an indeterminate form in most contexts.

--- Lesson 1.3: Relations vs. Functions (Summary) ---

  Key Concepts: Relations vs. Functions
Relations vs. Functions
A relation is any set of ordered pairs (x, y). A function is a special relation where every input (x) has exactly one output (y).
  • Vertical Line Test: If any vertical line crosses the graph more than once, it's NOT a function.  • Mapping diagrams: No input should have more than one arrow pointing to an output.  • Function notation: f(x) means 'the output of function f for input x.'  • One-to-one function: Each output corresponds to exactly one input. Passes the horizontal line test.

--- Lesson 1.3: Relations vs. Functions (Quiz) ---

  Q1. A function is a relation where:
    [ ] Every output has one input
    [*] Every input has exactly one output
    [ ] Inputs and outputs are equal
    [ ] There are no repeated values
    Explanation: The defining property: each x maps to exactly one y.

  Q2. The vertical line test determines if a graph represents:
    [ ] A relation
    [*] A function
    [ ] A one-to-one function
    [ ] An equation
    Explanation: If no vertical line hits the graph more than once, it's a function.

  Q3. Which is NOT a function? {(1,2), (3,4), (1,5)}
    [*] It is not a function (input 1 maps to both 2 and 5)
    [ ] It is a function
    [ ] It depends on the context
    [ ] It's one-to-one
    Explanation: Input 1 has two different outputs, violating the function definition.

  Q4. A circle's graph fails the vertical line test because:
    [ ] It's not a relation
    [*] Vertical lines cross it in two points
    [ ] It has no y-values
    [ ] It passes the test
    Explanation: At most x-values, there are two y-values on a circle.

  Q5. f(x) = 2x + 3. f(4) = ?
    [ ] 7
    [ ] 14
    [*] 11
    [ ] 5
    Explanation: f(4) = 2(4) + 3 = 8 + 3 = 11.

  Q6. One-to-one functions pass:
    [ ] Only the vertical line test
    [ ] Neither test
    [*] Both the vertical and horizontal line tests
    [ ] Only the horizontal line test
    Explanation: One-to-one means each output has one input (horizontal) and each input has one output (vertical).

  Q7. Which is a function? y = x², x = y², y = ±√x
    [ ] x = y²
    [ ] y = ±√x
    [*] y = x²
    [ ] All of them
    Explanation: y = x² gives one y for each x. The others give two y-values for some x.

  Q8. In f(x) = x² − 1, the input is:
    [ ] f
    [ ] x² − 1
    [*] x
    [ ] −1
    Explanation: x is the input variable in function notation.

  Q9. A mapping diagram shows a function if:
    [ ] Every output has multiple inputs
    [*] No input has more than one arrow to different outputs
    [ ] All arrows go to the same output
    [ ] There are equal numbers of inputs and outputs
    Explanation: Each input must map to exactly one output.

  Q10. Every function is a relation:
    [ ] False
    [*] True
    [ ] Only sometimes
    [ ] Only for linear functions
    Explanation: Functions are a subset of relations — every function is a relation, but not vice versa.

  Q11. {(2,3), (4,5), (6,3)} is:
    [ ] Not a function
    [*] A function (every input is unique, even though outputs repeat)
    [ ] One-to-one
    [ ] Not a relation
    Explanation: Repeated outputs are fine; only repeated inputs with different outputs fail.

  Q12. f(x) = 3x − 7. f(0) = ?
    [ ] 3
    [ ] 0
    [*] −7
    [ ] 7
    Explanation: f(0) = 3(0) − 7 = −7.

  Q13. The horizontal line test checks if a function is:
    [ ] A relation
    [ ] Continuous
    [*] One-to-one
    [ ] Increasing
    Explanation: If no horizontal line crosses the graph more than once, the function is one-to-one.

  Q14. y = |x| is:
    [ ] Not a function
    [*] A function (but not one-to-one)
    [ ] One-to-one
    [ ] Not a relation
    Explanation: Each x gives one y, but e.g. f(2) = f(−2) = 2, so not one-to-one.

  Q15. The set of all inputs of a function is the:
    [ ] Range
    [*] Domain
    [ ] Codomain
    [ ] Relation
    Explanation: Domain = set of all valid input values.

  Q16. The set of all outputs of a function is the:
    [ ] Domain
    [ ] Codomain
    [*] Range
    [ ] Relation
    Explanation: Range = set of all actual output values.

  Q17. f(x) = 1/x is a function because:
    [ ] It's defined for all x
    [*] Each input x (≠ 0) gives exactly one output
    [ ] It passes the horizontal line test only
    [ ] It's a polynomial
    Explanation: For every x ≠ 0, there is exactly one value of 1/x.

  Q18. Is y² = x a function of x?
    [ ] Yes
    [*] No — for positive x, there are two y-values (±√x)
    [ ] Only for x > 0
    [ ] Only for x = 0
    Explanation: y² = x gives y = ±√x, which is two outputs for each positive x.

  Q19. f(x) = x³ is:
    [ ] Not a function
    [ ] A function but not one-to-one
    [*] A function AND one-to-one
    [ ] Not a relation
    Explanation: Each input gives one output, and each output comes from one input.

  Q20. A constant function f(x) = 5 is:
    [ ] Not a function
    [ ] One-to-one
    [*] A function (every input gives the same output 5, but not one-to-one)
    [ ] A relation only
    Explanation: It's a valid function (one output per input) but not one-to-one.

--- Lesson 1.4: Domain & Range (interval notation) (Summary) ---

  Key Concepts: Domain & Range (interval notation)
Domain & Range
The domain is the set of all valid inputs (x-values); the range is the set of all outputs (y-values).
Finding the Domain  • Polynomials: Domain = (−∞, ∞).  • Rational functions: Exclude values where the denominator = 0.  • Square roots: The expression under the radical must be ≥ 0.  • Logarithms: The argument must be > 0.Interval Notation  • ( ) = open (endpoint not included); [ ] = closed (included).  • ∞ and −∞ always use parentheses.  • Union (∪) combines disjoint intervals.

--- Lesson 1.4: Domain & Range (interval notation) (Quiz) ---

  Q1. The domain of f(x) = x² is:
    [ ] [0, ∞)
    [*] (−∞, ∞)
    [ ] (0, ∞)
    [ ] [−1, 1]
    Explanation: Polynomials have domain all real numbers.

  Q2. The domain of f(x) = 1/(x−3) is:
    [ ] (−∞, ∞)
    [*] All real numbers except x = 3
    [ ] [3, ∞)
    [ ] (−∞, 3)
    Explanation: The denominator x−3 = 0 when x = 3.

  Q3. The domain of f(x) = √(x−2) is:
    [ ] (−∞, ∞)
    [ ] (2, ∞)
    [*] [2, ∞)
    [ ] [−2, ∞)
    Explanation: x − 2 ≥ 0 → x ≥ 2.

  Q4. In interval notation, [3, 7) means:
    [ ] 3 < x < 7
    [*] 3 ≤ x < 7
    [ ] 3 ≤ x ≤ 7
    [ ] 3 < x ≤ 7
    Explanation: [ means included, ) means excluded.

  Q5. The range of f(x) = x² is:
    [ ] (−∞, ∞)
    [*] [0, ∞)
    [ ] (0, ∞)
    [ ] [−∞, 0]
    Explanation: x² is always ≥ 0; it achieves 0 at x = 0.

  Q6. The domain of f(x) = ln(x) is:
    [ ] (−∞, ∞)
    [ ] [0, ∞)
    [*] (0, ∞)
    [ ] (1, ∞)
    Explanation: The argument of ln must be strictly positive.

  Q7. ∞ always uses:
    [ ] Brackets [ ]
    [*] Parentheses ( )
    [ ] Either
    [ ] Neither
    Explanation: Infinity is not a number, so it always gets parentheses.

  Q8. The domain of f(x) = √(9 − x²) is:
    [ ] (−∞, ∞)
    [ ] (−3, 3)
    [*] [−3, 3]
    [ ] [0, 3]
    Explanation: 9 − x² ≥ 0 → x² ≤ 9 → −3 ≤ x ≤ 3.

  Q9. Union (∪) is used for:
    [ ] Intersecting intervals
    [*] Combining disjoint intervals
    [ ] Multiplying intervals
    [ ] Subtracting intervals
    Explanation: ∪ combines separate pieces of the domain.

  Q10. Domain of f(x) = 1/(x² − 4):
    [ ] All real numbers
    [*] (−∞,−2) ∪ (−2,2) ∪ (2,∞)
    [ ] (−2, 2)
    [ ] x ≠ 0
    Explanation: x²−4 = 0 when x = ±2, so exclude both.

  Q11. The range of f(x) = |x| is:
    [ ] (−∞, ∞)
    [ ] (0, ∞)
    [*] [0, ∞)
    [ ] [−1, 1]
    Explanation: |x| ≥ 0 for all x.

  Q12. The domain of f(x) = x/(x² + 1) is:
    [ ] x ≠ 0
    [ ] x > 0
    [*] (−∞, ∞)
    [ ] [1, ∞)
    Explanation: x² + 1 > 0 for all real x — denominator never zero.

  Q13. The range of f(x) = 1/x is:
    [*] (−∞, 0) ∪ (0, ∞)
    [ ] (−∞, ∞)
    [ ] (0, ∞)
    [ ] [0, ∞)
    Explanation: 1/x can be any real number except 0.

  Q14. f(x) = √x + √(4−x). The domain is:
    [ ] [0, ∞)
    [*] [0, 4]
    [ ] (−∞, 4]
    [ ] [−4, 4]
    Explanation: Need x ≥ 0 AND 4−x ≥ 0, so 0 ≤ x ≤ 4.

  Q15. The range of f(x) = 3 is:
    [ ] {0}
    [*] (−∞, ∞)? No — the range is just {3}
    [ ] [0, 3]
    [ ] (−∞, ∞)
    Explanation: A constant function outputs only one value: {3}.

  Q16. Domain of f(x) = log₂(x − 5):
    [ ] (−∞, ∞)
    [ ] [5, ∞)
    [*] (5, ∞)
    [ ] (0, ∞)
    Explanation: x − 5 > 0 → x > 5.

  Q17. (−∞, 3] ∪ [5, ∞) represents:
    [ ] 3 < x < 5
    [*] x ≤ 3 or x ≥ 5
    [ ] All real numbers
    [ ] x = 3 or x = 5
    Explanation: Union of two intervals: everything ≤ 3 and everything ≥ 5.

  Q18. Domain of f(x) = √(x)/(x − 1):
    [ ] [0, ∞)
    [ ] (0, ∞)
    [*] [0, 1) ∪ (1, ∞)
    [ ] [0, 1]
    Explanation: Need x ≥ 0 for √x and x ≠ 1 for the denominator.

  Q19. The range of f(x) = −x² + 4 is:
    [ ] [4, ∞)
    [ ] [0, ∞)
    [*] (−∞, 4]
    [ ] (−∞, ∞)
    Explanation: The parabola opens down with vertex at (0, 4), so y ≤ 4.

  Q20. To find the range of a function, you can:
    [ ] Only look at the domain
    [*] Analyze the function's behavior, graph it, or solve for x in terms of y
    [ ] Always say (−∞, ∞)
    [ ] Use the vertical line test
    Explanation: Range requires understanding what y-values the function can produce.

--- Lesson 1.5: Function Notation & Evaluation (Summary) ---

  Key Concepts: Function Notation & Evaluation
Function Notation & Evaluation
Function notation f(x) names the function (f) and the input variable (x).
  • Evaluating: Replace x with the given value and simplify.  • f(a + h): Substitute (a + h) for every x in the formula.  • Difference quotient: [f(x+h) − f(x)] / h — fundamental to calculus (measures average rate of change).  • Piecewise evaluation: Determine which piece applies based on the input value.  • Reading function values from graphs: f(a) is the y-value where x = a on the graph.

--- Lesson 1.5: Function Notation & Evaluation (Quiz) ---

  Q1. f(x) = 3x − 2. f(5) = ?
    [ ] 17
    [ ] 8
    [*] 13
    [ ] 5
    Explanation: f(5) = 3(5) − 2 = 15 − 2 = 13.

  Q2. g(x) = x² + 1. g(−3) = ?
    [ ] −8
    [ ] 8
    [*] 10
    [ ] −10
    Explanation: g(−3) = (−3)² + 1 = 9 + 1 = 10.

  Q3. f(x) = 2x. f(a + h) = ?
    [ ] 2a + h
    [*] 2a + 2h (= 2(a+h))
    [ ] 2ah
    [ ] 2a − 2h
    Explanation: Replace x with (a+h): f(a+h) = 2(a+h) = 2a + 2h.

  Q4. For f(x) = x², the difference quotient [f(x+h)−f(x)]/h simplifies to:
    [ ] x + h
    [ ] x² + h
    [*] 2x + h
    [ ] h
    Explanation: f(x+h) = (x+h)² = x²+2xh+h²; subtract x²: 2xh+h²; divide by h: 2x+h.

  Q5. h(x) = √(x+4). h(5) = ?
    [ ] √5
    [*] 3

--- Lesson 1.8: Inverse Functions & Composition (Summary) ---

  Key Concepts: Inverse Functions & Composition
Inverse Functions & CompositionComposition
(f ∘ g)(x) = f(g(x)): Apply g first, then f to the result.
  • The domain of f ∘ g is restricted to x-values where g(x) is in the domain of f.Inverse Functions
f⁻¹ undoes f: f(f⁻¹(x)) = x and f⁻¹(f(x)) = x.
  • Finding f⁻¹: Replace f(x) with y, swap x and y, solve for y.  • Existence: Only one-to-one functions have inverses.  • Graphically: The graph of f⁻¹ is the reflection of f over the line y = x.  • Domain of f⁻¹ = Range of f; Range of f⁻¹ = Domain of f.

    [*] There is a discontinuity (gap or jump)
    [ ] The function is undefined
    [ ] Both values apply
    Explanation: A jump between pieces creates a discontinuity.

  Q15. f(x) = {√x if x ≥ 0; x+1 if x < 0}. f(4) = ?
    [ ] 5
    [ ] 3
    [*] 2
    [ ] 4
    Explanation: 4 ≥ 0, so use √x: √4 = 2.

  Q16. f(x) = {√x if x ≥ 0; x+1 if x < 0}. f(−3) = ?
    [ ] √3
    [ ] 4
    [*] −2
    [ ] −3
    Explanation: −3 < 0, so use x+1: −3+1 = −2.

  Q17. When graphing, the boundary point is included in:
    [ ] Both pieces

    [*] The piece whose condition includes it (≤ or ≥)
    [ ] Always the left piece
    Explanation: Check which condition uses ≤ or ≥ vs. < or >.

  Q18. A piecewise-linear function consists of:
    [ ] Curves connected at points
    [*] Straight-line segments connected at boundary points
    [ ] Only one line
    [ ] No boundaries
    Explanation: Each piece is a linear function, creating a chain of line segments.

  Q19. f(x) = {1/x if x ≠ 0; 0 if x = 0}. This is:
    [ ] Continuous
    [*] Piecewise-defined; not continuous at x = 0 since 1/x → ±∞ near 0
    [ ] Always 0
    [ ] Always 1/x
    Explanation: The limit of 1/x as x→0 doesn't exist, so defining f(0)=0 doesn't create continuity.

  Q20. Piecewise functions appear in:

--- Lesson 1.10: Applications in Modeling (Summary) ---

  Key Concepts: Applications in Modeling
Applications in Modeling
Functions model real-world situations by relating inputs to outputs.
  • Linear models: y = mx + b — constant rate of change. Example: distance = rate × time.  • Quadratic models: Parabolic paths. Example: projectile motion h(t) = −16t² + v₀t + h₀.  • Exponential models: Compound growth/decay. Example: population growth P(t) = P₀ert.  • Piecewise models: Different rules for different conditions (tax brackets, tiered pricing).  • Steps for modeling:  • Identify variables and relationships.  • Choose an appropriate function type.  • Write the equation.  • Solve and interpret in context.

--- Lesson 1.10: Applications in Modeling (Quiz) ---

  Q1. A car travels at 60 mph. Distance as a function of time:
    [ ] d = 60 + t
    [*] d = 60t
    [ ] d = t/60
    [ ] d = 60/t
    Explanation: Distance = rate × time = 60t.

  Q2. Projectile height: h(t) = −16t² + 48t + 5. The initial height is:
    [ ] 48
    [ ] −16
    [*] 5
    [ ] 0
    Explanation: At t = 0: h(0) = 5 (the constant term).

  Q3. Projectile height: h(t) = −16t² + 48t + 5. When does it hit the ground?
    [ ] When h = 5
    [*] When h(t) = 0 — solve −16t² + 48t + 5 = 0
    [ ] When t = 0
    [ ] When t = 48
    Explanation: The projectile hits the ground when h(t) = 0.

  Q4. Population doubles every 3 years. This is modeled by:
    [ ] A linear function
    [*] An exponential function
    [ ] A quadratic function
    [ ] A logarithmic function
    Explanation: Doubling = multiplicative growth = exponential.

  Q5. A bacteria colony starts at 100 and triples each hour. N(t) = ?
    [*] 100 · 3^t
    [ ] 100 + 3t
    [ ] 300t
    [ ] 100t³
    Explanation: Exponential: N(t) = initial × growth^t = 100·3^t.

  Q6. A pool has 10,000 gallons and drains at 500 gal/hr. Model:
    [ ] V = 10000 + 500t
    [*] V = 10000 − 500t
    [ ] V = 500t
    [ ] V = 10000/t
    Explanation: Linear decrease: V(t) = 10000 − 500t.

  Q7. The vertex of h(t) = −16t² + 48t + 5 gives:
    [ ] The initial height
    [ ] The time to hit the ground
    [*] The maximum height and the time it occurs
    [ ] The velocity
    Explanation: The vertex of a downward parabola gives the maximum.

  Q8. Which model is best for compound interest?
    [ ] Linear
    [ ] Quadratic
    [*] Exponential
    [ ] Piecewise
    Explanation: Compound interest grows multiplicatively: A = P(1+r/n)^(nt).

  Q9. A parking lot charges $5 for the first hour, $3 for each additional hour. This is:
    [ ] Linear
    [ ] Quadratic
    [ ] Exponential
    [*] Piecewise
    Explanation: Different rates for different time intervals = piecewise.

  Q10. Area of a square as a function of side length s:
    [ ] A = 4s
    [*] A = s²
    [ ] A = 2s
    [ ] A = s + 4
    Explanation: Area = side² — a quadratic function.

  Q11. Revenue R = price × quantity. If price = 50 − 2q, then R(q) = ?
    [*] R = (50−2q)q = 50q − 2q²
    [ ] R = 50q
    [ ] R = 50/q
    [ ] R = 2q²
    Explanation: R = pq = (50−2q)q = 50q − 2q² (quadratic model).

  Q12. To maximize revenue in R = 50q − 2q², find:
    [ ] R(0)
    [ ] R(50)
    [*] The vertex of the parabola (q = −b/(2a) = 50/4 = 12.5)
    [ ] The y-intercept
    Explanation: Vertex gives the maximum of a downward parabola.

  Q13. Radioactive half-life models use:
    [ ] Linear decay
    [ ] Quadratic decay
    [*] Exponential decay: N(t) = N₀(1/2)^(t/T)
    [ ] Logarithmic growth
    Explanation: Half-life is exponential: the amount halves every T time units.

  Q14. Modeling tip: always define your:
    [*] Variables (what x and y represent, including units)
    [ ] Favorite function
    [ ] Graph first
    [ ] Answer only
    Explanation: Clear variable definitions are essential for meaningful models.

  Q15. A phone depreciates 20% per year. After t years, value = ?
    [ ] 1000 − 200t
    [*] 1000(0.8)^t
    [ ] 1000(1.2)^t
    [ ] 1000/t
    Explanation: Declining by 20% means retaining 80% each year — exponential decay.

  Q16. Which function type has a constant rate of change?
    [ ] Exponential
    [ ] Quadratic
    [*] Linear
    [ ] Logarithmic
    Explanation: Linear functions have constant slope.

  Q17. The break-even point is where:
    [ ] Revenue = 0
    [ ] Cost = 0
    [*] Revenue = Cost
    [ ] Profit is maximized
    Explanation: Break-even: no profit, no loss (R = C).

  Q18. A function model should be checked by:
    [ ] Using only one data point
    [ ] Never checking
    [*] Comparing predictions to actual data and assessing reasonableness
    [ ] Using aesthetic criteria
    Explanation: Models should be validated against real observations.

  Q19. Choosing between models depends on:
    [ ] Personal preference
    [*] The pattern in the data (constant change, multiplicative change, parabolic shape, etc.)
    [ ] The number of variables
    [ ] Nothing — one model fits all
    Explanation: Match the function type to the data's behavior.

  Q20. In context, always interpret:
    [ ] Raw numbers only
    [*] What the answer means in the real-world scenario (units, meaning)
    [ ] Nothing — just compute
    [ ] Only the formula
    Explanation: Context makes mathematics useful.
