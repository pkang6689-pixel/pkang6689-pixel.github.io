--- Lesson 4.1: Rational Expressions & Simplification (Summary) ---

  Key Concepts: Rational Expressions & Simplification
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the LCD of 1/(x + 1) and 1/(x − 1)?
The denominators are distinct linear factors, so the LCD is (x + 1)(x − 1).
What is a rational expression?
A rational expression has the form P(x)/Q(x) where P and Q are polynomials and Q is not the zero polynomial.
What is the least common denominator (LCD) of two rational expressions?
The LCD is the least common multiple of the denominators, used to combine rational expressions with unlike denominators.
What is a complex fraction?
A complex fraction has fractions within fractions, such as (1/x + 1)/(1/x − 1).
What is the reciprocal of a rational expression a/b?
The reciprocal of a fraction is obtained by swapping the numerator and denominator; a/b becomes b/a.
What is the difference-of-cubes factoring formula?
The difference of cubes factors as a³ − b³ = (a − b)(a² + ab + b²), useful for simplifying expressions like (x³ − 8)/(x − 2).
A factory's average cost per unit is C̄(x) = (5000 + 8x)/x dollars, where x is the number of units produced. What is the average cost per unit when 250 units are produced?
C̄(250) = (5000 + 8 · 250)/250 = (5000 + 2000)/250 = 7000/250 = $28 per unit.
A biologist models the concentration of a nutrient in a solution as C(t) = 50t/(t² + 25) mg/L, where t is hours after mixing. What is the concentration at t = 5 hours?
C(5) = 50(5)/(25 + 25) = 250/50 = 5 mg/L.
A car's fuel efficiency is modeled by E(v) = 800v/(v² + 100) miles per gallon, where v is speed in mph. What is E(10)?
E(10) = 800(10)/(100 + 100) = 8000/200 = 40 mpg.
Real-World Applications
Example: Multiply: (x/3) · (6/x²).
6x/(3x²) = 2/x, x ≠ 0.
Example: A factory's average cost per unit is C̄(x) = (5000 + 8x)/x dollars, where x is the number of units produced. What is the average cost per unit when 250 units are produced?
C̄(250) = (5000 + 8 · 250)/250 = (5000 + 2000)/250 = 7000/250 = $28 per unit.
Example: A school fundraiser sells t-shirts with cost function C(n) = 200 + 5n dollars. The average cost per shirt is C̄(n) = (200 + 5n)/n. For what number of shirts does the average cost equal $10?
(200 + 5n)/n = 10 → 200 + 5n = 10n → 200 = 5n → n = 40 shirts.

    [ ] x − 3

--- Lesson 4.2: Vertical, Horizontal, and Oblique Asymptotes (Summary) ---

  Key Concepts: Vertical, Horizontal, and Oblique Asymptotes
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a vertical asymptote?
A vertical asymptote is a vertical line x = a where the function's value grows without bound (approaches ±∞).
What is a horizontal asymptote?
A horizontal asymptote is a horizontal line y = b that describes the end behavior of the function as x → ±∞.
What is an oblique (slant) asymptote?
An oblique asymptote is a slanted line y = mx + b that the function approaches when the degree of the numerator exceeds the degree of the denominator by exactly 1.
What is the relationship between leading coefficients and the HA when degrees are equal?
When the numerator and denominator have the same degree, HA = (leading coeff. of numerator)/(leading coeff. of denominator).
An engineer models voltage across a component as V(t) = (5t² + 3t)/(t² + 1). What is the horizontal asymptote, and what does it represent physically?
Same degree (2): HA = 5/1 = 5. The voltage stabilizes at 5 volts over time.
A physicist models the speed of a falling object with air resistance as v(t) = (50t)/(t + 5) m/s. What is the terminal velocity (the speed approached over a long time)?
As t → ∞, v(t) = 50t/(t + 5) → 50. The HA is y = 50, so the terminal velocity is 50 m/s.
A rocket's altitude model is h(t) = (t³ + 2t)/(t² + 1) km. As t grows large, the graph approaches what kind of asymptote, and what is it?
Degree 3 over degree 2: difference is 1, so there is an oblique asymptote. Long division: (t³ + 2t) ÷ (t² + 1) = t + remainder t/(t² + 1). As t → ∞ the remainder → 0, so OA is y = t.
Real-World Applications
Example: A factory's average cost per unit is C̄(x) = (3x + 1200)/x dollars. As production increases without bound, what value does the average cost approach?
As x → ∞, C̄(x) = 3 + 1200/x → 3. The HA is y = 3, meaning the minimum average cost approaches $3 per unit.
Example: An engineer models voltage across a component as V(t) = (5t² + 3t)/(t² + 1). What is the horizontal asymptote, and what does it represent physically?
Same degree (2): HA = 5/1 = 5. The voltage stabilizes at 5 volts over time.

  Q1. Find the vertical asymptote of f(x) = 1/(x − 3).
    [*] x = 3
    [ ] y = 3
    [ ] x = 0
    [ ] x = −3

  Q2. Find the horizontal asymptote of f(x) = 5/(x + 2).
    [ ] y = 2
    [*] y = 0
    [ ] No HA
    [ ] y = 5
    Explanation: Degree of numerator (0) < degree of denominator (1), so the HA is y = 0.

  Q3. Find the horizontal asymptote of f(x) = (3x + 1)/(2x − 5).
    [ ] y = −1/5
    [ ] y = 0
    [ ] y = 1
    [*] y = 3/2
    Explanation: Degrees are equal (both 1), so HA = leading coefficient ratio = 3/2.

  Q4. Does f(x) = (x² + 1)/(x − 1) have a horizontal asymptote?
    [ ] Yes, y = 1
    [ ] Yes, y = 0
    [ ] Yes, y = −1
    [*] No horizontal asymptote
    Explanation: Degree of numerator (2) > degree of denominator (1), so there is no horizontal asymptote.

  Q5. Find the oblique asymptote of f(x) = (x² + 1)/(x − 1).
    [ ] y = x + 2
    [ ] y = x − 1
    [*] y = x + 1
    [ ] y = x
    Explanation: Long division: x² + 1 ÷ (x − 1) = x + 1 with remainder 2. The oblique asymptote is y = x + 1.

    [ ] x = 0

    [ ] x = 0

--- Lesson 4.3: Holes and Discontinuities (Summary) ---

  Key Concepts: Holes and Discontinuities
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
For f(x) = (x² − 1)/(x − 1), what is the y-coordinate of the hole?
Simplified form: x + 1. At x = 1: 1 + 1 = 2. Hole at (1, 2).
For the function above, what is the vertical asymptote?
After canceling: (x + 2)/(x + 1). VA at x = −1.
What is the y-value of the hole at x = 3 for f(x) = (x − 3)/(x² − 9)?
Simplified form: 1/(x + 3). At x = 3: 1/(3 + 3) = 1/6.
f(x) = (x³ − 8)/(x − 2). Hole at x = 2. What is the y-value?
Simplified: x² + 2x + 4 (difference of cubes). At x = 2: 4 + 4 + 4 = 12.
What is a removable discontinuity?
A removable discontinuity (hole) occurs where the function is undefined but a finite limit exists; the discontinuity can be 'removed' by defining the function at that point.
What is a non-removable discontinuity?
A non-removable discontinuity cannot be fixed by redefining a single point because the limit either does not exist or is infinite.
What is a jump discontinuity?
Jump discontinuities arise when the left and right limits differ, which is typical of piecewise functions.
An economist models marginal revenue as MR(q) = (q² − 9)/(q − 3) dollars per unit. After simplification, what is the marginal revenue at q = 3 units (the location of the hole)?
(q − 3)(q + 3)/(q − 3) = q + 3. At q = 3: 3 + 3 = 6. The hole is at (3, 6), so the limit value is $6.
A pollution model gives concentration C(d) = (d − 5)/(d² − 25) ppm at distance d km. Is the discontinuity at d = 5 a hole or a VA, and what is its y-value?
(d − 5)/((d − 5)(d + 5)) = 1/(d + 5). Hole at d = 5; y = 1/(5 + 5) = 1/10.
A shipping cost model is S(w) = (3w − 6)/(w − 2) dollars per pound. After simplification, what is the graph of this function?
3(w − 2)/(w − 2) = 3, w ≠ 2. The graph is the horizontal line y = 3 with a hole at (2, 3).
A scientist models bacteria growth as B(t) = (t³ − 27)/(t − 3) colonies per mL. What is the y-value at the hole?
Difference of cubes: t³ − 27 = (t − 3)(t² + 3t + 9). Cancel (t − 3): t² + 3t + 9. At t = 3: 9 + 9 + 9 = 27.

--- Lesson 4.4: Graphing Rational Functions (Summary) ---

  Key Concepts: Graphing Rational Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
For f(x) = (2x)/(x − 4), what is f(0)?
f(0) = 2(0)/(0 − 4) = 0/(−4) = 0.
What is the first step when graphing a rational function?
Always factor first to identify holes, vertical asymptotes, and x-intercepts before proceeding.
What is a sign chart used for when graphing rational functions?
A sign chart tests a point in each interval between critical values to determine where the function is above or below the x-axis.
What is the parent function of most basic rational functions?
The reciprocal function f(x) = 1/x is the parent function, featuring a VA at x = 0 and HA at y = 0.
What is the domain of a rational function?
The domain excludes any x-value that makes the denominator zero, as division by zero is undefined.
A scientist studies light intensity I(d) = 100/d² lumens at distance d meters. Where is the function positive, and what is the graph's behavior near d = 0?
100/d² > 0 for all d ≠ 0 (squared denominator always positive). As d → 0, d² → 0⁺, so I → +∞.
An environmental model gives pollutant decay as C(t) = 1/(t − 1) + 2 ppm. What is the horizontal asymptote, and what does it represent?
As t → ∞, 1/(t − 1) → 0, so C(t) → 2. The HA is y = 2, meaning the pollutant settles to a baseline of 2 ppm.
Real-World Applications
Example: An engineer graphs the voltage function V(x) = (3x − 6)/(x − 2). After simplification, describe the graph.
3(x − 2)/(x − 2) = 3, x ≠ 2. The graph is the horizontal line y = 3 with an open circle at (2, 3).

  Key Concepts: Partial Fraction Decomposition
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is partial fraction decomposition?
Partial fraction decomposition reverses the process of combining fractions, expressing a complex fraction as a sum of simpler ones.
What is a 'proper' rational fraction?
A proper fraction has deg(numerator) < deg(denominator), which is required for partial fraction decomposition.
What is an 'irreducible quadratic' factor?
An irreducible quadratic (over the reals) has b² − 4ac < 0, so it cannot be factored into real linear factors, like x² + 1.
A physics student needs to decompose (2s)/(s² − 4) for a Laplace problem. What is the result?
2s/((s − 2)(s + 2)) = A/(s − 2) + B/(s + 2). s = 2: 4/(4) = A → A = 1. s = −2: −4/(−4) = B → B = 1. Result: 1/(s − 2) + 1/(s + 2).
An economics model involves x²/(x − 1). Before decomposing, what must be done, and what is the result of that step?
deg(num) = 2 ≥ deg(den) = 1, so divide first. x² ÷ (x − 1) = x + 1 with remainder 1. Result: x + 1 + 1/(x − 1).
An engineer encounters (s + 1)/(s²(s + 2)) in a circuit analysis. What is the correct partial fraction form?
s² is a repeated linear factor (s repeated twice), requiring A/s + B/s². The distinct factor (s + 2) adds C/(s + 2). Total: 3 terms.

--- Lesson 4.7: Rational Inequalities (Summary) ---

  Key Concepts: Rational Inequalities
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a rational inequality?
A rational inequality compares a rational expression to a value using <, >, ≤, or ≥, where the variable appears in the denominator.
What is a 'sign chart' (or sign analysis)?
A sign chart places critical numbers on a number line and uses test points in each interval to determine whether the expression is positive or negative.
What is a 'test point' in the context of solving rational inequalities?
A test point is any x-value within an interval (not at a critical number) that is substituted into the expression to check if the expression is positive or negative on that interval.
What is the difference between a strict inequality (< or >) and a non-strict inequality (≤ or ≥)?
Strict (< or >) excludes points where the expression equals zero. Non-strict (≤ or ≥) includes numerator zeros where the expression equals zero.
Real-World Applications
Example: Why can you NOT simply cross-multiply in a rational inequality?
If the denominator product is negative for some x-values, multiplying both sides flips the inequality, making cross-multiplication unreliable.
Example: A factory's average cost C̄(x) = (2000 + 5x)/x must not exceed $15 per unit. How many units must be produced?
(2000 + 5x)/x ≤ 15 → 2000/x + 5 ≤ 15 → 2000/x ≤ 10 → x ≥ 200. At least 200 units.

--- Lesson 4.8: Case Studies in Engineering (Summary) ---

  Key Concepts: Case Studies in Engineering
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the formula for total resistance of two resistors in parallel?
For parallel resistors, reciprocals add: 1/R_T = 1/R₁ + 1/R₂, which simplifies to R_T = R₁R₂/(R₁ + R₂).
What is Ohm's law?
Ohm's law states V = IR: voltage equals current times resistance, a fundamental electrical relationship.
What is a 'safety factor' in engineering?
A safety factor is the ratio of strength to stress; it must be greater than 1 to ensure the material can handle the load with a margin of safety.
What is the 'inverse-square law'?
The inverse-square law states that intensity is inversely proportional to the square of the distance, applying to light, gravity, and electromagnetic radiation.
What is a 'transfer function' in control engineering?
A transfer function is a rational expression relating a system's output to its input in the frequency (Laplace) domain.
What is 'efficiency' in engineering?
Efficiency η = output/input × 100%. A 100% efficient machine converts all input to useful output (impossible in practice due to losses).
What is a 'gear ratio'?
A gear ratio describes how the input speed is transformed to output speed; a 3:1 ratio means the output turns at 1/3 the input speed but with 3× the torque.
What is 'beam deflection'?
Beam deflection measures how far a beam bends under load. It depends on force (F), length (L), elasticity (E), and moment of inertia (I).
What is 'resonant frequency' in an LC circuit?
Resonant frequency f = 1/(2π√(LC)) is where the impedance of the inductor and capacitor cancel, allowing maximum current flow.
An electrical engineer connects three parallel resistors: 4 Ω, 6 Ω, and 12 Ω. What is the total resistance?
1/R_T = 1/4 + 1/6 + 1/12 = 3/12 + 2/12 + 1/12 = 6/12 = 1/2. R_T = 2 Ω.
A sound engineer knows that sound intensity follows I = k/d². At 2 meters, the intensity is 100 W/m². What is the intensity at 5 meters?
k = 100 × 4 = 400. At d = 5: I = 400/25 = 16 W/m².
A control system has transfer function H(s) = (s + 2)/(s² + 3s + 2). After factoring and simplifying, what is H(s)?
s² + 3s + 2 = (s + 1)(s + 2). Cancel (s + 2): H(s) = 1/(s + 1), a first-order system with a canceled pole-zero pair.
An LC circuit has L = 0.1 H and C = 0.01 F. What is the resonant frequency f = 1/(2π√(LC))?
LC = 0.1 × 0.01 = 0.001. √(0.001) ≈ 0.03162. f = 1/(2π × 0.03162) ≈ 1/0.1987 ≈ 5.03 Hz.
A motor has efficiency η = 0.85. If the input power is 200 W, what is the useful output power?
Output = η × input = 0.85 × 200 = 170 W.
A gear system has a 4:1 ratio. If the input shaft spins at 1200 RPM, what is the output shaft speed and how does torque change?
4:1 ratio: output speed = 1200/4 = 300 RPM. By conservation of power, torque is multiplied by 4.
Real-World Applications
Example: Ohm's law: V = IR. If resistance R doubles while voltage V stays constant, current I:
I = V/R. Doubling R halves I.
Example: Power in a circuit: P = V²/R. If voltage V doubles while R stays constant, P:
P is proportional to V². Doubling V: P_new = (2V)²/R = 4V²/R = 4P.
Example: Wind load on a building is proportional to v². If wind speed increases by 50% (from v to 1.5v), by what percentage does the wind load increase?
New load ∝ (1.5v)² = 2.25v². Increase = 2.25 − 1 = 1.25 = 125%.

    [ ] The total energy stored in the circuit
    [ ] The resistance of the circuit
    Explanation: Resonant frequency f = 1/(2π√(LC)) is where the impedance of the inductor and capacitor cancel, allowing maximum current flow.

  Q21. An electrical engineer connects three parallel resistors: 4 Ω, 6 Ω, and 12 Ω. What is the total resistance?
    [ ] 7.33 Ω
    [ ] 4 Ω
    [*] 2 Ω
    [ ] 22 Ω
    Explanation: 1/R_T = 1/4 + 1/6 + 1/12 = 3/12 + 2/12 + 1/12 = 6/12 = 1/2. R_T = 2 Ω.

  Q22. A structural engineer uses d = FL³/(48EI) for a simply supported beam. If the beam length L is doubled while F, E, and I remain constant, by what factor does deflection increase?
    [ ] 4 times
    [*] 8 times
    [ ] 2 times
    [ ] 16 times
    Explanation: Deflection is proportional to L³. Doubling L: (2L)³ = 8L³, so deflection increases by a factor of 8.

  Q23. A sound engineer knows that sound intensity follows I = k/d². At 2 meters, the intensity is 100 W/m². What is the intensity at 5 meters?
    [ ] 4 W/m²
    [ ] 40 W/m²
    [ ] 25 W/m²
    [*] 16 W/m²
    Explanation: k = 100 × 4 = 400. At d = 5: I = 400/25 = 16 W/m².

  Q24. A control system has transfer function H(s) = (s + 2)/(s² + 3s + 2). After factoring and simplifying, what is H(s)?
    [ ] (s + 2)/(s + 1)
    [ ] s + 1
    [ ] 1/(s + 2)
    [*] 1/(s + 1)
    Explanation: s² + 3s + 2 = (s + 1)(s + 2). Cancel (s + 2): H(s) = 1/(s + 1), a first-order system with a canceled pole-zero pair.

  Q25. Wind load on a building is proportional to v². If wind speed increases by 50% (from v to 1.5v), by what percentage does the wind load increase?
    [ ] 50%
    [*] 125%
    [ ] 100%
    [ ] 150%
    Explanation: New load ∝ (1.5v)² = 2.25v². Increase = 2.25 − 1 = 1.25 = 125%.

  Q26. An LC circuit has L = 0.1 H and C = 0.01 F. What is the resonant frequency f = 1/(2π√(LC))?
    [*] Approximately 5.03 Hz
    [ ] Approximately 50.3 Hz
    [ ] Approximately 0.503 Hz
    [ ] Approximately 100 Hz
    Explanation: LC = 0.1 × 0.01 = 0.001. √(0.001) ≈ 0.03162. f = 1/(2π × 0.03162) ≈ 1/0.1987 ≈ 5.03 Hz.

  Q27. A lever has the property force × distance = constant. If a mechanic moves the effort arm from 0.5 m to 1.5 m, the required force becomes:
    [*] One-third as large
    [ ] Three times as large
    [ ] The same
    [ ] Nine times as large
    Explanation: F × d = k. Tripling d (from 0.5 to 1.5): F_new = k/(3d) = F/3. The force is reduced to one-third.

  Q28. A motor has efficiency η = 0.85. If the input power is 200 W, what is the useful output power?
    [ ] 235 W
    [ ] 200 W
    [ ] 115 W
    [*] 170 W
    Explanation: Output = η × input = 0.85 × 200 = 170 W.

  Q29. A gear system has a 4:1 ratio. If the input shaft spins at 1200 RPM, what is the output shaft speed and how does torque change?
    [ ] 4800 RPM, torque divided by 4
    [ ] 300 RPM, torque unchanged
    [ ] 1200 RPM, torque unchanged
    [*] 300 RPM, torque multiplied by 4
    Explanation: 4:1 ratio: output speed = 1200/4 = 300 RPM. By conservation of power, torque is multiplied by 4.

  Q30. A civil engineer needs a safety factor of at least 2.5. If the working stress is 120 MPa, what minimum material strength is required?
    [ ] 240 MPa
    [ ] 48 MPa
    [*] 300 MPa
    [ ] 360 MPa
    Explanation: SF = strength/stress. Need SF ≥ 2.5: strength ≥ 2.5 × 120 = 300 MPa.
