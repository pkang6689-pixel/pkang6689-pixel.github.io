--- Lesson 4.1: Rational Expressions & Simplification (Summary) ---

  Key Concepts: Rational Expressions & Simplification
Rational Expressions & Simplification
A rational expression is a ratio of two polynomials: P(x)/Q(x), where Q(x) ≠ 0.
Simplifying  • Factor both the numerator and denominator completely.  • Cancel common factors (noting excluded values where Q = 0).Operations  • Multiply: Factor, cancel, then multiply remaining numerators and denominators.  • Divide: Multiply by the reciprocal of the divisor.  • Add/Subtract: Find the LCD, rewrite each fraction, then combine numerators.Complex Fractions  • A fraction within a fraction. Simplify by multiplying numerator and denominator by the LCD of all mini-fractions.

    [ ] 6/x⁵

--- Lesson 4.2: Vertical, Horizontal, and Oblique Asymptotes (Summary) ---

  Key Concepts: Vertical, Horizontal, and Oblique Asymptotes
Vertical, Horizontal, and Oblique Asymptotes
Asymptotes are lines that a graph approaches but typically never reaches.
Vertical Asymptotes (VA)  • Occur where the denominator = 0 and the numerator ≠ 0 (after simplification).  • The graph goes to ±∞ near a VA.Horizontal Asymptotes (HA)  • Compare degrees of numerator (n) and denominator (m):  • n < m → HA is y = 0.  • n = m → HA is y = (leading coeff of num)/(leading coeff of den).  • n > m → no horizontal asymptote.Oblique (Slant) Asymptotes  • Occur when n = m + 1 (degree of numerator is exactly one more than denominator).  • Perform polynomial long division; the quotient (ignoring remainder) is the oblique asymptote.

--- Lesson 4.3: Holes and Discontinuities (Summary) ---

  Key Concepts: Holes and Discontinuities
Holes and Discontinuities
When both numerator and denominator share a common factor, canceling it creates a hole (removable discontinuity) rather than a vertical asymptote.
Identifying Holes  • Factor numerator and denominator.  • A common factor (x − c) that cancels creates a hole at x = c.  • The y-coordinate of the hole is found by substituting c into the simplified expression.Types of Discontinuities  • Removable (hole): Factor cancels; the limit exists but the function is undefined there.  • Non-removable (VA): The function → ±∞; the limit does not exist as a finite number.  • Jump: Occurs in piecewise functions where left and right limits differ.

--- Lesson 4.4: Graphing Rational Functions (Summary) ---

  Key Concepts: Graphing Rational Functions
Graphing Rational Functions
Combine asymptotes, intercepts, holes, and sign analysis to sketch rational function graphs.
Steps  • Factor numerator and denominator; identify holes.  • Find vertical asymptotes (remaining denominator zeros).  • Find the horizontal or oblique asymptote.  • Find x-intercepts (numerator = 0) and y-intercept (x = 0).  • Use a sign chart to determine where f is positive or negative.  • Plot key points and sketch.Additional Tips  • Check symmetry: f(−x) = f(x) → even symmetry, f(−x) = −f(x) → odd symmetry.  • The graph approaches but typically doesn't cross a VA; it may cross a HA.

--- Lesson 4.5: Partial Fraction Decomposition (Summary) ---

  Key Concepts: Partial Fraction Decomposition
Partial Fraction Decomposition
Partial fraction decomposition reverses the process of combining fractions: it breaks a single rational expression into a sum of simpler fractions.
When to Use  • The degree of the numerator must be less than the degree of the denominator. (If not, perform long division first.)Cases  • Distinct linear factors: A/(x − a) + B/(x − b) + …  • Repeated linear factor: (x − a)² → A/(x − a) + B/(x − a)².  • Irreducible quadratic factor: (ax² + bx + c) → (Ax + B)/(ax² + bx + c).Solving for Coefficients  • Multiply both sides by the LCD, then equate coefficients or substitute convenient x-values.

--- Lesson 4.6: Applications in Real-World Ratios (rates, proportions) (Summary) ---

  Key Concepts: Applications in Real-World Ratios (rates, proportions)
Applications in Real-World Ratios
Rational functions model situations involving rates, concentrations, averages, and work problems.
Rate Problems  • Average speed = total distance / total time. If parts of a trip have different speeds, the overall average is a rational expression.Mixture / Concentration  • C(x) = (initial amount + added amount) / (initial volume + added volume).Work Problems  • If A takes a hours and B takes b hours to complete a job alone, together: 1/a + 1/b = 1/t → t = ab/(a+b).Average Cost  • C̄(x) = C(x)/x. As production increases, fixed costs are spread over more units, reducing average cost (up to a point).

--- Lesson 4.7: Rational Inequalities (Summary) ---

  Key Concepts: Rational Inequalities
Rational Inequalities
Solving inequalities involving rational expressions requires sign analysis rather than simple algebraic manipulation.
Steps  • Move all terms to one side so that 0 is on the other: f(x)/g(x) ≤ 0 (or >, ≥, <).  • Factor numerator and denominator; find critical numbers (zeros of numerator and denominator).  • Create a sign chart on a number line using the critical numbers.  • Test a value in each interval to determine the sign of the expression.  • Select the intervals that satisfy the inequality.  • Include endpoints where the expression = 0 (for ≤ or ≥) but exclude where denominator = 0.

--- Lesson 4.7: Rational Inequalities (Quiz) ---

  Q1. Solve (x − 1)/(x + 2) > 0.
    [ ] −2 < x < 1
    [*] x < −2 or x > 1
    [ ] x > 1 only
    [ ] x > −2
    Explanation: Critical: x = 1, −2. Positive when both same sign: x < −2 or x > 1.

  Q2. Solve (x + 3)/(x − 4) ≤ 0.
    [ ] x < −3 or x > 4
    [*] −3 ≤ x < 4
    [ ] −3 < x < 4
    [ ] x ≤ −3
    Explanation: Zero at x = −3 (include), undefined at x = 4 (exclude). Negative on [−3, 4).

  Q3. Solve 1/x > 0.
    [ ] All x
    [ ] x > 1
    [*] x > 0
    [ ] x < 0
    Explanation: 1/x > 0 when x > 0.

  Q4. Solve 1/x ≤ 0.
    [*] x < 0
    [ ] x ≤ 0
    [ ] x < 0 or x = 0
    [ ] No solution
    Explanation: 1/x < 0 when x < 0. 1/x = 0 never, so just x < 0.

  Q5. Solve (x − 2)/((x + 1)(x − 3)) ≥ 0.
    [ ] All x ≥ 2
    [*] −1 < x ≤ 2 or x > 3
    [ ] x ≤ 2 or x ≥ 3
    [ ] x = 2 only
    Explanation: Zeros: 2 (num), −1, 3 (den). Sign chart: positive on (−1, 2] ∪ (3, ∞). Include 2 (≥), exclude −1, 3.

  Q6. Solve x/(x − 5) < 1.
    [ ] x > 5
    [ ] x < 0 or x > 5
    [*] x < 0 or 0 < x < 5 is wrong. Let me reanalyze.
    [ ] x < 5
    Explanation: Rewrite: x/(x−5) − 1 < 0 → (x − x + 5)/(x−5) < 0 → 5/(x−5) < 0 → x−5 < 0 → x < 5. But x ≠ 5.

  Q7. Solve (x² − 4)/(x + 5) > 0.
    [*] x < −5 or −2 < x < 2 is wrong. Factor: (x−2)(x+2)/(x+5) > 0.
    [ ] x > 2
    [ ] −2 < x < 2
    [ ] x > 2 or x < −5
    Explanation: Critical: −5, −2, 2. Test intervals: (−∞,−5)→neg, (−5,−2)→pos, (−2,2)→neg, (2,∞)→pos. Answer: −5 < x < −2 or x > 2.

  Q8. For ≤ inequalities, include x where:
    [ ] Denominator = 0
    [*] Numerator = 0
    [ ] Both = 0
    [ ] Neither = 0
    Explanation: Where numerator = 0, the expression equals 0, satisfying ≤.

  Q9. Solve (x − 1)² / (x + 3) ≥ 0.
    [ ] x ≥ 1
    [*] x = 1 or x > −3, x ≠ −3... actually x > −3
    [ ] All x
    [ ] x > −3 only
    Explanation: (x−1)² ≥ 0 always. Sign depends on (x+3). ≥ 0 when x+3 > 0 or (x−1)²=0. So x > −3 or x = 1. Since x = 1 is in (−3, ∞), answer: x > −3, x ≠ −3. i.e., x ∈ (−3, ∞).

  Q10. Solve 2/(x − 1) ≥ 1.
    [*] 1 < x ≤ 3
    [ ] x ≥ 1
    [ ] x > 1
    [ ] x ≤ 3
    Explanation: 2/(x−1) − 1 ≥ 0 → (2 − x + 1)/(x−1) ≥ 0 → (3−x)/(x−1) ≥ 0. Critical: 1, 3. Positive on (1, 3]. Include 3, exclude 1.

    [ ] x ≤ −2 or x > −1

  Q13. Solve (x − 4)/(x + 2) > 0 and express in interval notation.
    [ ] (−2, 4)

    [ ] [−2, 4]

    Explanation: Both positive: x > 4. Both negative: x < −2. Answer: (−∞, −2) ∪ (4, ∞).

  Q14. What type of circle on a number line for a strict inequality at a critical number?
    [ ] Closed
    [*] Open
    [ ] No circle needed

  Q15. Non-strict inequality (≤ or ≥) at a numerator zero gets:
    [ ] Open circle
