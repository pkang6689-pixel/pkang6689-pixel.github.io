--- Lesson 5.1: Discrete Random Variables (Summary) ---

  Key Concepts: Discrete Random Variables
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a random variable?
A random variable assigns a numerical value to each outcome of a random experiment.
What is a probability distribution?
A probability distribution pairs every possible value of a random variable with its probability.
What is the shortcut formula for variance?
The shortcut formula is Var(X) = E(X^2) - [E(X)]^2, which is often easier to compute than the definition.
What is the 'mode' of a discrete probability distribution?
The mode is the most probable value -- the one with the largest P(X = x).
What is a 'probability histogram'?
A probability histogram uses bars to display P(X = x) for each possible value of a discrete random variable.
A bakery sells cupcakes in boxes. Let X be the number of broken cupcakes per box. The distribution is: P(0) = 0.60, P(1) = 0.25, P(2) = 0.10, P(3) = 0.05. What is the expected number of broken cupcakes per box?
E(X) = 0(0.60) + 1(0.25) + 2(0.10) + 3(0.05) = 0 + 0.25 + 0.20 + 0.15 = 0.60.
A game costs $5 to play. You roll a fair die: if you roll a 6, you win $20; otherwise you win nothing. What is the expected profit per game?
E(profit) = (1/6)(20) + (5/6)(0) - 5 = 3.33 - 5 = -$1.67. On average you lose about $1.67 per game.
A call center tracks X, the number of calls in a 5-minute window. If E(X) = 3.2 and Var(X) = 2.56, what is the standard deviation?
Standard deviation = sqrt(Var(X)) = sqrt(2.56) = 1.6 calls.
An insurance company pays $10,000 for a claim. The probability of a claim is 0.02. If each policy costs $300, what is the company's expected profit per policy?
Expected payout = 0.02 x $10,000 = $200. Expected profit = $300 - $200 = $100 per policy.
A teacher records the number of absences per class: P(0) = 0.4, P(1) = 0.3, P(2) = 0.2, P(3) = 0.1. If the teacher has 5 classes per day, what is the expected total number of absences per day?
E(X per class) = 0(0.4) + 1(0.3) + 2(0.2) + 3(0.1) = 1.0. For 5 classes: 5 x 1.0 = 5.0 absences per day.
A shipping company knows that X, the number of damaged packages per shipment, has E(X) = 1.2 and E(X^2) = 2.64. What is Var(X)?
Var(X) = E(X^2) - [E(X)]^2 = 2.64 - (1.2)^2 = 2.64 - 1.44 = 1.20.
A raffle sells 200 tickets at $2 each. One ticket wins a $100 prize, and two tickets each win a $25 prize. What is the expected net gain for a single ticket buyer?
E(winnings) = (1/200)(100) + (2/200)(25) + (197/200)(0) = 0.50 + 0.25 + 0 = 0.75. Net gain = 0.75 - 2.00 = -$1.25.
A factory worker earns a base pay of $15/hour plus $3 per widget produced. If the expected number of widgets per hour is 8, what is the expected hourly earnings?
Earnings = 15 + 3X. E(Earnings) = 15 + 3(8) = 15 + 24 = $39 per hour, using E(aX + b) = aE(X) + b.
A small store tracks the number of returns per day. The distribution is: P(0) = 0.50, P(1) = 0.30, P(2) = 0.15, P(3) = 0.05. Is this a valid distribution, and what is the expected number of returns?
Sum = 0.50 + 0.30 + 0.15 + 0.05 = 1.00, so it is valid. E(X) = 0(0.50) + 1(0.30) + 2(0.15) + 3(0.05) = 0.75.
A hospital monitors the number of emergency room visits per hour overnight. E(X) = 2.5. Each visit costs the hospital an average of $800 in resources. In a 10-hour overnight shift, what is the expected total cost?
Expected visits in 10 hours = 10 x 2.5 = 25. Expected cost = 25 x $800 = $20,000.
Real-World Applications
Example: A bakery sells cupcakes in boxes. Let X be the number of broken cupcakes per box. The distribution is: P(0) = 0.60, P(1) = 0.25, P(2) = 0.10, P(3) = 0.05. What is the expected number of broken cupcakes per box?
E(X) = 0(0.60) + 1(0.25) + 2(0.10) + 3(0.05) = 0 + 0.25 + 0.20 + 0.15 = 0.60.
Example: A shipping company knows that X, the number of damaged packages per shipment, has E(X) = 1.2 and E(X^2) = 2.64. What is Var(X)?
Var(X) = E(X^2) - [E(X)]^2 = 2.64 - (1.2)^2 = 2.64 - 1.44 = 1.20.
Example: A hospital monitors the number of emergency room visits per hour overnight. E(X) = 2.5. Each visit costs the hospital an average of $800 in resources. In a 10-hour overnight shift, what is the expected total cost?
Expected visits in 10 hours = 10 x 2.5 = 25. Expected cost = 25 x $800 = $20,000.

--- Lesson 5.2: Continuous Random Variables (Summary) ---

  Key Concepts: Continuous Random Variables
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a probability density function (PDF)?
A PDF describes the relative likelihood of a continuous random variable, and probabilities are computed as areas under the curve.
What is the cumulative distribution function (CDF)?
The CDF gives the cumulative probability up to and including x, expressed as F(x) = P(X <= x).
What is the 'median' of a continuous distribution?
The median of a continuous distribution is the value m where F(m) = 0.5, splitting the area under the curve in half.
What is the exponential distribution used to model?
The exponential distribution commonly models waiting times between events, such as time between customer arrivals.
What is the relationship between a PDF and a CDF?
The CDF F(x) is obtained by integrating the PDF f(x), and conversely, the PDF is the derivative of the CDF: f(x) = F'(x).
What is the key difference between discrete and continuous distributions?
Discrete distributions compute probability by summing individual values; continuous distributions compute probability by integrating the density.
A bus arrives at a stop uniformly between 8:00 and 8:20 AM. What is the probability that a passenger who arrives at 8:12 waits no more than 3 minutes?
The bus must arrive between 8:12 and 8:15 (3 minutes). P = 3/20 = 0.15 since the bus arrival is uniform on a 20-minute window.
The weight of cereal boxes is uniformly distributed between 15.5 oz and 16.5 oz. What is the expected weight and standard deviation?
E(X) = (15.5 + 16.5)/2 = 16.0. Var = (16.5 - 15.5)^2 / 12 = 1/12. SD = sqrt(1/12) = 0.289 oz.
A company's delivery time follows a continuous distribution with CDF F(x). If F(2) = 0.3 and F(5) = 0.85, what is P(2 < X <= 5)?
P(2 < X <= 5) = F(5) - F(2) = 0.85 - 0.30 = 0.55.
A machine part has a lifetime that is exponentially distributed with a mean of 500 hours. What is the probability the part lasts more than 500 hours?
For an exponential distribution, P(X > mean) = e^(-1) = 1/e = 0.368. The exponential distribution is right-skewed, so the median is less than the mean.
The temperature in a greenhouse is uniformly distributed between 68 and 76 degrees F. What is the probability the temperature is between 70 and 74 degrees?
P = (74 - 70) / (76 - 68) = 4/8 = 0.50.
A continuous random variable X has PDF f(x) = 2x for 0 <= x <= 1 (and 0 otherwise). What is P(X > 0.5)?
P(X > 0.5) = integral from 0.5 to 1 of 2x dx = [x^2] from 0.5 to 1 = 1 - 0.25 = 0.75.
A random number generator produces values uniformly on [0, 1]. If you generate 3 independent values, what is the probability all three are less than 0.5?
P(each < 0.5) = 0.5. Since they are independent: P(all three < 0.5) = 0.5^3 = 0.125.
The response time for a website is a continuous random variable with a mean of 2.0 seconds and CDF F(x). If F(3) = 0.90, what is the probability a user experiences a response time longer than 3 seconds?
P(X > 3) = 1 - F(3) = 1 - 0.90 = 0.10. About 10% of users will wait longer than 3 seconds.
Real-World Applications
Example: Height is an example of a:
Height can take any value in a range, making it a continuous random variable.
Example: A power company estimates that daily electricity demand (in megawatts) follows a continuous distribution with a mean of 450 MW and P(X > 500) = 0.10. If the company's maximum capacity is 500 MW, what percentage of days will demand exceed capacity?
P(X > 500) = 0.10, so demand exceeds capacity on about 10% of days.
Example: A bakery's oven temperature is uniformly distributed between 345 and 355 degrees F. The ideal baking range is 348 to 352 degrees. What percentage of the time is the oven within the ideal range?
P = (352 - 348) / (355 - 345) = 4/10 = 0.40, which is 40%.

--- Lesson 5.3: Binomial Distribution (Summary) ---

  Key Concepts: Binomial Distribution
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the normal approximation to the binomial?
When both np and n(1-p) are at least 10, the binomial distribution is well-approximated by a normal distribution.
What is the 'complement rule' as applied to binomial problems?
The complement rule states that P(at least one) = 1 - P(none), which simplifies many binomial calculations.
What is the meaning of the combination C(n, k) in the binomial formula?
C(n,k) counts the number of different ways to arrange k successes among n trials, which is n!/(k!(n-k)!).
What is the variance formula for a binomial distribution?
The variance of a binomial distribution is np(1 - p), which accounts for both the number of trials and the probability of success and failure.
A multiple-choice test has 20 questions, each with 4 options. A student guesses randomly on every question. What is the expected number of correct answers?
n = 20, p = 1/4 = 0.25. E(X) = np = 20 x 0.25 = 5 correct answers expected.
A basketball player has a free-throw accuracy of 80%. She shoots 10 free throws in a game. What is the probability she makes all 10?
P(X = 10) = (0.80)^10 = 0.1074. There is about a 10.7% chance she makes all 10.
A quality inspector checks 15 items from a production line where the defect rate is 5%. What is the probability that none of the 15 items are defective?
P(X = 0) = (1 - 0.05)^15 = (0.95)^15 = 0.4633. There is about a 46.3% chance of zero defectives.
In a survey, 60% of people prefer brand A. If 8 people are randomly selected, what is the probability that exactly 5 prefer brand A?
P(X = 5) = C(8,5)(0.6)^5(0.4)^3 = 56 x 0.07776 x 0.064 = 56 x 0.004977 = 0.2787.
A fair coin is flipped 6 times. What is the probability of getting exactly 3 heads?
P(X = 3) = C(6,3)(0.5)^3(0.5)^3 = 20 x (0.5)^6 = 20 x 0.015625 = 0.3125.
An airline knows that 4% of passengers do not show up for their flight. For a flight with 200 booked passengers, what is the expected number of no-shows and the standard deviation?
Mean = np = 200 x 0.04 = 8. Var = 200 x 0.04 x 0.96 = 7.68. SD = sqrt(7.68) = 2.77.
A seed company advertises a 90% germination rate. A gardener plants 20 seeds. What is the probability that at least 19 seeds germinate?
P(X >= 19) = P(19) + P(20). P(20) = (0.9)^20 = 0.1216. P(19) = C(20,19)(0.9)^19(0.1)^1 = 20 x 0.1351 x 0.1 = 0.2702. Total = 0.1216 + 0.2702 = 0.392.
A telemarketer has a 15% success rate per call. She makes 25 calls in an afternoon. What is the probability she makes at least one sale?
P(X >= 1) = 1 - P(X = 0) = 1 - (0.85)^25 = 1 - 0.0134 = 0.9866. There is a 98.7% chance of at least one sale.
Real-World Applications
Example: A multiple-choice test has 20 questions, each with 4 options. A student guesses randomly on every question. What is the expected number of correct answers?
n = 20, p = 1/4 = 0.25. E(X) = np = 20 x 0.25 = 5 correct answers expected.
Important Concepts
What does 'skewed right' mean for a binomial distribution with small p
When p is small, successes are rare, so most outcomes cluster near 0 and the distribution has a right tail.

--- Lesson 5.4: Normal Distribution (Summary) ---

  Key Concepts: Normal Distribution
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the Empirical Rule (68-95-99.7 Rule)?
The Empirical Rule provides approximate percentages for data within 1, 2, and 3 standard deviations of the mean in a normal distribution.
What is a z-score?
A z-score standardizes a value by expressing how many standard deviations it lies above or below the mean.
What is a 'z-table'?
A z-table provides the area under the standard normal curve to the left of any given z-score.
What is the 'inverse normal' or 'finding a value from a percentile'?
The inverse normal process works backward from a probability to find the corresponding value or z-score.
The time to complete an exam is normally distributed with mu = 50 minutes and sigma = 8 minutes. What is the probability a student finishes between 42 and 58 minutes?
42 = 50 - 8 and 58 = 50 + 8, so this range is mu +/- 1 sigma. By the Empirical Rule, about 68%.
Real-World Applications
Example: If mu = 100, sigma = 15, the z-score for X = 130 is:
Z = (130 - 100)/15 = 30/15 = 2.
Example: What is a z-score?
A z-score standardizes a value by expressing how many standard deviations it lies above or below the mean.
Example: What does a negative z-score indicate?
A negative z-score means the value lies below the mean by that many standard deviations.
Example: IQ scores are normally distributed with mu = 100 and sigma = 15. What percentage of people have an IQ above 130?
Z = (130 - 100)/15 = 2.0. P(Z > 2) = 0.0228, which is about 2.3%.
Example: A machine fills bottles with a mean of 500 mL and a standard deviation of 5 mL. Bottles with less than 490 mL are rejected. What percentage of bottles are rejected?
Z = (490 - 500)/5 = -2.0. P(Z < -2) = 0.0228, so about 2.3% of bottles are rejected.
Example: A student scores in the 84th percentile on a standardized test. Approximately what z-score does this correspond to?
P(Z < 1.0) = 0.8413, which is approximately the 84th percentile.
Example: SAT scores are normally distributed with mu = 1060 and sigma = 195. What score separates the top 10% of students?
The top 10% starts at the 90th percentile. Z = 1.28. X = 1060 + 1.28(195) = 1060 + 249.6 = about 1310.
Example: A tire company guarantees its tires for 40,000 miles. Tire lifespan is normally distributed with mu = 45,000 miles and sigma = 3,000 miles. What percentage of tires will fail before the guarantee expires?
Z = (40,000 - 45,000)/3,000 = -5,000/3,000 = -1.67. P(Z < -1.67) = 0.0475, so about 4.8%.
Example: The time to complete an exam is normally distributed with mu = 50 minutes and sigma = 8 minutes. What is the probability a student finishes between 42 and 58 minutes?
42 = 50 - 8 and 58 = 50 + 8, so this range is mu +/- 1 sigma. By the Empirical Rule, about 68%.
Example: A company's delivery times are normally distributed with mu = 4 days and sigma = 0.5 days. They promise delivery within 5 days. What percentage of deliveries arrive on time?
Z = (5 - 4)/0.5 = 2.0. P(Z < 2) = 0.9772, so about 97.7% arrive within 5 days.
Example: A grading system assigns an A to the top 10% of test scores. If scores are normally distributed with mu = 72 and sigma = 9, what minimum score earns an A?
The 90th percentile has z = 1.28. X = 72 + 1.28(9) = 72 + 11.52 = 83.52, approximately 83.5.
Example: Birth weights of full-term babies are normally distributed with mu = 7.5 lbs and sigma = 1.2 lbs. A hospital classifies babies under 5.5 lbs as low birth weight. What percentage of full-term babies are classified as low birth weight?
Z = (5.5 - 7.5)/1.2 = -2.0/1.2 = -1.67. P(Z < -1.67) = 0.0475, so about 4.8%.

  Key Concepts: Central Limit Theorem
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the Central Limit Theorem?
The CLT states that for large n, the distribution of x-bar is approximately normal, even if the population itself is not normal.
What is a 'sampling distribution'?
The sampling distribution shows the pattern of a statistic's values when the sampling process is repeated many times.
What is the Law of Large Numbers?
The Law of Large Numbers says that x-bar approaches mu as n grows, which is related to but distinct from the CLT.
A bottling machine fills cans with mu = 12 oz and sigma = 0.5 oz. If a quality inspector measures 25 cans, what is the probability that the sample mean exceeds 12.2 oz?
SE = 0.5 / sqrt(25) = 0.1. Z = (12.2 - 12) / 0.1 = 2.0. P(Z > 2) = 0.0228.

SE = 15 / sqrt(100) = 1.5. Z = (42 - 45) / 1.5 = -2.0. P(Z < -2) = 0.0228. The CLT applies since n = 100.

SE = 12 / sqrt(40) = 1.897. Z for 70 = (70 - 72)/1.897 = -1.054. Z for 74 = (74 - 72)/1.897 = 1.054. P(-1.054 < Z < 1.054) = about 0.706 or 70.6%.

SE = 0.3 / sqrt(9) = 0.1. Z = (10.2 - 10) / 0.1 = 2.0.

SE = 20,000 / sqrt(400) = 20,000 / 20 = $1,000. With n = 400, the CLT ensures x-bar is approximately normal even though income is typically skewed.

--- Lesson 5.6: Poisson Distribution (Summary) ---

  Key Concepts: Poisson Distribution
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the Poisson probability formula?
The Poisson formula is P(X = k) = e^(-lambda) x lambda^k / k!, where e is approximately 2.718.
What is lambda (the rate parameter) in the Poisson distribution?
Lambda is the average rate of events per interval. A special property is that both the mean and variance of the Poisson distribution equal lambda.
What is the relationship between the Poisson and binomial distributions?
When n is large and p is small, the binomial distribution can be well-approximated by a Poisson distribution with lambda = np.
A call center receives an average of 8 calls per hour. What is the probability of receiving exactly 5 calls in a given hour?
P(X = 5) = e^(-8) x 8^5 / 5! = 0.000335 x 32768 / 120 = 0.0916.
A website averages 3 server errors per day. What is the probability of zero errors on a given day?
P(X = 0) = e^(-3) x 3^0 / 0! = e^(-3) = 0.0498. There is about a 5% chance of a perfect day.
A hospital emergency room sees an average of 10 patients per hour. What is the expected number of patients in a 30-minute window, and what is the standard deviation?
For 30 minutes: lambda = 10 x 0.5 = 5. E(X) = 5. SD = sqrt(5) = 2.236.
A book has an average of 1.5 typos per page. What is the probability that a randomly selected page has 2 or more typos?
P(X >= 2) = 1 - P(0) - P(1). P(0) = e^(-1.5) = 0.2231. P(1) = e^(-1.5) x 1.5 = 0.3347. P(X >= 2) = 1 - 0.2231 - 0.3347 = 0.4422.
A traffic light intersection averages 2 accidents per month. What is the probability of exactly 4 accidents in a given month?
P(X = 4) = e^(-2) x 2^4 / 4! = 0.1353 x 16 / 24 = 0.0902.
A factory produces defective chips at a rate of 0.5 per 1,000 chips. In a batch of 10,000 chips, what is lambda for the Poisson approximation?
Lambda = 0.5 per 1,000, so for 10,000 chips: lambda = 0.5 x 10 = 5.
A meteor shower produces an average of 6 meteors visible per minute. What is the probability of seeing no meteors in a particular 30-second window?
For 30 seconds (half a minute): lambda = 6 x 0.5 = 3. P(X = 0) = e^(-3) = 0.0498.
An insurance company processes an average of 12 claims per day. What is the probability of processing more than 15 claims on a given day? (Use the fact that for lambda = 12, P(X <= 15) = 0.8444.)
P(X > 15) = 1 - P(X <= 15) = 1 - 0.8444 = 0.1556.
A rare genetic mutation occurs at a rate of 1 in 50,000 births. In a city with 200,000 births per year, what is the expected number of mutations per year and the appropriate distribution?
p = 1/50,000 = 0.00002. Lambda = np = 200,000 x 0.00002 = 4. This is a classic Poisson scenario (large n, small p).
A restaurant receives an average of 4 delivery orders per hour during lunch. What is the probability they receive at least 1 order in a 15-minute period?
For 15 minutes (1/4 hour): lambda = 4 x 0.25 = 1. P(X >= 1) = 1 - P(0) = 1 - e^(-1) = 1 - 0.3679 = 0.6321.
Real-World Applications

P(X = 5) = e^(-8) x 8^5 / 5! = 0.000335 x 32768 / 120 = 0.0916.

--- Lesson 5.7: Applications in Quality Control (Summary) ---

  Key Concepts: Applications in Quality Control
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is 'common cause variation'?
Common cause variation is the natural, random variation inherent in every process. It is expected and predictable.
What is 'special cause variation'?
Special cause variation comes from identifiable sources like equipment malfunction, new materials, or operator error.
What is 'acceptance sampling'?
Acceptance sampling draws a random sample from a batch and uses the sample results to make an accept/reject decision for the whole lot.
What is the role of the CLT in quality control?
The CLT ensures that sample means are approximately normal for large enough subgroups, justifying the 3-sigma control limits on x-bar charts.
What is a 'lot' in acceptance sampling?
The lot is the full batch of items from which a random sample is drawn for inspection.
A car manufacturer has specification limits of 2000 +/- 50 for a component dimension. The process has sigma = 15. What is the capability index Cp, and is the process capable?
Spec width = 2050 - 1950 = 100. Cp = 100 / (6 x 15) = 100/90 = 1.11. Generally Cp >= 1.33 is desired, so 1.11 is only marginally capable.
A circuit board manufacturer has a defect rate of 0.3 defects per board. Using the Poisson distribution, what is the probability a randomly selected board has zero defects?
P(X = 0) = e^(-0.3) = 0.7408. About 74.1% of boards are defect-free.
A food company uses acceptance sampling. They inspect 20 items from a lot and reject the lot if more than 1 item is defective. If the true defect rate is 5%, what is the probability the lot is accepted?
P(accept) = P(X <= 1) = P(0) + P(1). P(0) = (0.95)^20 = 0.3585. P(1) = C(20,1)(0.05)(0.95)^19 = 20 x 0.05 x 0.3774 = 0.3774. Total = 0.3585 + 0.3774 = 0.736.
A semiconductor fab tracks the number of particles per wafer. The process averages 2 particles per wafer. Management wants no more than 5 particles on any wafer. What is the probability a wafer passes this criterion?
Using Poisson with lambda = 2: P(X <= 5) = sum of P(0) through P(5) = 0.1353 + 0.2707 + 0.2707 + 0.1804 + 0.0902 + 0.0361 = 0.9834.
A coffee shop targets 12 oz per cup with sigma = 0.5 oz. The specification limits are 11 oz to 13 oz. What is the Cp, and what percentage of cups are expected to fall outside specifications?
Cp = (13 - 11) / (6 x 0.5) = 2/3 = 0.67. The specs are at mu +/- 2 sigma, so about 4.6% fall outside (since about 95.4% are within 2 sigma).
Real-World Applications
Example: A bottling plant monitors the proportion of underfilled bottles. Over 20 samples of 50 bottles each, the average proportion defective is 0.04. What are the control limits for a p-chart?
SE = sqrt(0.04 x 0.96 / 50) = sqrt(0.000768) = 0.0277. Control limits = 0.04 +/- 3(0.0277) = 0.04 +/- 0.0832. LCL cannot be negative, so LCL = 0, UCL = 0.123.
Example: A semiconductor fab tracks the number of particles per wafer. The process averages 2 particles per wafer. Management wants no more than 5 particles on any wafer. What is the probability a wafer passes this criterion?
Using Poisson with lambda = 2: P(X <= 5) = sum of P(0) through P(5) = 0.1353 + 0.2707 + 0.2707 + 0.1804 + 0.0902 + 0.0361 = 0.9834.
Example: A coffee shop targets 12 oz per cup with sigma = 0.5 oz. The specification limits are 11 oz to 13 oz. What is the Cp, and what percentage of cups are expected to fall outside specifications?
Cp = (13 - 11) / (6 x 0.5) = 2/3 = 0.67. The specs are at mu +/- 2 sigma, so about 4.6% fall outside (since about 95.4% are within 2 sigma).

--- Lesson 5.8: Case Studies in Real-World Distributions (Summary) ---

  Key Concepts: Case Studies in Real-World Distributions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the 'memoryless property' of the exponential distribution?
The memoryless property means P(X > s + t | X > s) = P(X > t). Past waiting time does not affect future waiting probability.
What is a 'goodness-of-fit test'?
Goodness-of-fit tests compare observed frequencies with expected frequencies under a proposed distribution model.
What is a 'Monte Carlo simulation'?
Monte Carlo methods repeatedly generate random samples from distributions to estimate probabilities, averages, and other quantities.
What is the exponential distribution?
The exponential distribution is continuous and right-skewed, commonly used to model the time between successive events.
What is a 'mixture distribution' in real-world data?
Mixture distributions arise when data comes from multiple groups, such as bimodal height distributions from combining male and female populations.
