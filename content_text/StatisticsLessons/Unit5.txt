--- Lesson 5.1: Discrete Random Variables (Summary) ---

  Key Concepts: Discrete Random Variables
Discrete Random Variables
A random variable (X) assigns a numerical value to each outcome in a sample space. A discrete random variable takes a countable number of distinct values (0, 1, 2, …).
  • Probability distribution: A table or function listing every value x and its probability P(X = x).  • Requirements: (1) 0 ≤ P(X = x) ≤ 1 for each value, and (2) ΣP(X = x) = 1.  • Expected value (mean): μ = E(X) = Σ[x · P(X = x)] — the long-run average.  • Variance: σ² = Σ[(x − μ)² · P(X = x)]; Standard deviation: σ = √σ².  • Examples: Number of heads in 3 coin flips, number of defective items in a batch, number rolled on a die.

--- Lesson 5.2: Continuous Random Variables (Summary) ---

  Key Concepts: Continuous Random Variables
Continuous Random Variables
A continuous random variable can take any value within an interval (e.g., height, weight, temperature).
  • Probability density function (PDF): f(x) describes the shape of the distribution. The total area under the curve = 1.  • P(a ≤ X ≤ b) = area under the curve between a and b. For continuous variables, P(X = exact value) = 0.  • Cumulative distribution function (CDF): F(x) = P(X ≤ x), the area under the curve from −∞ to x.  • Mean (μ): The balance point of the distribution. Variance (σ²): Measures spread around the mean.  • Uniform distribution: All values in [a, b] are equally likely. f(x) = 1/(b−a); E(X) = (a+b)/2.
Key difference from discrete: probability is found as an area, not a single point.

--- Lesson 5.3: Binomial Distribution (Summary) ---

  Key Concepts: Binomial Distribution
Binomial Distribution
The binomial distribution models the number of successes in n independent trials, each with the same probability of success p.

Conditions (BINS):
  • Binary outcomes: each trial has exactly two outcomes (success/failure).  • Independent trials: outcomes don't affect each other.  • N is fixed: the number of trials is predetermined.  • Same probability p for each trial.
Formula: P(X = k) = C(n,k) · pk · (1−p)n−k

Mean: μ = np   Variance: σ² = np(1−p)   Standard deviation: σ = √(np(1−p))

Example: Flipping a fair coin 10 times: n = 10, p = 0.5. P(exactly 3 heads) = C(10,3)(0.5)³(0.5)⁷ ≈ 0.117.

--- Lesson 5.4: Normal Distribution (Summary) ---

  Key Concepts: Normal Distribution
Normal Distribution
The normal distribution (Gaussian distribution) is the most important continuous distribution in statistics.
  • Bell-shaped and symmetric about the mean μ.  • Fully described by two parameters: μ (mean/center) and σ (standard deviation/spread).  • Empirical Rule (68-95-99.7): ~68% of data falls within 1σ of μ, ~95% within 2σ, ~99.7% within 3σ.  • Standard normal distribution: μ = 0, σ = 1. Any normal variable X can be standardized: Z = (X − μ)/σ.  • Z-table: Gives the area (probability) to the left of a given z-score.
Many natural phenomena approximate the normal distribution: heights, test scores, measurement errors, etc.

--- Lesson 5.5: Central Limit Theorem (Summary) ---

  Key Concepts: Central Limit Theorem
Central Limit Theorem (CLT)
The Central Limit Theorem is one of the most important results in statistics.

Statement: For a population with mean μ and standard deviation σ, the sampling distribution of the sample mean x̄ approaches a normal distribution as n increases, regardless of the population's shape.
  • Mean of x̄: μx̄ = μ (the population mean).  • Standard error: σx̄ = σ/√n (decreases as n increases).  • Rule of thumb: n ≥ 30 is generally sufficient for the CLT to apply, though if the population is already normal, any n works.  • Implication: Even if the population is skewed, x̄ will be approximately normal for large enough n.  • Why it matters: The CLT justifies using z-scores and normal-based methods for inference with sample means.

--- Lesson 5.6: Poisson Distribution (Summary) ---

  Key Concepts: Poisson Distribution
Poisson Distribution
The Poisson distribution models the number of events occurring in a fixed interval of time or space when events happen independently at a constant average rate.
  • Parameter: λ (lambda) = the average number of events per interval.  • Formula: P(X = k) = (e−λ · λk) / k!  • Mean: E(X) = λ   Variance: Var(X) = λ   (mean equals variance!)  • Conditions: Events occur independently, at a constant average rate, and cannot occur simultaneously.
Examples: Number of calls to a call center per hour, number of typos per page, number of car accidents at an intersection per month, number of emails received per day.

The Poisson can approximate the binomial when n is large and p is small (λ = np).

--- Lesson 5.7: Applications in Quality Control (Summary) ---

  Key Concepts: Applications in Quality Control
Applications in Quality Control
Statistical distributions are essential in quality control (QC) to monitor and improve manufacturing processes.
  • Control charts: Plot statistics (mean, proportion) over time to detect process deviations. Upper/lower control limits (UCL/LCL) are typically set at μ ± 3σ.  • Process capability: Compares the spread of the process (6σ) to specification limits.  • Acceptance sampling: Inspecting a sample from a batch to decide whether to accept or reject the lot. Uses binomial/Poisson distributions.  • Defect rate: The proportion or count of defective items. Modeled by binomial (proportion) or Poisson (count per unit).  • Six Sigma: A methodology aiming for ≤ 3.4 defects per million opportunities — about 6σ from the mean.
QC relies on sampling distributions and the CLT to make decisions about production quality from samples.

--- Lesson 5.8: Case Studies in Real-World Distributions (Summary) ---

  Key Concepts: Case Studies in Real-World Distributions
Case Studies in Real-World Distributions
Statistical distributions appear throughout real life. Recognizing which distribution fits a situation is a key statistical skill.
  • Normal: Heights of adults, standardized test scores, measurement errors. Symmetric, bell-shaped data.  • Binomial: Number of patients cured in n treated (fixed n, same p, independent). Pass/fail counts.  • Poisson: Number of mutations in a DNA strand, number of accidents per month, calls per hour.  • Uniform: Random number generators, equally likely outcomes in simulations.  • Exponential: Time between arrivals, duration until a radioactive particle decays. Memoryless property.  • Skewed distributions: Income distribution (right-skewed), wait times (often right-skewed).
Choosing the right model depends on the data type (discrete vs. continuous), shape, and underlying process.
