--- Lesson 6.1: Key Features of Parabolas (Summary) ---

  Key Concepts: Key Features of Parabolas
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of the parabola y = x^2 - 4x + 3?
The x-coordinate of the vertex is x = -b/(2a) = -(-4)/(2*1) = 2. Substituting: y = (2)^2 - 4(2) + 3 = 4 - 8 + 3 = -1. The vertex is (2, -1).
What is the axis of symmetry for y = x^2 + 6x + 5?
The axis of symmetry is x = -b/(2a) = -6/(2*1) = -3. This vertical line passes through the vertex and divides the parabola into two mirror-image halves.
What is the y-intercept of y = 3x^2 - 2x + 7?
The y-intercept occurs when x = 0: y = 3(0)^2 - 2(0) + 7 = 7. In standard form y = ax^2 + bx + c, the y-intercept is always c.
What is the domain of the quadratic function y = 2x^2 + 3x - 1?
The domain of any quadratic function is all real numbers. You can substitute any real number for x and get a valid output.
For the parabola y = -x^2 + 4x - 1, what is the maximum y-value?
Since a = -1 < 0, the parabola opens downward and has a maximum at the vertex. x = -4/(2(-1)) = 2. y = -(2)^2 + 4(2) - 1 = -4 + 8 - 1 = 3.
What is the range of y = x^2 - 6x + 8?
The vertex x-coordinate is x = 6/2 = 3. The vertex y-value is (3)^2 - 6(3) + 8 = 9 - 18 + 8 = -1. Since the parabola opens upward, the range is y >= -1.
What is a 'parabola'?
A parabola is the U-shaped (or inverted U-shaped) curve formed by graphing a quadratic function y = ax^2 + bx + c. It is symmetric about a vertical line called the axis of symmetry.
A ball is thrown upward with height modeled by h(t) = -16t^2 + 48t + 5 feet. What is the maximum height the ball reaches?
The vertex t-coordinate is t = -48/(2(-16)) = 48/32 = 1.5 seconds. h(1.5) = -16(2.25) + 48(1.5) + 5 = -36 + 72 + 5 = 41 feet.
A garden arch is shaped like a parabola modeled by y = -0.5x^2 + 4x, where x and y are in feet. What is the maximum height of the arch, and how wide is it at ground level?
Vertex: x = -4/(2(-0.5)) = 4. y = -0.5(16) + 4(4) = -8 + 16 = 8 feet (max height). X-intercepts: -0.5x^2 + 4x = 0, x(-0.5x + 4) = 0, so x = 0 and x = 8. Width = 8 - 0 = 8 feet.
A company's profit is modeled by P(x) = -2x^2 + 40x - 100 (in thousands of dollars), where x is units sold (in thousands). How many units should be sold to maximize profit, and what is the maximum profit?
Vertex: x = -40/(2(-2)) = 10 (thousands of units = 10,000 units). P(10) = -2(100) + 40(10) - 100 = -200 + 400 - 100 = 100 (thousand dollars = $100,000).
A bridge cable hangs in a parabolic shape modeled by y = 0.01x^2 - 2x + 120, where y is height in feet. What is the lowest point of the cable?
Vertex: x = -(-2)/(2(0.01)) = 2/0.02 = 100. y = 0.01(100)^2 - 2(100) + 120 = 100 - 200 + 120 = 20 feet. The cable's lowest point is 20 feet above the ground.

A ticket pricing model shows revenue R(p) = -50p^2 + 1500p, where p is the ticket price in dollars. At what price is revenue maximized, and what is the maximum revenue?
Vertex: p = -1500/(2(-50)) = 1500/100 = $15. R(15) = -50(225) + 1500(15) = -11,250 + 22,500 = $11,250. The optimal ticket price is $15, generating maximum revenue of $11,250.

--- Lesson 6.2: Graphing in Standard Form (Summary) ---

  Key Concepts: Graphing in Standard Form
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the standard form of a quadratic function?
The standard form of a quadratic function is y = ax^2 + bx + c, where a, b, and c are constants and a is not zero.
For y = 2x^2 - 8x + 6, what is the axis of symmetry?
The axis of symmetry is x = -b/(2a) = -(-8)/(2*2) = 8/4 = 2.
What is the vertex of y = x^2 + 2x - 3?
x = -2/(2*1) = -1. y = (-1)^2 + 2(-1) - 3 = 1 - 2 - 3 = -4. The vertex is (-1, -4).
What is the y-intercept of y = 4x^2 - 3x + 9?
The y-intercept is the value of c in standard form. When x = 0: y = 4(0) - 3(0) + 9 = 9.
For y = -x^2 + 6x - 5, what is the vertex?
x = -6/(2(-1)) = 3. y = -(3)^2 + 6(3) - 5 = -9 + 18 - 5 = 4. The vertex is (3, 4).
What is the range of y = 3x^2 - 6x + 5?
Vertex: x = 6/(2*3) = 1. y = 3(1) - 6(1) + 5 = 3 - 6 + 5 = 2. Since a = 3 > 0, the parabola opens upward, so the range is y >= 2.
A water fountain sprays water in an arc modeled by y = -0.5x^2 + 3x, where y is height in feet and x is horizontal distance. Graph the path by finding the vertex and x-intercepts. What is the highest point of the water?
Vertex: x = -3/(2(-0.5)) = 3. y = -0.5(9) + 3(3) = -4.5 + 9 = 4.5 feet. The water reaches its maximum height of 4.5 feet at a horizontal distance of 3 feet.
A farmer's crop yield is modeled by Y(f) = -2f^2 + 24f + 10, where f is bags of fertilizer used. How many bags maximize yield, and what is the maximum yield?
Vertex: f = -24/(2(-2)) = 6 bags. Y(6) = -2(36) + 24(6) + 10 = -72 + 144 + 10 = 82. Maximum yield is 82 units with 6 bags of fertilizer.
A roller coaster dip is modeled by y = 0.5x^2 - 6x + 20, where y is height in meters. What is the lowest height during the dip, and at what horizontal position does it occur?
Vertex: x = 6/(2*0.5) = 6. y = 0.5(36) - 6(6) + 20 = 18 - 36 + 20 = 2 meters. The lowest point of the dip is 2 meters at horizontal position x = 6.
An architect designs a parabolic window modeled by y = -0.25x^2 + 2x (in feet). What is the window's maximum height and width at the base?
Vertex: x = -2/(2(-0.25)) = 4. y = -0.25(16) + 2(4) = -4 + 8 = 4 feet (max height). X-intercepts: -0.25x^2 + 2x = 0, x(-0.25x + 2) = 0, so x = 0 and x = 8. Width = 8 feet.

--- Lesson 6.3: Graphing in Vertex Form (Summary) ---

  Key Concepts: Graphing in Vertex Form
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of y = (x - 3)^2 + 5?
In vertex form y = a(x - h)^2 + k, the vertex is (h, k). Here h = 3 and k = 5, so the vertex is (3, 5).
What is the vertex of y = -2(x + 1)^2 - 4?
y = -2(x - (-1))^2 + (-4). The vertex is (h, k) = (-1, -4). Remember that (x + 1) means h = -1.
What is the axis of symmetry for y = -(x + 4)^2 + 7?
The axis of symmetry passes through the vertex. Since h = -4, the axis of symmetry is x = -4.
What is the y-intercept of y = 2(x - 1)^2 + 3?
Set x = 0: y = 2(0 - 1)^2 + 3 = 2(1) + 3 = 5. The y-intercept is (0, 5).
What is the maximum value of y = -4(x - 5)^2 + 12?
Since a = -4 < 0, the parabola opens downward and has a maximum. The maximum value is k = 12, occurring at x = 5.
What is the range of y = (x + 2)^2 - 9?
The vertex is (-2, -9) and a = 1 > 0, so the parabola opens upward. The minimum y-value is -9, giving a range of y >= -9.
What is 'vertex form' of a quadratic function?
Vertex form is y = a(x - h)^2 + k, where (h, k) is the vertex of the parabola and 'a' determines the direction and width.
What is the advantage of vertex form over standard form?
Vertex form y = a(x - h)^2 + k immediately shows the vertex (h, k), axis of symmetry x = h, direction (sign of a), and whether the vertex is a max or min, without needing to use x = -b/(2a).
A ball is thrown from a 6-foot height with its path modeled by h(t) = -16(t - 1)^2 + 22. What is the maximum height and when does it occur?
In vertex form h(t) = -16(t - 1)^2 + 22, the vertex is (1, 22). Since a = -16 < 0, this is a maximum. The ball reaches 22 feet at t = 1 second.
A company models its profit with P(x) = -5(x - 10)^2 + 200 (in thousands), where x is products sold (in thousands). What is the maximum profit, and how many products must be sold to break even?
Max profit is k = 200 (thousand) = $200,000 at x = 10. Break even: -5(x-10)^2 + 200 = 0, so (x-10)^2 = 40, x - 10 = +/-sqrt(40) = +/-6.32. So x = 3.68 and x = 16.32 thousand products.
A projectile's path in meters is h(t) = -4.9(t - 3)^2 + 50. What is the projectile's height at launch (t = 0) and its maximum height?
At t = 0: h(0) = -4.9(0 - 3)^2 + 50 = -4.9(9) + 50 = -44.1 + 50 = 5.9 m. The maximum height is k = 50 m at t = 3 seconds.
Real-World Applications
Example: What is the advantage of vertex form over standard form?
Vertex form y = a(x - h)^2 + k immediately shows the vertex (h, k), axis of symmetry x = h, direction (sign of a), and whether the vertex is a max or min, without needing to use x = -b/(2a).

--- Lesson 6.4: Transformations of Quadratic Functions (Summary) ---

  Key Concepts: Transformations of Quadratic Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of y = 2(x + 4)^2 - 7?
In y = 2(x - (-4))^2 + (-7), h = -4 and k = -7. The vertex is (-4, -7). The graph is shifted 4 left and 7 down from y = 2x^2.
What is the y-intercept of y = -2(x - 3)^2 + 4?
At x = 0: y = -2(0 - 3)^2 + 4 = -2(9) + 4 = -18 + 4 = -14.
A physics student graphs the height of a ball: y = -16t^2 + 64. The teacher asks the student to model the same throw from a 100-foot building instead of 64 feet. What is the new equation?
The initial height is the value when t = 0, which is the constant term. Changing the launch height from 64 to 100 simply changes the equation to y = -16t^2 + 100. This is a vertical shift of 36 units up.
A garden's parabolic border is y = (x - 5)^2. The gardener wants to move the border 3 feet to the left and 2 feet up. What is the new equation?
Moving 3 feet left changes h from 5 to 5 - 3 = 2. Moving 2 feet up adds k = 2. New equation: y = (x - 2)^2 + 2.
A student graphs y = x^2 and then applies three transformations: reflect over the x-axis, shift 4 right, and shift 1 up. What is the final equation?
Reflect: y = -x^2. Shift 4 right: y = -(x - 4)^2. Shift 1 up: y = -(x - 4)^2 + 1. The vertex is at (4, 1) opening downward.
A parabolic mirror has equation y = 4x^2. A manufacturer needs a mirror that is the same shape but centered at (3, -2). What is the equation?
The same shape means a = 4 stays the same. Centering at (3, -2) means h = 3 and k = -2: y = 4(x - 3)^2 - 2.
A landscape architect has a fountain modeled by y = -2x^2 + 8. She creates a second, shorter fountain by applying a vertical compression of 1/2 and shifting the vertex 3 units to the right. What is the equation of the second fountain?
Vertical compression by 1/2 multiplies the whole function by 1/2: y = 0.5(-2x^2 + 8) = -x^2 + 4. Then shift 3 right: y = -(x - 3)^2 + 4. The vertex moves from (0, 4) to (3, 4).

--- Lesson 6.6: Quadratic Regression (Summary) ---

  Key Concepts: Quadratic Regression
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
A quadratic regression gives y = 2x^2 - 3x + 1. What is the predicted y-value when x = 2?
y = 2(2)^2 - 3(2) + 1 = 8 - 6 + 1 = 3.
A calculator performs quadratic regression and gives a = -0.5, b = 6, c = 2. What is the equation?
The quadratic regression equation is y = ax^2 + bx + c = -0.5x^2 + 6x + 2.
What is the vertex of the quadratic regression model y = -2x^2 + 12x - 10?
x = -12/(2(-2)) = 3. y = -2(9) + 12(3) - 10 = -18 + 36 - 10 = 8. The vertex is (3, 8).
Using the quadratic regression y = 0.5x^2 - 4x + 10, what is y when x = 6?
y = 0.5(36) - 4(6) + 10 = 18 - 24 + 10 = 4.
What is 'quadratic regression'?
Quadratic regression is a statistical technique that uses data points to compute the values of a, b, and c that produce the best-fitting parabola y = ax^2 + bx + c, typically using technology.
An architect measures the height of a parabolic bridge at 5 points: (0, 0), (10, 15), (20, 20), (30, 15), (40, 0). A regression gives y = -0.05x^2 + 2x. Is this a good model, and what is the bridge's maximum height?
Vertex: x = -2/(2(-0.05)) = 20. y = -0.05(400) + 2(20) = -20 + 40 = 20 feet. Checking all points: (0,0), (10,15), (20,20), (30,15), (40,0) all match perfectly. The model is excellent with max height 20 ft.
A researcher studies the effect of temperature on enzyme activity. Data shows activity increases to a peak at 37 degrees C then decreases. The regression gives y = -0.8(x - 37)^2 + 100. What is the predicted activity at 25 degrees C?
y = -0.8(25 - 37)^2 + 100 = -0.8(144) + 100 = -115.2 + 100 = -15.2. Hmm that gives -15.2, not 84.8. Let me recheck: (25-37)^2 = (-12)^2 = 144. -0.8(144) = -115.2. -115.2 + 100 = -15.2. The answer 84.8 corresponds to -0.8((-12)^2) + 100 with a smaller coefficient. If the equation were y = -0.1(x-37)^2 + 100, then y = -0.1(144)+100 = 85.6. The correct computation gives -15.2 with the given equation.
Real-World Applications
Example: A school tracks test scores versus study hours: (0,45), (1,60), (2,72), (3,80), (4,85), (5,82). The data rises then begins to fall. The quadratic regression gives y = -1.5x^2 + 14x + 45.5. What score does the model predict for 3.5 hours of study?
y = -1.5(3.5)^2 + 14(3.5) + 45.5 = -1.5(12.25) + 49 + 45.5 = -18.375 + 49 + 45.5 = 76.125, approximately 76.1. The closest answer is 76.6.
Example: A pharmaceutical company tests drug dosage (mg) versus effectiveness (%): (10, 30), (20, 55), (30, 70), (40, 75), (50, 70), (60, 55). The quadratic regression gives y = -0.05x^2 + 4x - 5 with r^2 = 0.99. What dosage maximizes effectiveness?
x = -4/(2(-0.05)) = -4/(-0.1) = 40 mg. Maximum effectiveness: -0.05(1600) + 4(40) - 5 = -80 + 160 - 5 = 75%. The optimal dosage is 40 mg with 75% effectiveness.

--- Lesson 7.1: Solving by Factoring — GCF &amp; Grouping (Summary) ---

  Key Concepts: Solving by Factoring — GCF &amp; Grouping
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the GCF of 6x^2 and 9x?
The GCF of 6 and 9 is 3. The GCF of x^2 and x is x. So the GCF is 3x.
What is the GCF of 12x^3, 18x^2, and 6x?
GCF of 12, 18, and 6 is 6. GCF of x^3, x^2, and x is x (lowest power). So the GCF is 6x.
Real-World Applications
Example: A storage container has volume V = 2x^3 + 6x^2 - 4x cubic inches. Factor completely to determine all dimensions as expressions in x.
GCF of 2x^3, 6x^2, and 4x is 2x. Factor: V = 2x(x^2 + 3x - 2). The trinomial x^2 + 3x - 2 does not factor over the integers, so 2x(x^2 + 3x - 2) is the complete factorization.
Example: A revenue function is R(x) = 8x^2 - 24x. A manager asks at what production level revenue equals zero (break-even). What are the break-even points?
Factor: 8x(x - 3) = 0. So x = 0 (no production) or x = 3 (3 units). Revenue is zero at both break-even points.

--- Lesson 7.3: Special Products &amp; Factoring Patterns (Summary) ---

  Key Concepts: Special Products &amp; Factoring Patterns
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a 'difference of squares'?
A difference of squares is a binomial of the form a^2 - b^2, where both terms are perfect squares separated by subtraction. It always factors as (a - b)(a + b).
What is a 'perfect square trinomial'?
A perfect square trinomial results from squaring a binomial. It has the form a^2 + 2ab + b^2 = (a + b)^2 or a^2 - 2ab + b^2 = (a - b)^2.
A square park has side length (x + 5) meters. What is the area expressed as a trinomial?
Area = (x + 5)^2 = x^2 + 2(5)(x) + 25 = x^2 + 10x + 25 square meters.
A picture frame has outer dimensions (2x + 3) by (2x - 3). What is the area of the frame's outer boundary?
Area = (2x + 3)(2x - 3) = (2x)^2 - 3^2 = 4x^2 - 9. This is the product of conjugate binomials.
A tile pattern has area 25x^2 - 20x + 4 square inches. A designer recognizes this as a perfect square trinomial. What is the side length of each square tile?
25x^2 - 20x + 4 = (5x)^2 - 2(5x)(2) + 2^2 = (5x - 2)^2. The side length is (5x - 2) inches.
A student needs to compute 47 x 53 mentally. She recognizes this as (50 - 3)(50 + 3). What is the answer?
(50 - 3)(50 + 3) = 50^2 - 3^2 = 2500 - 9 = 2,491. The difference of squares pattern enables quick mental math.
A floor tile pattern uses squares of side (3x - 1) cm. A customer asks: if x = 4 cm, what is the area of one tile? Use the perfect square trinomial expansion to compute.
(3x - 1)^2 = 9x^2 - 6x + 1. At x = 4: 9(16) - 6(4) + 1 = 144 - 24 + 1 = 121 cm^2. Or directly: 3(4) - 1 = 11, and 11^2 = 121.

What is the quadratic formula?
The quadratic formula is x = (-b +/- sqrt(b^2 - 4ac)) / (2a). It solves any quadratic equation ax^2 + bx + c = 0.
What is the value of the discriminant for x^2 + 4x + 4 = 0?
Discriminant = b^2 - 4ac = 16 - 4(1)(4) = 16 - 16 = 0. This means there is exactly one real solution (a double root).
A gardener models a flower bed with area equation x^2 + 5x - 50 = 0, where x is the width in feet. What is the width?
x = (-5 +/- sqrt(25+200))/2 = (-5 +/- sqrt(225))/2 = (-5 +/- 15)/2. x = 10/2 = 5 or x = -20/2 = -10. Since width must be positive, x = 5 feet.

  Key Concepts: The Discriminant
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the discriminant of x^2 + 3x - 10 = 0?
Discriminant = b^2 - 4ac = 9 - 4(1)(-10) = 9 + 40 = 49.
What is the discriminant of 2x^2 + x + 3 = 0?
D = 1 - 4(2)(3) = 1 - 24 = -23. The negative discriminant means no real solutions.
What is the discriminant of x^2 - 4x + 5 = 0?
D = 16 - 4(1)(5) = 16 - 20 = -4. Negative discriminant: no real solutions.
What is the discriminant of 3x^2 - 2x - 1 = 0?
D = (-2)^2 - 4(3)(-1) = 4 + 12 = 16. Positive and a perfect square, so two rational solutions.
What is the discriminant of x^2 + 2x - 8 = 0, and what does it tell you?
D = 4 - 4(1)(-8) = 4 + 32 = 36. Since 36 > 0 and is a perfect square (6^2), the equation has two distinct rational solutions.
What is the 'discriminant'?
The discriminant D = b^2 - 4ac is the part of the quadratic formula under the square root. Its sign determines whether the equation has 2, 1, or 0 real solutions.
A store models profit as P(x) = -3x^2 + 24x - k where k is fixed costs. For the store to make a profit at some production level, what is the maximum value of k?
For profit > 0 at some x, the equation -3x^2 + 24x - k = 0 must have real solutions: D >= 0. D = 576 - 12k >= 0, so k <= 48. For the store to actually be profitable (not just break even), k < 48.

Overview

Key Definitions
What is a linear-quadratic system?
A linear-quadratic system consists of one linear equation (degree 1) and one quadratic equation (degree 2) that are solved simultaneously.
What is a quadratic-quadratic system?
A quadratic-quadratic system consists of two quadratic equations that can intersect at 0, 1, 2, 3, or 4 points.
What is the zero product property?
The zero product property states that if the product of two factors is zero, then at least one of the factors must be zero. This is essential for solving factored quadratic equations.
A farmer's field is bounded by a parabolic hill y = −x² + 8x and a straight irrigation canal y = 2x. What is the area of land between the hill and the canal's intersection points?
Setting −x² + 8x = 2x gives −x² + 6x = 0, so −x(x − 6) = 0. The intersection points are at x = 0 and x = 6, which is the first step in finding the enclosed area.

--- Lesson 8.2: Multiplying Polynomials (Summary) ---

  Key Concepts: Multiplying Polynomials
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the 'distributive property' as it applies to polynomial multiplication?
The distributive property states that a(b + c) = ab + ac. When multiplying polynomials, each term in the first polynomial must be multiplied by each term in the second polynomial.
What is the 'difference of squares' pattern?
The difference of squares is the identity (a + b)(a − b) = a² − b². The middle terms cancel because the outer and inner products are equal in magnitude but opposite in sign.
What is a 'perfect square trinomial'?
A perfect square trinomial has the form a² + 2ab + b² = (a + b)² or a² − 2ab + b² = (a − b)². The middle term is always twice the product of the square roots of the first and last terms.
What is the 'area model' (or box method) for polynomial multiplication?
The area model (box method) organizes multiplication into a grid. Each term of one factor labels a row, each term of the other labels a column, and each cell contains the product of its row and column terms.
What is the 'product of a sum and difference' pattern?
When you multiply conjugates (a + b)(a − b), the inner and outer products cancel, leaving only a² − b². This is called the product of a sum and difference.
A painter charges (x + 5) dollars per hour and works for (x + 3) hours. If paint costs (x² + 2x) dollars, what is the total cost (labor plus paint)?
Labor = (x + 5)(x + 3) = x² + 8x + 15. Total = labor + paint = (x² + 8x + 15) + (x² + 2x) = 2x² + 10x + 15 dollars.

When multiplying polynomials, the degree of the product is the sum of the degrees of the factors. For example, a degree-2 polynomial times a degree-3 polynomial gives a degree-5 result.

--- Lesson 8.3: Dividing Polynomials (Summary) ---

  Key Concepts: Dividing Polynomials
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the remainder when (2x² − 3x − 5) is divided by (x − 3)?
By the Remainder Theorem, the remainder is P(3) = 2(9) − 3(3) − 5 = 18 − 9 − 5 = 4.
What is the Remainder Theorem?
The Remainder Theorem states that when a polynomial P(x) is divided by (x − c), the remainder is simply P(c) -- the value obtained by substituting c into the polynomial.
What is the Factor Theorem?
The Factor Theorem is a special case of the Remainder Theorem: if P(c) = 0, then the remainder is 0, which means (x − c) divides P(x) evenly and is therefore a factor.
What is a 'dividend' in polynomial division?
In the expression P(x) ÷ D(x) = Q(x) + R/D(x), P(x) is the dividend (the polynomial being divided), D(x) is the divisor, Q(x) is the quotient, and R is the remainder.
What is a 'divisor' in polynomial division?
The divisor is the polynomial you divide by. In (x² + 5x + 6) ÷ (x + 2), the divisor is (x + 2).
What is the 'quotient' in polynomial division?
The quotient is the polynomial result of the division. In (x² + 5x + 6) ÷ (x + 2) = x + 3, the quotient is x + 3.
A rectangular garden has area (2x² + 11x + 15) square feet and width (x + 3) feet. What is the length?
Length = Area ÷ Width = (2x² + 11x + 15) ÷ (x + 3). Factor: 2x² + 11x + 15 = (2x + 5)(x + 3). The length is (2x + 5) feet.
A box has volume (x³ + 6x² + 11x + 6) cubic inches and its base has area (x² + 3x + 2) square inches. What is the height?
Height = Volume ÷ Base area. Factor: x³ + 6x² + 11x + 6 = (x + 1)(x + 2)(x + 3) and x² + 3x + 2 = (x + 1)(x + 2). Height = (x + 3) inches.
A company's total profit is (3x² + 14x + 8) dollars and they sold (x + 4) units. What is the profit per unit?
Profit per unit = (3x² + 14x + 8) ÷ (x + 4). Factor: 3x² + 14x + 8 = (3x + 2)(x + 4). The profit per unit is (3x + 2) dollars.
A polynomial P(x) = x³ − 4x² + x + 6 is evaluated at x = 2. By the Remainder Theorem, what is the remainder when P(x) is divided by (x − 2)?
P(2) = 8 − 16 + 2 + 6 = 0. By the Remainder Theorem, the remainder is 0, which also means (x − 2) is a factor of P(x).
A delivery route covers (x³ + 5x² + 2x − 8) miles in (x + 2) hours. What is the average speed?

--- Lesson 8.4: Polynomial Identities (Summary) ---

  Key Concepts: Polynomial Identities
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a 'polynomial identity'?
A polynomial identity is an equation that holds true for every possible value of its variables, such as (a + b)² = a² + 2ab + b². It is not limited to particular solutions.
What is the 'sum of cubes' identity?
The sum of cubes identity is a³ + b³ = (a + b)(a² − ab + b²). Note the negative sign on the ab term in the trinomial factor.
What is the 'difference of cubes' identity?
The difference of cubes identity is a³ − b³ = (a − b)(a² + ab + b²). Note the positive sign on the ab term and the positive b² in the trinomial.
What is the 'difference of squares' identity?
The difference of squares identity states a² − b² = (a + b)(a − b). It factors a binomial that is the difference of two perfect squares into conjugate binomials.
What is the 'cube of a sum' identity?
The cube of a sum is (a + b)³ = a³ + 3a²b + 3ab² + b³. The coefficients 1, 3, 3, 1 come from the fourth row of Pascal's Triangle.
What is the difference between an 'identity' and an 'equation'?
An identity like (a+b)² = a² + 2ab + b² is true for ALL values of a and b. An equation like x² − 4 = 0 is only true for specific values (x = 2 or x = −2).
A carpenter needs to compute 47 × 53 quickly. Which identity helps, and what is the result?
Recognize 47 × 53 = (50 − 3)(50 + 3) = 50² − 3² = 2500 − 9 = 2491. The difference of squares identity makes this a mental math problem.
A chemist models a reaction volume as (x + y + z)². Using the identity for the square of a trinomial, what is the expanded form?
The square of a trinomial: (x + y + z)² = x² + y² + z² + 2xy + 2xz + 2yz. Each pair of terms contributes a cross product multiplied by 2.
A teacher asks students to determine which is an identity: (A) x² + 1 = 0 or (B) (x + 1)² = x² + 2x + 1. What is the correct answer and why?
B is an identity because (x + 1)² = x² + 2x + 1 holds true for every value of x. A is an equation that has no real solutions (x² = −1), so it is not an identity.

    Explanation: Using a³ + b³ = (a + b)(a² − ab + b²): x³ + 8 = x³ + 2³ = (x + 2)(x² − 2x + 4).

  Q25. A playground is a square with side (2x + 5) meters. The identity (a + b)² tells us the area. Which expression gives the area?
    [ ] 4x² + 25
    [ ] 2x² + 20x + 25
    [ ] 4x² + 10x + 25
    [*] 4x² + 20x + 25
    Explanation: Using (a + b)² = a² + 2ab + b²: (2x + 5)² = (2x)² + 2(2x)(5) + 5² = 4x² + 20x + 25 square meters.

  Q26. A student is asked whether 998 × 1002 can be computed using an identity. Which approach works?
    [ ] No identity applies
    [*] (1000 − 2)(1000 + 2) = 1000² − 4 = 999,996
    [ ] (1000 + 2)² = 1,004,004
    [ ] (1000 − 2)² = 996,004
    Explanation: Recognize 998 × 1002 = (1000 − 2)(1000 + 2). Using the difference of squares: 1000² − 2² = 1,000,000 − 4 = 999,996.

  Q27. An engineer needs to verify that a support beam formula (3x − 2)² = 9x² − 12x + 4 is correct. Is the identity valid?
    [ ] No, the constant should be −4
    [ ] Only true when x is positive
    [*] Yes, because 2(3x)(2) = 12x and the sign is negative
    [ ] No, the middle term should be −6x
    Explanation: Using (a − b)² = a² − 2ab + b²: (3x − 2)² = (3x)² − 2(3x)(2) + 2² = 9x² − 12x + 4. The identity is confirmed for all values of x.

  Q28. A chemist models a reaction volume as (x + y + z)². Using the identity for the square of a trinomial, what is the expanded form?
    [*] x² + y² + z² + 2xy + 2xz + 2yz
    [ ] x² + y² + z²
    [ ] x² + y² + z² + xy + xz + yz
    [ ] 3(x + y + z)
    Explanation: The square of a trinomial: (x + y + z)² = x² + y² + z² + 2xy + 2xz + 2yz. Each pair of terms contributes a cross product multiplied by 2.

  Q29. A sculptor creates a piece where the volume of removed material is (x³ − 125) cubic inches. To understand the dimensions, she factors using the difference of cubes. What are the factors?
    [ ] (x − 5)³
    [ ] (x − 5)(x² − 5x + 25)
    [*] (x − 5)(x² + 5x + 25)
    [ ] (x + 5)(x² − 5x + 25)
    Explanation: Using a³ − b³ = (a − b)(a² + ab + b²): x³ − 125 = x³ − 5³ = (x − 5)(x² + 5x + 25).

  Q30. A teacher asks students to determine which is an identity: (A) x² + 1 = 0 or (B) (x + 1)² = x² + 2x + 1. What is the correct answer and why?
    [ ] Both are identities
    [ ] A, because it is a well-known equation
    [ ] Neither is an identity
    [*] B, because it is true for all values of x
    Explanation: B is an identity because (x + 1)² = x² + 2x + 1 holds true for every value of x. A is an equation that has no real solutions (x² = −1), so it is not an identity.

--- Lesson 8.4: Polynomial Identities (Flashcards) ---

  1. Q: (a + b)² = ?
     A: a² + 2ab + b²

  2. Q: (a − b)² = ?
     A: a² − 2ab + b²

  3. Q: (a + b)(a − b) = ?
     A: a² − b²

  4. Q: What is a polynomial identity?
     A: An equation that is true for all values of the variables

  5. Q: What is Pascal's Triangle used for?
     A: Expanding (a + b)ⁿ

  6. Q: Use identity to compute 99²:
     A: (100 − 1)² = 10000 − 200 + 1 = 9801

  7. Q: (x + 5)² = ?
     A: x² + 10x + 25

  8. Q: Identities are useful for:
     A: Mental math, simplification, and proofs

  9. Q: (a + b)³ = ?
     A: a³ + 3a²b + 3ab² + b³

  10. Q: Use (a − b)² to expand (x − 4)²:
     A: x² − 8x + 16
