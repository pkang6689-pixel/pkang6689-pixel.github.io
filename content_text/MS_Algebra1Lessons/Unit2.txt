--- Lesson 4.1: Relations &amp; Functions (Summary) ---

  Key Concepts: Relations &amp; Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
In the relation {(2, 5), (3, 7), (4, 5), (5, 9)}, what is the range?
The range is the set of all output (y) values: {5, 7, 5, 9}. Removing duplicates gives {5, 7, 9}.
What is the domain of the relation {(-3, 1), (0, 4), (2, -1), (5, 3)}?
The domain is the set of all input (x) values: {-3, 0, 2, 5}.
The set {(1, 3), (2, 6), (3, 9), (4, 12)} represents a function. What is the output when the input is 3?
The ordered pair (3, 9) shows that when the input is 3, the output is 9.
What is a relation in mathematics?
A relation is any set of ordered pairs. It describes a connection between inputs and outputs, without the restriction that each input maps to only one output.
What is a function?
A function is a special relation where every input (x-value) has exactly one output (y-value). Multiple inputs may share the same output, but no input can have multiple outputs.
What is the 'vertical line test'?
The vertical line test determines whether a graph represents a function. If any vertical line intersects the graph at more than one point, the graph does not represent a function.
What is a 'mapping diagram'?
A mapping diagram shows inputs on one side and outputs on the other, with arrows connecting each input to its corresponding output(s).
A phone company charges $0.10 per text message. The total cost is a function of the number of texts sent. If a customer sends 150 texts, what is the total cost? Is the reverse (cost to number of texts) also a function?
Cost = 0.10 x 150 = $15.00. Since each cost uniquely determines the number of texts (divide by 0.10), the reverse is also a function.
Real-World Applications
Example: A vending machine takes a code (A1, A2, B1, B2) and dispenses a specific snack for each code. Is this vending machine an example of a function? Explain why.
Each code maps to exactly one snack, satisfying the definition of a function. It does not matter if two different codes give the same snack; what matters is that each input has one output.
Example: A phone company charges $0.10 per text message. The total cost is a function of the number of texts sent. If a customer sends 150 texts, what is the total cost? Is the reverse (cost to number of texts) also a function?
Cost = 0.10 x 150 = $15.00. Since each cost uniquely determines the number of texts (divide by 0.10), the reverse is also a function.
Example: A social media platform allows one username per account, but multiple accounts can have the same display name. Is the relation from usernames to display names a function?
Each username maps to exactly one display name, which satisfies the function definition. It is fine for multiple inputs to share the same output.
Example: A bank assigns each customer a unique account number. A customer can have multiple accounts. Is the relation from customers to account numbers a function?
Since one customer (input) can have multiple account numbers (outputs), this relation is not a function. A function requires each input to map to exactly one output.
Example: A delivery app assigns each order to one driver, but each driver can handle multiple orders. Is the relation from orders to drivers a function? What about from drivers to orders?
Each order maps to one driver, making orders-to-drivers a function. But one driver can have multiple orders, so drivers-to-orders is not a function.

--- Lesson 4.2: Function Notation (Summary) ---

  Key Concepts: Function Notation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = 3x + 1, what is f(4)?
f(4) = 3(4) + 1 = 12 + 1 = 13.
If g(x) = x^2 - 5, what is g(3)?
g(3) = 3^2 - 5 = 9 - 5 = 4.
If h(x) = -2x + 10, what is h(-3)?
h(-3) = -2(-3) + 10 = 6 + 10 = 16.
If f(x) = 4x and g(x) = x + 3, what is f(g(2))?
First find g(2) = 2 + 3 = 5. Then f(5) = 4(5) = 20.
If f(x) = x^2 + 2x, what is f(0)?
f(0) = 0^2 + 2(0) = 0 + 0 = 0.
If p(t) = -16t^2 + 64, what is p(1)?
p(1) = -16(1)^2 + 64 = -16 + 64 = 48.
If f(x) = 2x + 7, what is f(a + 1)?
f(a + 1) = 2(a + 1) + 7 = 2a + 2 + 7 = 2a + 9.
If f(x) = |x - 4|, what is f(1)?
f(1) = |1 - 4| = |-3| = 3.
If f(x) = 6x - 3, what is f(5) - f(2)?
f(5) = 6(5) - 3 = 27. f(2) = 6(2) - 3 = 9. f(5) - f(2) = 27 - 9 = 18.
A taxi fare is modeled by the function f(m) = 2.50m + 3, where m is miles traveled. A passenger rides 8 miles. What is the fare?
f(8) = 2.50(8) + 3 = 20 + 3 = $23.00.
A ball is dropped from 100 feet. Its height in feet after t seconds is h(t) = -16t^2 + 100. What is the height after 2 seconds?
h(2) = -16(2)^2 + 100 = -16(4) + 100 = -64 + 100 = 36 feet.
A cell phone plan costs $40/month plus $0.05 per text over 500 texts. The cost function for texts over 500 is C(t) = 0.05t + 40, where t is the number of extra texts. If a customer sends 300 extra texts, what is the monthly bill?
C(300) = 0.05(300) + 40 = 15 + 40 = $55.00.
The temperature in degrees Fahrenheit at hour h of the day is modeled by T(h) = -0.5h^2 + 12h + 40, where h = 0 is midnight. What is the temperature at 6 AM (h = 6)?
T(6) = -0.5(6)^2 + 12(6) + 40 = -0.5(36) + 72 + 40 = -18 + 72 + 40 = 94 degrees F.
A population of bacteria is modeled by B(t) = 500(2)^t, where t is the number of hours. What is B(3)?
B(3) = 500(2)^3 = 500(8) = 4000 bacteria.
A discount store has a pricing function where the sale price is S(p) = 0.75p, where p is the original price. If the original price of a jacket is $80, what is the sale price?
S(80) = 0.75(80) = $60. The jacket is 25% off, making the sale price $60.
A roller coaster's height above the ground in feet is modeled by h(t) = -4t^2 + 24t + 10, where t is time in seconds. What is the height at t = 3 seconds?
h(3) = -4(3)^2 + 24(3) + 10 = -4(9) + 72 + 10 = -36 + 72 + 10 = 46 feet.

--- Lesson 4.3: Domain &amp; Range (Summary) ---

  Key Concepts: Domain &amp; Range
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the domain of f(x) = 2x + 5?
A linear function f(x) = 2x + 5 has no restrictions on x, so the domain is all real numbers.
What is the range of f(x) = x^2?
Since squaring any real number gives a non-negative result, the range of f(x) = x^2 is y >= 0.
What is the domain of f(x) = 1/(x - 3)?
Division by zero is undefined, so x - 3 cannot equal 0, meaning x cannot equal 3. The domain is all real numbers except x = 3.
What is the domain of f(x) = sqrt(x)?
The square root function requires a non-negative input, so the domain is x >= 0.
A graph shows a parabola opening upward with its vertex at (0, -4). What is the range?
An upward-opening parabola has its minimum y-value at the vertex. Since the vertex is at (0, -4), the range is y >= -4.
What is the range of f(x) = -x^2 + 9?
This is a downward-opening parabola with vertex at (0, 9). The maximum y-value is 9, so the range is y <= 9.
What is the domain of f(x) = 5/(x^2 - 16)?
x^2 - 16 = 0 when x = 4 or x = -4. Division by zero is undefined at these values, so the domain excludes both.
A function's graph extends infinitely to the left and right but only goes as high as y = 6 and as low as y = -2. What is the range?
The function's y-values are bounded between -2 and 6 inclusive, so the range is -2 <= y <= 6.
A scuba diver's depth below sea level (in feet) is given by d(t) = -3t, where t is time in minutes. The diver can only go to a maximum depth of 60 feet. What is the practical domain?
The diver starts at t = 0. At d = -60 (max depth): -3t = -60, so t = 20. The practical domain is 0 <= t <= 20 minutes.
A rocket's height is modeled by h(t) = -5t^2 + 40t, where t is seconds and h is meters. The rocket is on the ground when h = 0. What is the practical domain?
Set h(t) = 0: -5t^2 + 40t = 0, so -5t(t - 8) = 0, giving t = 0 or t = 8. The rocket launches at t = 0 and lands at t = 8, so the domain is 0 <= t <= 8.
A water tank holds up to 500 gallons. Water drains at 10 gallons per minute. The volume function is V(t) = 500 - 10t. What is the practical range?
The tank starts full at 500 gallons and drains to 0 gallons. The volume cannot be negative, so the practical range is 0 <= V <= 500.
A company produces widgets. The cost function is C(x) = 2x + 500 and the revenue function is R(x) = 8x, where x is the number of widgets. What is the domain restriction if the company can produce at most 1,000 widgets?
The company can produce 0 to 1,000 widgets, and you cannot produce a negative number of widgets. The domain is 0 <= x <= 1000.
A store offers a discount: the sale price is S(p) = p - 0.20p = 0.80p, where p is the original price. If items range from $5 to $500, what is the range of the sale price function?
S(5) = 0.80(5) = $4 and S(500) = 0.80(500) = $400. The range of sale prices is $4 to $400.
A bounce house rental company charges C(h) = 50 + 25h, where h is hours rented. The minimum rental is 2 hours and maximum is 8 hours. What is the range of the cost function?
C(2) = 50 + 25(2) = $100. C(8) = 50 + 25(8) = $250. The range is $100 to $250.
Real-World Applications
Example: A bakery can make at most 200 cupcakes per day. The profit per cupcake is $3, and daily fixed costs are $100. Profit function: P(c) = 3c - 100, where c is cupcakes sold. What are the practical domain and range?
The bakery sells 0 to 200 cupcakes. P(0) = -100 (a loss) and P(200) = 3(200) - 100 = 500. So the range is -100 <= P <= 500.

--- Lesson 4.4: Evaluating &amp; Interpreting Functions (Summary) ---

  Key Concepts: Evaluating &amp; Interpreting Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = 4x - 7, what is f(5)?
f(5) = 4(5) - 7 = 20 - 7 = 13.
If g(x) = x^2 + 3x - 2, what is g(-1)?
g(-1) = (-1)^2 + 3(-1) - 2 = 1 - 3 - 2 = -4.
A function is defined by the table: x = {0, 1, 2, 3}, f(x) = {5, 8, 11, 14}. What is f(2)?
From the table, when x = 2, f(x) = 11.
If h(t) = -16t^2 + 80t + 5, what is h(0)?
h(0) = -16(0)^2 + 80(0) + 5 = 0 + 0 + 5 = 5.
A graph of f(x) shows that f(3) = 7 and f(6) = 13. What is the average rate of change between x = 3 and x = 6?
Average rate of change = (f(6) - f(3)) / (6 - 3) = (13 - 7) / 3 = 6/3 = 2.
If f(x) = |2x - 6|, what is f(1)?
f(1) = |2(1) - 6| = |2 - 6| = |-4| = 4.
If f(x) = 2x + 1, what is f(3) + f(4)?
f(3) = 2(3) + 1 = 7. f(4) = 2(4) + 1 = 9. f(3) + f(4) = 7 + 9 = 16.
If f(x) = x^2 - 4x + 3, what is f(2)?
f(2) = (2)^2 - 4(2) + 3 = 4 - 8 + 3 = -1.
A swimming pool is being drained. The water volume in gallons is modeled by V(t) = 10000 - 250t, where t is time in minutes. What does V(20) represent, and what is its value?
V(20) = 10000 - 250(20) = 10000 - 5000 = 5000 gallons. It represents the water remaining after 20 minutes.
A company's revenue is R(x) = 50x and cost is C(x) = 20x + 3000, where x is units sold. The profit function is P(x) = R(x) - C(x) = 30x - 3000. What does P(150) mean, and what is its value?
P(150) = 30(150) - 3000 = 4500 - 3000 = $1,500. It represents the profit when 150 units are sold.
A ball is thrown upward. Its height is h(t) = -16t^2 + 48t + 4, where t is seconds. What is the height at t = 1 second, and what is the average rate of change from t = 1 to t = 2?
h(1) = -16(1) + 48(1) + 4 = -16 + 48 + 4 = 36. h(2) = -16(4) + 48(2) + 4 = -64 + 96 + 4 = 36. Average rate = (36 - 36)/(2 - 1) = 0 ft/sec.
A car's value depreciates according to V(t) = 25000(0.85)^t, where t is years. What does V(0) = 25000 tell you, and what is V(3)?
V(0) = 25000 is the original price. V(3) = 25000(0.85)^3 = 25000(0.614125) = $15,353.13 approximately.
An online subscription costs $9.99/month after a one-time setup fee of $15. The total cost is C(m) = 9.99m + 15. After 12 months, what is the total cost? What is the average monthly cost over 12 months?
C(12) = 9.99(12) + 15 = 119.88 + 15 = $134.88. Average monthly cost = 134.88/12 = $11.24 per month.
A car travels on a highway. The distance function is d(t) = 65t, where t is hours and d is miles. The driver needs to travel 325 miles. At what time does the driver arrive, and what is d(2.5)?
Set 65t = 325: t = 5 hours. d(2.5) = 65(2.5) = 162.5 miles, which is halfway through the trip.
Real-World Applications
Example: A graph of f(x) shows that f(3) = 7 and f(6) = 13. What is the average rate of change between x = 3 and x = 6?
Average rate of change = (f(6) - f(3)) / (6 - 3) = (13 - 7) / 3 = 6/3 = 2.
Example: What does 'average rate of change' mean?
The average rate of change measures how much the output changes per unit change in input over a specified interval, calculated as (f(b) - f(a)) / (b - a).
Example: A ball is thrown upward. Its height is h(t) = -16t^2 + 48t + 4, where t is seconds. What is the height at t = 1 second, and what is the average rate of change from t = 1 to t = 2?
h(1) = -16(1) + 48(1) + 4 = -16 + 48 + 4 = 36. h(2) = -16(4) + 48(2) + 4 = -64 + 96 + 4 = 36. Average rate = (36 - 36)/(2 - 1) = 0 ft/sec.
Example: An online subscription costs $9.99/month after a one-time setup fee of $15. The total cost is C(m) = 9.99m + 15. After 12 months, what is the total cost? What is the average monthly cost over 12 months?
C(12) = 9.99(12) + 15 = 119.88 + 15 = $134.88. Average monthly cost = 134.88/12 = $11.24 per month.

--- Lesson 4.5: Rate of Change &amp; Slope (Summary) ---

  Key Concepts: Rate of Change &amp; Slope
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the slope of a line that passes through (2, 3) and (6, 11)?
Slope = (11 - 3) / (6 - 2) = 8 / 4 = 2.
A line passes through (-1, 5) and (3, -3). What is its slope?
Slope = (-3 - 5) / (3 - (-1)) = -8 / 4 = -2.
What is the slope of a horizontal line?
A horizontal line has no vertical change, so the rise is 0. Slope = 0 / run = 0.
What is the slope of a vertical line?
A vertical line has no horizontal change (run = 0). Since division by zero is undefined, the slope of a vertical line is undefined.
A table shows: x = {0, 2, 4, 6}, y = {1, 5, 9, 13}. What is the rate of change?
Rate of change = (5 - 1) / (2 - 0) = 4 / 2 = 2. The y-value increases by 4 for every increase of 2 in x, giving a rate of 2.
The equation y = -3x + 7 represents a line. What is its slope?
In slope-intercept form y = mx + b, the coefficient of x is the slope. Here m = -3.
Two lines have slopes of 3/4 and -4/3. What is the relationship between these lines?
Two lines are perpendicular if their slopes are negative reciprocals. (3/4) x (-4/3) = -1, confirming they are perpendicular.
A line rises 6 units for every 2 units it runs to the right. What is the slope?
Slope = rise / run = 6 / 2 = 3.
A road rises 150 feet over a horizontal distance of 3,000 feet. What is the slope (grade) of the road?
Slope = rise / run = 150 / 3000 = 1/20 = 0.05 = 5% grade.
A phone's battery drops from 100% to 20% over 4 hours of use. What is the average rate of change of battery percentage per hour?
Rate of change = (20 - 100) / (4 - 0) = -80 / 4 = -20% per hour. The negative sign indicates the battery is decreasing.
A hiker starts at an elevation of 2,000 feet. After 3 hours, the elevation is 3,500 feet. What is the average rate of elevation change?
Rate of change = (3500 - 2000) / (3 - 0) = 1500 / 3 = 500 feet per hour.
A water tank fills at a rate modeled by the line through (0, 10) and (5, 60), where x is minutes and y is gallons. What is the fill rate, and what does the y-intercept represent?
Slope = (60 - 10) / (5 - 0) = 50 / 5 = 10 gallons per minute. The y-intercept of 10 means there were already 10 gallons in the tank at time 0.
A wheelchair ramp must have a maximum slope of 1/12 (1 inch rise per 12 inches of run). If a doorway is 30 inches above ground, what is the minimum length of the ramp?
With slope = rise/run = 1/12: 30/run = 1/12, so run = 30 x 12 = 360 inches = 30 feet.
A stock's value over 5 days is: Day 1 = $40, Day 2 = $42, Day 3 = $38, Day 4 = $44, Day 5 = $50. What is the average rate of change from Day 1 to Day 5?
Average rate of change = (50 - 40) / (5 - 1) = 10 / 4 = $2.50 per day.
A car travels at a constant speed. At time t = 2 hours, the car is 120 miles from home. At t = 5 hours, the car is 300 miles from home. What is the car's speed (rate of change)?
Rate of change = (300 - 120) / (5 - 2) = 180 / 3 = 60 mph.
Real-World Applications
Example: A phone's battery drops from 100% to 20% over 4 hours of use. What is the average rate of change of battery percentage per hour?
Rate of change = (20 - 100) / (4 - 0) = -80 / 4 = -20% per hour. The negative sign indicates the battery is decreasing.
Example: A hiker starts at an elevation of 2,000 feet. After 3 hours, the elevation is 3,500 feet. What is the average rate of elevation change?
Rate of change = (3500 - 2000) / (3 - 0) = 1500 / 3 = 500 feet per hour.
Example: A stock's value over 5 days is: Day 1 = $40, Day 2 = $42, Day 3 = $38, Day 4 = $44, Day 5 = $50. What is the average rate of change from Day 1 to Day 5?
Average rate of change = (50 - 40) / (5 - 1) = 10 / 4 = $2.50 per day.

--- Lesson 4.6: Arithmetic Sequences as Functions (Summary) ---

  Key Concepts: Arithmetic Sequences as Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the common difference of the arithmetic sequence 3, 7, 11, 15, 19?
Common difference = 7 - 3 = 4. Each term increases by 4.
The first term of an arithmetic sequence is 5 and the common difference is 3. What is the 10th term?
a(n) = a(1) + (n-1)d = 5 + (10-1)(3) = 5 + 27 = 32.
What is the 20th term of the arithmetic sequence where a(1) = -4 and d = 6?
a(20) = -4 + (20-1)(6) = -4 + 114 = 110.
The 5th term of an arithmetic sequence is 23 and the common difference is 4. What is the first term?
a(5) = a(1) + 4d, so 23 = a(1) + 4(4) = a(1) + 16. Therefore a(1) = 7.
What is the recursive formula for an arithmetic sequence with first term 10 and common difference -3?
A recursive formula defines each term from the previous one. With a(1) = 10 and d = -3: a(n) = a(n-1) + (-3) = a(n-1) - 3.
An arithmetic sequence has a(3) = 14 and a(7) = 30. What is the common difference?
From term 3 to term 7 is 4 steps. Change = 30 - 14 = 16. Common difference = 16 / 4 = 4.
What is a(50) for the arithmetic sequence a(n) = -2n + 100?
a(50) = -2(50) + 100 = -100 + 100 = 0.
What is an arithmetic sequence?
An arithmetic sequence has a constant difference between consecutive terms. Each term equals the previous term plus the common difference.
What is the 'common difference' in an arithmetic sequence?
The common difference (d) is the constant value that is added to each term to get the next term. It can be found by subtracting any term from the next term.
What is an 'explicit formula' for a sequence?
An explicit formula calculates any term directly from its position number n without needing to know the previous terms. For arithmetic sequences: a(n) = a(1) + (n-1)d.
What is a 'recursive formula' for a sequence?
A recursive formula defines each term using the previous term. For arithmetic sequences: a(n) = a(n-1) + d, with a specified first term.

a(n) = 50 + 30n. After 12 months: a(12) = 50 + 30(12) = 50 + 360 = $410.
A concert venue sells tickets starting at $50 for row 1. Each row farther from the stage costs $5 less. What is the ticket price for row 8?
a(n) = 50 + (n-1)(-5) = 50 - 5n + 5 = 55 - 5n. a(8) = 55 - 40 = $15.
An employee earns $35,000 in year 1 and receives a $2,000 raise each year. What is the salary in year 10, and what type of function models this?
a(n) = 35000 + (n-1)(2000). a(10) = 35000 + 9(2000) = 35000 + 18000 = $53,000. The constant increase makes this a linear function.

a(n) = 50 + (n-1)(-5) = 50 - 5n + 5 = 55 - 5n. a(8) = 55 - 40 = $15.

--- Lesson 4.8: Function Transformations (Summary) ---

  Key Concepts: Function Transformations
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = x^2, what is g(x) = (x + 2)^2 - 5? Describe the transformation.
The (x + 2) shifts the graph 2 units left, and the -5 shifts it 5 units down.
What is the vertex of g(x) = (x - 1)^2 + 6?
In vertex form (x - h)^2 + k, the vertex is (h, k). Here h = 1 and k = 6, so the vertex is (1, 6).
What is a 'transformation' in the context of functions?
A transformation is a change applied to a function's graph that alters its position (translation), shape (stretch/compression), or orientation (reflection).
What is a 'translation'?
A translation shifts the entire graph up, down, left, or right without changing its size, shape, or orientation.
What is a 'reflection' in function transformations?
A reflection flips the graph across a specified axis, producing a mirror image. Reflecting across the x-axis uses -f(x); reflecting across the y-axis uses f(-x).
What is a 'parent function'?
A parent function is the simplest function in a family (e.g., f(x) = x^2 for quadratics, f(x) = |x| for absolute value). All other functions in the family are transformations of it.
A ball's height is modeled by h(t) = -16t^2 + 64. If the ball is launched from a platform 20 feet higher, what is the new function?
Launching 20 feet higher adds 20 to the function: h(t) = -16t^2 + 64 + 20 = -16t^2 + 84.
A sound wave is modeled by f(t). If the volume doubles, the new model is g(t) = 2f(t). If the wave also shifts 1 second later, what is the combined transformation?
Doubling the volume means multiplying by 2 (vertical stretch). Shifting 1 second later means replacing t with (t - 1) (horizontal shift right). Combined: g(t) = 2f(t - 1).
A parabola f(x) = x^2 is reflected across the x-axis and then shifted up 5. What is the resulting function?
Reflecting across the x-axis gives -x^2. Shifting up 5 adds 5: g(x) = -x^2 + 5.
The temperature function on a normal day is T(h) = -2h^2 + 24h + 40. During a heat wave, every temperature is 8 degrees higher. What is the new function?
Every temperature being 8 degrees higher is a vertical shift up by 8: T(h) = -2h^2 + 24h + 40 + 8 = -2h^2 + 24h + 48.
An architect designs an arch modeled by f(x) = -x^2 + 16. The client wants the arch 3 units taller and centered 2 units to the right. What is the new function?
3 units taller adds 3 to the output: -x^2 + 16 + 3 = -x^2 + 19. Shifting 2 right replaces x with (x - 2): g(x) = -(x - 2)^2 + 19.
A population model is P(t) = 1000(2)^t. If the population starts 5 years later and with twice as many individuals, what is the new model?
Starting 5 years later means replacing t with (t - 5) for a horizontal shift right. Twice as many individuals doubles the initial value to 2000. Result: Q(t) = 2000(2)^(t-5).
Real-World Applications
Example: What does 'image' mean in the context of transformations?
The image is the new graph that results after one or more transformations have been applied to the original (pre-image) graph.

--- Lesson 5.4: Geometric Sequences (Summary) ---

  Key Concepts: Geometric Sequences
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the common ratio of the geometric sequence 3, 12, 48, 192?
Common ratio = 12/3 = 4. Each term is multiplied by 4.
Is the sequence 100, 50, 25, 12.5 geometric? If so, what is the common ratio?
50/100 = 1/2, 25/50 = 1/2, 12.5/25 = 1/2. The constant ratio is 1/2, so it is geometric.
What is the 4th term of the geometric sequence where a(1) = 1000 and r = 0.1?
a(4) = 1000(0.1)^3 = 1000(0.001) = 1.
A geometric sequence has a(2) = 10 and a(5) = 1250. What is the common ratio?
From term 2 to term 5 is 3 multiplications: a(5) = a(2) x r^3. So 1250 = 10 x r^3, r^3 = 125, r = 5.
What is the 8th term of the sequence 2, -6, 18, -54, ...?
r = -6/2 = -3. a(8) = 2(-3)^7 = 2(-2187) = -4374.
What is a geometric sequence?
A geometric sequence is one where each term is found by multiplying the previous term by a fixed number called the common ratio.
What is the 'common ratio' in a geometric sequence?
The common ratio r is the constant multiplier between consecutive terms. It is found by dividing any term by the previous term: r = a(n)/a(n-1).
What is the 'explicit formula' for a geometric sequence?
The explicit formula a(n) = a(1) x r^(n-1) lets you calculate any term directly using the first term and the common ratio.
What is the 'recursive formula' for a geometric sequence?
The recursive formula defines each term from the previous one: a(n) = a(n-1) x r, with a specified first term a(1).
An employee's annual bonus starts at $500 and increases by 10% each year. What is the bonus in year 5?
a(5) = 500(1.10)^4 = 500(1.4641) = $732.05. (Year 1 = $500, so year 5 is after 4 increases.)

--- Lesson 5.5: Compound Interest (Summary) ---

  Key Concepts: Compound Interest
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the result of $1,000 invested at 5% annual interest compounded yearly for 1 year?
A = 1000(1 + 0.05)^1 = 1000(1.05) = $1,050. The principal grows by 5% in one year.
If $500 is invested at 10% annual interest compounded yearly, what is the balance after 2 years?
A = 500(1.10)^2 = 500(1.21) = $605. After year 1 the balance is $550, and 10% of $550 is $55, giving $605.
What is the value of (1 + 0.08/4) when interest is 8% compounded quarterly?
Dividing 0.08 by 4 gives 0.02, so (1 + 0.02) = 1.02. Each quarter the balance grows by 2%.
What is the balance of $800 invested at 4% compounded annually for 3 years? (Round to the nearest cent.)
A = 800(1.04)^3 = 800(1.124864) = $899.89. Each year the interest is calculated on the growing balance.
What is the total amount after investing $1,200 at 3% compounded annually for 2 years?
A = 1200(1.03)^2 = 1200(1.0609) = $1,273.08. The extra $1.08 beyond simple interest comes from compounding.
If P = $400, r = 0.05, n = 2, and t = 1, what is A using A = P(1 + r/n)^(nt)?
A = 400(1 + 0.05/2)^(2x1) = 400(1.025)^2 = 400(1.050625) = $420.25. Semi-annual compounding yields slightly more than annual.
What is compound interest?
Compound interest is interest calculated on both the initial principal and the accumulated interest from previous periods, causing the balance to grow exponentially.
What is the difference between simple interest and compound interest?
Simple interest is computed only on the original principal (I = Prt), whereas compound interest is computed on the principal plus any previously earned interest, leading to exponential growth.
Jordan invests $5,000 at 6% interest compounded semi-annually. What is the balance after 3 years? (Round to the nearest cent.)
A = 5000(1 + 0.06/2)^(2x3) = 5000(1.03)^6 = 5000(1.194052) = $5,970.26. Semi-annual compounding means n = 2, so the rate per period is 3%.
A business takes out a $20,000 loan at 8% annual interest compounded quarterly. What is the total amount owed after 2 years? (Round to the nearest cent.)
A = 20000(1 + 0.08/4)^(4x2) = 20000(1.02)^8 = 20000(1.17165938) = $23,433.19. Quarterly compounding at 8% annual means 2% per quarter over 8 quarters.
Real-World Applications
Example: What does 'annual percentage rate (APR)' mean in the context of compound interest?
The APR is the nominal annual interest rate stated by the lender or bank. It does not reflect the effect of compounding; the actual yield may be higher depending on how often interest compounds.
Example: What does 'annual percentage yield (APY)' mean?
APY (annual percentage yield) is the effective annual rate that reflects the actual amount earned after compounding. It is always equal to or higher than the stated APR when compounding occurs more than once a year.
Important Concepts
Which compounding frequency produces the most interest over the same time period and rate
The more frequently interest is compounded, the more total interest is earned because interest-on-interest accumulates faster. Daily compounding beats monthly, quarterly, and annual.

--- Lesson 5.6: Exponential Equations from Tables &amp; Graphs (Summary) ---

Overview

Key Definitions
A table shows the values (0, 3), (1, 6), (2, 12), (3, 24). What is the common ratio?
Each y-value is multiplied by 2 to get the next: 3 x 2 = 6, 6 x 2 = 12, 12 x 2 = 24. The common ratio is 2.
What is the initial value (y-intercept) of the exponential function passing through (0, 5), (1, 15), (2, 45)?
The initial value of an exponential function y = a(b)^x is the y-value when x = 0. Here, when x = 0, y = 5, so a = 5.
An exponential graph passes through (0, 10) and (1, 5). What is the base of the exponential function?
With y = a(b)^x, a = 10 (from x = 0). At x = 1: 10b = 5, so b = 0.5. The function is y = 10(0.5)^x, representing exponential decay.
A table shows (0, 100), (1, 80), (2, 64), (3, 51.2). What is the decay factor?
The decay factor is the common ratio: 80/100 = 0.8. Each value is 80% of the previous one: 80 x 0.8 = 64, and 64 x 0.8 = 51.2.
Given the points (0, 2) and (3, 54) on an exponential curve y = 2(b)^x, what is b?
At x = 3: 2(b)^3 = 54, so b^3 = 27, meaning b = 3. The equation is y = 2(3)^x.
An exponential graph has a y-intercept of 6 and passes through (2, 24). What is the equation?
The y-intercept gives a = 6. At x = 2: 6b^2 = 24, so b^2 = 4 and b = 2. The equation is y = 6(2)^x.
What is the value of y = 3(2)^x when x = 4?
y = 3(2)^4 = 3(16) = 48. Substitute x = 4 into the equation and evaluate the exponent first.
A table shows (1, 6), (2, 18), (3, 54), (4, 162). What is the value when x = 0?
The common ratio is 18/6 = 3. Going backward from x = 1: y = 6/3 = 2 when x = 0. The equation is y = 2(3)^x.
What is the 'initial value' in an exponential function y = a(b)^x?
The initial value 'a' is the y-intercept of the exponential function. When x = 0, y = a(b)^0 = a(1) = a, so 'a' is the starting value.
A car worth $20,000 depreciates by 15% each year. A table shows its value over time. What is the equation and the car's value after 3 years? (Round to the nearest dollar.)
With 15% depreciation, the decay factor is 1 - 0.15 = 0.85. The equation is y = 20000(0.85)^x. After 3 years: 20000(0.85)^3 = 20000(0.614125) = $12,283 (rounded).
A researcher collects data: (0, 500), (2, 125). If the data follows an exponential pattern y = 500(b)^x, what is the value of b?
At x = 2: 500(b)^2 = 125, so b^2 = 125/500 = 0.25, meaning b = 0.5. The equation is y = 500(0.5)^x.
A graph shows an exponential curve with y-intercept at (0, 8) that passes through (2, 72). A student needs to find the value at x = 3. What is it?
With a = 8 and the point (2, 72): 8b^2 = 72, so b^2 = 9 and b = 3. The equation is y = 8(3)^x. At x = 3: y = 8(3)^3 = 8(27) = 216.

--- Lesson 5.7: Transformations of Exponential Functions (Summary) ---

  Key Concepts: Transformations of Exponential Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the effect of adding 3 to the function y = 2^x to get y = 2^x + 3?
Adding a constant outside the exponential expression shifts the graph vertically. y = 2^x + 3 moves every point on y = 2^x up by 3 units.
What is the horizontal asymptote of y = 2^x - 5?
The parent function y = 2^x has a horizontal asymptote at y = 0. Subtracting 5 shifts the entire graph down 5 units, so the asymptote moves to y = -5.
What is the y-intercept of y = 3(2)^x + 1?
At x = 0: y = 3(2)^0 + 1 = 3(1) + 1 = 4. The y-intercept is the point (0, 4).
What is the effect of y = 5(2)^x compared to y = 2^x?
Multiplying the function by 5 scales all y-values by a factor of 5. This is a vertical stretch; the y-intercept moves from (0,1) to (0,5).
What is the horizontal asymptote of y = 4(3)^x + 7?
In y = a(b)^x + k, the horizontal asymptote is y = k. Here k = 7, so the graph approaches but never reaches y = 7.
What is the y-intercept of y = 2^(x - 1)?
At x = 0: y = 2^(0 - 1) = 2^(-1) = 1/2 = 0.5. The horizontal shift changes the y-intercept from 1 to 0.5.
What is the range of y = 2^x + 4?
The parent function y = 2^x has range y > 0. Adding 4 shifts the graph up 4 units, so the new range is y > 4 (the asymptote moves to y = 4).
A medicine concentration in the blood is modeled by C(t) = 250(0.8)^(t - 1) + 10 mg. What is the concentration at time t = 1?
At t = 1: C(1) = 250(0.8)^(1-1) + 10 = 250(0.8)^0 + 10 = 250(1) + 10 = 260 mg. The horizontal shift means the peak concentration occurs at t = 1.
A researcher models data with y = 4(2)^(x + 1) - 3. What is the y-intercept and horizontal asymptote of this function?
At x = 0: y = 4(2)^(0+1) - 3 = 4(2) - 3 = 8 - 3 = 5. The horizontal asymptote is y = k = -3. So the y-intercept is 5 and the asymptote is y = -3.
A student transforms y = 2^x by reflecting it over the x-axis, shifting it 2 units right, and moving it up 10. What is the resulting equation and its value at x = 2?
Reflecting over x-axis: y = -2^x. Shifting 2 right: y = -2^(x-2). Shifting up 10: y = -2^(x-2) + 10. At x = 2: y = -2^0 + 10 = -1 + 10 = 9.
