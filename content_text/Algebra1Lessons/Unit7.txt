--- Lesson 7.1: Solving by Factoring — GCF &amp; Grouping (Summary) ---

  Notes
Key Concepts: Solving by Factoring — GCF & Grouping

Always look for a Greatest Common Factor (GCF) first. Factor it out before doing anything else.

Example: 6x² + 9x = 3x(2x + 3). Set each factor equal to 0.

Factoring by grouping: for four terms, group into pairs, factor each pair, then factor the common binomial.

The Zero Product Property: if AB = 0, then A = 0 or B = 0.

--- Lesson 7.2: Solving by Factoring — Trinomials (Summary) ---

  Notes
Key Concepts: Solving by Factoring — Trinomials

Factor x² + bx + c: find two numbers that multiply to c and add to b.

Example: x² + 5x + 6 = (x + 2)(x + 3) because 2 × 3 = 6 and 2 + 3 = 5.

For ax² + bx + c (a ≠ 1), use the AC method: find two numbers that multiply to ac and add to b, then factor by grouping.

After factoring, set each factor equal to 0 to solve.

--- Lesson 7.3: Special Products &amp; Factoring Patterns (Summary) ---

  Notes
Key Concepts: Special Products & Factoring Patterns

Difference of squares: a² − b² = (a + b)(a − b).

Perfect square trinomial: a² + 2ab + b² = (a + b)² and a² − 2ab + b² = (a − b)².

Sum/Difference of cubes: a³ + b³ = (a + b)(a² − ab + b²); a³ − b³ = (a − b)(a² + ab + b²).

Recognizing these patterns makes factoring much faster.

--- Lesson 7.4: Completing the Square (Summary) ---

  Key Concepts: Completing the Square
Completing the square rewrites ax² + bx + c in vertex form a(x − h)² + k.

Steps: (1) Move c to the other side. (2) Take half of b, square it, add to both sides. (3) Factor the perfect square. (4) Solve.

Example: x² + 6x = 7 → x² + 6x + 9 = 16 → (x+3)² = 16 → x + 3 = ±4 → x = 1 or x = −7.

If a ≠ 1, divide the entire equation by a first.

  Key Concepts: The Quadratic Formula
The quadratic formula: x = (−b ± √(b²−4ac)) / (2a) solves any equation ax² + bx + c = 0.

--- Lesson 7.6: The Discriminant (Summary) ---

  Key Concepts: The Discriminant
The discriminant is the expression under the square root in the quadratic formula: D = b² − 4ac.

If D > 0: two distinct real solutions. If D = 0: one repeated real solution. If D no real solutions (two complex solutions).

The discriminant tells you how many times the parabola crosses the x-axis.

Example: For x² + 4x + 5 = 0, D = 16 − 20 = −4 < 0 → no real solutions.
