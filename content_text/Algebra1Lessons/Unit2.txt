  Notes
Key Concepts: One-Step & Two-Step Equations

A one-step equation requires a single operation to solve. Use inverse operations: addition â†” subtraction, multiplication â†” division.

Examples: x + 5 = 12 â†’ x = 7; 3x = 18 â†’ x = 6.

A two-step equation requires two operations. Undo addition/subtraction first, then undo multiplication/division.

Example: 2x + 3 = 11 â†’ 2x = 8 â†’ x = 4. Always check by substituting back.

--- Lesson 2.2: Multi-Step Equations (Summary) ---

  Key Concepts: Multi-Step Equations
Multi-step equations may require distributing, combining like terms, and using inverse operations.

Steps: (1) Distribute if needed. (2) Combine like terms on each side. (3) Move variable terms to one side. (4) Solve using inverse operations.

Example: 3(x + 2) âˆ’ 4 = 11 â†’ 3x + 6 âˆ’ 4 = 11 â†’ 3x + 2 = 11 â†’ 3x = 9 â†’ x = 3.

Always check your solution by substituting it back into the original equation.

--- Lesson 2.3: Equations with Variables on Both Sides (Summary) ---

  Key Concepts: Equations with Variables on Both Sides
When variables appear on both sides, move all variable terms to one side and constants to the other.

An equation with no solution simplifies to a false statement like 0 = 5. It is a contradiction.

An equation with infinitely many solutions simplifies to a true statement like 0 = 0. It is an identity.

Example: 4x + 2 = 2x + 10 â†’ 2x = 8 â†’ x = 4.

--- Lesson 2.4: Literal Equations (Summary) ---

  Key Concepts: Literal Equations
A literal equation contains two or more variables. You solve for one variable in terms of the others.

Use the same inverse operations as with regular equations — just treat the other variables as constants.

Examples: Solve d = rt for r → r = d/t. Solve A = ½bh for h → h = 2A/b.

Literal equations are common in science and geometry formulas.

--- Lesson 2.8: Systems of Equations — Substitution &amp; Elimination (Summary) ---

  Notes
Key Concepts: Systems of Equations — Substitution & Elimination

A system of equations is a set of two or more equations with the same variables. The solution is the point(s) that satisfy all equations.

Substitution: Solve one equation for a variable, then substitute into the other.

Elimination: Add or subtract equations to eliminate a variable. Multiply first if needed.

Systems can have one solution (intersecting lines), no solution (parallel lines), or infinitely many solutions (same line).
