  Notes
Key Concepts: Arithmetic & Geometric Sequences Review

Arithmetic sequence: common difference d. Formula: a(n) = a(1) + (nвҲ’1)d.

Geometric sequence: common ratio r. Formula: a(n) = a(1) В· r^(nвҲ’1).

Arithmetic: add the same value each time. Geometric: multiply by the same value each time.

Identify the type by checking if consecutive terms have a common difference or common ratio.

--- Lesson 11.2: Recursive &amp; Explicit Formulas (Summary) ---

  Notes
Key Concepts: Recursive & Explicit Formulas

A recursive formula defines each term based on the previous term(s): a(n) = a(nвҲ’1) + d (arithmetic) or a(n) = a(nвҲ’1) В· r (geometric).

An explicit formula calculates any term directly using its position number n.

Recursive formulas require the initial term and the rule. They are good for computing the next term.

Explicit formulas are more efficient for finding the nth term without computing all preceding terms.

  Key Concepts: Introduction to Series
A series is the sum of the terms of a sequence.

Arithmetic series sum: S(n) = n/2 В· (a(1) + a(n)) or S(n) = n/2 В· (2a(1) + (nвҲ’1)d).

Geometric series sum (finite): S(n) = a(1)(1 вҲ’ rвҒҝ)/(1 вҲ’ r), where r вү  1.

Sigma notation (ОЈ) provides a compact way to write a series: ОЈ from i=1 to n of a(i).
