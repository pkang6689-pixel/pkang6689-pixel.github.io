--- Lesson 4.1: Relations &amp; Functions (Summary) ---

  Notes
Key Concepts: Relations & Functions

A relation is any set of ordered pairs (x, y). The set of x-values is the domain; the set of y-values is the range.

A function is a relation where each input (x) has exactly one output (y).

The vertical line test: if any vertical line crosses the graph more than once, it is NOT a function.

Functions can be represented as tables, graphs, equations, or mappings.

--- Lesson 4.2: Function Notation (Summary) ---

  Key Concepts: Function Notation
Function notation uses f(x) instead of y. Read f(x) as 'f of x.'

f(3) means substitute x = 3 into the function and find the output.

Different letters can name different functions: f(x), g(x), h(x).

Example: If f(x) = 2x + 1, then f(4) = 2(4) + 1 = 9.

--- Lesson 4.3: Domain &amp; Range (Summary) ---

  Notes
Key Concepts: Domain & Range

The domain is the set of all possible input values (x-values) for a function.

The range is the set of all possible output values (y-values).

Restrictions on domain: cannot divide by zero, cannot take the square root of a negative (in real numbers).

Domain and range can be expressed in set-builder notation {x | condition} or interval notation (a, b).

--- Lesson 4.4: Evaluating &amp; Interpreting Functions (Summary) ---

  Notes
Key Concepts: Evaluating & Interpreting Functions

To evaluate a function, substitute the given input value and simplify.

For real-world functions, interpret the input and output in context (what they represent).

Interpreting f(a) = b means: when the input is a, the output is b.

Example: If C(t) = 50t + 200 represents cost over time, C(3) = 350 means the cost at 3 hours is $350.

  9. Q: Evaluate the piecewise function f(x) = {2x if x < 0, x + 1 if x ≥ 0} at x = 3
     A: f(3) = 3 + 1 = 4

  10. Q: What is a common real-world piecewise function?
     A: Cell phone plans with different rates for different usage levels

--- Lesson 4.8: Function Transformations (Summary) ---

  Key Concepts: Function Transformations
Vertical translation: f(x) + k shifts the graph up k units; f(x) − k shifts it down.

Horizontal translation: f(x − h) shifts the graph right h units; f(x + h) shifts it left.

 1, vertical stretch; if 0 Vertical stretch/compression: a·f(x) — if |a| > 1, vertical stretch; if 0 Reflection: −f(x) reflects over the x-axis; f(−x) reflects over the y-axis.
