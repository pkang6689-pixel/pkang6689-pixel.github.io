--- Lesson 4.1: Relations &amp; Functions (Summary) ---

  Key Concepts: Relations &amp; Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a relation?
A relation is simply any set of ordered pairs. It does not have any restrictions on inputs or outputs. A function is a special type of relation with additional rules.
What is a function?
A function is a special type of relation where every input (element of the domain) is paired with exactly one output (element of the range). No input can have more than one output.
What is the domain of a relation?
The domain of a relation is the set of all input values (x-values or first coordinates) in the ordered pairs.
What is the range of a relation?
The range of a relation is the set of all output values (y-values or second coordinates) in the ordered pairs.
What is the vertical line test?
The vertical line test determines whether a graph represents a function. If any vertical line can be drawn that intersects the graph at more than one point, the graph is not a function.
What is a mapping diagram?
A mapping diagram displays the domain values on one side and range values on the other, with arrows showing how each input maps to its corresponding output.
What is an input in a function?
An input is the value from the domain that is substituted into the function. In function notation f(x), the input is x, and the output is f(x).
What is an output in a function?
An output is the value the function produces when a specific input is substituted. It is the y-value in the ordered pair and represents the result of applying the function rule.
What is a one-to-one function?
A one-to-one function (also called an injection) is a function where each output is paired with exactly one input. No two different inputs produce the same output.
A vending machine accepts coins and dispenses one specific snack for each button pressed. Button A gives chips, Button B gives cookies, Button C gives pretzels. Does this vending machine represent a function? What is the domain and range?
The vending machine is a function because each button (input) gives exactly one snack (output). The domain is the set of buttons {A, B, C} and the range is the set of snacks {chips, cookies, pretzels}.
A phone plan charges $0.10 per text message. The total monthly text charge y depends on the number of texts x sent. A customer sends 150 texts. What is y, and does this relationship represent a function?
y = 0.10 * 150 = $15. This is a function because each number of texts (x) produces exactly one charge (y). The rule y = 0.10x assigns one output to each input.
Real-World Applications
Example: A phone plan charges $0.10 per text message. The total monthly text charge y depends on the number of texts x sent. A customer sends 150 texts. What is y, and does this relationship represent a function?
y = 0.10 * 150 = $15. This is a function because each number of texts (x) produces exactly one charge (y). The rule y = 0.10x assigns one output to each input.
Example: A teacher assigns grades based on test scores: 90-100 = A, 80-89 = B, 70-79 = C, 60-69 = D, below 60 = F. A student scores 85. Is the grading system a function from scores to letter grades?
Each specific test score maps to exactly one letter grade. A score of 85 gives exactly one grade: B. Multiple scores mapping to the same grade is fine; the key is each score gives only one grade.
Example: A social media platform shows that a user can have multiple posts, but each post has exactly one author. Consider the relation from posts to authors. Is this a function?
When mapping posts to authors, each post has exactly one author, making this a function. The fact that one author may write multiple posts means it's not one-to-one, but it is still a function.
Example: A GPS system maps each address to a set of coordinates (latitude, longitude). However, some buildings have multiple addresses (like a corner store with two street addresses). Does the relation from addresses to coordinates represent a function?
Each individual address maps to exactly one location (latitude, longitude), so the relation from addresses to coordinates is a function. Multiple addresses mapping to the same coordinates makes it not one-to-one, but it is still a function.

--- Lesson 4.2: Function Notation (Summary) ---

  Key Concepts: Function Notation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = 3x + 2, what is f(4)?
f(4) = 3(4) + 2 = 12 + 2 = 14.
If g(x) = x^2 - 5, what is g(-3)?
g(-3) = (-3)^2 - 5 = 9 - 5 = 4.
If f(x) = -2x + 10, what is f(0)?
f(0) = -2(0) + 10 = 0 + 10 = 10. The output when x = 0 is the y-intercept.
If f(x) = x^2 + 3x, what is f(2) + f(-1)?
f(2) = (2)^2 + 3(2) = 4 + 6 = 10. f(-1) = (-1)^2 + 3(-1) = 1 - 3 = -2. f(2) + f(-1) = 10 + (-2) = 8.
If f(x) = 5x - 3 and g(x) = x + 7, what is f(2) - g(2)?
f(2) = 5(2) - 3 = 10 - 3 = 7. g(2) = 2 + 7 = 9. f(2) - g(2) = 7 - 9 = -2.
If f(x) = |x - 4|, what is f(1)?
f(1) = |1 - 4| = |-3| = 3. The absolute value makes the result positive.
If f(x) = 6x and g(x) = x/2, what is f(g(4))?
First find g(4) = 4/2 = 2. Then f(g(4)) = f(2) = 6(2) = 12.
What is function notation?
Function notation uses the format f(x) to name a function (f) and indicate the input variable (x). It provides a concise way to express the output of a function for any given input.
What is the relationship between f(x) and y?
f(x) and y are equivalent representations of the output of a function. The equation y = 3x + 2 and f(x) = 3x + 2 describe the same function; f(x) simply names the function.
What is a function rule?
A function rule is the mathematical expression or equation that describes the operation(s) performed on the input to produce the output. For example, in f(x) = 3x + 2, the rule is '3x + 2'.
What is the difference between f(x) = 3x and f(3x)?
f(x) = 3x defines the function rule: multiply the input by 3. f(3x) means substituting 3x for x in the function, giving f(3x) = 3(3x) = 9x.
A taxi company uses the function C(m) = 2.50m + 3.00, where m is miles traveled and C is the cost in dollars. A passenger travels 8 miles. What is the fare?
C(8) = 2.50(8) + 3.00 = 20.00 + 3.00 = $23.00.
A ball is dropped from a building, and its height in feet after t seconds is given by h(t) = -16t^2 + 64. What is the height after 1.5 seconds?
h(1.5) = -16(1.5)^2 + 64 = -16(2.25) + 64 = -36 + 64 = 28 feet.
A company's revenue function is R(x) = 15x and cost function is C(x) = 8x + 350, where x is units sold. The profit function is P(x) = R(x) - C(x). What is the profit when 100 units are sold?
P(x) = R(x) - C(x) = 15x - (8x + 350) = 7x - 350. P(100) = 7(100) - 350 = 700 - 350 = $350.
A cell phone plan has a function for monthly cost: C(d) = 0.05d + 30, where d is data usage in MB. In January, a user uses 800 MB. In February, they use 1200 MB. What is C(1200) - C(800)?
C(800) = 0.05(800) + 30 = 40 + 30 = $70. C(1200) = 0.05(1200) + 30 = 60 + 30 = $90. C(1200) - C(800) = $90 - $70 = $20.
A theme park's admission function is A(n) = 45n, where n is the number of guests. A group discount function applies: D(x) = x - 0.1x = 0.9x. If a group of 20 guests gets the discount, what is the total cost? (Find D(A(20)))
A(20) = 45(20) = $900. D(A(20)) = D(900) = 0.9(900) = $810. The discounted total for 20 guests is $810.
Real-World Applications
Example: A cell phone plan has a function for monthly cost: C(d) = 0.05d + 30, where d is data usage in MB. In January, a user uses 800 MB. In February, they use 1200 MB. What is C(1200) - C(800)?
C(800) = 0.05(800) + 30 = 40 + 30 = $70. C(1200) = 0.05(1200) + 30 = 60 + 30 = $90. C(1200) - C(800) = $90 - $70 = $20.

--- Lesson 4.3: Domain &amp; Range (Summary) ---

  Key Concepts: Domain &amp; Range
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the domain of f(x) = 2x + 5?
A linear function like f(x) = 2x + 5 has no restrictions on the input, so the domain is all real numbers.
What is the range of f(x) = x^2?
Since squaring any real number gives a non-negative result, the range of f(x) = x^2 is y >= 0 (all non-negative real numbers).
What is the domain of f(x) = 1/x?
Division by zero is undefined, so x cannot equal 0. The domain of f(x) = 1/x is all real numbers except x = 0.
What is the domain of f(x) = sqrt(x - 3)?
The expression under the square root must be non-negative: x - 3 >= 0, so x >= 3.
A graph shows a parabola opening upward with its vertex at (2, -3). What is the range?
An upward-opening parabola has its minimum at the vertex. Since the vertex is at y = -3, the range is all y-values greater than or equal to -3.
What is the domain of f(x) = 5/(x - 4)?
The denominator cannot be zero: x - 4 cannot equal 0, so x cannot equal 4. The domain is all real numbers except x = 4.
What is the range of f(x) = |x|?
The absolute value of any number is always non-negative. The minimum value is 0 (when x = 0), so the range is y >= 0.
What is the range of the function f(x) = 3 (a constant function)?
A constant function f(x) = 3 always outputs 3 regardless of the input, so the range is just the single value {3}.
What is the domain of a function?
The domain of a function is the complete set of all possible input values (x-values) for which the function produces a valid output.
What is the range of a function?
The range of a function is the complete set of all possible output values (y-values) that result from substituting every domain value into the function.
What is interval notation?
Interval notation uses brackets [ ] for included endpoints and parentheses ( ) for excluded endpoints to compactly describe a continuous range of values, such as [2, 5) or (-infinity, 3].
What is a discrete domain?
A discrete domain consists of individual, separated values rather than a continuous range. For example, the number of students in a class can only be whole numbers like {0, 1, 2, 3, ...}.
What is a continuous domain?
A continuous domain includes all real numbers within an interval with no gaps. For example, time and temperature have continuous domains because they can take any value within a range.
What is a reasonable domain?
A reasonable domain considers the real-world context. For example, mathematically f(x) = 2x is defined for all reals, but if x represents the number of concert tickets, the reasonable domain is whole numbers greater than or equal to 0.
A swimming pool is being filled at a rate of 500 gallons per hour. The pool holds a maximum of 10,000 gallons. The function V(t) = 500t models the volume after t hours. What is the reasonable domain?
Time starts at t = 0. The pool is full when 500t = 10000, so t = 20 hours. The reasonable domain is 0 <= t <= 20 because negative time or filling past capacity does not make sense.
A company's profit function is P(x) = -2x^2 + 120x - 1000, where x is the number of units sold (in tens). The graph is a downward parabola with vertex at x = 30. What is the reasonable domain if the company can sell between 0 and 50 tens of units?
The company can sell between 0 and 50 tens of units, so the reasonable domain is 0 <= x <= 50. Even though the vertex is at x = 30, the company can still sell up to 50 tens of units.
An amusement park ride function gives height in feet: h(t) = -16t^2 + 48t + 5, where t is seconds. The ride starts at t = 0 and the car returns to ground level when h(t) = 0. What is the range of this function within the reasonable domain?
The maximum height occurs at the vertex: t = -48/(2*-16) = 1.5 seconds. h(1.5) = -16(2.25) + 48(1.5) + 5 = -36 + 72 + 5 = 41 feet. The ride starts at h(0) = 5 and returns to ground (h = 0). Range: 0 <= h <= 41.
A cell phone tower can transmit signals within a 25-mile radius. The function S(d) = 100 - 4d models signal strength (%) based on distance d (miles) from the tower. What is the domain, and what is the range?
Distance must be between 0 and 25 miles (tower range). At d = 0: S = 100%. At d = 25: S = 100 - 100 = 0%. Domain: [0, 25], Range: [0, 100].
A subscription box service charges C(n) = 30n - 5, where n is the number of months subscribed. The service offers plans from 1 to 12 months. What is the domain and range?
Since you can only subscribe for whole months (1-12), the domain is discrete: {1, 2, ..., 12}. C(1) = 25, C(2) = 55, ..., C(12) = 355. The range is the corresponding set of discrete values.
A hot air balloon launches from the ground and rises for 2 hours, then descends for 1 hour. The height function is h(t) = -100t^2 + 400t for 0 <= t <= 3 (measured in hundreds of feet). What is the maximum height, and what is the range?
Vertex at t = -400/(2*-100) = 2. h(2) = -100(4) + 400(2) = -400 + 800 = 400. h(0) = 0 and h(3) = -100(9) + 400(3) = -900 + 1200 = 300. The range within [0,3] is [0, 400].
A school fundraiser sells cookies for $3 each. The function R(c) = 3c models revenue. The school has ingredients to make at most 200 cookies, and they cannot sell a negative number. What is the reasonable domain and range?
Cookies must be whole numbers from 0 to 200 (you cannot sell half a cookie or negative cookies). R(0) = $0 and R(200) = $600. The domain is discrete {0, 1, 2, ..., 200} and the range is {0, 3, 6, ..., 600}.
A weather model predicts temperature T (in F) based on hours after midnight: T(h) = -0.5h^2 + 6h + 50 for 0 <= h <= 24. What is the maximum predicted temperature, and at what time does it occur?
Vertex: h = -6/(2*-0.5) = 6 hours (6:00 AM). T(6) = -0.5(36) + 6(6) + 50 = -18 + 36 + 50 = 68 degrees F.
A trampoline park models a jumper's height with h(t) = -4t^2 + 16t, where t is seconds and h is feet above the trampoline. What is the domain if the jumper lands back on the trampoline when h = 0?
Set h(t) = 0: -4t^2 + 16t = 0, -4t(t - 4) = 0, so t = 0 or t = 4. The jumper launches at t = 0 and lands at t = 4. Reasonable domain: [0, 4].
An investment grows according to A(t) = 1000(1.05)^t, where t is years and A is the account balance. The investor plans to keep the money invested for at most 30 years. What is the domain, and what are the approximate minimum and maximum values of the range?
Domain: 0 <= t <= 30 (years invested). A(0) = 1000(1) = $1000 (minimum). A(30) = 1000(1.05)^30 approximately equals 1000(4.322) = $4322 (maximum). Range: approximately [$1000, $4322].

--- Lesson 4.4: Evaluating &amp; Interpreting Functions (Summary) ---

  Key Concepts: Evaluating &amp; Interpreting Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = 4x - 3, what is f(5)?
f(5) = 4(5) - 3 = 20 - 3 = 17.
If g(x) = -2x^2 + 8, what is g(3)?
g(3) = -2(3)^2 + 8 = -2(9) + 8 = -18 + 8 = -10.
If h(x) = 3x + 1, what is h(2a)?
h(2a) = 3(2a) + 1 = 6a + 1. Replace every x in the function with 2a.
A function table shows: f(1) = 5, f(2) = 8, f(3) = 11, f(4) = 14. What is f(5)?
The outputs increase by 3 each time (5, 8, 11, 14, ...). The pattern is f(x) = 3x + 2. So f(5) = 3(5) + 2 = 17.
The function f(x) = -16x^2 + 96x represents the height of a projectile. What is f(3)?
f(3) = -16(3)^2 + 96(3) = -16(9) + 288 = -144 + 288 = 144.
If f(x) = 2^x, what is f(0) + f(1) + f(2)?
f(0) = 2^0 = 1, f(1) = 2^1 = 2, f(2) = 2^2 = 4. Sum = 1 + 2 + 4 = 7.
If f(x) = 3x - 1, what is f(x + 2)?
f(x + 2) = 3(x + 2) - 1 = 3x + 6 - 1 = 3x + 5.
What is a zero of a function?
A zero (or root) of a function is an x-value where f(x) = 0. On the graph, these are the x-intercepts where the function crosses the x-axis.
What is a maximum of a function?
The maximum of a function is the greatest output (y-value) that the function achieves, either within its entire domain (absolute maximum) or within a local interval (local maximum).
What is a minimum of a function?
The minimum of a function is the smallest output (y-value) that the function reaches, representing the lowest point on the graph.
A gym membership costs M(m) = 25m + 50, where m is months. A rival gym offers N(m) = 30m + 20. After how many months do both gyms cost the same, and what is that cost?
Set M(m) = N(m): 25m + 50 = 30m + 20, 30 = 5m, m = 6. M(6) = 25(6) + 50 = 200. Both cost $200 at 6 months.
A store's daily profit from selling t-shirts is P(t) = 8t - 120, where t is the number of shirts sold. What does the x-intercept of this function represent, and what is its value?
Set P(t) = 0: 8t - 120 = 0, 8t = 120, t = 15. The x-intercept means the store must sell 15 shirts to break even (profit = $0). Selling fewer means a loss; more means a profit.
A concert venue's revenue function is R(p) = -50p^2 + 3000p, where p is ticket price in dollars. The venue wants to maximize revenue. What ticket price maximizes revenue, and what is the maximum revenue?
Vertex: p = -3000/(2(-50)) = 3000/100 = $30. R(30) = -50(900) + 3000(30) = -45000 + 90000 = $45,000.
Real-World Applications
Example: The temperature in a refrigerator after a power outage is modeled by T(m) = 0.5m + 35, where m is minutes and T is degrees Fahrenheit. Interpret the slope and y-intercept in context.
The slope 0.5 means the temperature increases by 0.5 degrees Fahrenheit per minute after the outage. The y-intercept 35 means the refrigerator was at 35 degrees F when the power went out (m = 0).

--- Lesson 4.5: Rate of Change &amp; Slope (Summary) ---

  Key Concepts: Rate of Change &amp; Slope
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the slope of the line passing through (2, 5) and (6, 13)?
Slope = (13 - 5)/(6 - 2) = 8/4 = 2.
What is the slope of a vertical line?
A vertical line has a run of 0, which means the slope formula would involve division by zero. Therefore, the slope of a vertical line is undefined.
What is the slope of the line through (-1, 4) and (3, -8)?
Slope = (-8 - 4)/(3 - (-1)) = -12/4 = -3.
A table shows: x = {0, 1, 2, 3}, y = {5, 8, 11, 14}. What is the rate of change?
Rate of change = change in y / change in x = (8 - 5)/(1 - 0) = 3/1 = 3. Each time x increases by 1, y increases by 3.
What is the slope of a line that goes through (4, 7) and (4, 12)?
Both points have x = 4, so this is a vertical line. Slope = (12 - 7)/(4 - 4) = 5/0, which is undefined.
What is the slope of the line y = -4x + 9?
In y = -4x + 9, the coefficient of x is -4, which is the slope.
What is slope?
Slope is a measure of how steep a line is, calculated as the ratio of the vertical change (rise) to the horizontal change (run) between any two points on the line.
What is rate of change?
Rate of change is a ratio that describes how quickly one quantity changes relative to another. In a linear function, the rate of change is constant and equals the slope.
What is the slope formula?
The slope formula is m = (y2 - y1)/(x2 - x1), where (x1, y1) and (x2, y2) are two points on the line. It calculates rise over run.
A hiker climbs a trail and records elevation: at 0 miles the elevation is 2000 ft, at 3 miles it is 3200 ft. What is the average rate of change in elevation per mile?
Average rate of change = (3200 - 2000)/(3 - 0) = 1200/3 = 400 ft per mile.
A car travels at constant speed. After 2 hours, it has gone 110 miles. After 5 hours, it has gone 275 miles. What is the car's speed (rate of change)?
Rate of change = (275 - 110)/(5 - 2) = 165/3 = 55 mph.
A water tank is being drained. At t = 0 minutes it holds 500 gallons, and at t = 20 minutes it holds 200 gallons. What is the rate of change, and what does the sign mean?
Rate of change = (200 - 500)/(20 - 0) = -300/20 = -15 gallons per minute. The negative sign indicates the water level is decreasing (draining).
A ramp for wheelchair access rises 1 foot for every 12 feet of horizontal distance. What is the slope of the ramp? A building code requires a maximum slope of 1/12. Does this ramp meet code?
Slope = rise/run = 1/12. Since the building code requires a maximum slope of 1/12, this ramp meets the requirement exactly.
A student's test scores over 5 tests are: Test 1 = 72, Test 2 = 78, Test 3 = 85, Test 4 = 88, Test 5 = 92. What is the average rate of change in score per test from Test 1 to Test 5?
Average rate of change = (92 - 72)/(5 - 1) = 20/4 = 5 points per test.
A phone battery is at 100% at 8:00 AM and at 25% at 1:00 PM. What is the average rate of battery drain per hour?
Time elapsed: 8 AM to 1 PM = 5 hours. Rate = (25 - 100)/5 = -75/5 = -15% per hour.
A candle is 12 inches tall and burns at a constant rate. After 3 hours it is 9 inches tall. What is the rate of change in height, and when will the candle burn out completely?
Rate = (9 - 12)/(3 - 0) = -3/3 = -1 inch per hour. Height = 12 - 1t. Burns out when 12 - t = 0, t = 12 hours total.
A runner's distance from home is modeled by d(t), where t is minutes. From t = 0 to t = 10, d goes from 0 to 1.5 miles. From t = 10 to t = 20, d stays at 1.5 miles. From t = 20 to t = 30, d goes from 1.5 to 0 miles. What are the rates of change for each segment, and what is the runner doing?
Segment 1: (1.5-0)/(10-0) = 0.15 mi/min running away from home. Segment 2: (1.5-1.5)/(20-10) = 0 mi/min resting. Segment 3: (0-1.5)/(30-20) = -0.15 mi/min running back home.
Real-World Applications
Example: What does average rate of change mean?
The average rate of change between two points is the slope of the secant line connecting them: (f(b) - f(a))/(b - a). It represents the overall rate of change over that interval.
Example: A hiker climbs a trail and records elevation: at 0 miles the elevation is 2000 ft, at 3 miles it is 3200 ft. What is the average rate of change in elevation per mile?

--- Lesson 4.6: Arithmetic Sequences as Functions (Summary) ---

  Key Concepts: Arithmetic Sequences as Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the common difference in the sequence 3, 7, 11, 15, 19?
The common difference is found by subtracting consecutive terms: 7 - 3 = 4, 11 - 7 = 4. The common difference is 4.
What is the 10th term of the arithmetic sequence with a(1) = 5 and d = 3?
a(n) = a(1) + (n-1)d = 5 + (10-1)(3) = 5 + 27 = 32.
The explicit formula for an arithmetic sequence is a(n) = 2n + 7. What is a(1)?
a(1) = 2(1) + 7 = 2 + 7 = 9.
An arithmetic sequence has a(3) = 14 and a(7) = 30. What is the common difference?
From term 3 to term 7 is 4 steps. d = (30 - 14)/(7 - 3) = 16/4 = 4.
An arithmetic sequence can be represented as a linear function. If a(n) = 4n - 1, what is the slope and y-intercept when graphed with n on the x-axis?
a(n) = 4n - 1 is a linear function of n with slope 4 (which equals the common difference) and y-intercept -1.
The 5th term of an arithmetic sequence is 22 and the 1st term is 6. What is the common difference?
From term 1 to term 5 is 4 steps. d = (22 - 6)/(5 - 1) = 16/4 = 4.
What is an arithmetic sequence?
An arithmetic sequence is a sequence of numbers where the difference between any two consecutive terms is constant. This constant is called the common difference (d).
What is the common difference?
The common difference (d) is the constant value that is added to each term to produce the next term. It is found by subtracting any term from the term that follows it.
What is an explicit formula for an arithmetic sequence?
The explicit (or general) formula a(n) = a(1) + (n-1)d allows you to calculate any term directly by plugging in the term number n, the first term a(1), and the common difference d.
What is a recursive formula for an arithmetic sequence?
A recursive formula defines each term using the previous term: a(n) = a(n-1) + d. It requires knowing the first term (or any starting term) and the common difference.
What is a term of a sequence?
A term is a single number in the sequence, identified by its position. For example, in the sequence 3, 7, 11, 15, the first term (a(1)) is 3, the second term (a(2)) is 7, and so on.
What is a finite arithmetic sequence?
A finite arithmetic sequence has a definite number of terms, with a clear beginning (first term) and end (last term), unlike an infinite sequence that continues forever.
What is a sequence?
A sequence is an ordered list of numbers arranged in a definite pattern. Each number in the sequence is called a term, and the order matters.
A phone plan charges $0.20 for the first text and decreases the cost by $0.01 for each subsequent text (minimum $0.05). The costs form a decreasing arithmetic sequence. What is the cost of the 10th text?
a(1) = 0.20, d = -0.01. a(10) = 0.20 + (10-1)(-0.01) = 0.20 - 0.09 = $0.11.
Real-World Applications
Example: A car's odometer reads 15,000 miles on January 1st. The owner drives an average of 1,200 miles per month. Model the odometer reading as an arithmetic sequence and predict the reading on September 1st (after 8 months).
a(1) = 15000, d = 1200. After 8 months: a(9) = 15000 + (9-1)(1200) = 15000 + 9600 = 24,600 miles. (Month 1 = January reading, Month 9 = September.)

A piecewise function uses different formulas or expressions for different parts (intervals) of its domain. Each 'piece' of the function applies to a specific range of input values.

A step function is a special type of piecewise function where each piece is a constant (horizontal segment). The graph looks like a staircase, jumping from one constant value to another.

The greatest integer function (floor function), written floor(x) or [x], returns the largest integer that is less than or equal to x. For example, floor(2.9) = 2 and floor(-1.5) = -2.
What is a discontinuity in a piecewise function?
A discontinuity occurs at a point where the graph has a break, jump, or gap. At this point, the left-hand and right-hand values of the function do not match or the function is not defined.
What is a boundary point in a piecewise function?
A boundary point is the x-value where one piece of the function ends and another begins. It is where the rule or formula changes. Evaluating at boundary points requires checking which piece applies.

The ceiling function returns the smallest integer that is greater than or equal to x. For example, ceiling(2.1) = 3 and ceiling(-1.5) = -1.

  Key Concepts: Function Transformations
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
The graph of f(x) = x^2 is shifted up 3 units. What is the new equation?
A vertical shift up by 3 units adds 3 to the output: f(x) = x^2 + 3.
The graph of f(x) = |x| is shifted 4 units to the right. What is the new equation?
A horizontal shift to the right by 4 replaces x with (x - 4): f(x) = |x - 4|.
The graph of g(x) = x^2 is reflected over the x-axis. What is the new equation?
Reflecting a function over the x-axis negates all the output values: g(x) = -x^2. The parabola opens downward instead of upward.
The function f(x) is vertically stretched by a factor of 3. If f(x) = x^2, what is the new equation?
A vertical stretch by a factor of 3 multiplies the entire function by 3: f(x) = 3x^2. The graph becomes narrower.
What is a transformation of a function?
A transformation is a change applied to a function that alters its graph's position (translation), size (stretch/compression), or orientation (reflection) while maintaining its basic shape.
What is a translation?
A translation shifts the entire graph up, down, left, or right without changing the shape, size, or orientation of the graph.
What is a vertical translation?
A vertical translation shifts the graph up (when adding a positive constant) or down (when subtracting) by changing the output: f(x) + k shifts up k units, f(x) - k shifts down k units.
What is a horizontal translation?
A horizontal translation replaces x with (x - h) to shift right h units, or (x + h) to shift left h units. Note: the direction is opposite the sign inside the parentheses.
What is a reflection of a function?
A reflection flips the graph over a line of reflection. Reflecting over the x-axis negates the output (-f(x)), and reflecting over the y-axis negates the input (f(-x)).
What is a vertical stretch?
A vertical stretch multiplies the output of the function by a factor a > 1 (i.e., a*f(x)). This pulls the graph away from the x-axis, making it appear narrower or taller.
What is a vertical compression?
A vertical compression multiplies the output by a factor between 0 and 1 (i.e., a*f(x) where 0 < a < 1). This pushes the graph toward the x-axis, making it appear wider or flatter.
What is a parent function?
A parent function is the simplest form of a function family. For example, f(x) = x^2 is the parent quadratic function, and f(x) = |x| is the parent absolute value function.
What is the order of transformations when graphing?
When applying multiple transformations, a common order is: (1) horizontal translations, (2) stretches, compressions, and reflections, (3) vertical translations. This order follows from the structure of the equation.
A roller coaster's profile is based on the parent function f(x) = x^2 but is flipped upside down, stretched vertically by a factor of 2, shifted right 5 units, and shifted up 100 units (where 100 represents 100 feet). What is the equation modeling the coaster's peak?
Flipped = negative (-), stretched by 2 = multiply by 2, right 5 = (x-5), up 100 = +100. The equation is g(x) = -2(x-5)^2 + 100.

Making the bridge 20 feet taller is a vertical shift up by 20: add 20 to the function. f(x) = -0.01x^2 + 50 + 20 = -0.01x^2 + 70.
