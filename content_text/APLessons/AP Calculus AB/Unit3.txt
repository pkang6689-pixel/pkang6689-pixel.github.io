--- Lesson 3.1: Chain Rule and Implicit Differentiation (Summary) ---

  Key Concepts: Chain Rule and Implicit Differentiation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[sin(x^3)]?
Chain rule: d/dx[sin(u)] = cos(u)*du/dx. With u = x^3: cos(x^3)*3x^2 = 3x^2*cos(x^3).
For x^2 + y^2 = 16, what is dy/dx?
Implicit differentiation: 2x + 2y*dy/dx = 0, so dy/dx = -x/y.
What is d/dx[(e^x + 1)^5]?
Chain rule: 5(e^x+1)^4 * d/dx(e^x+1) = 5(e^x+1)^4 * e^x = 5e^x(e^x+1)^4.
For xy + y^2 = 6, what is dy/dx?
Differentiate: y + x*dy/dx + 2y*dy/dx = 0. Factor: (x + 2y)*dy/dx = -y. So dy/dx = -y/(x + 2y).
What is d/dx[ln(sin(x))]?
Chain rule: (1/sin(x)) * cos(x) = cos(x)/sin(x) = cot(x).
For e^(xy) = x + y, what is dy/dx at the point (0, 1)?
Differentiate: e^(xy)*(y + x*dy/dx) = 1 + dy/dx. At (0,1): e^0*(1 + 0) = 1 + dy/dx, so 1 = 1 + dy/dx, giving dy/dx = 0.
What is d/dx[cos(e^x)]?
Chain rule: d/dx[cos(u)] = -sin(u)*du/dx = -sin(e^x)*e^x = -e^x*sin(e^x).
What is d/dx[sqrt(tan(x))]?
Chain rule: (1/2)(tan(x))^(-1/2) * sec^2(x) = sec^2(x)/(2*sqrt(tan(x))).
For x^3 + y^3 = 9, what is dy/dx at the point (1, 2)?
Differentiate: 3x^2 + 3y^2*dy/dx = 0, so dy/dx = -x^2/y^2. At (1,2): dy/dx = -1/4.
What is d/dx[e^(sin(2x))]?
Double chain rule: e^(sin(2x)) * d/dx[sin(2x)] = e^(sin(2x)) * cos(2x) * 2 = 2cos(2x)*e^(sin(2x)).
What is a 'composite function'?
A composite function (f o g)(x) = f(g(x)) applies g to x first, then applies f to the result. The chain rule is designed to differentiate such functions.
What is 'Leibniz notation' for the chain rule?
In Leibniz notation, the chain rule is dy/dx = (dy/du)(du/dx). The du terms appear to 'cancel,' which is a helpful way to remember the rule.
A circular swimming pool has the equation x^2 + y^2 = 100 (in meters). A swimmer at position (6, 8) wants to know which direction to swim tangentially. What is the slope of the tangent?
For x^2 + y^2 = 100: dy/dx = -x/y. At (6, 8): dy/dx = -6/8 = -3/4. The swimmer should head in a direction with slope -3/4.
A rollercoaster track follows the curve y^3 + x*y = 10. At the point (1, 2), a safety engineer needs to know the steepness. What is dy/dx?
Differentiate: 3y^2*dy/dx + y + x*dy/dx = 0. At (1,2): 12*dy/dx + 2 + dy/dx = 0, so 13*dy/dx = -2, giving dy/dx = -2/13.
An artist's logo is defined by the curve sin(x + y) = y*cos(x). What is dy/dx at (0, 0)?
Check: sin(0) = 0*cos(0) -> 0 = 0, valid. Differentiate: cos(x+y)(1+dy/dx) = dy/dx*cos(x) - y*sin(x). At (0,0): cos(0)(1+dy/dx) = dy/dx*cos(0) - 0. So 1 + dy/dx = dy/dx, giving 1 = 0, which is impossible. The tangent is vertical at (0,0).
A highway interchange follows the curve x^4 + y^4 = 97. The slope at point (2, 3) determines the banking angle. What is dy/dx there?
Verify: 16 + 81 = 97. Differentiate: 4x^3 + 4y^3*dy/dx = 0. dy/dx = -x^3/y^3. At (2,3): dy/dx = -8/27. Note that -32/108 = -8/27, so both represent the same value.
The path of a comet is modeled by xy - y^2 = 3. At the point (7, 3), what is the slope of the comet's trajectory?
Verify: 7(3) - 9 = 21 - 9 = 12, not 3. Let me use (4, 3): 12 - 9 = 3. Differentiate: y + x*dy/dx - 2y*dy/dx = 0. At (4,3): 3 + 4*dy/dx - 6*dy/dx = 0, so 3 - 2*dy/dx = 0, dy/dx = 3/2.
A camera lens is ground to the elliptical profile x^2/9 + y^2/4 = 1. What is the tangent slope at the point (3cos(pi/3), 2sin(pi/3)) = (3/2, sqrt(3))?
Differentiate: 2x/9 + 2y/4 * dy/dx = 0. dy/dx = -4x/(9y). At (3/2, sqrt(3)): dy/dx = -4(3/2)/(9*sqrt(3)) = -6/(9sqrt(3)) = -2/(3sqrt(3)).
A bridge support has shape y = sin(pi*x/L) embedded in the equation x*sin(y) + y*cos(x) = 0. At the origin (0, 0), what is dy/dx using implicit differentiation?
Differentiate: sin(y) + x*cos(y)*dy/dx + dy/dx*cos(x) - y*sin(x) = 0. At (0,0): sin(0) + 0 + dy/dx*cos(0) - 0 = 0, so dy/dx = 0.
A spacecraft trajectory is defined by e^(y/x) = x + y. Using implicit differentiation, what is dy/dx at (1, 0)?
Verify: e^(0/1) = e^0 = 1, and 1 + 0 = 1. Valid. Differentiate using quotient rule for y/x: e^(y/x)*(x*dy/dx - y)/x^2 = 1 + dy/dx. At (1,0): e^0*(dy/dx - 0)/1 = 1 + dy/dx. So dy/dx = 1 + dy/dx, which gives 0 = 1. This means the tangent may be vertical. Let me re-differentiate more carefully.
A chemical reaction follows the equation x^2*y + x*y^2 = 30, where x and y are concentrations. If the concentration x = 2 and y = 3, what is dy/dx?
Verify: 4(3) + 2(9) = 12 + 18 = 30. Differentiate: 2xy + x^2*dy/dx + y^2 + 2xy*dy/dx = 0. At (2,3): 12 + 4*dy/dx + 9 + 12*dy/dx = 0, so 21 + 16*dy/dx = 0, giving dy/dx = -21/16.

  Q11. What does the chain rule state for composite functions?
    [*] d/dx[f(g(x))] = f'(g(x)) * g'(x)
    [ ] d/dx[f(g(x))] = f'(g(x)) + g'(x)
    [ ] d/dx[f(g(x))] = f(g'(x))
    [ ] d/dx[f(g(x))] = f'(x) * g'(x)
    Explanation: The chain rule states d/dx[f(g(x))] = f'(g(x)) * g'(x). Differentiate the outer function at the inner function, then multiply by the derivative of the inner function.

  Q12. What does 'implicit differentiation' mean?
    [*] Differentiating both sides of an equation that relates x and y without solving for y first
    [ ] Differentiating only explicit functions
    [ ] Using the product rule on all terms
    [ ] Differentiating constants only
    Explanation: Implicit differentiation allows finding dy/dx directly from an equation F(x,y) = 0, treating y as a function of x and applying the chain rule to y-terms.

  Q13. What does 'outer function' mean when applying the chain rule?
    [*] The function that wraps around the inner function
    [ ] The function with the larger domain
    [ ] The function on the right side of the equation
    [ ] The function applied to x directly
    Explanation: In f(g(x)), the outer function f is the one 'wrapping around' the inner function g(x). When differentiating, you first take the derivative of the outer function while keeping the inner function intact.

  Q14. What does 'treating y as a function of x' mean in implicit differentiation?
    [ ] Assuming y is a constant according to widely held views on this topic
    [ ] Setting y equal to x in most scenarios involving function factors
    [ ] Replacing y with an explicit formula within the broader context
    [*] Assuming y depends on x so the chain rule gives dy/dx when differentiating y-terms
    Explanation: When we treat y as a function of x, every time we differentiate a y-term, the chain rule requires us to multiply by dy/dx. For example, d/dx[y^2] = 2y * dy/dx.

  Q15. What is a 'composite function'?
    [ ] A function with multiple variables
    [*] A function formed by applying one function to the output of another
    [ ] A function formed by adding two functions
    [ ] A function that is always increasing
    Explanation: A composite function (f o g)(x) = f(g(x)) applies g to x first, then applies f to the result. The chain rule is designed to differentiate such functions.

  Q16. What does 'horizontal tangent line' mean?
    [ ] A tangent line at the origin
    [ ] A tangent line along the x-axis only
    [*] A tangent line with slope zero
    [ ] A tangent line with an undefined slope
    Explanation: A horizontal tangent line has slope dy/dx = 0. In implicit differentiation, this occurs when the numerator of dy/dx equals zero while the denominator does not.

  Q17. What is 'Leibniz notation' for the chain rule?
    [ ] dy/dx = du/dy * dx/du
    [ ] dy/dx = dy * du * dx
    [ ] dy/dx = dy + du/dx
    [*] dy/dx = (dy/du)(du/dx)
    Explanation: In Leibniz notation, the chain rule is dy/dx = (dy/du)(du/dx). The du terms appear to 'cancel,' which is a helpful way to remember the rule.

  Q18. What does 'isolate dy/dx' mean in implicit differentiation?
    [ ] Substitute a value for dy/dx
    [*] Collect all dy/dx terms on one side and solve for dy/dx
    [ ] Set dy/dx equal to zero
    [ ] Remove dy/dx from the equation
    Explanation: After differentiating implicitly, dy/dx appears in multiple terms. To isolate it, move all dy/dx terms to one side, factor dy/dx out, and divide by the remaining coefficient.

  Q19. What does 'implicit equation' mean?
    [ ] An equation with no solution

    [ ] An equation involving only constants
    [ ] An equation written as y = f(x)
    Explanation: An implicit equation like x^2 + y^2 = 25 or xy + y^3 = 7 relates x and y without explicitly writing y as a function of x. Implicit differentiation is the tool for finding slopes from such equations.

--- Lesson 3.1: Chain Rule and Implicit Differentiation (Flashcards) ---

  1. Q: Define: Limit
     A: The value that a function approaches as the input approaches some value

  2. Q: Define: Derivative
     A: The instantaneous rate of change of a function at a specific point, representing the slope of the tangent line

  3. Q: Define: Integral
     A: The reverse process of differentiation; it represents the accumulation of quantities

  4. Q: Define: Continuity
     A: A property of a function that can be drawn without lifting the pencil, with no breaks or discontinuities

  5. Q: Define: Critical Point
     A: A point where the derivative equals zero or is undefined, often indicating a local maximum or minimum

  6. Q: Define: Antiderivative
     A: A function whose derivative equals the given function

  7. Q: Define: Riemann Sum
     A: An approximation of the area under a curve using rectangles of varying heights

  8. Q: Define: Chain Rule
     A: A method for finding the derivative of composite functions by multiplying the derivatives in sequence

  9. Q: The product rule for derivatives states that if $h(x) = f(x) \cdot g(x)$, then $h'(x) = ?$
     A: $f'(x) \cdot g(x) + f(x) \cdot g'(x)$

  10. Q: The derivative of $f(x) = e^{2x}$ is:
     A: $2e^{2x}$

--- Lesson 3.2: Inverse Function Differentiation (Summary) ---

  Key Concepts: Inverse Function Differentiation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[arcsin(x)]?
The derivative of arcsin(x) is 1/sqrt(1 - x^2) for |x| < 1.
What is d/dx[arctan(x)]?
The derivative of arctan(x) is 1/(1 + x^2), valid for all real x.
What is d/dx[arccos(x)]?
The derivative of arccos(x) is -1/sqrt(1 - x^2). Note it differs from arcsin(x) only by a negative sign.
If f(x) = x^3 + x and f^(-1)(2) = 1, what is (f^(-1))'(2)?
By the inverse function theorem: (f^(-1))'(y) = 1/f'(f^(-1)(y)). f'(x) = 3x^2 + 1. Since f^(-1)(2) = 1: (f^(-1))'(2) = 1/f'(1) = 1/(3+1) = 1/4.
What is d/dx[arctan(3x)]?
Chain rule: d/dx[arctan(u)] = 1/(1+u^2) * du/dx = 1/(1+(3x)^2) * 3 = 3/(1+9x^2).
What is d/dx[arcsin(x/2)]?
Chain rule: 1/sqrt(1-(x/2)^2) * (1/2) = 1/(2*sqrt(1-x^2/4)) = 1/sqrt(4-x^2).
If g is the inverse of f, and f(3) = 7 with f'(3) = 5, what is g'(7)?
By the inverse function theorem: g'(7) = 1/f'(g(7)) = 1/f'(3) = 1/5.
What is d/dx[arctan(e^x)]?
Chain rule: 1/(1+(e^x)^2) * e^x = e^x/(1 + e^(2x)).
If f(x) = 2x + sin(x), what is (f^(-1))'(0)?
f(0) = 0, so f^(-1)(0) = 0. f'(x) = 2 + cos(x). (f^(-1))'(0) = 1/f'(0) = 1/(2+1) = 1/3.
What is d/dx[arcsin(cos(x))] at x = pi/2?
Chain rule: 1/sqrt(1-cos^2(x)) * (-sin(x)) = -sin(x)/|sin(x)|. At x = pi/2: sin(pi/2) = 1 > 0, so the result is -1.
What is the 'inverse function theorem' for derivatives?
The inverse function theorem states (f^(-1))'(y) = 1/f'(f^(-1)(y)), provided f'(f^(-1)(y)) is not zero. The derivative of the inverse is the reciprocal of the derivative of f evaluated at the corresponding x-value.
What is the 'horizontal line test'?
The horizontal line test determines if a function is one-to-one: if no horizontal line crosses the graph more than once, the function has an inverse. This is the graphical criterion for invertibility.
A function f(x) = x^5 + 2x + 1 is strictly increasing. If f(1) = 4, what is (f^(-1))'(4)?
f'(x) = 5x^4 + 2. Since f^(-1)(4) = 1: (f^(-1))'(4) = 1/f'(1) = 1/(5+2) = 1/7.
A temperature conversion function is f(C) = 1.8C + 32 (Celsius to Fahrenheit). What is (f^(-1))'(212)?
f'(C) = 1.8 for all C. By the inverse function theorem: (f^(-1))'(F) = 1/f'(f^(-1)(F)) = 1/1.8 = 5/9 for all F. This confirms the Fahrenheit-to-Celsius rate is 5/9.
An economist uses a supply function S(p) = p^3 + p that gives quantity as a function of price. If S(2) = 10, what is the rate of change of price with respect to quantity at q = 10?
S'(p) = 3p^2 + 1. At p = 2: S'(2) = 12 + 1 = 13. The inverse rate is dp/dq = 1/S'(2) = 1/13.
A physics experiment involves a monotonically increasing function f(x) = e^x + x. Given that f(0) = 1, what is (f^(-1))'(1)?
f'(x) = e^x + 1. Since f^(-1)(1) = 0: (f^(-1))'(1) = 1/f'(0) = 1/(1+1) = 1/2.
A camera autofocus system uses the function d = f(theta) = 50tan(theta) mm. If the focus distance is changing, what is (f^(-1))'(50)?
f'(theta) = 50sec^2(theta). When d = 50: 50 = 50tan(theta), so tan(theta) = 1, theta = pi/4. f'(pi/4) = 50sec^2(pi/4) = 50*2 = 100. (f^(-1))'(50) = 1/100 rad/mm.
A biochemist models enzyme activity as v = f(s) = 100s/(s+5), where s is substrate concentration. If f(5) = 50, what is (f^(-1))'(50)?
f'(s) = [100(s+5) - 100s]/(s+5)^2 = 500/(s+5)^2. At s = 5: f'(5) = 500/100 = 5. So (f^(-1))'(50) = 1/f'(5) = 1/5.

--- Lesson 3.2: Inverse Function Differentiation (Quiz) ---

  Q1. What is d/dx[arcsin(x)]?

    [*] 1/sqrt(1 - x^2)

    Explanation: The derivative of arcsin(x) is 1/sqrt(1 - x^2) for |x| < 1.

    [ ] -1/(1 + x^2)
    [*] 1/(1 + x^2)
    [ ] sec^2(x)
    [ ] 1/(1 - x^2)
    Explanation: The derivative of arctan(x) is 1/(1 + x^2), valid for all real x.

  Q3. What is d/dx[arccos(x)]?
    [ ] -1/sqrt(1 + x^2)
    [*] -1/sqrt(1 - x^2)
    [ ] 1/sqrt(1 - x^2)
    [ ] 1/sqrt(x^2 - 1)
    Explanation: The derivative of arccos(x) is -1/sqrt(1 - x^2). Note it differs from arcsin(x) only by a negative sign.

  Q4. If f(x) = x^3 + x and f^(-1)(2) = 1, what is (f^(-1))'(2)?
    [ ] 1/2
    [*] 1/4
    [ ] 4
    [ ] 2
    Explanation: By the inverse function theorem: (f^(-1))'(y) = 1/f'(f^(-1)(y)). f'(x) = 3x^2 + 1. Since f^(-1)(2) = 1: (f^(-1))'(2) = 1/f'(1) = 1/(3+1) = 1/4.

  Q5. What is d/dx[arctan(3x)]?
    [ ] 1/(1 + 3x^2)
    [*] 3/(1 + 9x^2)
    [ ] 1/(1 + 9x^2)
    [ ] 3/(1 + 3x^2)
    Explanation: Chain rule: d/dx[arctan(u)] = 1/(1+u^2) * du/dx = 1/(1+(3x)^2) * 3 = 3/(1+9x^2).

  Q6. What is d/dx[arcsin(x/2)]?
    [ ] 1/(2*sqrt(1 - x^2))
    [ ] 1/sqrt(1 - x^2/4)
    [*] 1/sqrt(4 - x^2)
    [ ] 2/sqrt(4 - x^2)
    Explanation: Chain rule: 1/sqrt(1-(x/2)^2) * (1/2) = 1/(2*sqrt(1-x^2/4)) = 1/sqrt(4-x^2).

  Q7. If g is the inverse of f, and f(3) = 7 with f'(3) = 5, what is g'(7)?
    [ ] 5
    [ ] 1/7
    [*] 1/5
    [ ] 1/3
    Explanation: By the inverse function theorem: g'(7) = 1/f'(g(7)) = 1/f'(3) = 1/5.

  Q8. What is d/dx[arctan(e^x)]?
    [ ] 1/(1 + e^(2x))
    [*] e^x/(1 + e^(2x))
    [ ] e^x/(1 + e^x)
    [ ] e^(2x)/(1 + e^(2x))
    Explanation: Chain rule: 1/(1+(e^x)^2) * e^x = e^x/(1 + e^(2x)).

  Q9. If f(x) = 2x + sin(x), what is (f^(-1))'(0)?
    [ ] 1
    [*] 1/3
    [ ] 1/2
    [ ] 3
    Explanation: f(0) = 0, so f^(-1)(0) = 0. f'(x) = 2 + cos(x). (f^(-1))'(0) = 1/f'(0) = 1/(2+1) = 1/3.

  Q10. What is d/dx[arcsin(cos(x))] at x = pi/2?
    [ ] 0
    [ ] Undefined
    [*] -1
    [ ] 1
    Explanation: Chain rule: 1/sqrt(1-cos^2(x)) * (-sin(x)) = -sin(x)/|sin(x)|. At x = pi/2: sin(pi/2) = 1 > 0, so the result is -1.

  Q11. What is the 'inverse function theorem' for derivatives?
    [ ] (f^(-1))'(y) = f'(1/y)
    [*] (f^(-1))'(y) = 1/f'(f^(-1)(y))
    [ ] (f^(-1))'(y) = -1/f'(y)
    [ ] (f^(-1))'(y) = f'(y)
    Explanation: The inverse function theorem states (f^(-1))'(y) = 1/f'(f^(-1)(y)), provided f'(f^(-1)(y)) is not zero. The derivative of the inverse is the reciprocal of the derivative of f evaluated at the corresponding x-value.

  Q12. What does 'inverse function' mean?
    [ ] A function that is always decreasing
    [*] A function that undoes another function: if f(a) = b, then f^(-1)(b) = a
    [ ] A function that equals 1/f(x)
    [ ] The negative of a function
    Explanation: An inverse function reverses the input-output relationship. If f(a) = b, then f^(-1)(b) = a. The composition f(f^(-1)(x)) = x and f^(-1)(f(x)) = x for all x in the appropriate domains.

  Q13. What does 'one-to-one' mean for a function to have an inverse?
    [*] Each output value corresponds to exactly one input value
    [ ] The function has exactly one zero
    [ ] The function passes through the point (1, 1)
    [ ] The function equals one at some point
    Explanation: A function is one-to-one (injective) if different inputs always produce different outputs. This is necessary for the inverse to be a function. Graphically, one-to-one functions pass the horizontal line test.

  Q14. What does 'arcsine' (arcsin or sin^(-1)) represent?
    [ ] 1/sin(x), based on conventional assumptions about arcsin
    [*] The inverse of the sine function, giving the angle whose sine is x
    [ ] The reciprocal of sin(x) without accounting for additional
    [ ] sin(x) squared, at least according to standard theory
    Explanation: Arcsine (arcsin(x) or sin^(-1)(x)) is the inverse of the sine function restricted to [-pi/2, pi/2]. It returns the angle in [-pi/2, pi/2] whose sine equals x.

  Q15. What does 'arctangent' (arctan or tan^(-1)) represent?
    [ ] cot(x), which is a common assumption in the field
    [*] The inverse of the tangent function, giving the angle whose tangent is x
    [ ] tan(x) to the power of -1 in most scenarios involving arctangent
    [ ] 1/tan(x) without accounting for additional influences
    Explanation: Arctangent returns the angle in (-pi/2, pi/2) whose tangent is x. It is defined for all real numbers and has horizontal asymptotes at y = +/- pi/2.

  Q16. What does 'restricted domain' mean for inverse trigonometric functions?
    [ ] The domain is restricted to integers only
    [*] The original trig function's domain is limited to an interval where it is one-to-one
    [ ] The function is only defined for positive x
    [ ] The function has no domain
    Explanation: Since trig functions like sin(x) are periodic and not one-to-one over their full domain, we restrict the domain (e.g., [-pi/2, pi/2] for sin) to create a one-to-one function that has a proper inverse.

  Q17. What is the 'horizontal line test'?
    [ ] If the function is horizontal, it has an inverse
    [*] If every horizontal line intersects the graph at most once, the function is one-to-one
    [ ] A test to find horizontal asymptotes
    [ ] If any horizontal line intersects the graph, the function has an inverse
    Explanation: The horizontal line test determines if a function is one-to-one: if no horizontal line crosses the graph more than once, the function has an inverse. This is the graphical criterion for invertibility.

  Q18. What does the graph of an inverse function look like compared to the original?
    [ ] It is a horizontal shift
    [ ] It is identical to the original
    [*] It is a reflection across the line y = x
    [ ] It is a vertical reflection
    Explanation: The graph of f^(-1) is the reflection of f across the line y = x. This is because if (a, b) is on the graph of f, then (b, a) is on the graph of f^(-1).

  Q19. What does 'monotonic' mean for a function?
    [ ] The function oscillates regularly
    [ ] The function has exactly one root
    [ ] The function has a constant value
    [*] The function is either always increasing or always decreasing on its domain
    Explanation: A monotonic function is either entirely non-decreasing or entirely non-increasing. Monotonic functions are automatically one-to-one and therefore have inverses.

  Q20. What does 'bijection' mean?
    [ ] A function that has exactly two values in its range
    [ ] A function that maps every element to itself
    [ ] A function with two inputs
    [*] A function that is both one-to-one (injective) and onto (surjective)
    Explanation: A bijection is a function that is both one-to-one (each output has at most one input) and onto (every element in the codomain is an output). Bijections have true inverses that are also functions.

  Q21. A surveyor measures the angle of elevation to a building top as theta = arctan(h/100), where h is the building height in meters. If h is increasing at 2 m/s during construction, how fast is the angle changing when h = 100 m?
    [ ] 0.02 rad/s
    [ ] 0.1 rad/s
    [*] 0.01 rad/s
    [ ] 0.005 rad/s
    Explanation: d(theta)/dt = 1/(1+(h/100)^2) * (1/100) * dh/dt. At h = 100: d(theta)/dt = 1/(1+1) * (1/100) * 2 = (1/2)(1/50) = 1/100 = 0.01 rad/s.

  Q22. A GPS system calculates an aircraft's bearing using theta = arcsin(d/R), where d is horizontal distance and R = 6371 km (Earth's radius). If the aircraft moves so that d changes at 800 km/h, how fast is theta changing when d = 3185.5 km (approximately R/2)?
    [ ] 800/6371 rad/hr
    [ ] 800/(6371*sqrt(2)) rad/hr
    [ ] 400/6371 rad/hr
    [*] 800/(6371*sqrt(3/4)) rad/hr

  Q23. A robot arm has angle theta = arccos(d/(2L)), where d is the distance to a target and L = 1 m is the arm length. If d is decreasing at 0.5 m/s, how fast is theta changing when d = 1 m?
    [ ] sqrt(3)/2 rad/s
    [ ] 1/sqrt(3) rad/s
    [ ] -1/(2*sqrt(3)) rad/s
    [*] 1/(2*sqrt(3)) rad/s
    Explanation: d(theta)/dt = -1/sqrt(1-(d/2)^2) * (1/2) * dd/dt. At d = 1: d(theta)/dt = -1/sqrt(1-1/4) * (1/2) * (-0.5) = 1/(sqrt(3/4) * 4) = 1/(2*sqrt(3)) rad/s.

  Q24. A function f(x) = x^5 + 2x + 1 is strictly increasing. If f(1) = 4, what is (f^(-1))'(4)?
    [ ] 1/4
    [ ] 7
    [*] 1/7
    [ ] 4
    Explanation: f'(x) = 5x^4 + 2. Since f^(-1)(4) = 1: (f^(-1))'(4) = 1/f'(1) = 1/(5+2) = 1/7.

  Q25. A navigation system uses the function theta(x) = arctan(x/500), where x meters is the offset from a centerline. If a ship drifts at 10 m/s, how fast is the bearing angle changing when x = 500 m?
    [ ] 0.005 rad/s

    [ ] 0.04 rad/s

  Q26. A temperature conversion function is f(C) = 1.8C + 32 (Celsius to Fahrenheit). What is (f^(-1))'(212)?
    [*] 5/9
    [ ] 1/32
    [ ] 1.8
    [ ] 9/5
    Explanation: f'(C) = 1.8 for all C. By the inverse function theorem: (f^(-1))'(F) = 1/f'(f^(-1)(F)) = 1/1.8 = 5/9 for all F. This confirms the Fahrenheit-to-Celsius rate is 5/9.

  Q27. An economist uses a supply function S(p) = p^3 + p that gives quantity as a function of price. If S(2) = 10, what is the rate of change of price with respect to quantity at q = 10?
    [ ] 13

  Q29. A camera autofocus system uses the function d = f(theta) = 50tan(theta) mm. If the focus distance is changing, what is (f^(-1))'(50)?

  Q30. A biochemist models enzyme activity as v = f(s) = 100s/(s+5), where s is substrate concentration. If f(5) = 50, what is (f^(-1))'(50)?

     A: The value that a function approaches as the input approaches some value

  4. Q: Define: Continuity
     A: A property of a function that can be drawn without lifting the pencil, with no breaks or discontinuities

     A: A function whose derivative equals the given function

     A: A method for finding the derivative of composite functions by multiplying the derivatives in sequence

--- Lesson 3.3: Derivatives of Exponential and Logarithmic Functions (Summary) ---

  Key Concepts: Derivatives of Exponential and Logarithmic Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[e^(4x)]?
Chain rule: d/dx[e^(4x)] = e^(4x) * 4 = 4e^(4x).
What is d/dx[ln(3x + 7)]?
Chain rule: d/dx[ln(u)] = u'/u = 3/(3x + 7).
What is d/dx[5^x]?
For any constant base a > 0: d/dx[a^x] = a^x * ln(a). So d/dx[5^x] = 5^x * ln(5).
What is d/dx[ln(x^2 + 4)]?
Chain rule: (1/(x^2+4)) * 2x = 2x/(x^2+4).
What is d/dx[e^(x^2 - 1)]?
Chain rule: e^(x^2-1) * d/dx(x^2-1) = e^(x^2-1) * 2x.
What is d/dx[log_2(x)]?
d/dx[log_a(x)] = 1/(x * ln(a)). So d/dx[log_2(x)] = 1/(x * ln(2)).
What is d/dx[e^(sin(x))]?
Chain rule: e^(sin(x)) * cos(x).
What is d/dx[x * ln(x)] at x = e?
Product rule: d/dx[x*ln(x)] = ln(x) + 1. At x = e: ln(e) + 1 = 1 + 1 = 2.
What is d/dx[3^(2x)]?
Chain rule: 3^(2x) * ln(3) * 2 = 2 * ln(3) * 3^(2x).
What is d/dx[ln|x|] for x not equal to 0?
Remarkably, d/dx[ln|x|] = 1/x for all x not equal to 0. For x > 0: d/dx[ln(x)] = 1/x. For x < 0: d/dx[ln(-x)] = (1/(-x))*(-1) = 1/x.
What is Euler's number e?
e is an irrational number approximately equal to 2.71828... It can be defined as lim(n->inf)(1+1/n)^n and is the unique base for which d/dx[a^x] = a^x.
What is 'doubling time' for exponential growth?
Doubling time is how long it takes for a quantity growing exponentially to double. For y = Ce^(kt), the doubling time is t = ln(2)/k.
What is 'half-life' in the context of exponential decay?
Half-life is the time required for a quantity undergoing exponential decay (y = Ce^(-kt)) to reach half its current value. It equals ln(2)/k and is constant regardless of the current amount.
A sound's intensity level is L = 10*ln(I/I_0)/ln(10) decibels. If I is doubling every second at the current moment when I = I_0, what is dL/dt?
L = 10*log_10(I/I_0). dL/dt = 10/(I*ln(10)) * dI/dt. If I doubles every second, dI/dt = I*ln(2). At I = I_0: dL/dt = 10/(I_0*ln(10)) * I_0*ln(2) = 10*ln(2)/ln(10) dB/s.
A medicine's concentration follows C(t) = 20(e^(-0.1t) - e^(-0.5t)) mg/L. What is the maximum rate of decrease?
C'(t) = 20(-0.1e^(-0.1t) + 0.5e^(-0.5t)) = -2e^(-0.1t) + 10e^(-0.5t). The maximum rate of decrease occurs where C''(t) = 0.
An investment has value V(t) = 1000 * 1.06^t dollars. Using the derivative, what is the growth rate at t = 10 years?
V'(t) = 1000 * 1.06^t * ln(1.06). At t = 10: V'(10) = 1000 * ln(1.06) * 1.06^10 (approximately $104.35/year).
A learning model shows that the number of words a student memorizes follows W(t) = 200(1 - e^(-0.1t)). What is the learning rate after 10 sessions?
W'(t) = 200 * 0.1 * e^(-0.1t) = 20e^(-0.1t). At t = 10: W'(10) = 20e^(-1) words/session (approximately 7.36).

  Key Concepts: Higher Derivatives and Concavity
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = x^4, what is f''(x)?
f'(x) = 4x^3, then f''(x) = 12x^2.
What is the second derivative of f(x) = sin(x)?
f'(x) = cos(x), then f''(x) = -sin(x).
If f(x) = e^(2x), what is f''(x)?
f'(x) = 2e^(2x), then f''(x) = 4e^(2x).
What is the third derivative of f(x) = x^5?
f'(x) = 5x^4, f''(x) = 20x^3, f'''(x) = 60x^2.
If f(x) = ln(x), what is f''(x)?
f'(x) = 1/x = x^(-1), then f''(x) = -x^(-2) = -1/x^2.
What is the inflection point of f(x) = x^3 - 3x^2 + 2?
f'(x) = 3x^2 - 6x, f''(x) = 6x - 6. Setting f''(x) = 0: 6x - 6 = 0, so x = 1. Check: f'' changes sign at x = 1, confirming an inflection point.
What is the second derivative of f(x) = x * e^x?
f'(x) = e^x + xe^x = e^x(x+1). f''(x) = e^x(x+1) + e^x = e^x(x+2).
What is the 'second derivative' of a function?
The second derivative f''(x) is the derivative of f'(x). It measures the rate of change of the rate of change, or how the slope itself is changing.
What is an 'inflection point'?
An inflection point is where the graph changes concavity. At an inflection point, f''(x) = 0 or is undefined, and f'' must actually change sign.
What is 'jerk' in physics?
Jerk is the third derivative of position: j(t) = s'''(t) = a'(t). It measures how quickly acceleration is changing, relevant for ride comfort in vehicles and elevators.
A ball is thrown upward with position s(t) = -16t^2 + 64t + 5 feet. What is the acceleration at any time t?
s'(t) = -32t + 64, s''(t) = -32 ft/s^2. The acceleration is constant at -32 ft/s^2 (due to gravity).
An economist models GDP as G(t) = 100e^(0.03t) trillion dollars. What is the second derivative G''(t), and what does it represent?
G'(t) = 3e^(0.03t), G''(t) = 0.09e^(0.03t). Since G''(t) > 0 for all t, GDP growth is always accelerating (concave up). G''(t) = 0.0009 * 100 * e^(0.03t) = 0.09e^(0.03t).
An engineer tests a spring and finds displacement x(t) = 5sin(4t) cm. What is the maximum acceleration?
x'(t) = 20cos(4t), x''(t) = -80sin(4t). The maximum |acceleration| = 80 cm/s^2, occurring when sin(4t) = +/-1.

--- Lesson 3.5: Related Rates (Summary) ---

  Key Concepts: Related Rates
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If x^2 + y^2 = 25 and dx/dt = 3, what is dy/dt when x = 3 and y = 4?
Differentiate: 2x(dx/dt) + 2y(dy/dt) = 0. Substituting: 2(3)(3) + 2(4)(dy/dt) = 0, so 18 + 8(dy/dt) = 0, dy/dt = -9/4.
What is a 'snapshot' in a related rates problem?
A 'snapshot' is the specific moment described in the problem (e.g., 'when x = 3' or 'at t = 5'). After differentiating, you substitute the snapshot values to find the unknown rate.
Real-World Applications
Example: A streetlight is 12 m high. A person 2 m tall walks away at 1.5 m/s. How fast is the tip of their shadow moving along the ground?
By similar triangles: 12/(x+s) = 2/s, so 12s = 2x + 2s, 10s = 2x, s = x/5. The tip moves at d/dt(x+s) = d/dt(x+x/5) = (6/5)(dx/dt) = (6/5)(1.5) = 1.8 m/s.

--- Lesson 3.6: Logarithmic Differentiation (Summary) ---

  Key Concepts: Logarithmic Differentiation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[x^x] using logarithmic differentiation?
Let y = x^x. ln(y) = x*ln(x). Differentiate: y'/y = ln(x) + 1. So y' = x^x(ln(x) + 1).
Using logarithmic differentiation, what is d/dx[(x+1)^3 * (x+2)^4]?
Let y = (x+1)^3(x+2)^4. ln(y) = 3ln(x+1) + 4ln(x+2). y'/y = 3/(x+1) + 4/(x+2). So y' = (x+1)^3(x+2)^4[3/(x+1) + 4/(x+2)]. Note: the last option is also correct via the product rule.
What is d/dx[x^(sin(x))]?
Let y = x^(sin(x)). ln(y) = sin(x)*ln(x). y'/y = cos(x)*ln(x) + sin(x)/x. So y' = x^(sin(x))[cos(x)*ln(x) + sin(x)/x].
What is d/dx[(2x)^(3x)] using logarithmic differentiation?
Let y = (2x)^(3x). ln(y) = 3x*ln(2x). y'/y = 3*ln(2x) + 3x*(1/x) = 3*ln(2x) + 3 = 3[ln(2x)+1]. So y' = (2x)^(3x)*3[ln(2x)+1].
What is d/dx[x^(1/x)] using logarithmic differentiation?
Let y = x^(1/x). ln(y) = ln(x)/x. y'/y = (1/x * x - ln(x))/x^2 = (1 - ln(x))/x^2. So y' = x^(1/x)(1-ln(x))/x^2.
Using logarithmic differentiation, what is d/dx[(x^2+1)^5 / (x^3-1)^2]?
ln(y) = 5ln(x^2+1) - 2ln(x^3-1). y'/y = 5*2x/(x^2+1) - 2*3x^2/(x^3-1) = 10x/(x^2+1) - 6x^2/(x^3-1). y' = y * [10x/(x^2+1) - 6x^2/(x^3-1)].
What is d/dx[(cos(x))^x]?
Let y = (cos(x))^x. ln(y) = x*ln(cos(x)). y'/y = ln(cos(x)) + x*(-sin(x)/cos(x)) = ln(cos(x)) - x*tan(x). So y' = (cos(x))^x[ln(cos(x)) - x*tan(x)].
What is d/dx[x^(e^x)]?
Let y = x^(e^x). ln(y) = e^x*ln(x). y'/y = e^x*ln(x) + e^x*(1/x) = e^x(ln(x) + 1/x). So y' = x^(e^x)*e^x(ln(x) + 1/x).
If y = (x^2 + 1)^(x^2), what is y'(0)?
ln(y) = x^2*ln(x^2+1). y'/y = 2x*ln(x^2+1) + x^2*2x/(x^2+1). At x = 0: y = 1^0 = 1, y'/1 = 0 + 0 = 0. So y'(0) = 0.
What is 'logarithmic differentiation'?
Logarithmic differentiation involves: (1) taking ln of both sides: ln(y) = ln(f(x)), (2) using log properties to simplify, (3) differentiating both sides, and (4) solving for y'. It is especially useful for functions of the form f(x)^g(x).
A population model uses P(t) = t^t for a certain rapidly-growing organism over a short period. What is P'(2)?
P'(t) = t^t(ln(t) + 1). At t = 2: P'(2) = 2^2(ln(2) + 1) = 4(ln(2) + 1) (approximately 6.77).
A chemical reaction rate is R = k * [A]^2 * [B]^3, where [A] and [B] are concentrations. Using logarithmic differentiation, what is dR/dt in terms of R, d[A]/dt, and d[B]/dt?
ln(R) = ln(k) + 2ln[A] + 3ln[B]. Differentiate: (1/R)*dR/dt = 2*(1/[A])*d[A]/dt + 3*(1/[B])*d[B]/dt. So dR/dt = R[2/A * dA/dt + 3/B * dB/dt].
An economist models utility as U = x^(0.4) * y^(0.6), where x and y are quantities of two goods. If x = 10, y = 20, dx/dt = 2, dy/dt = -1, what is dU/dt?
ln(U) = 0.4*ln(x) + 0.6*ln(y). dU/dt = U[0.4/x * dx/dt + 0.6/y * dy/dt] = U[0.4(2)/10 + 0.6(-1)/20] = U[0.08 - 0.03] = 0.05U.
A physics model uses f(x) = (sin(x))^(cos(x)) for a specialized wave. What is f'(pi/2)?
At x = pi/2: sin(pi/2) = 1, cos(pi/2) = 0. f(pi/2) = 1^0 = 1. ln(f) = cos(x)*ln(sin(x)). f'/f = -sin(x)*ln(sin(x)) + cos(x)*cos(x)/sin(x). At pi/2: f'/1 = -1*ln(1) + 0 = 0. So f'(pi/2) = 0.
A company's revenue is R = p^n * q^m where p = price, q = quantity, n = 1.5, m = 0.8. If price increases at 5%/year and quantity at 3%/year, what is the percentage rate of change of revenue?
ln(R) = 1.5*ln(p) + 0.8*ln(q). (1/R)*dR/dt = 1.5*(1/p)*dp/dt + 0.8*(1/q)*dq/dt = 1.5(0.05) + 0.8(0.03) = 0.075 + 0.024 = 0.099 = 9.9%.
A biologist studies growth with G(t) = t^(sqrt(t)) for a certain cell culture. What is G'(4)?
ln(G) = sqrt(t)*ln(t). G'/G = (1/(2sqrt(t)))*ln(t) + sqrt(t)/t = ln(t)/(2sqrt(t)) + 1/sqrt(t). At t = 4: G = 4^2 = 16. G'/16 = ln(4)/4 + 1/2. Actually G'/16 = ln(4)/(2*2) + 2/4 = ln(4)/4 + 1/2. Hmm, let me recheck: 1/sqrt(4) = 1/2. So G'/16 = ln(4)/4 + 1/2.
An investment banker models compound returns as V(t) = (1 + r(t))^t, where r(t) = 0.05 + 0.01*sin(t). What is the logarithmic derivative (1/V * dV/dt) at t = 0?
ln(V) = t*ln(1+r(t)). d/dt[ln(V)] = ln(1+r) + t*r'/(1+r). At t = 0: r = 0.05, r' = 0.01*cos(0) = 0.01. So d/dt[ln(V)] = ln(1.05) + 0*0.01/1.05 = ln(1.05). Wait -- at t = 0, the second term is 0. So the answer is just ln(1.05). Actually with t=0: ln(1.05) + 0 = ln(1.05).
A pharmacologist models drug potency as P = (C/C_0)^n * e^(-kt), where C = concentration, C_0 = reference, n = 2. Using logarithmic differentiation, what is (1/P)*dP/dt?
ln(P) = 2*ln(C/C_0) - kt = 2*ln(C) - 2*ln(C_0) - kt. (1/P)*dP/dt = 2*(1/C)*dC/dt - k.
A thermal model uses T(x) = x^(ln(x)). What is T'(e)?
ln(T) = (ln(x))^2. T'/T = 2*ln(x)/x. At x = e: T = e^1 = e, T'/e = 2*1/e = 2/e. So T'(e) = e * 2/e = 2.
A materials scientist uses the stress-strain formula sigma = A * epsilon^n, where A = 500 MPa, n = 0.3, and epsilon (strain) is increasing at 0.001/s. What is the rate of change of stress when epsilon = 0.1?
d(sigma)/dt = A * n * epsilon^(n-1) * d(epsilon)/dt = 500 * 0.3 * (0.1)^(-0.7) * 0.001. This equals 0.15 * (0.1)^(-0.7) MPa/s (approximately 0.15 * 5.01 = 0.75 MPa/s).
Real-World Applications
Example: What does 'percentage rate of change' mean?
The percentage rate of change is 100 * f'(x)/f(x). It expresses the relative rate as a percentage. For exponential growth y = Ce^(kt), the percentage rate is 100k%.
Example: A company's revenue is R = p^n * q^m where p = price, q = quantity, n = 1.5, m = 0.8. If price increases at 5%/year and quantity at 3%/year, what is the percentage rate of change of revenue?

    Explanation: Let y = (2x)^(3x). ln(y) = 3x*ln(2x). y'/y = 3*ln(2x) + 3x*(1/x) = 3*ln(2x) + 3 = 3[ln(2x)+1]. So y' = (2x)^(3x)*3[ln(2x)+1].

  Q6. What is d/dx[x^(1/x)] using logarithmic differentiation?
    [*] x^(1/x) * (1 - ln(x))/x^2
    [ ] (1/x) * x^(1/x - 1)
    [ ] x^(1/x) * ln(x)/x^2
    [ ] x^(1/x-1) / x
    Explanation: Let y = x^(1/x). ln(y) = ln(x)/x. y'/y = (1/x * x - ln(x))/x^2 = (1 - ln(x))/x^2. So y' = x^(1/x)(1-ln(x))/x^2.

  Q7. Using logarithmic differentiation, what is d/dx[(x^2+1)^5 / (x^3-1)^2]?
    [ ] [(x^2+1)^5/(x^3-1)^2] * [5/(x^2+1) - 2/(x^3-1)]
    [*] [(x^2+1)^5/(x^3-1)^2] * [10x/(x^2+1) - 6x^2/(x^3-1)]
    [ ] [5(x^2+1)^4 * 2x * (x^3-1)^2 - (x^2+1)^5 * 2(x^3-1)*3x^2] / (x^3-1)^4
    [ ] 10x(x^2+1)^4 / (x^3-1)^2
    Explanation: ln(y) = 5ln(x^2+1) - 2ln(x^3-1). y'/y = 5*2x/(x^2+1) - 2*3x^2/(x^3-1) = 10x/(x^2+1) - 6x^2/(x^3-1). y' = y * [10x/(x^2+1) - 6x^2/(x^3-1)].

  Q8. What is d/dx[(cos(x))^x]?
    [ ] (cos(x))^x * ln(cos(x))
    [*] (cos(x))^x * [ln(cos(x)) - x*tan(x)]
    [ ] x * (cos(x))^(x-1) * (-sin(x))
    [ ] (cos(x))^x * x * sin(x)
    Explanation: Let y = (cos(x))^x. ln(y) = x*ln(cos(x)). y'/y = ln(cos(x)) + x*(-sin(x)/cos(x)) = ln(cos(x)) - x*tan(x). So y' = (cos(x))^x[ln(cos(x)) - x*tan(x)].

  Q9. What is d/dx[x^(e^x)]?
    [ ] x^(e^x) * [e^x/x]
    [ ] e^x * x^(e^x - 1)
    [*] x^(e^x) * e^x * [ln(x) + 1/x]
    [ ] x^(e^x) * e^x * ln(x)
    Explanation: Let y = x^(e^x). ln(y) = e^x*ln(x). y'/y = e^x*ln(x) + e^x*(1/x) = e^x(ln(x) + 1/x). So y' = x^(e^x)*e^x(ln(x) + 1/x).

  Q10. If y = (x^2 + 1)^(x^2), what is y'(0)?
    [ ] 2
    [*] 0
    [ ] 1
    [ ] ln(1)
    Explanation: ln(y) = x^2*ln(x^2+1). y'/y = 2x*ln(x^2+1) + x^2*2x/(x^2+1). At x = 0: y = 1^0 = 1, y'/1 = 0 + 0 = 0. So y'(0) = 0.

  Q11. What is 'logarithmic differentiation'?
    [ ] Using the log rule for integration
    [ ] Differentiating logarithmic functions directly
    [ ] Converting all functions to exponentials
    [*] A technique where you take the natural log of both sides of y = f(x), then differentiate
    Explanation: Logarithmic differentiation involves: (1) taking ln of both sides: ln(y) = ln(f(x)), (2) using log properties to simplify, (3) differentiating both sides, and (4) solving for y'. It is especially useful for functions of the form f(x)^g(x).

  Q12. When is logarithmic differentiation most useful?
    [ ] For simple polynomial functions
    [ ] For integration problems
    [ ] Only for constant exponents
    [*] For products, quotients, and powers where f(x)^g(x) appears
    Explanation: Logarithmic differentiation is most useful for: (1) functions raised to variable powers like x^x, (2) complicated products/quotients that simplify with log properties, and (3) any expression where standard rules are cumbersome.

  Q13. What does 'ln(y)' represent in logarithmic differentiation?
    [ ] The derivative of y under most circumstances involving differentiation
    [ ] The integral of y in most scenarios involving differentiation factors
    [ ] The reciprocal of y, despite what intuition might initially suggest
    [*] The natural logarithm of the function y, used to simplify the expression before differentiating
    Explanation: Taking ln(y) converts products to sums, quotients to differences, and exponents to coefficients via log properties. This makes the subsequent differentiation much simpler.

  Q14. What does the equation (1/y) * y' = ... represent in logarithmic differentiation?
    [ ] The product rule applied to 1/y
    [ ] The inverse function theorem
    [*] The result of differentiating ln(y) using the chain rule: d/dx[ln(y)] = y'/y
    [ ] The quotient rule applied to y
    Explanation: When we differentiate ln(y) with respect to x, the chain rule gives d/dx[ln(y)] = (1/y) * dy/dx = y'/y. This is the key step in logarithmic differentiation.

  Q15. What logarithmic property converts products into sums?
    [ ] ln(a/b) = ln(a) + ln(b)
    [ ] ln(a * b) = ln(a) * ln(b)
    [*] ln(a * b) = ln(a) + ln(b)
    [ ] ln(a + b) = ln(a) * ln(b)
    Explanation: The product rule for logarithms states ln(ab) = ln(a) + ln(b). This is why logarithmic differentiation simplifies products: taking ln turns them into sums, which are easier to differentiate.

  Q16. What logarithmic property converts quotients into differences?
    [*] ln(a/b) = ln(a) - ln(b)
    [ ] ln(a/b) = ln(a) / ln(b)
    [ ] ln(a/b) = ln(a) + ln(b)
    [ ] ln(a - b) = ln(a)/ln(b)
    Explanation: The quotient rule for logarithms states ln(a/b) = ln(a) - ln(b). This converts division into subtraction, making quotients easier to differentiate after taking the natural log.

  Q17. What logarithmic property converts powers into coefficients?
    [ ] ln(a^n) = ln(a)^n
    [ ] ln(a^n) = n + ln(a)
    [ ] ln(a^n) = a * ln(n)
    [*] ln(a^n) = n * ln(a)
    Explanation: The power rule for logarithms states ln(a^n) = n*ln(a). The exponent becomes a multiplier. When n is a function of x (like in x^x), this is essential for logarithmic differentiation.

  Q18. Why can't the power rule d/dx[x^n] = nx^(n-1) be used for x^x?
    [ ] Because the chain rule applies instead according to widely held views on this topic
    [*] Because the power rule requires a constant exponent, and in x^x both the base and exponent depend on x
    [ ] Because x must be positive, at least according to standard theory
    [ ] Because x^x is undefined, which is a common assumption in the field
    Explanation: The power rule applies when the exponent is constant. In x^x, the exponent x varies, so neither the power rule nor the exponential rule d/dx[a^x] = a^x*ln(a) applies directly. Logarithmic differentiation handles this case.

  Q19. What does 'relative rate of change' mean?
    [*] The derivative divided by the function value: f'(x)/f(x)
    [ ] The derivative multiplied by the function value
    [ ] The second derivative
    [ ] The average rate of change
    Explanation: The relative rate of change is f'(x)/f(x) = d/dx[ln(f(x))]. It measures the rate of change relative to the current value, often expressed as a percentage. This is exactly what appears in logarithmic differentiation.

  Q20. What does 'percentage rate of change' mean?
    [ ] The change from 0 to 100
    [ ] f'(x) * 100
    [ ] f(x)/100
    [*] 100 * f'(x)/f(x)
    Explanation: The percentage rate of change is 100 * f'(x)/f(x). It expresses the relative rate as a percentage. For exponential growth y = Ce^(kt), the percentage rate is 100k%.

  Q21. A population model uses P(t) = t^t for a certain rapidly-growing organism over a short period. What is P'(2)?
    [*] 4(ln(2) + 1)
    [ ] 4
    [ ] 2*ln(2) + 2
    [ ] 4*ln(2)
    Explanation: P'(t) = t^t(ln(t) + 1). At t = 2: P'(2) = 2^2(ln(2) + 1) = 4(ln(2) + 1) (approximately 6.77).

  Q22. A chemical reaction rate is R = k * [A]^2 * [B]^3, where [A] and [B] are concentrations. Using logarithmic differentiation, what is dR/dt in terms of R, d[A]/dt, and d[B]/dt?
    [ ] 5R/t
    [ ] 2k*A*B^3*dA/dt + 3k*A^2*B^2*dB/dt
    [ ] R * [2*dA/dt + 3*dB/dt]
    [*] R * [2/(A) * dA/dt + 3/(B) * dB/dt]
    Explanation: ln(R) = ln(k) + 2ln[A] + 3ln[B]. Differentiate: (1/R)*dR/dt = 2*(1/[A])*d[A]/dt + 3*(1/[B])*d[B]/dt. So dR/dt = R[2/A * dA/dt + 3/B * dB/dt].

  Q23. An economist models utility as U = x^(0.4) * y^(0.6), where x and y are quantities of two goods. If x = 10, y = 20, dx/dt = 2, dy/dt = -1, what is dU/dt?
    [*] U * [0.08 - 0.03]
    [ ] U * [0.8 + 0.6]
    [ ] U * [0.04 - 0.03]
    [ ] 0.4*0.6 * U
    Explanation: ln(U) = 0.4*ln(x) + 0.6*ln(y). dU/dt = U[0.4/x * dx/dt + 0.6/y * dy/dt] = U[0.4(2)/10 + 0.6(-1)/20] = U[0.08 - 0.03] = 0.05U.

  Q24. A physics model uses f(x) = (sin(x))^(cos(x)) for a specialized wave. What is f'(pi/2)?
    [ ] 1 * [0 * 0 + 0 * 0] = 0, but need careful analysis
    [*] -1 * 0 = 0
    [ ] -ln(1) = 0
    [ ] 0
    Explanation: At x = pi/2: sin(pi/2) = 1, cos(pi/2) = 0. f(pi/2) = 1^0 = 1. ln(f) = cos(x)*ln(sin(x)). f'/f = -sin(x)*ln(sin(x)) + cos(x)*cos(x)/sin(x). At pi/2: f'/1 = -1*ln(1) + 0 = 0. So f'(pi/2) = 0.

  Q25. A company's revenue is R = p^n * q^m where p = price, q = quantity, n = 1.5, m = 0.8. If price increases at 5%/year and quantity at 3%/year, what is the percentage rate of change of revenue?
    [*] 9.9%
    [ ] 8%
    [ ] 4%
    [ ] 11.5%
    Explanation: ln(R) = 1.5*ln(p) + 0.8*ln(q). (1/R)*dR/dt = 1.5*(1/p)*dp/dt + 0.8*(1/q)*dq/dt = 1.5(0.05) + 0.8(0.03) = 0.075 + 0.024 = 0.099 = 9.9%.

  Q26. A biologist studies growth with G(t) = t^(sqrt(t)) for a certain cell culture. What is G'(4)?
    [ ] 16[ln(4)/4 + 1/2]
    [ ] 4^2 * 2*ln(4)
    [*] 16[ln(4)/4 + 1/4]
    [ ] 16[ln(4)/(2*sqrt(4)) + sqrt(4)/4]
    Explanation: ln(G) = sqrt(t)*ln(t). G'/G = (1/(2sqrt(t)))*ln(t) + sqrt(t)/t = ln(t)/(2sqrt(t)) + 1/sqrt(t). At t = 4: G = 4^2 = 16. G'/16 = ln(4)/4 + 1/2. Actually G'/16 = ln(4)/(2*2) + 2/4 = ln(4)/4 + 1/2. Hmm, let me recheck: 1/sqrt(4) = 1/2. So G'/16 = ln(4)/4 + 1/2.

  Q27. An investment banker models compound returns as V(t) = (1 + r(t))^t, where r(t) = 0.05 + 0.01*sin(t). What is the logarithmic derivative (1/V * dV/dt) at t = 0?
    [ ] 0.05 under most
    [ ] ln(1.05) without
    [ ] 0.06 when
    [*] ln(1.05) + 0.01/(1.05)
    Explanation: ln(V) = t*ln(1+r(t)). d/dt[ln(V)] = ln(1+r) + t*r'/(1+r). At t = 0: r = 0.05, r' = 0.01*cos(0) = 0.01. So d/dt[ln(V)] = ln(1.05) + 0*0.01/1.05 = ln(1.05). Wait -- at t = 0, the second term is 0. So the answer is just ln(1.05). Actually with t=0: ln(1.05) + 0 = ln(1.05).

  Q28. A pharmacologist models drug potency as P = (C/C_0)^n * e^(-kt), where C = concentration, C_0 = reference, n = 2. Using logarithmic differentiation, what is (1/P)*dP/dt?
    [ ] 2*dC/dt - k
    [*] 2*(1/C)*(dC/dt) - k
    [ ] 2*C/C_0 - kt
    [ ] n/C - k
    Explanation: ln(P) = 2*ln(C/C_0) - kt = 2*ln(C) - 2*ln(C_0) - kt. (1/P)*dP/dt = 2*(1/C)*dC/dt - k.

  Q29. A thermal model uses T(x) = x^(ln(x)). What is T'(e)?
    [ ] e^2 (Euler squared)
    [*] 2e/e = 2
    [ ] e (Euler number)
    [ ] 1 (identity value)
    Explanation: ln(T) = (ln(x))^2. T'/T = 2*ln(x)/x. At x = e: T = e^1 = e, T'/e = 2*1/e = 2/e. So T'(e) = e * 2/e = 2.

  Q30. A materials scientist uses the stress-strain formula sigma = A * epsilon^n, where A = 500 MPa, n = 0.3, and epsilon (strain) is increasing at 0.001/s. What is the rate of change of stress when epsilon = 0.1?
    [ ] 0.15 MPa/s
    [ ] 150 * (0.1)^0.3 MPa/s
    [ ] 500 * (0.1)^0.3 * 0.001 MPa/s
    [*] 500 * 0.3 * (0.1)^(-0.7) * 0.001 MPa/s
    Explanation: d(sigma)/dt = A * n * epsilon^(n-1) * d(epsilon)/dt = 500 * 0.3 * (0.1)^(-0.7) * 0.001. This equals 0.15 * (0.1)^(-0.7) MPa/s (approximately 0.15 * 5.01 = 0.75 MPa/s).

--- Lesson 3.6: Logarithmic Differentiation (Flashcards) ---

  1. Q: Define: Limit
     A: The value that a function approaches as the input approaches some value

  2. Q: Define: Derivative
     A: The instantaneous rate of change of a function at a specific point, representing the slope of the tangent line

  3. Q: Define: Integral
     A: The reverse process of differentiation; it represents the accumulation of quantities

  4. Q: Define: Continuity
     A: A property of a function that can be drawn without lifting the pencil, with no breaks or discontinuities

  5. Q: Define: Critical Point
     A: A point where the derivative equals zero or is undefined, often indicating a local maximum or minimum

  6. Q: Define: Antiderivative
     A: A function whose derivative equals the given function

  7. Q: Define: Riemann Sum
     A: An approximation of the area under a curve using rectangles of varying heights

  8. Q: Define: Chain Rule
     A: A method for finding the derivative of composite functions by multiplying the derivatives in sequence

  9. Q: The product rule for derivatives states that if $h(x) = f(x) \cdot g(x)$, then $h'(x) = ?$
     A: $f'(x) \cdot g(x) + f(x) \cdot g'(x)$

  10. Q: The derivative of $f(x) = e^{2x}$ is:
     A: $2e^{2x}$
