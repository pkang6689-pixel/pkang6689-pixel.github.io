--- Lesson 2.1: Applications of Derivatives (Summary) ---

  Key Concepts: Applications of Derivatives
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If s(t) = 3t^2 - 12t + 5 represents position in meters, what is the velocity at t = 3?
Velocity v(t) = s'(t) = 6t - 12. At t = 3: v(3) = 18 - 12 = 6 m/s.
If the position function is s(t) = t^3 - 6t^2, what is the acceleration at t = 2?
v(t) = 3t^2 - 12t, a(t) = 6t - 12. At t = 2: a(2) = 12 - 12 = 0.
A ball thrown upward has h(t) = -16t^2 + 64t. What is the maximum height?
h'(t) = -32t + 64 = 0 at t = 2. h(2) = -16(4) + 64(2) = -64 + 128 = 64 feet.
What is marginal cost?
Marginal cost = C'(x), the derivative of the total cost function. It approximates the cost increase from producing one more unit.
What is a critical point?
Critical points occur where the derivative is zero or undefined. They are candidates for local maxima, local minima, or neither.
What is linear approximation (linearization)?
Linear approximation uses the tangent line at x = a as an estimate for f(x) near a. It works well when x is close to a.
What is a normal line?
The normal line at a point is perpendicular to the tangent. If the tangent slope is m, the normal slope is -1/m.
What is optimization in calculus?
Optimization uses calculus to find extrema: set up an objective function, find critical points using derivatives, and determine which gives the max or min.
What is the First Derivative Test?
If f' changes from + to - at c, it is a local max. If f' changes from - to +, it is a local min. If no sign change, it is neither.
A company's profit is P(x) = -2x^2 + 100x - 200 dollars for x units sold. How many units maximize profit, and what is the maximum profit?
P'(x) = -4x + 100 = 0 at x = 25. P(25) = -2(625) + 100(25) - 200 = -1250 + 2500 - 200 = $1,050.
An engineer uses linear approximation to estimate sqrt(17). Using f(x) = sqrt(x) and a = 16, what is the estimate?
L(x) = 4 + (1/8)(x-16). L(17) = 4 + 1/8 = 4.125. The actual value is about 4.123.

--- Lesson 2.1: Applications of Derivatives (Quiz) ---

  Q1. If s(t) = 3t^2 - 12t + 5 represents position in meters, what is the velocity at t = 3?
    [ ] -6 m/s
    [*] 6 m/s
    [ ] 0 m/s
    [ ] 14 m/s
    Explanation: Velocity v(t) = s'(t) = 6t - 12. At t = 3: v(3) = 18 - 12 = 6 m/s.

  Q2. A particle moves along a line with v(t) = t^2 - 4t + 3. When is the particle at rest?
    [ ] t = 2 only
    [ ] t = 0 and t = 4
    [ ] t = 4 only
    [*] t = 1 and t = 3
    Explanation: At rest means v(t) = 0: t^2 - 4t + 3 = (t-1)(t-3) = 0, so t = 1 and t = 3.

  Q3. The tangent line to y = x^2 at x = 3 has what equation?
    [ ] y = 3x - 9
    [*] y = 6x - 9
    [ ] y = 6x + 9
    [ ] y = 2x + 3
    Explanation: At x = 3: y = 9, y' = 2x = 6. Tangent line: y - 9 = 6(x - 3), so y = 6x - 9.

  Q4. If the position function is s(t) = t^3 - 6t^2, what is the acceleration at t = 2?
    [ ] 12
    [ ] 6
    [ ] -12
    [*] 0
    Explanation: v(t) = 3t^2 - 12t, a(t) = 6t - 12. At t = 2: a(2) = 12 - 12 = 0.

  Q5. The linear approximation of f(x) = sqrt(x) at a = 25 gives an estimate for sqrt(26) of:
    [*] 5.1
    [ ] 5.0
    [ ] 5.2
    [ ] 5.5
    Explanation: L(x) = f(25) + f'(25)(x-25) = 5 + (1/10)(x-25). L(26) = 5 + 0.1 = 5.1.

  Q6. If f'(c) = 0 and f''(c) > 0, then f has a:
    [ ] Neither max nor min
    [ ] Local maximum at x = c
    [*] Local minimum at x = c
    [ ] Inflection point at x = c
    Explanation: By the Second Derivative Test, f'(c) = 0 with f''(c) > 0 means the curve is concave up at c, so c is a local minimum.

  Q7. A rectangle has perimeter 20. What dimensions maximize its area?
    [ ] 1 by 9
    [*] 5 by 5 (a square)
    [ ] 2 by 8
    [ ] 4 by 6
    Explanation: A = x(10-x) where 2x + 2y = 20. A'(x) = 10 - 2x = 0 at x = 5, so y = 5. Maximum area is 25.

  Q8. The normal line to y = x^2 at x = 1 has slope:
    [*] -1/2
    [ ] 1/2
    [ ] -2
    [ ] 2
    Explanation: The tangent slope is f'(1) = 2. The normal line is perpendicular, so its slope is -1/2.

  Q9. If f(x) = x^3 - 3x, find the x-values of the critical points.
    [ ] x = -3 and x = 3
    [*] x = -1 and x = 1
    [ ] x = 0 only
    [ ] x = 3 only
    Explanation: f'(x) = 3x^2 - 3 = 3(x^2-1) = 3(x-1)(x+1) = 0 at x = 1 and x = -1.

  Q10. A ball thrown upward has h(t) = -16t^2 + 64t. What is the maximum height?
    [ ] 48 feet
    [*] 64 feet
    [ ] 32 feet
    [ ] 128 feet
    Explanation: h'(t) = -32t + 64 = 0 at t = 2. h(2) = -16(4) + 64(2) = -64 + 128 = 64 feet.

  Q11. What is marginal cost?
    [ ] The total cost divided by the number of units
    [*] The derivative of the cost function, representing the approximate cost of producing one additional unit
    [ ] The cost of the last unit produced
    [ ] The fixed cost of production
    Explanation: Marginal cost = C'(x), the derivative of the total cost function. It approximates the cost increase from producing one more unit.

  Q12. What is a critical point?
    [ ] The maximum value of f, particularly in the context of critical
    [ ] A point where f is discontinuous without accounting for additional
    [ ] Any point where f(x) = 0, despite what intuition might initially
    [*] A point where f'(x) = 0 or f'(x) does not exist, where local extrema may occur
    Explanation: Critical points occur where the derivative is zero or undefined. They are candidates for local maxima, local minima, or neither.

  Q13. What is linear approximation (linearization)?
    [ ] Computing the second derivative
    [ ] Finding the equation of a secant line
    [*] Using the tangent line at a known point to estimate function values nearby: L(x) = f(a) + f'(a)(x - a)
    [ ] Replacing a curve with a straight line everywhere
    Explanation: Linear approximation uses the tangent line at x = a as an estimate for f(x) near a. It works well when x is close to a.

  Q14. What does the Second Derivative Test determine?
    [ ] The concavity of the function everywhere
    [*] Whether a critical point is a local max (f'' < 0), local min (f'' > 0), or inconclusive (f'' = 0)
    [ ] Whether the function is continuous
    [ ] The location of inflection points
    Explanation: At a critical point c: if f''(c) > 0, it is a local min (concave up); if f''(c) < 0, local max (concave down); if f''(c) = 0, the test is inconclusive.

  Q15. What is a normal line?
    [ ] The horizontal line through a point
    [*] A line perpendicular to the tangent line at the point of tangency
    [ ] Any line that crosses the curve
    [ ] A line parallel to the tangent line
    Explanation: The normal line at a point is perpendicular to the tangent. If the tangent slope is m, the normal slope is -1/m.

  Q16. What does it mean for a particle to be 'at rest'?
    [ ] Its acceleration is zero
    [ ] Its position is zero
    [*] Its velocity is zero at that instant, meaning v(t) = s'(t) = 0
    [ ] It has returned to its starting point
    Explanation: A particle is at rest when its instantaneous velocity v(t) = 0. This may be momentary; the particle can be accelerating while momentarily at rest.

  Q17. What is optimization in calculus?
    [ ] Making a function as simple as possible
    [ ] Finding the zeros of a function
    [ ] Computing the integral of a function
    [*] Using derivatives to find the maximum or minimum value of a function subject to given constraints
    Explanation: Optimization uses calculus to find extrema: set up an objective function, find critical points using derivatives, and determine which gives the max or min.

  Q18. What does acceleration represent as a derivative?
    [*] The derivative of velocity (second derivative of position), measuring the rate of change of velocity
    [ ] The first derivative of position, even when conditions shift significantly
    [ ] The average velocity under most circumstances involving acceleration
    [ ] The integral of position, even when conditions shift significantly
    Explanation: Acceleration a(t) = v'(t) = s''(t). It measures how quickly velocity changes: positive means speeding up (in the positive direction).

  Q19. What is the First Derivative Test?
    [ ] Checking if the first derivative exists, based on conventional assumptions about changes
    [ ] A test for continuity because the underlying forces at work tend to produce exactly this type
    [ ] Finding where f'(x) = 0, based on conventional assumptions about changes
    [*] A method using the sign change of f' around a critical point to classify it as a local max, local min, or neither
    Explanation: If f' changes from + to - at c, it is a local max. If f' changes from - to +, it is a local min. If no sign change, it is neither.

  Q20. What does 'related rates' mean?
    [*] Problems where two or more quantities change with respect to time, and their rates of change are connected through an equation
    [ ] Finding the derivative of a composite function
    [ ] Problems involving only position and velocity
    [ ] Computing the ratio of two derivatives
    Explanation: Related rates problems involve differentiating an equation relating multiple changing quantities with respect to time to find unknown rates.

  Q21. A company's profit is P(x) = -2x^2 + 100x - 200 dollars for x units sold. How many units maximize profit, and what is the maximum profit?
    [ ] 100 units for $0 profit
    [ ] 10 units for $600 profit
    [*] 25 units for $1,050 profit
    [ ] 50 units for $2,300 profit
    Explanation: P'(x) = -4x + 100 = 0 at x = 25. P(25) = -2(625) + 100(25) - 200 = -1250 + 2500 - 200 = $1,050.

  Q22. A ladder 10 ft long leans against a wall. The base slides away at 2 ft/s. How fast is the top sliding down when the base is 6 ft from the wall?
    [ ] -4/3 ft/s under most
    [ ] -1 ft/s, even when conditions
    [ ] -2 ft/s, as conventional base
    [*] -3/2 ft/s (sliding down at 1.5 ft/s)
    Explanation: x^2 + y^2 = 100. Differentiate: 2x(dx/dt) + 2y(dy/dt) = 0. When x = 6, y = 8. So 12(2) + 16(dy/dt) = 0, dy/dt = -24/16 = -3/2 ft/s.

  Q23. A farmer wants to fence a rectangular area next to a river (no fence needed on the river side) with 200 meters of fencing. What dimensions maximize the area?
    [ ] 100 m by 100 m when viewed through
    [ ] 66.7 m by 66.7 m according to widely held
    [*] 50 m by 100 m, with the 100 m side along the river
    [ ] 40 m by 120 m, especially during periods
    Explanation: Let x = width (two sides) and y = length (river side). 2x + y = 200, so y = 200 - 2x. A = xy = x(200-2x). A'(x) = 200 - 4x = 0 at x = 50, y = 100.

  Q24. A car's position is s(t) = t^3 - 9t^2 + 24t meters. During what interval(s) is the car moving in the positive direction?
    [ ] (0, 4) only
    [ ] (2, 4) only
    [*] (0, 2) and (4, infinity)
    [ ] Always positive
    Explanation: v(t) = 3t^2 - 18t + 24 = 3(t-2)(t-4). v > 0 when t < 2 or t > 4. The car moves positively on (0,2) and (4, infinity).

  Q25. A doctor models a patient's temperature as T(t) = 98.6 + 2*e^(-0.5t) degrees F after t hours of treatment. At what rate is the temperature changing at t = 2 hours?
    [ ] 0 degrees F per hour
    [ ] -2 degrees F per hour
    [ ] -1 degree F per hour
    [*] -e^(-1) which is approximately -0.368 degrees F per hour
    Explanation: T'(t) = 2*(-0.5)*e^(-0.5t) = -e^(-0.5t). At t = 2: T'(2) = -e^(-1) which is about -0.368 degrees F/hr. Temperature is decreasing.

  Q26. A spherical balloon is being inflated at a rate of 100 cm^3/s. How fast is the radius increasing when the radius is 5 cm?
    [ ] 4 cm/s
    [ ] 100/(4*pi) cm/s
    [*] 1/pi cm/s
    [ ] 20/pi cm/s
    Explanation: V = (4/3)*pi*r^3. dV/dt = 4*pi*r^2*(dr/dt). 100 = 4*pi*(25)*(dr/dt). dr/dt = 100/(100*pi) = 1/pi cm/s.

  Q27. An engineer uses linear approximation to estimate sqrt(17). Using f(x) = sqrt(x) and a = 16, what is the estimate?
    [*] 4.125
    [ ] 4.0
    [ ] 4.5
    [ ] 4.25
    Explanation: L(x) = 4 + (1/8)(x-16). L(17) = 4 + 1/8 = 4.125. The actual value is about 4.123.

  Q28. A box with a square base and no top has volume 32 cubic inches. What dimensions minimize the surface area?
    [ ] Base 8 in by 8 in, height 0.5 in
    [ ] Base 2 in by 2 in, height 8 in
    [ ] Base 3 in by 3 in, height 32/9 in
    [*] Base 4 in by 4 in, height 2 in
    Explanation: V = x^2*h = 32, so h = 32/x^2. SA = x^2 + 4xh = x^2 + 128/x. SA' = 2x - 128/x^2 = 0 gives x^3 = 64, x = 4, h = 2.

  Q29. A population of bacteria grows as P(t) = 1000*e^(0.2t). A biologist needs to know when the population is growing at 400 bacteria per hour. At what time does this occur?
    [ ] t = 2 hours, especially during periods
    [ ] t = 5 hours, despite what intuition
    [*] t = 5*ln(2) which is approximately 3.47 hours
    [ ] t = 10 hours, even when conditions shift
    Explanation: P'(t) = 200*e^(0.2t) = 400. e^(0.2t) = 2. 0.2t = ln(2). t = 5*ln(2) which is about 3.47 hours.

  Q30. A spotlight on the ground is 20 m from a wall. A person 2 m tall walks toward the wall at 1 m/s. How fast is the length of their shadow on the wall changing when they are 10 m from the wall?
    [ ] The shadow is shrinking at 1 m/s
    [ ] The shadow is shrinking at 2 m/s
    [ ] The shadow is growing at 0.4 m/s
    [*] The shadow is shrinking at 0.4 m/s
    Explanation: By similar triangles: s/20 = 2/x where x is distance from spotlight. s = 40/x. ds/dt = -40/x^2 * dx/dt. Person is 10m from wall, so x = 10. dx/dt = -1 (approaching wall). ds/dt = -40/100 * (-1) = 0.4... Wait: the person walks toward the wall, so away from the spotlight means x increases. At 10m from wall, x = 10 from the light? Actually x = distance from light = 20-10 = 10m from light. ds/dt = -40/100*1 = -0.4 m/s.

--- Lesson 2.2: Integrals (Summary) ---

  Key Concepts: Integrals
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is integral from 0 to 2 of x dx?
integral of x dx = x^2/2. Evaluated from 0 to 2: 4/2 - 0/2 = 2.
What is integral from 1 to 3 of 2x dx?
Antiderivative is x^2. Evaluate: 3^2 - 1^2 = 9 - 1 = 8.
What is integral of x^(-2) dx for x != 0?
Using the power rule for integration: x^(-2+1)/(-2+1) = x^(-1)/(-1) = -1/x + C.
What is integral of sqrt(x) dx?
sqrt(x) = x^(1/2). integral = x^(3/2)/(3/2) = (2/3)x^(3/2) + C.
If F'(x) = f(x) and integral from a to b of f(x) dx = 10, what is F(b) - F(a)?
By the Fundamental Theorem of Calculus, integral from a to b of f(x) dx = F(b) - F(a) = 10.
What is an antiderivative?
An antiderivative F of f satisfies F'(x) = f(x). For example, x^3/3 is an antiderivative of x^2 because d/dx(x^3/3) = x^2.
What is a definite integral?
A definite integral has bounds a and b and produces a number representing the net signed area between f(x) and the x-axis from x = a to x = b.
What is an indefinite integral?
An indefinite integral has no bounds and includes + C (constant of integration) because many functions share the same derivative.
What is the constant of integration C?
Since d/dx(C) = 0, any constant can be added to an antiderivative without changing its derivative. The + C accounts for all possible antiderivatives.
What is the power rule for integration?
The power rule for integration increases the exponent by 1 and divides by the new exponent. The case n = -1 gives ln|x| + C instead.
What is a Riemann sum?
A Riemann sum divides [a,b] into subintervals and sums f(x_i*)*delta_x for sample points x_i*. As the partition gets finer, it approaches the definite integral.
What is the relationship between differentiation and integration?
The FTC establishes that differentiation and integration are inverse processes: d/dx[integral from a to x of f(t)dt] = f(x).
A runner's speed is v(t) = 12 - 0.5t ft/s. What is the total distance covered from t = 0 to t = 20 seconds?
integral from 0 to 20 of (12-0.5t) dt = [12t - 0.25t^2] from 0 to 20 = 240 - 100 = 140 feet.
A company's marginal revenue is R'(x) = 50 - 0.4x dollars per unit. What is the total revenue from selling the first 100 units?
R(100) = integral from 0 to 100 of (50-0.4x) dx = [50x - 0.2x^2] from 0 to 100 = 5000 - 2000 = $3,000.
A city's population grows at a rate of dP/dt = 500 + 20t people per year. If the population is 50,000 at t = 0, what is it after 10 years?
Population increase = integral from 0 to 10 of (500+20t) dt = [500t+10t^2] from 0 to 10 = 5000+1000 = 6000. Total: 50000+6000 = 56,000.

  Q1. Evaluate the integral: integral of 3x^2 dx.
    [*] x^3 + C
    [ ] x^3
    [ ] 6x + C
    [ ] 3x^3 + C
    Explanation: integral of 3x^2 dx = 3*(x^3/3) + C = x^3 + C. Always include the constant of integration.

  Q2. What is integral from 0 to 2 of x dx?
    [ ] 0
    [ ] 4
    [*] 2
    [ ] 1
    Explanation: integral of x dx = x^2/2. Evaluated from 0 to 2: 4/2 - 0/2 = 2.

  Q3. Evaluate: integral of (4x^3 - 2x) dx.
    [*] x^4 - x^2 + C
    [ ] 12x^2 - 2 + C
    [ ] x^4 - x^2
    [ ] 4x^4 - 2x^2 + C
    Explanation: integral of 4x^3 dx = x^4, integral of 2x dx = x^2. So the answer is x^4 - x^2 + C.

  Q4. What is integral from 1 to 3 of 2x dx?
    [ ] 4
    [*] 8
    [ ] 6
    [ ] 10
    Explanation: Antiderivative is x^2. Evaluate: 3^2 - 1^2 = 9 - 1 = 8.

  Q5. Evaluate: integral of 5 dx.
    [ ] 5 + C
    [ ] 5x^2/2 + C
    [*] 5x + C
    [ ] 0 + C
    Explanation: The integral of a constant k is kx + C. So integral of 5 dx = 5x + C.

  Q6. What is integral of x^(-2) dx for x != 0?
    [ ] -x^(-3)/3 + C
    [*] -1/x + C
    [ ] ln|x| + C
    [ ] x^(-1) + C
    Explanation: Using the power rule for integration: x^(-2+1)/(-2+1) = x^(-1)/(-1) = -1/x + C.

  Q7. Evaluate: integral from 0 to 1 of (3x^2 + 2x + 1) dx.
    [ ] 6
    [ ] 2
    [ ] 4
    [*] 3
    Explanation: Antiderivative: x^3 + x^2 + x. At x = 1: 1 + 1 + 1 = 3. At x = 0: 0. Answer: 3 - 0 = 3.

  Q8. What is integral of sqrt(x) dx?
    [ ] (1/2)x^(-1/2) + C
    [ ] (3/2)x^(1/2) + C
    [*] (2/3)x^(3/2) + C
    [ ] x^(3/2) + C
    Explanation: sqrt(x) = x^(1/2). integral = x^(3/2)/(3/2) = (2/3)x^(3/2) + C.

  Q9. If F'(x) = f(x) and integral from a to b of f(x) dx = 10, what is F(b) - F(a)?
    [ ] -10
    [ ] 0
    [*] 10
    [ ] F(b) + F(a)
    Explanation: By the Fundamental Theorem of Calculus, integral from a to b of f(x) dx = F(b) - F(a) = 10.

  Q10. Evaluate: integral from -1 to 1 of x^3 dx.
    [ ] -1/4
    [ ] 1/4
    [*] 0
    [ ] 1/2
    Explanation: x^3 is an odd function on a symmetric interval [-1, 1], so the integral is 0. Alternatively: x^4/4 evaluated gives 1/4 - 1/4 = 0.

  Q11. What is an antiderivative?
    [ ] A function that undoes multiplication
    [*] A function F(x) whose derivative is f(x): F'(x) = f(x)
    [ ] The derivative of a function
    [ ] The area of a rectangle
    Explanation: An antiderivative F of f satisfies F'(x) = f(x). For example, x^3/3 is an antiderivative of x^2 because d/dx(x^3/3) = x^2.

  Q12. What is a definite integral?
    [*] The integral from a to b of f(x) dx, which gives a specific numerical value representing the signed area under the curve
    [ ] An integral without limits of integration
    [ ] The derivative of a function evaluated at two points
    [ ] An integral that always gives a positive answer
    Explanation: A definite integral has bounds a and b and produces a number representing the net signed area between f(x) and the x-axis from x = a to x = b.

  Q13. What is an indefinite integral?
    [ ] An integral with specific bounds, which is a common assumption in the field
    [ ] The area under a curve within the broader context of indefinite analysis
    [*] The general antiderivative of f(x), written as integral f(x) dx = F(x) + C, representing a family of functions
    [ ] An integral that cannot be computed, especially during periods of significant change
    Explanation: An indefinite integral has no bounds and includes + C (constant of integration) because many functions share the same derivative.

  Q14. What does the Fundamental Theorem of Calculus state?
    [ ] The derivative of an integral is always zero
    [*] If F is an antiderivative of f on [a,b], then integral from a to b of f(x) dx = F(b) - F(a)
    [ ] Every function has an antiderivative
    [ ] Integrals and derivatives are unrelated
    Explanation: The FTC connects differentiation and integration: to evaluate a definite integral, find any antiderivative F and compute F(b) - F(a).

  Q15. What is the constant of integration C?
    [ ] The slope of the antiderivative
    [ ] A specific number that must be calculated
    [*] An arbitrary constant added to an indefinite integral because the derivative of a constant is zero
    [ ] The value of the integral at x = 0
    Explanation: Since d/dx(C) = 0, any constant can be added to an antiderivative without changing its derivative. The + C accounts for all possible antiderivatives.

  Q16. What does the integral represent geometrically?
    [ ] The length of the curve
    [*] The signed area between the curve and the x-axis over the given interval
    [ ] The slope of the tangent line
    [ ] The maximum value of the function
    Explanation: The definite integral gives the net area: regions above the x-axis contribute positive area, regions below contribute negative area.

  Q17. What is the power rule for integration?
    [ ] integral of x^n dx = nx^(n-1) + C
    [ ] integral of x^n dx = x^n/n + C
    [ ] integral of x^n dx = x^(n+1) + C
    [*] integral of x^n dx = x^(n+1)/(n+1) + C, for n != -1
    Explanation: The power rule for integration increases the exponent by 1 and divides by the new exponent. The case n = -1 gives ln|x| + C instead.

  Q18. What does 'signed area' mean in the context of integrals?
    [*] Area above the x-axis counts as positive, and area below the x-axis counts as negative
    [ ] The sign depends on whether a < b or a > b
    [ ] Signed area equals the absolute value of the integral
    [ ] All area is always positive
    Explanation: When f(x) < 0, the integral accumulates negative values. The signed area can be zero even if the curve encloses regions both above and below the axis.

  Q19. What is a Riemann sum?
    [*] An approximation of a definite integral using the sum of areas of rectangles under the curve
    [ ] The sum of all function values under most circumstances involving Riemann
    [ ] The exact value of an integral regardless of prevailing conditions in practice
    [ ] The derivative of the integral without accounting for additional influences
    Explanation: A Riemann sum divides [a,b] into subintervals and sums f(x_i*)*delta_x for sample points x_i*. As the partition gets finer, it approaches the definite integral.

  Q20. What is the relationship between differentiation and integration?
    [ ] They are completely unrelated
    [*] They are inverse operations: integration undoes differentiation and vice versa

--- Lesson 2.3: Power Rule and Basic Derivatives (Summary) ---

  Key Concepts: Power Rule and Basic Derivatives
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx(x^5)?
By the power rule, d/dx(x^n) = nx^(n-1). So d/dx(x^5) = 5x^(5-1) = 5x^4.
What is d/dx(3x^2)?
Using the constant multiple rule with the power rule: d/dx(3x^2) = 3 * 2x^(2-1) = 6x.
What is the derivative of f(x) = 7?
The derivative of any constant is zero. Since f(x) = 7 is a constant function, f'(x) = 0.
What is d/dx(x^(-3))?
By the power rule, d/dx(x^(-3)) = -3x^(-3-1) = -3x^(-4). The power rule works for all real exponents, including negative ones.
What is d/dx(4x^3 - 2x + 9)?
Differentiate term by term: d/dx(4x^3) = 12x^2, d/dx(-2x) = -2, and d/dx(9) = 0. So the answer is 12x^2 - 2.
What is d/dx(sqrt(x)), rewritten as d/dx(x^(1/2))?
Rewrite sqrt(x) as x^(1/2), then apply the power rule: d/dx(x^(1/2)) = (1/2)x^(1/2 - 1) = (1/2)x^(-1/2).
If f(x) = x^8, what is f'(2)?
First find f'(x) = 8x^7 by the power rule. Then f'(2) = 8(2)^7 = 8 * 128 = 1024.
What is d/dx(x^(2/3))?
By the power rule: d/dx(x^(2/3)) = (2/3)x^(2/3 - 1) = (2/3)x^(-1/3).
What is the derivative of f(x) = 5x^4 + 3x^2 - x evaluated at x = 1?
f'(x) = 20x^3 + 6x - 1. At x = 1: f'(1) = 20(1) + 6(1) - 1 = 20 + 6 - 1 = 25.
What is d/dx(1/x^2), rewritten as d/dx(x^(-2))?
Rewrite 1/x^2 as x^(-2). By the power rule: d/dx(x^(-2)) = -2x^(-3) = -2/x^3.
What is the power rule for differentiation?
The power rule states that for any real number n, d/dx(x^n) = nx^(n-1). You bring down the exponent as a coefficient and reduce the exponent by one.
What is meant by the term 'derivative' of a function?
A derivative measures the instantaneous rate of change of a function at a given point. Geometrically, it represents the slope of the tangent line to the curve at that point.
What is a 'monomial' in the context of differentiation?
A monomial is a single-term algebraic expression of the form ax^n, where a is a constant and n is a non-negative integer. The power rule directly applies to monomials.
What is meant by 'notation' f'(x) in calculus?
The notation f'(x), read as 'f prime of x,' is Lagrange notation for the first derivative. Other notations include dy/dx (Leibniz) and Df(x) (operator notation).
A ball is thrown upward from the ground, and its height in meters after t seconds is given by h(t) = -5t^2 + 20t. What is the ball's velocity at t = 1 second?
Velocity is the derivative of position: h'(t) = -10t + 20. At t = 1: h'(1) = -10(1) + 20 = 10 m/s.
A company's revenue in thousands of dollars is modeled by R(x) = -2x^3 + 15x^2 + 100x, where x is the number of units sold (in hundreds). What is the marginal revenue when 2 hundred units are sold?
Marginal revenue is R'(x) = -6x^2 + 30x + 100. At x = 2: R'(2) = -6(4) + 30(2) + 100 = -24 + 60 + 100 = 136 thousand dollars per hundred units.
An engineer models the displacement of a vibrating beam as s(t) = 0.5t^4 - 3t^2 + 2 centimeters, where t is in seconds. What is the velocity of the beam at t = 2 seconds?
Velocity is s'(t) = 2t^3 - 6t. At t = 2: s'(2) = 2(8) - 6(2) = 16 - 12 = 4 cm/s.
A physics student models the potential energy of a spring as U(x) = 0.5kx^2, where k = 200 N/m and x is displacement in meters. What is the force (F = -dU/dx) when the spring is compressed by 0.3 meters?
U(x) = 0.5(200)x^2 = 100x^2. The force is F = -dU/dx = -200x. At x = 0.3: F = -200(0.3) = -60 N. The negative sign indicates a restoring force.
A diver jumps from a 10-meter platform, and her height is modeled by h(t) = -4.9t^2 + 3t + 10 meters, where t is in seconds. What is her velocity when she hits the water (h = 0), given that she hits the water at approximately t = 1.75 seconds?
Velocity is h'(t) = -9.8t + 3. At t = 1.75: h'(1.75) = -9.8(1.75) + 3 = -17.15 + 3 = -14.15 m/s. The negative sign indicates downward motion.
Real-World Applications
Example: What does the constant multiple rule state about derivatives?
The constant multiple rule states that the derivative of a constant times a function equals the constant times the derivative of the function: d/dx[c * f(x)] = c * f'(x).

--- Lesson 2.4: Product, Quotient, and Chain Rules (Summary) ---

  Key Concepts: Product, Quotient, and Chain Rules
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Using the product rule, what is d/dx[x^2 * sin(x)]?
By the product rule: d/dx[f*g] = f'g + fg'. Here f = x^2, g = sin(x), so d/dx = 2x*sin(x) + x^2*cos(x).
What is d/dx[x / (x + 1)] using the quotient rule?
By the quotient rule: d/dx[f/g] = (f'g - fg')/g^2 = [(1)(x+1) - (x)(1)]/(x+1)^2 = (x+1-x)/(x+1)^2 = 1/(x+1)^2.
Using the chain rule, what is d/dx[(3x + 1)^4]?
By the chain rule: d/dx[u^4] = 4u^3 * du/dx where u = 3x + 1, du/dx = 3. So d/dx = 4(3x+1)^3 * 3 = 12(3x+1)^3.
What is d/dx[e^x * x^3] using the product rule?
By the product rule: d/dx[e^x * x^3] = e^x * x^3 + e^x * 3x^2 = e^x(x^3 + 3x^2). Both the second and third options are equivalent, but the simplified form is e^x(x^3 + 3x^2).
What is d/dx[sin(5x)] using the chain rule?
By the chain rule: d/dx[sin(u)] = cos(u) * du/dx. With u = 5x, du/dx = 5. So d/dx[sin(5x)] = cos(5x) * 5 = 5cos(5x).
What is d/dx[(x^2 + 1)/(x^2 - 1)] using the quotient rule?
Quotient rule: [2x(x^2-1) - (x^2+1)(2x)]/(x^2-1)^2 = [2x^3 - 2x - 2x^3 - 2x]/(x^2-1)^2 = -4x/(x^2-1)^2.
What is d/dx[(2x - 3)^5] using the chain rule?
Chain rule: d/dx[u^5] = 5u^4 * du/dx, where u = 2x - 3, du/dx = 2. So d/dx = 5(2x-3)^4 * 2 = 10(2x-3)^4.
Using the product rule, what is d/dx[x * ln(x)]?
Product rule: d/dx[x * ln(x)] = (1)(ln(x)) + x(1/x) = ln(x) + 1.
What is d/dx[sqrt(4x + 9)], rewritten as d/dx[(4x + 9)^(1/2)]?
Chain rule: (1/2)(4x+9)^(-1/2) * 4 = 4/(2*sqrt(4x+9)) = 2/sqrt(4x+9).
What is d/dx[x^2 * cos(x)] using the product rule?
Product rule: d/dx[x^2 * cos(x)] = 2x*cos(x) + x^2*(-sin(x)) = 2x*cos(x) - x^2*sin(x).
What is the product rule for derivatives?
The product rule states that d/dx[f(x)*g(x)] = f'(x)*g(x) + f(x)*g'(x). You differentiate the first and multiply by the second, then add the first times the derivative of the second.
What is the quotient rule for derivatives?
The quotient rule states d/dx[f/g] = (f'g - fg')/g^2. A common mnemonic is 'lo d-hi minus hi d-lo over lo-lo' meaning (denominator * numerator' - numerator * denominator') / denominator^2.
What is Leibniz notation for the chain rule?
In Leibniz notation, the chain rule is dy/dx = (dy/du)(du/dx). This notation makes the chain rule appear as if the du terms 'cancel,' which is a helpful mnemonic for remembering the formula.
A manufacturer models its daily profit as P(x) = x^2 * e^(-0.1x) dollars, where x is the number of units produced. Using the product rule, what is P'(10)?
P'(x) = 2x*e^(-0.1x) + x^2*(-0.1)e^(-0.1x) = e^(-0.1x)(2x - 0.1x^2). At x = 10: P'(10) = e^(-1)(20 - 10) = 10/e dollars per unit.
A drug's concentration in the bloodstream is modeled by C(t) = 5t/(t^2 + 4) mg/L, where t is hours after administration. Using the quotient rule, what is the rate of change at t = 2 hours?
C'(t) = [5(t^2+4) - 5t(2t)]/(t^2+4)^2 = [5t^2 + 20 - 10t^2]/(t^2+4)^2 = (20 - 5t^2)/(t^2+4)^2. At t = 2: C'(2) = (20-20)/(8)^2 = 0. The concentration is at its peak at t = 2.
The temperature of a cooling object is modeled by T(t) = 70 + 130e^(-0.05t) degrees Fahrenheit, where t is in minutes. What is the rate of cooling at t = 10 minutes?
Using the chain rule: T'(t) = 130 * (-0.05) * e^(-0.05t) = -6.5e^(-0.05t). At t = 10: T'(10) = -6.5e^(-0.5) degrees per minute (approximately -3.94 degrees/min).
A particle's position is given by s(t) = (t^2 + 1)^3 meters. Using the chain rule, what is the velocity at t = 1?
s'(t) = 3(t^2+1)^2 * 2t = 6t(t^2+1)^2. At t = 1: s'(1) = 6(1)(1+1)^2 = 6(1)(4) = 24 m/s.
A market analyst models stock price as P(t) = t * sqrt(t + 5) dollars, where t is months after listing. Using the product rule, what is the rate of price change at t = 4 months?
P'(t) = sqrt(t+5) + t * 1/(2*sqrt(t+5)). At t = 4: P'(4) = sqrt(9) + 4/(2*sqrt(9)) = 3 + 4/6 = 3 + 2/3 = 11/3 approximately 3.67 dollars per month.

By the chain rule with outer function sin(u) and inner function u = x^2: f'(x) = cos(x^2) * 2x = 2x*cos(x^2).
A rocket's altitude is modeled by h(t) = (4t + 1)^(3/2) meters. Using the chain rule, what is the velocity at t = 2 seconds?
h'(t) = (3/2)(4t+1)^(1/2) * 4 = 6(4t+1)^(1/2). At t = 2: h'(2) = 6*sqrt(9) = 6*3 = 18 m/s.
An investment's value is V(t) = 1000t/(t + 2) dollars after t years. A financial advisor wants to know the instantaneous rate of return at t = 3 years. What is V'(3)?
V'(t) = [1000(t+2) - 1000t(1)]/(t+2)^2 = 2000/(t+2)^2. At t = 3: V'(3) = 2000/(5)^2 = 2000/25 = $80 per year.

--- Lesson 2.5: Product and Quotient Rules (Summary) ---

  Key Concepts: Product and Quotient Rules
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[x^3 * e^x] using the product rule?
Product rule: d/dx[x^3 * e^x] = 3x^2 * e^x + x^3 * e^x = e^x(3x^2 + x^3) = e^x(x^3 + 3x^2).
What is d/dx[(3x + 2)(x^2 - 1)]?
Product rule: (3)(x^2-1) + (3x+2)(2x) = 3x^2 - 3 + 6x^2 + 4x = 9x^2 + 4x - 3.
What is d/dx[(2x - 5)/(x + 3)] using the quotient rule?
Quotient rule: [2(x+3) - (2x-5)(1)]/(x+3)^2 = [2x + 6 - 2x + 5]/(x+3)^2 = 11/(x+3)^2.
What is d/dx[sin(x) * cos(x)]?
Product rule: cos(x)*cos(x) + sin(x)*(-sin(x)) = cos^2(x) - sin^2(x). This is also equal to cos(2x) by the double angle identity.
What is d/dx[x * sqrt(x)] simplified?
Product rule: (1)(sqrt(x)) + x * (1/(2sqrt(x))) = sqrt(x) + x/(2sqrt(x)) = sqrt(x) + sqrt(x)/2 = (3/2)sqrt(x). Alternatively, x*sqrt(x) = x^(3/2), so d/dx = (3/2)x^(1/2) = (3/2)sqrt(x).
What is d/dx[e^x / x^2] using the quotient rule?
Quotient rule: [e^x * x^2 - e^x * 2x]/x^4 = e^x(x^2 - 2x)/x^4 = e^x * x(x-2)/x^4 = e^x(x-2)/x^3.
If f(x) = (x + 1)(x^2 + 3), what is f'(0)?
f'(x) = (1)(x^2+3) + (x+1)(2x) = x^2 + 3 + 2x^2 + 2x = 3x^2 + 2x + 3. At x = 0: f'(0) = 0 + 0 + 3 = 3.
What is d/dx[tan(x)] using sin(x)/cos(x) and the quotient rule?
d/dx[sin(x)/cos(x)] = [cos(x)*cos(x) - sin(x)*(-sin(x))]/cos^2(x) = [cos^2(x) + sin^2(x)]/cos^2(x) = 1/cos^2(x) = sec^2(x).
What is d/dx[x^2/(x + 4)] evaluated at x = 1?
Quotient rule: [2x(x+4) - x^2(1)]/(x+4)^2 = (2x^2 + 8x - x^2)/(x+4)^2 = (x^2 + 8x)/(x+4)^2. At x = 1: (1+8)/25 = 9/25.
What is d/dx[(x^2 + 1)(x^3 - 2x)] evaluated at x = 1?
Product rule: 2x(x^3-2x) + (x^2+1)(3x^2-2). At x = 1: 2(1)(1-2) + (2)(3-2) = 2(-1) + 2(1) = -2 + 2 = 0.
What is meant by 'numerator' in the context of the quotient rule?
In a fraction f(x)/g(x), the numerator is f(x) -- the top function. In the quotient rule, you differentiate the numerator and multiply by the denominator, then subtract.
What is 'denominator' in the context of the quotient rule?
In a fraction f(x)/g(x), the denominator is g(x) -- the bottom function. In the quotient rule, the denominator squared appears in the result: [f'g - fg']/g^2.
What is the 'secant function' sec(x)?
The secant function is defined as sec(x) = 1/cos(x). It appears frequently in derivatives: for example, d/dx[tan(x)] = sec^2(x), which can be derived using the quotient rule on sin(x)/cos(x).
What is meant by 'undefined' in the context of the quotient rule?
When applying the quotient rule, the result is undefined wherever the denominator g(x) = 0, since division by zero is not defined. These points may correspond to vertical asymptotes or holes in the function.
What is 'algebraic manipulation' in the context of simplifying derivatives?
Algebraic manipulation involves rewriting expressions using techniques like expanding products, factoring, combining like terms, and canceling common factors to simplify derivative results.
A population of fish in a lake is modeled by P(t) = 5000t/(t + 10) fish, where t is years. A wildlife biologist wants to know the growth rate at t = 15 years. What is P'(15)?
P'(t) = [5000(t+10) - 5000t]/(t+10)^2 = 50000/(t+10)^2. At t = 15: P'(15) = 50000/625 = 80 fish per year.

--- Lesson 2.6: Trigonometric Derivatives (Summary) ---

  Key Concepts: Trigonometric Derivatives
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[sin(x)]?
The derivative of sin(x) is cos(x). This is one of the fundamental trigonometric derivative formulas.
What is d/dx[cos(x)]?
The derivative of cos(x) is -sin(x). Note the negative sign, which distinguishes it from d/dx[sin(x)] = cos(x).
What is d/dx[tan(x)]?
The derivative of tan(x) is sec^2(x). This can be derived using the quotient rule on sin(x)/cos(x).
What is d/dx[3sin(x) + 2cos(x)]?
Differentiate term by term: d/dx[3sin(x)] = 3cos(x) and d/dx[2cos(x)] = -2sin(x). So the result is 3cos(x) - 2sin(x).
What is d/dx[sec(x)]?
The derivative of sec(x) is sec(x)tan(x). This can be derived by differentiating 1/cos(x) using the quotient rule.
What is d/dx[csc(x)]?
The derivative of csc(x) is -csc(x)cot(x). Like d/dx[cos(x)] = -sin(x), the co-functions (csc, cot) have negative signs in their derivatives.
What is d/dx[cot(x)]?
The derivative of cot(x) is -csc^2(x). This mirrors d/dx[tan(x)] = sec^2(x) but with a negative sign and the corresponding co-function.
If f(x) = sin(x), what is f'(pi/6)?
f'(x) = cos(x), so f'(pi/6) = cos(pi/6) = sqrt(3)/2.
What is d/dx[sin^2(x) + cos^2(x)]?
Since sin^2(x) + cos^2(x) = 1 (the Pythagorean identity), its derivative is d/dx[1] = 0.
What is d/dx[5tan(x) - 2sec(x)] evaluated at x = 0?

What is the Pythagorean identity involving sin and cos?
The Pythagorean identity states sin^2(x) + cos^2(x) = 1 for all values of x. This identity is frequently used to simplify trigonometric expressions and their derivatives.
What is the 'secant function' in terms of cosine?
The secant function is the reciprocal of cosine: sec(x) = 1/cos(x). It is undefined wherever cos(x) = 0, which occurs at x = pi/2 + n*pi for integer n.
What is the 'cotangent function' in terms of sine and cosine?
Cotangent is defined as cot(x) = cos(x)/sin(x), which is the reciprocal of tangent. Its derivative is -csc^2(x).

theta'(t) = 0.3 * 2 * cos(2t) = 0.6cos(2t). At t = pi/4: theta'(pi/4) = 0.6cos(pi/2) = 0.6(0) = 0 rad/s. The pendulum momentarily stops at its maximum displacement.

h'(t) = 12sin(pi*t/30) * (pi/30) = (12pi/30)sin(pi*t/30) = (2pi/5)sin(pi*t/30). At t = 15: h'(15) = (2pi/5)sin(pi/2) = (2pi/5)(1) = 2pi/5 m/s.

V'(t) = 120 * 60pi * cos(60pi*t) = 7200pi*cos(60pi*t). The maximum value occurs when cos(60pi*t) = 1, giving a maximum rate of 7200pi volts/s.
A buoy bobs in the ocean with its height modeled by y(t) = 2sin(t) + cos(2t) meters. What is the velocity of the buoy at t = 0?
y'(t) = 2cos(t) - 2sin(2t). At t = 0: y'(0) = 2cos(0) - 2sin(0) = 2(1) - 2(0) = 2 m/s.

x'(t) = -12sin(3t) + 3cos(3t). The maximum speed is the amplitude: sqrt(12^2 + 3^2) = sqrt(144 + 9) = sqrt(153) = 3*sqrt(17) cm/s.
A sound wave is modeled by P(t) = 0.02sin(440 * 2pi * t) pascals. What is the maximum pressure change rate?
P'(t) = 0.02 * 880pi * cos(880pi*t). The maximum occurs when cos = 1, giving 0.02 * 880pi = 17.6pi pascals/s.

--- Lesson 2.7: Exponential and Logarithmic Derivatives (Summary) ---

  Key Concepts: Exponential and Logarithmic Derivatives
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[e^x]?
The derivative of e^x is e^x itself. This is the unique and defining property of the natural exponential function.
What is d/dx[ln(x)]?
The derivative of the natural logarithm ln(x) is 1/x, defined for x > 0.
What is d/dx[e^(3x)]?
By the chain rule: d/dx[e^(3x)] = e^(3x) * d/dx(3x) = e^(3x) * 3 = 3e^(3x).
What is d/dx[ln(5x)]?
By the chain rule: d/dx[ln(5x)] = (1/(5x)) * 5 = 1/x. Alternatively, ln(5x) = ln(5) + ln(x), so d/dx = 0 + 1/x = 1/x.
What is d/dx[2^x]?
For any base a > 0, d/dx[a^x] = a^x * ln(a). So d/dx[2^x] = 2^x * ln(2).
What is d/dx[log_10(x)]?
For any base a, d/dx[log_a(x)] = 1/(x * ln(a)). So d/dx[log_10(x)] = 1/(x * ln(10)).
What is d/dx[e^(-x^2)]?
By the chain rule: d/dx[e^(-x^2)] = e^(-x^2) * d/dx(-x^2) = e^(-x^2) * (-2x) = -2x * e^(-x^2).
What is d/dx[ln(x^2 + 1)]?
By the chain rule: d/dx[ln(u)] = (1/u) * du/dx = [1/(x^2+1)] * 2x = 2x/(x^2+1).
If f(x) = 5e^(2x), what is f'(0)?
f'(x) = 5 * 2 * e^(2x) = 10e^(2x). At x = 0: f'(0) = 10e^0 = 10 * 1 = 10.
What is d/dx[x * e^x] evaluated at x = 1?
By the product rule: d/dx[x*e^x] = e^x + x*e^x = e^x(1+x). At x = 1: e^1(1+1) = 2e.
What is the 'natural exponential function'?
The natural exponential function is e^x, where e (Euler's number) is approximately 2.71828. Its unique property is that d/dx[e^x] = e^x -- it is its own derivative.
What is the 'change of base formula' for logarithms?
The change of base formula states log_a(x) = ln(x)/ln(a). This is useful for differentiating logarithms of any base: d/dx[log_a(x)] = 1/(x*ln(a)).
A colony of bacteria doubles every 4 hours and starts with 500 cells. The population is P(t) = 500 * 2^(t/4). What is the growth rate at t = 8 hours?
P'(t) = 500 * 2^(t/4) * ln(2) * (1/4) = 125 * ln(2) * 2^(t/4). At t = 8: P'(8) = 125 * ln(2) * 2^2 = 125 * 4 * ln(2) = 500 * ln(2) cells/hour.
The decibel level of sound is L = 10 * ln(I/I_0)/ln(10), where I is intensity. If I = 0.001 watts/m^2 is increasing at 0.0001 watts/m^2 per second, what is dL/dt?
L = 10*log_10(I/I_0), so dL/dI = 10/(I * ln(10)). Then dL/dt = dL/dI * dI/dt = [10/(I * ln(10))] * 0.0001. At I = 0.001: dL/dt = 10/(0.001 * ln(10)) * 0.0001 = 1/ln(10) dB/s.
The Richter scale magnitude is M = (2/3)log_10(E/E_0). If the energy E of an earthquake is increasing at 10^12 joules/s and currently E = 10^15 joules, what is dM/dt? (Use E_0 = 1.)
dM/dt = (2/3) * 1/(E * ln(10)) * dE/dt = (2/3) * 1/(10^15 * ln(10)) * 10^12 = (2/3) * 10^(-3)/ln(10) = 2/(3 * 10^3 * ln(10)).

    [ ] -100/ln(10) per second
    [ ] -10/ln(10) per second
    Explanation: dpH/dt = -1/([H+] * ln(10)) * d[H+]/dt. At [H+] = 0.01 with d[H+]/dt = 0.001: dpH/dt = -1/(0.01 * ln(10)) * 0.001 = -0.1/ln(10). This equals -1/(0.01*ln(10)) * 0.001.

  Q25. A cup of coffee cools according to Newton's Law: T(t) = 22 + 68e^(-0.1t) degrees Celsius. At what rate is the coffee cooling when its temperature is 56 degrees?

    [ ] 10/ln(10)

    [*] 10/(I * ln(10)) * 0.0001

  Q27. A learning curve is modeled by L(t) = 100(1 - e^(-0.2t)) words per minute, where t is days of practice. At what rate is the student improving after 5 days?

  Q28. A drug's concentration follows C(t) = 8(e^(-0.3t) - e^(-0.6t)) mg/L. When does the concentration reach its maximum (C'(t) = 0)?

--- Lesson 2.8: Chain Rule (Summary) ---

  Key Concepts: Chain Rule
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is d/dx[sin(3x)]?
By the chain rule: d/dx[sin(u)] = cos(u) * du/dx. With u = 3x: d/dx[sin(3x)] = cos(3x) * 3 = 3cos(3x).
What is d/dx[(x^2 + 5)^7]?
Chain rule: 7(x^2+5)^6 * d/dx(x^2+5) = 7(x^2+5)^6 * 2x = 14x(x^2+5)^6.
What is d/dx[e^(sin(x))]?
Chain rule: d/dx[e^u] = e^u * du/dx. With u = sin(x): d/dx[e^(sin(x))] = e^(sin(x)) * cos(x).
What is d/dx[ln(cos(x))]?
Chain rule: d/dx[ln(u)] = (1/u)*du/dx = (1/cos(x))*(-sin(x)) = -sin(x)/cos(x) = -tan(x).
What is d/dx[sqrt(x^3 + 1)]?
Rewrite as (x^3+1)^(1/2). Chain rule: (1/2)(x^3+1)^(-1/2) * 3x^2 = 3x^2/(2*sqrt(x^3+1)).
What is d/dx[cos^3(x)], rewritten as d/dx[(cos(x))^3]?
Chain rule: 3(cos(x))^2 * d/dx[cos(x)] = 3cos^2(x) * (-sin(x)) = -3cos^2(x)sin(x).
What is d/dx[tan(x^2)]?
Chain rule: d/dx[tan(u)] = sec^2(u) * du/dx. With u = x^2: sec^2(x^2) * 2x = 2x*sec^2(x^2).
What is d/dx[e^(x^2 + 3x)]?
Chain rule: d/dx[e^u] = e^u * du/dx = e^(x^2+3x) * (2x + 3).
What is d/dx[(2x - 1)^(-3)]?
Chain rule: -3(2x-1)^(-4) * 2 = -6(2x-1)^(-4).
What is d/dx[sin(cos(x))]?
Double chain rule: d/dx[sin(cos(x))] = cos(cos(x)) * d/dx[cos(x)] = cos(cos(x)) * (-sin(x)) = -sin(x)*cos(cos(x)).
What is the chain rule in Leibniz notation?
In Leibniz notation, the chain rule is dy/dx = (dy/du)(du/dx). If y = f(u) and u = g(x), the derivative 'chains' through the intermediate variable u.
What is the 'generalized power rule'?
The generalized power rule combines the power rule with the chain rule: d/dx[f(x)]^n = n[f(x)]^(n-1) * f'(x). It applies the power rule to the outer function and multiplies by the derivative of the inner function.
What is the meaning of 'du/dx' in the chain rule?
In the chain rule dy/dx = dy/du * du/dx, the term du/dx represents the derivative of the inner function u = g(x) with respect to x. This is the 'extra factor' that distinguishes the chain rule from direct differentiation.
What is a 'chain' of derivatives in the chain rule?
The 'chain' in the chain rule refers to the product of derivatives linked through composition: dy/dx = dy/du * du/dv * dv/dx for y = f(u), u = g(v), v = h(x). Each link in the chain is a derivative of one layer.
A roller coaster's height is modeled by h(t) = 50sin(pi*t/30)^2 meters, where t is seconds. What is the vertical velocity at t = 15 seconds?
h(t) = 50sin^2(pi*t/30). h'(t) = 50 * 2sin(pi*t/30) * cos(pi*t/30) * (pi/30) = (100pi/30)sin(pi*t/30)cos(pi*t/30). At t = 15: sin(pi/2) = 1, cos(pi/2) = 0, so h'(15) = 0 m/s. The coaster is at peak height.
The temperature inside an oven follows T(t) = 400 - 350e^(-0.1t) degrees Fahrenheit. A baker places bread inside when T = 300 degrees. What is the rate of temperature change at that moment?
T'(t) = 35e^(-0.1t). When T = 300: 300 = 400 - 350e^(-0.1t), so 350e^(-0.1t) = 100, e^(-0.1t) = 2/7. Then T' = 35 * (2/7) = 10 degrees/min.
A satellite's distance from Earth is d(t) = sqrt(4t^2 + 9) thousand km. What is dd/dt at t = 2 hours?
d'(t) = (1/2)(4t^2+9)^(-1/2) * 8t = 4t/sqrt(4t^2+9). At t = 2: d'(2) = 8/sqrt(16+9) = 8/sqrt(25) = 8/5 thousand km/hr.
A population model uses P(t) = 10000/(1 + 99e^(-0.5t)). This is a logistic function. What is P'(0)?
Rewrite P = 10000(1+99e^(-0.5t))^(-1). P'(t) = -10000(1+99e^(-0.5t))^(-2) * 99(-0.5)e^(-0.5t) = 495000e^(-0.5t)/(1+99e^(-0.5t))^2. At t=0: P'(0) = 495000/(100)^2 = 495000/10000 = 49.5.
The intensity of light through a filter is I(x) = I_0 * e^(-0.3x), where x is the number of filters. If I_0 = 1000 lumens and you have a stack where x = ln(t)/ln(2) filters at time t, what is dI/dt at t = 4?
By the chain rule: dI/dt = dI/dx * dx/dt = -300e^(-0.3x) * 1/(t*ln(2)). At t = 4: x = 2, so dI/dt = -300e^(-0.6)/(4*ln(2)).
A spring-mass system has displacement x(t) = 3cos(sqrt(5)*t) cm. What is the acceleration (second derivative) at t = 0?
x'(t) = -3sqrt(5)sin(sqrt(5)*t). x''(t) = -3*5*cos(sqrt(5)*t) = -15cos(sqrt(5)*t). At t = 0: x''(0) = -15cos(0) = -15 cm/s^2.
A company's stock price follows S(t) = 50e^(0.02*sin(pi*t/6)) dollars, where t is months. What is S'(0)?
S'(t) = 50e^(0.02sin(pi*t/6)) * 0.02cos(pi*t/6) * pi/6. At t = 0: e^0 = 1, cos(0) = 1, so S'(0) = 50 * 0.02 * pi/6 = pi/6 dollars/month.
A skydiver's velocity during freefall is modeled by v(t) = 50(1 - e^(-0.2t)) m/s. What is the acceleration at t = 5 seconds?
Acceleration is v'(t) = 50 * 0.2 * e^(-0.2t) = 10e^(-0.2t). At t = 5: v'(5) = 10e^(-1) m/s^2 (approximately 3.68 m/s^2).
An electrical circuit has charge Q(t) = 5(1 - e^(-t/RC)) coulombs, where RC = 0.01 seconds. What is the current I = dQ/dt at t = 0.02 seconds?
I = dQ/dt = 5 * (1/0.01) * e^(-t/0.01) = 500e^(-100t). At t = 0.02: I = 500e^(-2) = 500/e^2 amperes.
The volume of a tumor is modeled by V(t) = 0.5e^(0.3*sqrt(t)) cm^3, where t is weeks. What is the growth rate at t = 4 weeks?
V'(t) = 0.5 * e^(0.3sqrt(t)) * 0.3 * 1/(2sqrt(t)) = 0.075e^(0.3sqrt(t))/sqrt(t). At t = 4: V'(4) = 0.075e^(0.6)/2 = 0.0375e^(0.6) cm^3/week (approximately 0.068 cm^3/week).

Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
For the equation x^2 + y^2 = 25, what is dy/dx using implicit differentiation?
Differentiate both sides: 2x + 2y(dy/dx) = 0. Solving: 2y(dy/dx) = -2x, so dy/dx = -x/y.
For the equation xy = 10, what is dy/dx?
Using the product rule on xy: (1)(y) + x(dy/dx) = 0. So dy/dx = -y/x.
For x^3 + y^3 = 6xy, what is dy/dx?
Differentiate: 3x^2 + 3y^2(dy/dx) = 6y + 6x(dy/dx). Rearrange: (3y^2 - 6x)(dy/dx) = 6y - 3x^2. So dy/dx = (6y - 3x^2)/(3y^2 - 6x).
For x^2 + y^2 = 25, what is the slope of the tangent line at the point (3, 4)?
From the previous result, dy/dx = -x/y. At (3, 4): dy/dx = -3/4.
For sin(y) = x, what is dy/dx?
Differentiate both sides: cos(y) * dy/dx = 1. So dy/dx = 1/cos(y) = sec(y).
For e^y + xy = 5, what is dy/dx?
Differentiate: e^y(dy/dx) + y + x(dy/dx) = 0. Factor: (e^y + x)(dy/dx) = -y. So dy/dx = -y/(e^y + x).
For x^2 - xy + y^2 = 7, what is dy/dx?
Differentiate: 2x - (y + x*dy/dx) + 2y*dy/dx = 0. So 2x - y - x*dy/dx + 2y*dy/dx = 0. (2y-x)*dy/dx = y-2x. dy/dx = (y-2x)/(2y-x).
For y^3 = x, what is dy/dx?
Differentiate: 3y^2 * dy/dx = 1. So dy/dx = 1/(3y^2).
For cos(xy) = x, what is dy/dx?
Differentiate: -sin(xy)*(y + x*dy/dx) = 1. So -y*sin(xy) - x*sin(xy)*dy/dx = 1. dy/dx = -(1 + y*sin(xy))/(x*sin(xy)).
For x^2 + y^2 = 9, what is the slope at (0, 3)?
dy/dx = -x/y. At (0, 3): dy/dx = -0/3 = 0. The tangent line at the top of the circle is horizontal.
What is a 'relation' versus a 'function' in mathematics?
A relation is any set of (x, y) pairs. A function is a special relation where each x has exactly one y. For example, x^2 + y^2 = 25 is a relation (each x near 0 gives two y values) but not a function.
An elliptical running track is described by x^2/100 + y^2/36 = 1 (in meters). What is the slope of the track at the point (6, 4.8)?

A circular pond has equation x^2 + y^2 = 400. A duck swims along the circle. At the point (12, 16), what is the direction (slope) of the duck's path?
dy/dx = -x/y = -12/16 = -3/4. The duck's path has slope -3/4 at that point, tangent to the circle.
A lens has a hyperbolic cross-section described by x^2/4 - y^2/9 = 1. What is the slope of the tangent at (4, 3*sqrt(3))?
Differentiate: 2x/4 - 2y/9 * dy/dx = 0. So x/2 = (2y/9)*dy/dx, giving dy/dx = 9x/(4y). At (4, 3sqrt(3)): dy/dx = 36/(12sqrt(3)) = 3/sqrt(3) = sqrt(3).
An architect designs an arch using the equation x^2 + (y-6)^2 = 36. What is the slope of the arch at the point (3, 6 + 3*sqrt(3))?
Differentiate: 2x + 2(y-6)*dy/dx = 0. dy/dx = -x/(y-6). At (3, 6+3sqrt(3)): dy/dx = -3/(3sqrt(3)) = -1/sqrt(3).
A chemical equilibrium is described by x^3 + 3xy + y^3 = 1, where x and y are concentrations in mol/L. At the point (1, -1), what is dy/dx?
First verify: 1 + 3(1)(-1) + (-1)^3 = 1 - 3 - 1 = -3, not 1. The point (1,-1) is not on this curve. For the actual curve, differentiate: 3x^2 + 3y + 3x*dy/dx + 3y^2*dy/dx = 0, so dy/dx = -(3x^2 + 3y)/(3x + 3y^2) = -(x^2 + y)/(x + y^2). At a valid point, substitute accordingly.
A mirror is shaped by the curve y^2 = 4x (a parabola). An optics engineer needs the tangent slope at (4, 4). What is dy/dx there?
Differentiate y^2 = 4x: 2y*dy/dx = 4, so dy/dx = 2/y. At (4, 4): dy/dx = 2/4 = 1/2.
A production possibilities curve is given by x^(2/3) + y^(2/3) = 10, where x is food output and y is technology output. What is dy/dx at the point where x = 8 and y = 8?
Differentiate: (2/3)x^(-1/3) + (2/3)y^(-1/3)*dy/dx = 0. dy/dx = -x^(-1/3)/y^(-1/3) = -(y/x)^(1/3). At x = 8, y = 8: dy/dx = -(8/8)^(1/3) = -1.
A suspension bridge cable follows the catenary-like curve e^y + e^(-y) = x. At the lowest point (2, 0), what is the slope dy/dx?
Differentiate: e^y*dy/dx - e^(-y)*dy/dx = 1. So (e^y - e^(-y))*dy/dx = 1, giving dy/dx = 1/(e^y - e^(-y)). At (2, 0): dy/dx = 1/(e^0 - e^0) = 1/(1-1) = 1/0, which is undefined. However, the lowest point of e^y + e^(-y) = x occurs at y = 0, x = 2, and the tangent is vertical. Actually, the cable is symmetric, so at the bottom the slope depends on orientation. Re-examining: dy/dx = 1/(e^y - e^(-y)) is undefined at y = 0, indicating a vertical tangent at the lowest point.
