--- Lesson 1.1: Review of Functions & Notation (Summary) ---

  Key Concepts: Review of Functions & Notation
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If f(x) = x² - 5, what is f(3)?
f(3) = (3)² - 5 = 9 - 5 = 4. Substitute the input value and evaluate using order of operations.
If f(x) = 3x - 2, what is f(5)?
f(5) = 3(5) - 2 = 15 - 2 = 13. Plug the given value into the function and simplify.
What is the domain of f(x) = 1/(x - 4)?
The denominator cannot equal zero, so x - 4 ≠ 0, which means x ≠ 4. All other real numbers are valid inputs.
What is the domain of f(x) = √(x - 3)?
The expression under a square root must be non-negative: x - 3 ≥ 0, so x ≥ 3. This ensures the output is a real number.
If f(x) = x² + 1, what is f(-3)?
f(-3) = (-3)² + 1 = 9 + 1 = 10. Remember that squaring a negative number produces a positive result.
What is a function?
A function is a special type of relation where each element in the domain (input) corresponds to exactly one element in the range (output). This is verified using the vertical line test.
What is the domain of a function?
The domain is the complete set of all possible input values (x-values) for which the function is defined. Restrictions come from denominators equaling zero or negative values under even radicals.
What is function composition?
Function composition, written as (f ∘ g)(x) = f(g(x)), means substituting the output of g(x) as the input of f(x). It chains two functions together sequentially.
What is the range of a function?
The range is the complete set of all possible output values (y-values) that a function can produce. It depends on both the function's rule and its domain.
What is an inverse function?
An inverse function f⁻¹(x) undoes the action of f(x), swapping inputs and outputs. If f(a) = b, then f⁻¹(b) = a, and its graph is a reflection of f over the line y = x.
What is a one-to-one function?
A one-to-one function passes the horizontal line test, meaning no two different inputs produce the same output. Only one-to-one functions have inverse functions.
What is a piecewise function?
A piecewise function uses different formulas or rules for different parts of its domain. Each piece applies only on the specified interval, and together they define the complete function.
What is the horizontal line test?
The horizontal line test determines whether a function is one-to-one. If every horizontal line crosses the graph at most once, the function is one-to-one and has an inverse function.
A car rental company charges a flat fee of $50 plus $0.25 per mile driven. If the cost function is C(m) = 0.25m + 50, what is the domain of this function in a real-world context?
In a real-world context, you cannot drive a negative number of miles, so the domain is m ≥ 0. The mathematical domain would be all real numbers, but the practical domain restricts m to non-negative values.
A temperature conversion formula is F(C) = (9/5)C + 32. A scientist records a temperature of 20°C. What is this in Fahrenheit?
F(20) = (9/5)(20) + 32 = 36 + 32 = 68°F. Substitute the Celsius value into the conversion function and simplify.
A concert venue has a capacity restriction such that the number of attendees a must satisfy 0 ≤ a ≤ 5000. The revenue function is R(a) = 25a. What is the range of R in this context?
The minimum revenue is R(0) = $0 and the maximum is R(5000) = 25(5000) = $125,000. The restricted domain produces a restricted range of 0 ≤ R ≤ 125,000.
A shipping company converts weights from kilograms to pounds using P(k) = 2.2k. If a package weighs 55 pounds, what is its weight in kilograms? (Use the inverse function.)
The inverse is P⁻¹(p) = p/2.2. So P⁻¹(55) = 55/2.2 = 25 kg. Finding the inverse function lets you convert in the opposite direction.
A biologist models a fish population with P(t) = √(100t + 400), where t is in years. What is the population at time t = 0, and what real-world restriction applies to the domain?
P(0) = √(0 + 400) = √400 = 20. Mathematically the radicand requires t ≥ -4, but since t represents time in years, the practical domain is t ≥ 0.
Real-World Applications
Example: A shipping company converts weights from kilograms to pounds using P(k) = 2.2k. If a package weighs 55 pounds, what is its weight in kilograms? (Use the inverse function.)
The inverse is P⁻¹(p) = p/2.2. So P⁻¹(55) = 55/2.2 = 25 kg. Finding the inverse function lets you convert in the opposite direction.

--- Lesson 1.2: Linear Equations & Graphing (Summary) ---

  Key Concepts: Linear Equations & Graphing
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the slope of y = 3x - 7?
In slope-intercept form y = mx + b, the coefficient of x is the slope. Here m = 3, so the slope is 3.
What is the slope of a line through (1, 3) and (4, 9)?
Slope = (y2 - y1)/(x2 - x1) = (9 - 3)/(4 - 1) = 6/3 = 2. The slope formula measures the rate of change between two points.
What is the slope of a horizontal line?
A horizontal line has no vertical change (rise = 0), so slope = rise/run = 0/run = 0. The line stays at the same y-value for all x.
What is the slope of a line perpendicular to y = (2/3)x + 1?
Perpendicular lines have slopes that are negative reciprocals. The negative reciprocal of 2/3 is -3/2.
What is slope-intercept form?
Slope-intercept form is y = mx + b, where m represents the slope (rate of change) and b represents the y-intercept (the point where the line crosses the y-axis).
What is point-slope form?
Point-slope form y - y1 = m(x - x1) is useful when you know the slope and one point on the line. It can be easily converted to slope-intercept or standard form.
What is standard form of a linear equation?
Standard form Ax + By = C organizes both variables on one side of the equation. It is particularly useful for finding intercepts and working with systems of equations.
What is a linear equation?
A linear equation is a first-degree polynomial equation in one or two variables whose graph produces a straight line. The variables have exponents of 1 and are not multiplied together.
What is the x-intercept of a line?
The x-intercept is where the line crosses the x-axis. At this point, the y-coordinate is zero, so you find it by substituting y = 0 into the equation and solving for x.
What is the slope formula?
The slope formula m = (y2 - y1)/(x2 - x1) calculates the ratio of vertical change to horizontal change between two points. It is often called 'rise over run.'
A cell phone plan costs $25 per month plus $0.10 per text message. If Elena sent 150 texts this month, what is her total bill?
C(150) = 0.10(150) + 25 = 15 + 25 = $40. The per-text charge of $0.10 multiplied by 150 texts gives $15, plus the $25 monthly base.
A delivery truck travels at a constant speed. It is 200 miles from its destination at noon and 80 miles away at 4:00 PM. What is its speed in miles per hour?
The truck traveled 200 - 80 = 120 miles in 4 hours. Speed = distance/time = 120/4 = 30 mph. The slope of the distance-time relationship gives the rate.
A company's revenue increases linearly from $10,000 in its first year to $34,000 in its seventh year. What is the annual rate of increase?
Rate of increase = (34,000 - 10,000)/(7 - 1) = 24,000/6 = $4,000 per year. This is the slope of the linear revenue model.
A store marks up items linearly. An item costing the store $20 is sold for $32, and an item costing $50 is sold for $74. What is the selling price of an item that costs the store $35?
Slope = (74 - 32)/(50 - 20) = 42/30 = 1.4. Using point-slope: S = 1.4(35 - 20) + 32 = 1.4(15) + 32 = 21 + 32 = $53.
A ramp must rise 3 feet over a horizontal distance of 36 feet to meet accessibility standards. What is the slope of the ramp, and does it meet the maximum allowed slope of 1/12?
Slope = rise/run = 3/36 = 1/12. The ramp rises 1 foot for every 12 feet of horizontal distance, which exactly meets the maximum slope standard.
Real-World Applications
Example: A cell phone plan costs $25 per month plus $0.10 per text message. If Elena sent 150 texts this month, what is her total bill?
C(150) = 0.10(150) + 25 = 15 + 25 = $40. The per-text charge of $0.10 multiplied by 150 texts gives $15, plus the $25 monthly base.

--- Lesson 1.3: Systems of Linear Equations (Summary) ---

  Key Concepts: Systems of Linear Equations
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Solve: 3x + 2y = 12 and x = 2. What is y?
Substitute x = 2: 3(2) + 2y = 12, 6 + 2y = 12, 2y = 6, y = 3.
What is a system of linear equations?
A system of linear equations consists of two or more linear equations sharing the same variables. The goal is to find values that satisfy all equations at once.
What is a consistent system?
A consistent system has at least one solution, meaning the graphs share at least one point. It may be independent (one solution) or dependent (infinitely many).
What is an inconsistent system?
An inconsistent system has no solution because the equations represent parallel lines that never intersect. Algebraically, solving leads to a false statement like 0 = 5.
What is a dependent system?
A dependent system has equations that are multiples of each other, representing the same line. Every point on that line is a solution, resulting in infinitely many solutions.
What is an independent system?
An independent system is a consistent system where two equations have different slopes. The lines intersect at exactly one point, which is the unique solution.
What is the substitution method?
The substitution method involves isolating one variable in one equation, then replacing that variable in the other equation with the expression. This reduces the system to one equation in one variable.
What is the elimination method?
The elimination method adds or subtracts equations (possibly after multiplying by constants) so that one variable cancels out, leaving a single equation in one variable.
What is a solution to a system of equations?
A solution is an ordered pair that makes every equation in the system true when substituted. Graphically, it is the intersection point of the lines.
What is a simultaneous equation?
Simultaneous equations are two or more equations that share the same variables and must be true at the same time. The solution satisfies every equation simultaneously.
A boat travels 24 miles downstream in 2 hours and upstream in 3 hours. What is the speed of the current?
Downstream: b + c = 12. Upstream: b - c = 8. Add: 2b = 20, b = 10. Then c = 12 - 10 = 2 mph.

--- Lesson 1.4: Solving Systems by Substitution (Summary) ---

  Key Concepts: Solving Systems by Substitution
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the substitution method for solving systems?
The substitution method solves one equation for one variable, then substitutes that expression into the other equation, creating a single equation in one unknown.
What is an equivalent expression?
Equivalent expressions have the same value for all valid inputs. For example, if y = 2x + 1, then 2x + 1 is equivalent to y and can replace it in another equation.
What is back-substitution?
After finding one variable's value, back-substitution plugs it into an original equation to find the other variable, completing the solution.
What is a contradiction in a system of equations?
A contradiction is a false statement arising during the solving process, such as 0 = 5. It indicates the system is inconsistent with no solution.
What is an identity in solving systems?
When solving produces a true identity like 0 = 0, the equations represent the same line with infinitely many solutions.
What is a coefficient?
A coefficient is the numerical factor in a term containing a variable. In 3x, the coefficient is 3.
What is an ordered pair?
An ordered pair (x, y) represents a specific point. The first number is the x-coordinate and the second is the y-coordinate; swapping them gives a different point.
What is a linear combination?
A linear combination is formed by multiplying each equation by a constant and adding the results. This is the basis of the elimination method.
Real-World Applications
Example: A phone plan charges $0.05/min (Plan X) versus $10/month + $0.02/min (Plan Y). At what usage per month do both plans cost the same?
Set 0.05m = 10 + 0.02m: 0.03m = 10, m = 333.33 minutes. Both plans cost about $16.67 at that usage.

Key Definitions
What is the elimination method?
The elimination (or addition) method combines equations so that one variable is eliminated, reducing the system to a single equation with one unknown that can be solved directly.
What is the least common multiple (LCM)?
The LCM is the smallest positive integer that both numbers divide into evenly. In elimination, we often use the LCM of coefficients to determine what multipliers to apply to each equation.
What is a multiplier in the elimination method?
A multiplier is a constant you multiply every term of an equation by, so that a variable's coefficient matches (or opposes) the corresponding coefficient in the other equation.
What is scaling an equation?
Scaling means multiplying every term in an equation by the same nonzero constant. This produces an equivalent equation and is used to create matching or opposite coefficients for elimination.
What is a trivially true statement in solving systems?
A trivially true statement like 0 = 0 means both equations describe the same line. Every point on that line satisfies both equations, giving infinitely many solutions.
What is a false statement in solving systems?
A false statement like 0 = 7 arises when the system is inconsistent (parallel lines). No ordered pair satisfies both equations, so there is no solution.
What is the addition property of equality?
The addition property of equality states that adding the same quantity to both sides of an equation preserves equality. This principle justifies adding two equations together in the elimination method.

--- Lesson 1.6: Applications of Linear Systems (Summary) ---

  Key Concepts: Applications of Linear Systems
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Two numbers have a sum of 50 and a difference of 14. What is the larger number?
x + y = 50 and x - y = 14. Add: 2x = 64, x = 32. The larger number is 32.
A boat goes 30 miles downstream in 2 hours and 30 miles upstream in 3 hours. What is the boat's speed in still water?
Downstream: b + c = 15; Upstream: b - c = 10. Add: 2b = 25, b = 12.5 mph.
A rectangle's perimeter is 36 cm. Its length is 4 cm more than its width. What is the length?
2l + 2w = 36 and l = w + 4. Substitute: 2(w+4) + 2w = 36, 4w = 28, w = 7, l = 11 cm.
An airplane flies 1200 miles with the wind in 3 hours. The return trip takes 4 hours. What is the wind speed?
With wind: p + w = 400. Against: p - w = 300. Subtract: 2w = 100, w = 50 mph.
A store sells 2 shirts and 3 hats for $56. One shirt and 2 hats cost $32. What is the price of a shirt?
2s + 3h = 56 and s + 2h = 32. Multiply second by -2: -2s - 4h = -64. Add: -h = -8, h = 8. Then s + 16 = 32, s = $16.
What is a mixture problem?
A mixture problem involves combining two or more substances with different properties (like concentration or price) to create a mixture with a desired property. These are modeled with systems of equations.
What is a rate-time-distance problem?
Rate-time-distance problems use the formula d = rt. When two objects travel with or against a current/wind, systems of equations model the effective speeds and solve for unknowns.
What is a break-even point?
The break-even point occurs when revenue equals cost (R = C), meaning the business earns exactly enough to cover expenses. It is found by setting the revenue and cost equations equal.
What is a coin problem in algebra?
Coin problems use two equations: one for the total number of coins and one for the total monetary value. Each coin type has a known value, creating a system that can be solved.
What is a system of equations word problem?
System word problems describe real situations with multiple unknowns and relationships. Translating the words into equations creates a system that can be solved algebraically.
What is the effective speed when traveling with a current?
When traveling with a current (downstream or with the wind), the effective speed is the sum of the object's own speed and the current speed, since both push in the same direction.
What is the perimeter of a rectangle?
The perimeter P = 2l + 2w is the total distance around a rectangle. Perimeter problems often create one equation, and a relationship between l and w creates the second.
What is the relationship between cost, revenue, and profit?
Profit equals revenue minus cost (P = R - C). When revenue exceeds cost, there is a positive profit. At the break-even point, profit is zero.
What is a weighted average?
A weighted average accounts for the quantity of each component. In mixture problems, the final concentration is the weighted average of the component concentrations.
What is the interest formula for simple interest?

--- Lesson 1.7: Inequalities & System of Inequalities (Summary) ---

  Key Concepts: Inequalities & System of Inequalities
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is an inequality?
An inequality is a mathematical statement showing that one expression is less than, greater than, or not equal to another. Unlike equations, inequalities typically have a range of solutions.
What is a compound inequality?
A compound inequality combines two inequalities. With 'and,' both must be true (intersection). With 'or,' at least one must be true (union).
What is a feasible region?
The feasible region is the set of all points satisfying every inequality in the system. In linear programming, the optimal solution lies within or on the boundary of this region.
What is a boundary line?
The boundary line is the equation you get by replacing the inequality symbol with an equals sign. It divides the plane into two half-planes, one of which contains the solutions.
What is a half-plane?
A boundary line divides the coordinate plane into two half-planes. A linear inequality selects one of these half-planes as its solution set.
What is an absolute value inequality?
An absolute value inequality like |x - a| < b means the distance from x to a is less than b, which translates to the compound inequality a - b < x < a + b.
What is a solution set?
The solution set is the complete collection of all values that satisfy a given inequality or system. For inequalities, this is typically a range or region rather than a single point.
What is interval notation?
Interval notation uses brackets [ ] for included endpoints and parentheses ( ) for excluded endpoints. For example, -2 < x ≤ 5 is written as (-2, 5].

A budget allows at most $500 for advertising. Online ads cost $10 each and print ads cost $25 each. At least 5 of each type must be purchased. What is the maximum number of online ads?
Constraints: 10o + 25p ≤ 500, o ≥ 5, p ≥ 5. Minimize print ads at p = 5: 10o + 125 ≤ 500, 10o ≤ 375, o ≤ 37.5. Since o must be a whole number, maximum online ads = 37.

--- Lesson 1.8: Linear Programming (Summary) ---

  Key Concepts: Linear Programming
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Maximize P = 3x + 5y subject to x + y ≤ 10, x ≥ 0, y ≥ 0. The vertices are (0,0), (10,0), and (0,10). What is the maximum P?
Evaluate P at each vertex: P(0,0)=0, P(10,0)=30, P(0,10)=50. The maximum is P=50 at (0,10).
Minimize C = 4x + 3y at vertices (1,2), (5,1), (4,5), (2,4). What is the minimum C?
C(1,2)=4+6=10, C(5,1)=20+3=23, C(4,5)=16+15=31, C(2,4)=8+12=20. Minimum is 10 at (1,2).
What is linear programming?
Linear programming is a mathematical optimization technique that finds the maximum or minimum value of a linear function subject to a set of linear inequality constraints.
What is an objective function?
The objective function is the linear expression (like P = 3x + 5y) that represents the quantity to be optimized. It is evaluated at each vertex of the feasible region to find the optimal value.
What is a constraint in linear programming?
Constraints are linear inequalities that model real-world limitations such as available resources, budget limits, or minimum requirements. Together they define the feasible region.
What is the feasible region?
The feasible region is the intersection of all constraint half-planes. It is the set of all points (x, y) that satisfy every constraint, and the optimal solution lies within or on its boundary.
What is the Corner Point Theorem?
The Corner Point Theorem guarantees that on a bounded polygonal feasible region, the maximum and minimum of a linear objective function occur at vertices. This allows checking only finitely many points.
What is a vertex of the feasible region?
A vertex is a corner point of the feasible region formed by the intersection of two or more constraint boundaries. These are the candidate points for the optimal solution.
What is an infeasible problem?
An infeasible problem has contradictory constraints, so no point satisfies all of them. The feasible region is empty, and no optimal solution exists.
What is a bounded feasible region?
A bounded feasible region is enclosed on all sides and does not extend to infinity. Both the maximum and minimum of the objective function are guaranteed to exist in a bounded region.
A shipping company uses small vans (capacity 50 boxes, cost $100/trip) and large trucks (capacity 200 boxes, cost $300/trip). They must ship at least 1000 boxes with at most 8 vehicles. What is the minimum cost?
50v + 200t ≥ 1000, v + t ≤ 8, v,t ≥ 0. C = 100v + 300t. At (0,5): 1000 boxes shipped, cost = 300(5) = $1,500. At (4,4): cost = 400+1200 = $1,600. Minimum is $1,500.
A company makes desks ($50 profit) and chairs ($30 profit). Assembly: desk = 4 hrs, chair = 2 hrs (80 hrs available). Finishing: desk = 2 hrs, chair = 3 hrs (90 hrs available). What is the maximum profit?
4d + 2c ≤ 80, 2d + 3c ≤ 90. Intersection of boundary lines: d = 7.5, c = 25. P(7.5, 25) = 375 + 750 = $1,125. This exceeds P(20,0) = $1,000 and P(0,30) = $900.
Real-World Applications
Example: A toy store orders dolls ($5 cost, $12 revenue) and cars ($3 cost, $7 revenue). The budget is $300 and storage fits 80 items. How many of each maximizes revenue?
5d + 3c ≤ 300, d + c ≤ 80, d,c ≥ 0. R = 12d + 7c. Vertices: (0,0), (60,0), (0,80). Intersection: 5d+3(80-d)=300, 2d=60, d=30, c=50. R(30,50)=360+350=710, R(60,0)=720, R(0,80)=560. Max = $720.

--- Lesson 1.9: Advanced Linear Applications (Summary) ---

  Key Concepts: Advanced Linear Applications
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Solve: x + y + z = 6, 2x - y + z = 3, x + 2y - z = 5. What is x?
Add eq1 and eq2: 3x + 2z = 9. Add eq1 and eq3: 2x + 3y = 11. Add eq2 and eq3: 3x + y = 8. From 3x+y=8 and eq1 rearranged, solve to get x = 2, y = 2, z = 2.
Solve: x + y = 5, y + z = 7, x + z = 8. What is z?
Add all three: 2x + 2y + 2z = 20, so x + y + z = 10. Since x + y = 5, z = 10 - 5 = 5.
What is an augmented matrix?
An augmented matrix combines the coefficient matrix and constant column into one structure, separated by a vertical bar. Row operations on this matrix solve the system.
What is Gaussian elimination?
Gaussian elimination systematically applies row operations to transform an augmented matrix into row echelon form, from which variables are found by back-substitution.
What is row echelon form?
Row echelon form has a staircase pattern where the leading nonzero entry of each row is to the right of the one in the row above, and all entries below each leading entry are zero.
What is a determinant?
A determinant is a scalar computed from a square matrix. For [a b; c d], det = ad - bc. A nonzero determinant means the matrix is invertible and the system has a unique solution.
What is Cramer's Rule?
Cramer's Rule computes each variable as a ratio of determinants. The numerator determinant replaces the variable's column with the constant column; the denominator is the coefficient determinant.
What is an elementary row operation?
The three elementary row operations are swapping two rows, multiplying a row by a nonzero scalar, and adding a scalar multiple of one row to another. These preserve the system's solutions.
What is a three-variable system?
A three-variable system has three equations with unknowns x, y, and z. Geometrically, each equation represents a plane, and the solution is where all three planes intersect.
What is a coefficient matrix?
The coefficient matrix contains only the coefficients of the variables, not the constant terms. It is used in Cramer's Rule and matrix methods for solving systems.
What is a free variable?
A free variable is not uniquely determined by the system and can take any real value. Other variables are expressed in terms of it, generating infinitely many solutions.
What is an ordered triple?
An ordered triple (x, y, z) assigns values to three variables in a specific order. It represents a point in 3D space and is the solution format for three-variable systems.
A factory produces items A ($10 profit), B ($15 profit), and C ($20 profit). They produce 100 units total. Item B production exceeds A by 20 units. Item C is 10 fewer than A. What is the total profit?
a + b + c = 100, b = a + 20, c = a - 10. Substitute: a + (a + 20) + (a - 10) = 100, 3a = 90, a = 30, b = 50, c = 20. Profit = 300 + 750 + 400 = $1,450.
Real-World Applications
Example: A small business tracks monthly expenses: rent (r), wages (w), and supplies (s). Total expenses are $10,000. Wages are three times supplies. Rent is $2,000 more than supplies. What are the monthly wages?
r + w + s = 10000, w = 3s, r = s + 2000. Substitute: (s + 2000) + 3s + s = 10000, 5s = 8000, s = 1600. Wages = 3(1600) = $4,800.
Example: A shipping company loads three types of packages: type A weighs 2 lbs, type B weighs 3 lbs, and type C weighs 10 lbs. A truck carries 20 packages weighing 80 lbs total. If there are equal numbers of A and B packages, how many C packages are on the truck?
Set up: a + b + c = 20, 2a + 3b + 10c = 80, a = b. Substituting a = b gives 2b + c = 20, so c = 20 - 2b. Then 5b + 10(20 - 2b) = 80, which simplifies to -15b = -120, so b = 8, a = 8, and c = 4.

  Q21. A small business tracks monthly expenses: rent (r), wages (w), and supplies (s). Total expenses are $10,000. Wages are three times supplies. Rent is $2,000 more than supplies. What are the monthly wages?

  1. Q: Solve 3×3 system to find x, y, z form solution set
     A: Process: Gaussian elimination to row echelon form, then back-substitute.

  2. Q: Augmented matrix for x+y+z=6, 2x-y+z=3, x+2y-z=2?

     A: Inconsistent system (0=5 is false). No solution exists.

  4. Q: Row [0 0 0 | 0] indicates what?
     A: Dependent system (infinitely many solutions). Free variables exist.

  5. Q: Back-substitution: from z=1 in 2x+y+3z=8, 2y+z=3
     A: From 2y+1=3: y=1. From 2x+1+3=8: x=2.

  6. Q: Parametric solution if z is free: x=3-z, y=2+z, z=t?
     A: Infinitely many: (3-t, 2+t, t). One parameter, line of solutions.

  7. Q: Classify system: x+y=1, x+y=2
     A: Inconsistent (parallel lines). No solution.

  8. Q: Solve: x-2y+z=0, 2x+y-z=5, x+y=2
     A: x=7/3, y=-1/3, z=5/3. Use elimination/substitution systematically.

  9. Q: AP Connection: how many solutions for generic 3×3 system?
     A: Unique (1), infinitely many, or none. Determinant crucial for existence.

  10. Q: Linear combination test: is (1,2,3) solution to x+y+z=6, 2x-y+z=3?
