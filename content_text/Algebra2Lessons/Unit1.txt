--- Lesson 1.1: Review of Functions & Notation (Summary) ---

  Key Concepts: Review of Functions & Notation
Functions are relations that assign exactly one output to each input. A function f(x) can be evaluated by substituting a value for x and simplifying.

Function Notation: f(x) represents the output of function f when the input is x. This replaces the older y = notation and makes it clear which variable is the input.

Domain and Range: The domain is the set of all possible input values, while the range is the set of all possible output values. Restrictions occur when denominators are zero or when values are under even-root radicals.

Operations with Functions: Functions can be combined through addition, subtraction, multiplication, and division. Composition of functions (f âˆ˜ g)(x) = f(g(x)) means apply g first, then apply f to the result.

Point-Slope Form: y - yâ‚ = m(x - xâ‚) is useful when you know a point on the line and the slope. This form can be rearranged to slope-intercept form.

--- Lesson 1.3: Systems of Linear Equations (Summary) ---

  Key Concepts: Systems of Linear Equations
What is a System? A system of equations consists of two or more equations with the same variables. A solution must satisfy all equations simultaneously.

Graphical Solutions: Graph each line and find intersection point(s). One intersection = unique solution. Parallel lines = no solution. Same line = infinitely many solutions.

Types of Systems: Consistent and independent: exactly one solution. Consistent and dependent: infinitely many solutions. Inconsistent: no solution.

Algebraic Methods: Prepare to learn substitution and elimination methods in the next lessons. Both methods transform the system to isolate variables and find the solution.

--- Lesson 1.4: Solving Systems by Substitution (Summary) ---

  Key Concepts: Solving Systems by Substitution
The Substitution Method: Solve one equation for a variable, then substitute that expression into the other equation. This reduces the system to one equation with one variable.

Step-by-Step Process: (1) Solve one equation for one variable (choose the easiest). (2) Substitute the expression into the other equation. (3) Solve the resulting one-variable equation. (4) Back-substitute to find the other variable.

When to Use Substitution: This method works best when one variable has a coefficient of 1 or -1, making it easy to isolate. If all coefficients are complex, elimination might be easier.

Checking Solutions: Substitute the x and y values into both original equations. The solution is correct if both equations are satisfied.

--- Lesson 1.5: Solving Systems by Elimination (Summary) ---

  Key Concepts: Solving Systems by Elimination
The Elimination Method: Multiply one or both equations by constants so that one variable has opposite coefficients. Add the equations to eliminate that variable, leaving one equation with one unknown.

Making Coefficients Opposite: If the same variable has coefficients like 2 and 3, multiply the first equation by 3 and the second by -2 to create coefficients 6 and -6. When added, they cancel.

Step-by-Step Process: (1) Multiply equation(s) to make coefficients opposites. (2) Add the equations. (3) Solve for the remaining variable. (4) Substitute back to find the other variable.

Advantages: Elimination is more efficient when coefficients are large or when substitution creates messy fractions. It avoids the need to isolate variables first.

--- Lesson 1.8: Linear Programming (Summary) ---

  Key Concepts: Linear Programming
What is Linear Programming? A method to find the maximum or minimum value of an objective function subject to linear constraints. Used in business for optimization problems.

Components: Objective function: what we're maximizing/minimizing (P = ax + by). Constraints: limitations on variables (system of inequalities). Feasible region: all points satisfying all constraints.

The Fundamental Theorem: The maximum or minimum occurs at a corner point (vertex) of the feasible region. Find all vertices and test each in the objective function.

Solving Steps: (1) Write objective function. (2) Write constraints as inequalities. (3) Graph feasible region. (4) Find corner points. (5) Evaluate objective function at each corner. (6) Identify the maximum/minimum.
