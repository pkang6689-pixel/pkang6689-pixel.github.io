--- Lesson 2.1: Quadratic Functions & Parabolas (Summary) ---

  Key Concepts: Quadratic Functions & Parabolas
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of f(x) = x^2 - 4x + 3?
The vertex x-coordinate is -b/(2a) = 4/2 = 2. Substituting x = 2 gives f(2) = 4 - 8 + 3 = -1, so the vertex is (2, -1).
What is the axis of symmetry for y = x^2 + 8x + 12?
The axis of symmetry is x = -b/(2a) = -8/(2*1) = -4. This vertical line passes through the vertex of the parabola.
What is the y-intercept of f(x) = 2x^2 - 5x + 7?
The y-intercept occurs when x = 0. Substituting gives f(0) = 0 - 0 + 7 = 7, so the y-intercept is 7.
What is the maximum or minimum value of f(x) = -x^2 + 6x - 4?
Since a = -1 < 0, the parabola opens downward and has a maximum. The vertex x-coordinate is 3, and f(3) = -9 + 18 - 4 = 5, so the maximum value is 5.
If f(x) = 3x^2 - 12x + 7, what is f(2)?
Substituting x = 2: f(2) = 3(4) - 12(2) + 7 = 12 - 24 + 7 = -5.
What is the range of y = x^2 - 2x - 3?
The vertex is at x = 1, and f(1) = 1 - 2 - 3 = -4. Since a = 1 > 0, the parabola opens upward, so the range is y >= -4.
What is a quadratic function?
A quadratic function is a polynomial of degree 2 in the form f(x) = ax^2 + bx + c, where a, b, and c are constants and a cannot equal zero.
What is a parabola?
A parabola is the U-shaped (or inverted U-shaped) curve produced by graphing a quadratic function. It is symmetric about a vertical line through its vertex.
What is the vertex of a parabola?
The vertex is the turning point of a parabola, representing either the maximum point (when a < 0) or the minimum point (when a > 0) of the quadratic function.
What is the axis of symmetry of a parabola?
The axis of symmetry is the vertical line x = -b/(2a) that passes through the vertex and divides the parabola into two congruent halves that are mirror images of each other.
What is the standard form of a quadratic function?
The standard form of a quadratic function is f(x) = ax^2 + bx + c, where a, b, and c are real number constants and a is not zero. This form clearly shows the coefficients used in many formulas.
What is the vertex form of a quadratic function?
The vertex form is f(x) = a(x - h)^2 + k, which directly reveals the vertex at (h, k) and shows the direction and width through the coefficient a.
A ball is thrown upward from a 6-foot platform with an initial velocity of 32 ft/s. Its height is modeled by h(t) = -16t^2 + 32t + 6. What is the maximum height the ball reaches?
The maximum occurs at t = -32/(2*(-16)) = 1 second. Substituting: h(1) = -16(1) + 32(1) + 6 = -16 + 32 + 6 = 22 feet.
A farmer has 60 meters of fencing to enclose a rectangular garden against a barn wall (only three sides need fencing). What is the maximum area of the garden?
Let x be the width. Then the length is 60 - 2x, and area A = x(60 - 2x) = -2x^2 + 60x. Maximum at x = 15: A = 15(30) = 450 square meters.
A bridge arch is modeled by y = -0.01x^2 + 2x, where x is the horizontal distance in feet and y is the height. What is the maximum height of the arch?
Maximum height at x = -2/(2*(-0.01)) = -2/(-0.02) = 100. Then y = -0.01(10000) + 2(100) = -100 + 200 = 100 feet.
A store sells phone cases. The daily profit is P(x) = -x^2 + 24x - 80, where x is the number sold. What is the maximum daily profit?
Maximum profit at x = -24/(2*(-1)) = 12 cases. P(12) = -144 + 288 - 80 = 64. The maximum daily profit is $64.

--- Lesson 2.2: Transformations of Quadratics (Summary) ---

  Key Concepts: Transformations of Quadratics
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of y = 3(x + 2)^2 - 7?
In vertex form y = a(x - h)^2 + k, the vertex is (h, k). Here x + 2 = x - (-2), so h = -2 and k = -7. The vertex is (-2, -7).
What is the equation of y = x^2 shifted left 3 units and down 2 units?
Shifting left 3 replaces x with (x + 3), and shifting down 2 subtracts 2. The resulting equation is y = (x + 3)^2 - 2.
If f(x) = x^2, what is g(x) = f(x - 5) + 1?
Replacing x with (x - 5) in f gives (x - 5)^2, then adding 1 produces g(x) = (x - 5)^2 + 1. This shifts the parabola right 5 and up 1.
What is the vertex of y = -2(x + 1)^2 + 4?
In vertex form y = a(x - h)^2 + k, (x + 1) means h = -1 and k = 4. The vertex is (-1, 4), and since a = -2 < 0, it opens downward.
What is a vertical translation of a function?
A vertical translation moves every point on a graph the same number of units up or down. It is achieved by adding or subtracting a constant to the function: f(x) + k.
What is a horizontal translation of a function?
A horizontal translation shifts the entire graph left or right. It is achieved by replacing x with (x - h): f(x - h) shifts right h units, and f(x + h) shifts left h units.
What is a reflection over the x-axis?
A reflection over the x-axis replaces f(x) with -f(x), which flips every point across the x-axis. For a parabola, this changes the direction it opens.
What is a vertical stretch?
A vertical stretch multiplies all y-values by a factor greater than 1, pulling points farther from the x-axis. For a parabola, this makes the curve narrower and steeper.
What is a vertical compression?
A vertical compression multiplies all y-values by a factor between 0 and 1, pushing points closer to the x-axis. For a parabola, this produces a wider, flatter curve.
What is the parent function of a quadratic?
The parent function of all quadratics is y = x^2, the simplest quadratic with vertex at the origin. All other quadratics are transformations of this basic parabola.
What is a non-rigid transformation?
A non-rigid transformation changes the shape or size of the graph. Vertical stretches, compressions, and horizontal stretches/compressions are examples of non-rigid transformations.
An architect designs a parabolic archway. The basic arch follows y = -x^2 + 16. To fit the building, it must be shifted right 5 feet and up 3 feet. What is the equation of the repositioned arch?
Shifting right 5 replaces x with (x - 5), and shifting up 3 adds 3 to the constant. The equation becomes y = -(x - 5)^2 + 16 + 3 = -(x - 5)^2 + 19.
A roller coaster designer takes the shape y = x^2 and reflects it, shifts it right 6, and up 50 to create a drop. If the ride car is at x = 10, what is the height?
The transformed equation is y = -(x - 6)^2 + 50. At x = 10: y = -(10 - 6)^2 + 50 = -16 + 50 = 34 units high.
A landscape artist models a fountain spray with y = -2x^2. To move the spray center 3 feet left and raise its base 4 feet, what is the new equation?
Shifting left 3 replaces x with (x + 3), and raising the base 4 adds 4. The result is y = -2(x + 3)^2 + 4, with vertex at (-3, 4).
A manufacturer models production costs with C(x) = x^2 (in thousands). After upgrading equipment, costs are halved and there is a fixed setup cost of $5,000. What is the new cost model?
Halving costs applies a vertical compression by 0.5, giving 0.5x^2. Adding a fixed cost of $5,000 (represented as 5 in thousands) shifts up by 5: C(x) = 0.5x^2 + 5.
An engineer models a cable's sag as y = 0.01x^2. To reduce the sag by half and account for a 5-meter elevation, what is the new model?
Reducing sag by half means compressing vertically by 1/2: 0.01 * 0.5 = 0.005. Adding 5-meter elevation shifts up 5. The new model is y = 0.005x^2 + 5.
A searchlight beam on a wall follows y = 2x^2. If the light is moved 4 meters to the right and aimed 6 meters higher, what is the new equation? What is the height of the beam at x = 7?
The transformed equation is y = 2(x - 4)^2 + 6. At x = 7: y = 2(7 - 4)^2 + 6 = 2(9) + 6 = 18 + 6 = 24 meters.

--- Lesson 2.3: Completing the Square (Summary) ---

  Key Concepts: Completing the Square
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Complete the square for 2x^2 + 12x + 10. What is the vertex form?
Factor out 2: 2(x^2 + 6x) + 10 = 2(x^2 + 6x + 9 - 9) + 10 = 2(x + 3)^2 - 18 + 10 = 2(x + 3)^2 - 8.
What is the vertex of y = x^2 + 12x + 32?
Completing the square: x^2 + 12x + 36 - 36 + 32 = (x + 6)^2 - 4. The vertex is (-6, -4).
What is a perfect square trinomial?
A perfect square trinomial has the form a^2 + 2ab + b^2 = (a + b)^2 or a^2 - 2ab + b^2 = (a - b)^2. It results from squaring a binomial.
What is the general rule for the number you add to complete the square in x^2 + bx?
To complete the square for x^2 + bx, you add (b/2)^2, which is the square of half the coefficient of x. This creates the perfect square trinomial (x + b/2)^2.
What is the vertex form of a quadratic equation?
The vertex form is y = a(x - h)^2 + k, where (h, k) is the vertex. Completing the square converts standard form to vertex form.
What is a binomial?
A binomial is a polynomial expression with exactly two terms, such as (x + 3) or (2x - 5). Squaring a binomial produces a perfect square trinomial.
What is a trinomial?
A trinomial is a polynomial with exactly three terms. For example, x^2 + 5x + 6 is a trinomial. Quadratic expressions in standard form are always trinomials.
A farmer wants to fence a rectangular pasture with 100 meters of fencing. What is the maximum area he can enclose?
Perimeter 2w + 2l = 100, so l = 50 - w. Area A = w(50 - w) = -w^2 + 50w. Completing the square: -(w^2 - 50w + 625) + 625 = -(w - 25)^2 + 625. Maximum area is 625 sq m.
A playground is being built with a rectangular area alongside a school wall. With 80 meters of fencing for three sides, what is the maximum area?
With the wall as one side: 2w + l = 80, so l = 80 - 2w. Area A = w(80 - 2w) = -2w^2 + 80w = -2(w - 20)^2 + 800. Maximum area is 800 sq m when w = 20.

What is the discriminant of 3x^2 + 2x - 5 = 0?

--- Lesson 2.5: Graphing Quadratic Functions (Summary) ---

  Key Concepts: Graphing Quadratic Functions
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the vertex of y = x^2 - 4x - 5?
Vertex x = -(-4)/(2*1) = 2. Then y = 4 - 8 - 5 = -9. The vertex is (2, -9).
What is the y-intercept of y = -3x^2 + 6x + 9?
The y-intercept occurs at x = 0: y = 0 + 0 + 9 = 9. In standard form y = ax^2 + bx + c, the y-intercept is always c.
What is the range of y = -2x^2 + 8x - 3?
Vertex x = -8/(-4) = 2, y = -8 + 16 - 3 = 5. Since a = -2 < 0, the parabola opens down and the range is y <= 5.
What is the vertex of y = -x^2 + 4x + 12?
Vertex x = -4/(2*(-1)) = 2. Then y = -4 + 8 + 12 = 16. The vertex is (2, 16) and is a maximum since a < 0.
If a parabola has vertex (3, -2) and passes through (5, 6), what is the value of a?
Using vertex form: 6 = a(5 - 3)^2 + (-2), so 6 = 4a - 2, giving 8 = 4a, and a = 2.
What is the 'end behavior' of a quadratic function?
End behavior describes what happens to y as x goes to positive or negative infinity. For a > 0, both ends go up; for a < 0, both ends go down.
What is the 'intercept form' of a quadratic?
The intercept form (or factored form) is y = a(x - p)(x - q), where p and q are the x-intercepts (zeros) of the function. This form makes it easy to identify where the graph crosses the x-axis.
What is a 'turning point' of a quadratic graph?
The turning point is the vertex of the parabola. It is where the function changes direction: from decreasing to increasing (minimum) or from increasing to decreasing (maximum).
What is the 'domain' of a quadratic function?
The domain of every quadratic function is all real numbers (-infinity, infinity), because you can substitute any real number for x in ax^2 + bx + c.
What is a 'table of values' used for when graphing?
A table of values organizes several x-values and their corresponding y-values to provide points that can be plotted on a coordinate plane to sketch the graph of the function.
What is the 'concavity' of a quadratic function?
Concavity describes the direction a parabola opens. If a > 0, the parabola is concave up (opens upward like a cup). If a < 0, it is concave down (opens downward).
A football is punted and follows h(t) = -16t^2 + 64t + 2. Sketch-related: at what time does the ball reach its highest point, and what is that height?
The vertex t = -64/(2*(-16)) = 2. Then h(2) = -16(4) + 64(2) + 2 = -64 + 128 + 2 = 66 feet. The highest point is 66 feet at t = 2 seconds.
A company's revenue is R(x) = -4x^2 + 120x, where x is the price in dollars. What price produces the maximum revenue, and what is that revenue?
Vertex x = -120/(2*(-4)) = 15. R(15) = -4(225) + 120(15) = -900 + 1800 = 900. Maximum revenue is $900 at a price of $15.
A bridge cable hangs in the shape y = 0.005x^2 - x + 60 for 0 <= x <= 200. What is the lowest point of the cable?
Vertex x = -(-1)/(2*0.005) = 1/0.01 = 100. y(100) = 0.005(10000) - 100 + 60 = 50 - 100 + 60 = 10. The lowest point is 10 units at x = 100.
A park sprinkler sprays water in a parabolic arc modeled by h(d) = -0.2d^2 + 3d, where d is the distance in feet. What is the maximum height of the water?
Vertex d = -3/(2*(-0.2)) = -3/(-0.4) = 7.5. h(7.5) = -0.2(56.25) + 3(7.5) = -11.25 + 22.5 = 11.25 feet.
An architect designs a parabolic window. The equation is y = -x^2 + 8x for the arch. What is the width of the window at its base?
The base is where y = 0: -x^2 + 8x = 0, so x(8 - x) = 0, giving x = 0 and x = 8. The width is 8 - 0 = 8 units.
A stadium roof follows the curve y = -0.01x^2 + 4x for 0 <= x <= 400. What is the maximum height of the roof?
Vertex x = -4/(2*(-0.01)) = 200. y(200) = -0.01(40000) + 4(200) = -400 + 800 = 400 units high.

--- Lesson 2.6: Applications of Quadratics (Summary) ---

  Key Concepts: Applications of Quadratics
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
A ball is thrown upward with h(t) = -16t^2 + 80t + 5. What is the maximum height?
Vertex t = -80/(2*(-16)) = 2.5. h(2.5) = -16(6.25) + 80(2.5) + 5 = -100 + 200 + 5 = 105 feet.
A rectangular field has perimeter 120 meters. What is the maximum area?
Let w = width. Then l = 60 - w, and A = w(60 - w) = -w^2 + 60w. Maximum at w = 30: A = 30(30) = 900 sq m.
Profit is P(x) = -x^2 + 50x - 400. What is the maximum profit?
Maximum at x = 25: P(25) = -625 + 1250 - 400 = $225.
Two consecutive positive integers have a product of 132. What is the smaller integer?
Let n(n+1) = 132: n^2 + n - 132 = 0. Factoring: (n - 11)(n + 12) = 0. Since n is positive, n = 11.
A water balloon launched from the ground follows h(t) = -4.9t^2 + 19.6t. What is the maximum height?
Vertex t = -19.6/(2*(-4.9)) = 2. h(2) = -4.9(4) + 19.6(2) = -19.6 + 39.2 = 19.6 meters.
What is a 'projectile motion' problem?
Projectile motion problems model the height of an object launched into the air using a quadratic function h(t) = -gt^2 + v0*t + h0, where g is gravitational acceleration.
What is a 'constraint' in an optimization problem?
A constraint is a condition that limits the values variables can take. For example, having a fixed perimeter is a constraint when maximizing area.
What is a 'parabolic trajectory'?
A parabolic trajectory is the curved path an object follows when launched through the air under gravity (ignoring air resistance). It has the shape of a parabola.
What is 'free fall' in physics?
Free fall is when an object falls under the influence of gravity alone, with no initial vertical velocity. The height is modeled by h(t) = -16t^2 + h0 (in feet) or h(t) = -4.9t^2 + h0 (in meters).
A toy manufacturer finds that profit P(x) = -2x^2 + 100x - 800 depends on units x produced. How many units should be produced for maximum profit, and what is that profit?
Maximum at x = -100/(2*(-2)) = 25 units. P(25) = -2(625) + 100(25) - 800 = -1250 + 2500 - 800 = $450.

--- Lesson 2.7: Quadratic Inequalities (Summary) ---

  Key Concepts: Quadratic Inequalities
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a quadratic inequality?
A quadratic inequality is an inequality that can be written in a form like ax^2 + bx + c > 0 (or <, >=, <=), where the highest power of the variable is 2.
What is a 'sign chart' or 'sign analysis'?
A sign chart divides the number line into intervals using the zeros of the quadratic. Test points from each interval determine whether the expression is positive or negative in that region.
What is the difference between a strict inequality and a non-strict inequality?
Strict inequalities (< or >) do not include the boundary values (endpoints), while non-strict inequalities (<= or >=) include the boundary values in the solution set.

--- Lesson 2.7: Quadratic Inequalities (Flashcards) ---

  1. Q: Solve x² - 9 > 0
     A: x < -3 or x > 3. Parabola opens up, above x-axis outside roots.

  2. Q: Solve -x² + 4 ≤ 0
     A: x ≤ -2 or x ≥ 2. Opens down, below x-axis outside zeros.

  3. Q: Sign analysis: (x-1)(x+2) > 0
     A: x < -2 or x > 1. Test intervals: negative, zero, positive regions.

  4. Q: Inequality 2x² + 5x - 3 < 0
     A: -3 < x < 1/2. Factor: (2x-1)(x+3) < 0. Roots -3 and 1/2.

  5. Q: Quadratic inequality solution notation: x² ≥ 16
     A: x ≤ -4 or x ≥ 4. Interval: (-∞,-4] ∪ [4, ∞).

  6. Q: When does ax² + bx + c < 0 have no solution?
     A: When a > 0 and discriminant < 0 (parabola entirely above x-axis).

  7. Q: Solve (x-2)² ≥ 0
     A: All real x. Perfect square always ≥ 0.

  8. Q: Test point method: test x = 0 in x(x-5) > 0
     A: 0(0-5) = 0, not > 0. So x = 0 not in solution set.

  9. Q: Sign chart for x² - 4x - 5 = 0
     A: Roots: 5, -1. Chart: + (x<-1), - (-1<x<5), + (x>5).

  10. Q: Solution to 3x² ≤ 12x?
     A: 0 ≤ x ≤ 4. 3x² - 12x ≤ 0, 3x(x-4) ≤ 0, interval [0,4].
