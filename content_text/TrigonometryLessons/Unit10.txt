--- Lesson 10.1: Trigonometric Proofs & Verification (Summary) ---

  Key Concepts: Trigonometric Proofs & Verification
Trigonometric Proofs & Verification
Proving trig identities means showing the LHS equals the RHS through algebraic manipulation.
Strategies  • Work on one side only (usually the more complex one).  • Convert everything to sin and cos.  • Factor, combine fractions, or multiply by a conjugate.  • Use fundamental identities: Pythagorean, reciprocal, quotient.

--- Lesson 10.2: Trigonometric Substitution (Summary) ---

  Key Concepts: Trigonometric Substitution
Trigonometric Substitution (Calculus Preview)
In calculus, trig substitution simplifies integrals involving √(a²−x²), √(a²+x²), or √(x²−a²).
The Three Cases  • √(a²−x²): let x = a sinθ → √(a²−x²) = a cosθ.  • √(a²+x²): let x = a tanθ → √(a²+x²) = a secθ.  • √(x²−a²): let x = a secθ → √(x²−a²) = a tanθ.

--- Lesson 10.3: Parametric Equations (Summary) ---

  Key Concepts: Parametric Equations
Parametric Equations
Instead of y = f(x), a curve is defined by x = f(t), y = g(t) with parameter t.
Key Skills  • Graphing: Make a table of t → (x, y) and plot.  • Eliminating the parameter: Express t in terms of x (or y) and substitute.  • Direction: As t increases, trace the orientation of the curve.Examples
x = cosθ, y = sinθ → unit circle. x = t², y = t → parabola x = y².

--- Lesson 10.3: Parametric Equations (Quiz) ---

  Q1. In parametric form, a curve is defined by:
    [ ] y = f(x)
    [*] x = f(t), y = g(t)
    [ ] r = f(θ)
    [ ] y = mx + b
    Explanation: Separate functions for x and y in terms of t.

  Q2. x = 2t, y = t + 1. At t = 0:

    [*] (0, 1)

    [ ] A line

  Q10. x = cos(−θ), y = sin(−θ) traces the same circle but:

  Q12. x = t, y = t² for t ∈ [−2, 2] traces part of:

  Q13. Cycloid: x = t − sin t, y = 1 − cos t. At t = 0:

  Q14. At t = π for the cycloid:

  Q15. Parametric equations are useful for:

  Key Concepts: Calculus Connections & Applications
Calculus ConnectionsDerivatives of Trig Functions  • d/dx sin x = cos x  • d/dx cos x = −sin x  • d/dx tan x = sec²xLimits Involving Trig
lim (sin x)/x = 1 as x → 0 (fundamental limit).
Integration
∫sin x dx = −cos x + C, ∫cos x dx = sin x + C. Trig identities simplify complex integrals.
