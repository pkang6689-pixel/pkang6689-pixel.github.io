--- Lesson 3.1: Definition & Domains of Inverse Trig Functions (Summary) ---

  Key Concepts: Definition & Domains of Inverse Trig Functions
Inverse Trigonometric Functions
Because trig functions are periodic (not one-to-one), we restrict domains to create invertible functions.
Principal Restrictions  • sin⁻¹ (arcsin): domain [−1, 1], range [−π/2, π/2].  • cos⁻¹ (arccos): domain [−1, 1], range [0, π].  • tan⁻¹ (arctan): domain (−∞, ∞), range (−π/2, π/2).
Inverse trig functions answer the question: "What angle has this trig value?"

Notation: sin⁻¹ x = arcsin x (not the reciprocal 1/sin x).

  Key Concepts: Graphs of Inverse Trig Functions
Graphs of Inverse Trig Functionsy = arcsin x  • Domain [−1, 1], range [−π/2, π/2]. S-shaped curve passing through the origin.  • Increasing. Endpoints: (−1, −π/2) and (1, π/2).y = arccos x  • Domain [−1, 1], range [0, π]. Decreasing curve.  • Endpoints: (−1, π) and (1, 0).y = arctan x  • Domain (−∞, ∞), range (−π/2, π/2). Increasing S-curve with horizontal asymptotes at y = ±π/2.
Each inverse trig graph is the reflection of the corresponding restricted trig graph over y = x.

--- Lesson 3.3: Solving Equations with Inverse Trig (Summary) ---

  Key Concepts: Solving Equations with Inverse Trig
Solving Equations with Inverse Trig Functions
To solve equations like sin θ = 0.6, use inverse trig functions to find θ.
Steps  • Step 1: Isolate the trig function.  • Step 2: Apply the inverse to get the principal value.  • Step 3: Find all solutions using symmetry and periodicity.Example
Solve sin θ = 0.5 for 0 ≤ θ ≤ 2π.
  • Principal: θ₁ = arcsin(0.5) = π/6.  • Sine is also positive in Q II: θ₂ = π − π/6 = 5π/6.  • Solutions: θ = π/6, 5π/6.

--- Lesson 3.4: Applications in Geometry & Physics (Summary) ---

  Key Concepts: Applications in Geometry & Physics
Applications of Inverse Trig in Geometry & PhysicsGeometry  • Finding angles in right triangles: θ = arctan(opposite/adjacent).  • Angle of elevation/depression: using trig ratios and inverses.  • Angle between vectors: θ = arccos(u·v / (|u||v|)).Physics  • Projectile motion: launch angle θ = arctan(v_y / v_x).  • Inclined planes: component analysis using trig inverses.  • Optics: Snell's law → θ₂ = arcsin((n₁ sin θ₁)/n₂).

--- Lesson 3.5: Restrictions & Principal Values (Summary) ---

  Key Concepts: Restrictions & Principal Values
Restrictions & Principal Values
Because trig functions map many angles to the same value, inverse trig functions must use restricted ranges.
Key Identities for Composition  • sin(arcsin x) = x for x ∈ [−1, 1].  • arcsin(sin θ) = θ only if θ ∈ [−π/2, π/2].  • cos(arccos x) = x for x ∈ [−1, 1].  • arccos(cos θ) = θ only if θ ∈ [0, π].Common Pitfall
arcsin(sin(5π/6)) ≠ 5π/6 because 5π/6 ∉ [−π/2, π/2]. Answer: arcsin(1/2) = π/6.

--- Lesson 3.6: Applications in Engineering (Summary) ---

  Key Concepts: Applications in Engineering
Engineering Applications of Inverse TrigSignal Processing  • Phase detection: arctan(Im/Re) extracts the phase angle of a complex signal.  • Beamforming: arcsin(d sin θ / λ) determines antenna array steering angles.Mechanical Engineering  • Linkage analysis: inverse trig finds joint angles from link lengths.  • Cam design: follower position involves arctan and arccos of geometric ratios.Electrical Engineering  • Power factor angle: θ = arccos(P/S) where P = real power, S = apparent power.  • Impedance angle: θ = arctan(X_L / R) for an RL circuit.

--- Lesson 3.6: Applications in Engineering (Quiz) ---

  Q1. The phase angle of a complex signal is found using:
    [ ] arcsin
    [ ] arccos
    [*] arctan(Im/Re)
    [ ] arctan(Re/Im)
    Explanation: Phase = arctan(imaginary part / real part).

  Q2. Power factor angle in AC circuits: θ =
    [ ] arcsin(P/S)
    [*] arccos(P/S)
    [ ] arctan(P/S)
    [ ] arcsin(S/P)
    Explanation: Power factor = cos θ = P/S, so θ = arccos(P/S).

  Q3. If P = 300 W and S = 500 VA, the power factor angle is:
    [ ] arcsin(0.6)
    [*] arccos(0.6) ≈ 53.1°
    [ ] arctan(0.6)
    [ ] arccos(500/300)
    Explanation: θ = arccos(300/500) = arccos(0.6) ≈ 53.1°.

  Q4. For an RL circuit with R = 100 Ω and X_L = 100 Ω, impedance angle:
    [ ] 30°
    [*] 45°
    [ ] 60°
    [ ] 90°
    Explanation: θ = arctan(100/100) = arctan(1) = 45°.

  Q5. In beamforming, arcsin is used to find:
    [ ] Signal strength
    [*] Steering angle for antenna arrays
    [ ] Power consumption
    [ ] Resistance
    Explanation: arcsin relates element spacing and wavelength to beam direction.

  Q6. A four-bar linkage has links of known lengths. Joint angles are found using:
    [ ] Calculus
    [*] Inverse trig functions applied to the geometry
    [ ] Linear algebra only
    [ ] Integration
    Explanation: Geometric relationships in linkages yield trig equations solved with inverse trig.

  Q7. arctan2(y, x) is preferred over arctan(y/x) in engineering because:
    [ ] It's faster
    [*] It handles all four quadrants correctly
    [ ] It gives radians
    [ ] It's more accurate
    Explanation: arctan2 uses both signs of x and y to determine the correct quadrant.

  Q8. A robot arm with link length L needs to reach point (x, y). The joint angle is:
    [ ] arcsin(L/x)
    [*] arctan(y/x) (simplified)
    [ ] arccos(x+y)
    [ ] π/4 always
    Explanation: The angle to the target is arctan(y/x) (simplified 2D case).

  Q9. In a cam mechanism, the follower position often involves:
    [ ] Only linear equations
    [*] arccos or arctan of geometric ratios
    [ ] Exponentials
    [ ] Logarithms
    Explanation: The geometry of cam profiles leads to inverse trig expressions.

  Q10. If a signal has Re = 3 and Im = 4, the phase angle is:
    [ ] arctan(3/4)
    [*] arctan(4/3) ≈ 53.1°
    [ ] arcsin(4/5)
    [ ] arccos(3/5)
    Explanation: Phase = arctan(Im/Re) = arctan(4/3).

  Q11. An impedance angle of 0° means:
    [*] Purely resistive circuit (no reactance)
    [ ] Purely capacitive
    [ ] Purely inductive
    [ ] Open circuit
    Explanation: θ = 0 → arctan(X/R) = 0 → X = 0, purely resistive.

  Q12. An impedance angle of 90° means:
    [ ] Purely resistive
    [*] Purely reactive (inductive or capacitive)
    [ ] Short circuit
    [ ] No current
    Explanation: θ = 90° → X/R = tan 90° → R ≈ 0, purely reactive.

  Q13. In surveying, a transit measures angles then converts to distances using:
    [ ] Only addition

  Q14. A solar panel's tilt angle for maximum output: θ ≈ latitude. This is related to:

  Q15. Arccos appears in calculating:
