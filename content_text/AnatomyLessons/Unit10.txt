--- Lesson 10.1: Immune System Overview (Summary) ---

  Key Concepts: Immune System Overview
Immune System OverviewInnate (Non-Specific) Immunity  • First line: Skin, mucous membranes, secretions (lysozyme, HCl, sebum).  • Second line: Phagocytes (neutrophils, macrophages), NK cells, inflammation, fever, complement, interferons.Adaptive (Specific) Immunity  • Humoral: B cells → plasma cells → antibodies (IgG, IgA, IgM, IgE, IgD).  • Cell-mediated: T cells — CD4⁺ helper (coordinate response), CD8⁺ cytotoxic (kill infected cells).  • Memory cells: Long-lived B and T cells for rapid secondary response.

--- Lesson 10.1: Immune System Overview (Quiz) ---

  Q1. The first line of defense includes:
    [ ] Antibodies
    [*] Skin, mucous membranes, and secretions (lysozyme, acid, sebum)
    [ ] T cells
    [ ] B cells
    Explanation: Physical and chemical barriers.

  Q2. Neutrophils are:
    [ ] Adaptive immune cells
    [*] The most abundant WBCs; first responders to infection (phagocytes)
    [ ] Red blood cells
    [ ] Platelets
    Explanation: Arrive first at infection site.

  Q3. Macrophages function as:
    [ ] Only antigen presenters
    [*] Phagocytes AND antigen-presenting cells (bridge innate and adaptive immunity)
    [ ] Only in the spleen
    [ ] Only in the liver
    Explanation: Versatile immune cells.

  Q4. Inflammation is characterized by:
    [ ] Only pain
    [*] Redness, heat, swelling, and pain (rubor, calor, tumor, dolor)
    [ ] Only fever
    [ ] Only swelling
    Explanation: Cardinal signs of inflammation.

  Q5. The complement system:
    [ ] Only fights parasites
    [*] Is a cascade of ~30 proteins that enhances phagocytosis, causes lysis, and promotes inflammation
    [ ] Is part of adaptive immunity
    [ ] Only makes antibodies
    Explanation: Innate defense amplifier.

  Q6. B cells mature in the:
    [ ] Thymus
    [*] Bone marrow
    [ ] Lymph nodes
    [ ] Spleen
    Explanation: B = Bone marrow.

  Q7. T cells mature in the:
    [ ] Bone marrow
    [*] Thymus
    [ ] Lymph nodes
    [ ] Liver
    Explanation: T = Thymus.

  Q8. Plasma cells produce:
    [ ] T cell receptors
    [*] Antibodies (immunoglobulins)
    [ ] Complement
    [ ] Interferons
    Explanation: Differentiated B cells.

  Q9. IgG is:
    [ ] Only in secretions
    [*] The most abundant antibody in blood; crosses the placenta; provides passive immunity to the fetus
    [ ] Only on mast cells
    [ ] Only in mucous membranes
    Explanation: Primary antibody of adaptive immunity.

  Q10. IgE is associated with:
    [ ] Normal infection defense
    [*] Allergic reactions and parasitic infections (binds mast cells → histamine release)
    [ ] Blood type
    [ ] Complement activation
    Explanation: Mediator of type I hypersensitivity.

  Q11. CD4⁺ T helper cells:
    [ ] Kill infected cells
    [*] Coordinate the immune response by activating B cells, CD8⁺ T cells, and macrophages
    [ ] Produce antibodies
    [ ] Are phagocytes
    Explanation: Orchestrators of adaptive immunity.

  Q12. CD8⁺ cytotoxic T cells:
    [ ] Coordinate responses
    [*] Directly kill virus-infected cells and cancer cells (via perforin/granzymes)
    [ ] Produce antibodies
    [ ] Are macrophages
    Explanation: Cytotoxic = cell-killing.

  Q13. Vaccines work by:
    [ ] Causing disease
    [*] Exposing the immune system to antigens (weakened/inactivated pathogen or antigen) → memory cells formed
    [ ] Producing antibiotics
    [ ] Enhancing innate immunity only
    Explanation: Immunological memory without disease.

  Q14. Autoimmune diseases occur when:
    [ ] The immune system is weak
    [*] The immune system attacks the body's own tissues (loss of self-tolerance)
    [ ] Only infections cause them
    [ ] Vaccines cause them
    Explanation: Examples: lupus, RA, MS, type 1 diabetes.

  Q15. HIV attacks _____ cells.
    [ ] CD8⁺ T
    [*] CD4⁺ T helper
    [ ] B
    [ ] NK
    Explanation: Immune system collapse → AIDS.

  Q16. Anaphylaxis is a:
    [ ] Mild reaction
    [*] Severe, life-threatening systemic allergic reaction (IgE-mediated)
    [ ] Type of infection
    [ ] Autoimmune disease
    Explanation: Emergency: epinephrine is first-line treatment.

  Q17. Immunodeficiency disorders can be:
    [ ] Only acquired
    [*] Primary (genetic, e.g., SCID) or secondary (acquired, e.g., HIV/AIDS)
    [ ] Only genetic
    [ ] Caused by diet alone
    Explanation: Both categories exist.

  Q18. The lymphatic system's role in immunity includes:
    [ ] Only fluid drainage
    [*] Filtering lymph (lymph nodes), housing immune cells, and transporting antigens to lymphoid organs
    [ ] Only fighting cancer
    [ ] No role
    Explanation: Lymph nodes are immune surveillance hubs.

  Q19. Interferons:
    [ ] Kill bacteria
    [*] Are proteins released by virus-infected cells that signal neighboring cells to increase antiviral defenses
    [ ] Are antibodies
    [ ] Are only produced by T cells
    Explanation: Innate antiviral defense.

  Q20. Understanding the immune system is essential for:
    [ ] Nothing
    [*] Managing infections, autoimmune diseases, allergies, transplant rejection, cancer immunotherapy, and vaccination
    [ ] Only immunologists
    [ ] Only researchers
    Explanation: One of the most important body systems.

--- Lesson 10.2: Interactions Between Organ Systems (Summary) ---

  Key Concepts: Interactions Between Organ Systems
Interactions Between Organ SystemsHomeostasis Through Integration
No organ system works in isolation. Maintaining homeostasis requires coordinated communication between all 11 organ systems.
Key Interactions  • Exercise response: Muscular (contraction) + skeletal (support) + nervous (motor commands) + cardiovascular (↑ CO) + respiratory (↑ ventilation) + endocrine (epinephrine, cortisol).  • Eating a meal: Digestive (digestion/absorption) + nervous (vagal stimulation) + endocrine (insulin, GI hormones) + cardiovascular (nutrient transport) + urinary (waste excretion).  • Hemorrhage response: Cardiovascular (↑ HR, vasoconstriction) + nervous (baroreceptor reflex) + endocrine (RAAS, ADH) + urinary (↓ urine output) + immune (clotting).

--- Lesson 10.2: Interactions Between Organ Systems (Quiz) ---

  Q1. During exercise, cardiac output increases because:
    [ ] Only HR increases
    [*] Both heart rate AND stroke volume increase to meet muscle oxygen demand
    [ ] Only stroke volume increases
    [ ] Blood pressure drops
    Explanation: CO = HR × SV; both rise.

  Q2. The respiratory system responds to exercise by:
    [ ] Decreasing ventilation
    [*] Increasing ventilation rate and depth (to supply more O₂ and remove CO₂)
    [ ] Not changing
    [ ] Only increasing depth
    Explanation: Proportional to metabolic demand.

  Q3. Epinephrine during exercise:
    [ ] Slows the heart
    [*] Increases HR, BP, bronchodilation, and mobilizes glucose from glycogen
    [ ] Only relaxes muscles
    [ ] Has no effect
    Explanation: Sympathetic activation.

  Q4. After eating, the parasympathetic nervous system:
    [ ] Inhibits digestion
    [*] Stimulates digestion ('rest and digest' — increases GI secretion and motility)
    [ ] Increases heart rate
    [ ] Causes sweating
    Explanation: Vagal stimulation of GI tract.

  Q5. Insulin released after a meal:
    [ ] Raises blood glucose
    [*] Lowers blood glucose by promoting cellular uptake, glycogenesis, and lipogenesis
    [ ] Has no effect on glucose
    [ ] Only affects the liver
    Explanation: Key postprandial hormone.

  Q6. In hemorrhage, baroreceptors detect:
    [ ] High blood pressure
    [*] Decreased blood pressure → trigger sympathetic response (↑ HR, vasoconstriction)
    [ ] High CO₂
    [ ] Low O₂
    Explanation: Baroreceptor reflex compensates for blood loss.

  Q7. RAAS activation during hemorrhage leads to:
    [ ] Vasodilation
    [*] Na⁺ and water retention + vasoconstriction → restores blood volume and pressure
    [ ] Diuresis
    [ ] Decreased HR
    Explanation: Compensatory mechanism.

  Q8. The integumentary system (skin) interacts with other systems by:
    [ ] Having no interactions
    [*] Protecting all systems (barrier), synthesizing vitamin D (skeletal), thermoregulation (cardiovascular), and sensory input (nervous)
    [ ] Only protection
    [ ] Only sensation
    Explanation: Skin is the body's interface.

  Q9. The skeletal system supports other systems by:
    [ ] Only providing structure
    [*] Providing structure, protecting organs (brain, heart, lungs), producing blood cells (marrow), and storing minerals (Ca²⁺)
    [ ] Only storing calcium
    [ ] Only protecting the brain
    Explanation: Hematopoiesis + mineral storage.

  Q10. Fever is an example of:
    [ ] System failure
    [*] Coordinated response: immune (pyrogens) + nervous (hypothalamic thermostat) + cardiovascular (vasodilation) + muscular (shivering)
    [ ] Only immune system
    [ ] Only nervous system
    Explanation: Multi-system response to infection.

  Q11. Pregnancy involves coordination of:
    [ ] Only the reproductive system
    [*] Reproductive, endocrine, cardiovascular, respiratory, urinary, musculoskeletal, immune, and digestive systems
    [ ] Only two systems
    [ ] Only hormones
    Explanation: Nearly every system adapts.

  Q12. The nervous and endocrine systems work together by:
    [ ] Operating independently
    [*] The nervous system triggers rapid responses while the endocrine system sustains them (e.g., fight-or-flight)
    [ ] Only one functioning at a time
    [ ] Competing against each other
    Explanation: Neural-endocrine integration.

  Q13. Blood serves as a communication highway by:
    [ ] Only carrying oxygen
    [*] Transporting O₂, CO₂, nutrients, hormones, waste, immune cells, and heat
    [ ] Only carrying waste
    [ ] Only carrying hormones
    Explanation: Circulatory system connects all organs.

  Q14. Dehydration affects multiple systems:
    [ ] Only the urinary system
    [*] Cardiovascular (↓ blood volume), nervous (confusion), renal (↓ GFR), muscular (cramps), and thermoregulatory
    [ ] Only causing thirst
    [ ] Only the kidneys
    Explanation: Multi-system impact.

  Q15. Diabetes mellitus demonstrates system interactions:
    [ ] In the pancreas only
    [*] Endocrine (insulin deficit) → cardiovascular (atherosclerosis), nervous (neuropathy), renal (nephropathy), visual (retinopathy), immune (infections)
    [ ] Only blood sugar
    [ ] Only the kidneys
    Explanation: Systemic disease.

  Q16. Stress (chronic) affects:
    [ ] Nothing
    [*] Nervous (HPA axis), endocrine (cortisol), cardiovascular (hypertension), immune (suppression), digestive (ulcers), and mental health
    [ ] Only mood
    [ ] Only the heart
    Explanation: Chronic cortisol has widespread effects.

  Q17. Aging involves:
    [ ] Only one system declining
    [*] Gradual decline in all organ systems: decreased GFR, cardiac output, bone density, immune function, etc.
    [ ] Only joint pain
    [ ] Nothing significant
    Explanation: Multi-system decline.

  Q18. A systems approach to medicine means:
    [ ] Treating one organ at a time
    [*] Considering how all body systems interact when diagnosing and treating disease
    [ ] Ignoring other systems
    [ ] Only using surgery
    Explanation: Holistic understanding.

  Q19. Clinical integration is important because:
    [ ] Diseases affect only one system
    [*] Most diseases affect multiple organ systems, and treatment must consider the whole patient
    [ ] It's not important
    [ ] Only for research
    Explanation: Real patients have multi-system pathology.

  Q20. Studying organ system interactions prepares students for:
    [ ] Only anatomy exams
    [*] Understanding complex clinical scenarios, practicing systems-based medicine, and succeeding on AP exams
    [ ] Only memorizing facts
    [ ] Nothing
    Explanation: Foundation for clinical reasoning.

--- Lesson 10.3: Case Studies in Pathophysiology (Summary) ---

  Key Concepts: Case Studies in Pathophysiology
Case Studies in PathophysiologyCase 1: Congestive Heart Failure (CHF)
68 y/o with dyspnea, edema, elevated JVP. Chest X-ray: pulmonary congestion. Echo: reduced EF. CHF involves cardiovascular (pump failure), respiratory (pulmonary edema), renal (fluid retention), and endocrine (RAAS activation) systems.
Case 2: Type 1 Diabetes (DKA)
16 y/o with polyuria, polydipsia, weight loss, fruity breath. Labs: glucose 450, pH 7.1, ketones +++. Endocrine (no insulin), renal (glucosuria, osmotic diuresis), respiratory (Kussmaul breathing), and nervous system (altered mental status) involvement.
Case 3: Sepsis
45 y/o post-surgical with fever, tachycardia, hypotension, altered mental status. Blood cultures positive. Immune (overwhelming infection), cardiovascular (vasodilation, hypotension), renal (AKI), respiratory (ARDS), and coagulation (DIC) involved.

--- Lesson 10.3: Case Studies in Pathophysiology (Quiz) ---

  Q1. In CHF, the heart:
    [ ] Pumps too forcefully
    [*] Cannot pump effectively enough to meet the body's demands
    [ ] Only produces arrhythmias
    [ ] Is structurally normal
    Explanation: Pump failure.

  Q2. Pulmonary edema in left-sided heart failure occurs because:
    [ ] Blood drains from lungs
    [*] Blood backs up into pulmonary veins → fluid leaks into alveoli
    [ ] The lungs produce excess mucus
    [ ] Airway is obstructed
    Explanation: Left ventricle failure → pulmonary congestion.

  Q3. In CHF, the kidneys respond by:
    [ ] Excreting excess water
    [*] Retaining Na⁺ and water (RAAS activation) → worsening fluid overload
    [ ] Stopping function
    [ ] Only filtering waste
    Explanation: Compensatory mechanism that backfires.

  Q4. ACE inhibitors treat CHF by:
    [ ] Increasing fluid retention
    [*] Blocking angiotensin II → vasodilation + reduced preload/afterload + less aldosterone
    [ ] Increasing HR
    [ ] Increasing aldosterone
    Explanation: First-line CHF medications.

  Q5. In DKA, the body uses _____ for energy because insulin is absent.
    [ ] Glucose
    [*] Fat (fatty acid oxidation → ketone bodies → acidosis)
    [ ] Protein only
    [ ] Carbohydrates
    Explanation: Can't use glucose without insulin.

  Q6. Kussmaul breathing in DKA is:
    [ ] Shallow breathing
    [*] Deep, rapid breathing to blow off CO₂ (respiratory compensation for metabolic acidosis)
    [ ] Normal breathing
    [ ] Absent breathing
    Explanation: Compensatory hyperventilation.

  Q7. Treatment of DKA includes:
    [ ] Only oral glucose
    [*] IV insulin, IV fluids, electrolyte replacement (especially K⁺), and monitoring
    [ ] Only antacids
    [ ] Only bicarbonate
    Explanation: Must correct glucose, volume, and electrolytes.

  Q8. In sepsis, vasodilation causes:
    [ ] Hypertension
    [*] Hypotension (distributive shock) — despite high cardiac output initially
    [ ] Vasoconstriction
    [ ] Normal BP
    Explanation: Inflammatory mediators → vasodilation → shock.

  Q9. Sepsis can lead to multi-organ failure because:
    [ ] Only one organ is affected
    [*] Systemic inflammation damages multiple organs: kidneys (AKI), lungs (ARDS), liver, brain
    [ ] The infection is localized
    [ ] Antibiotics fail
    Explanation: Cascade of organ dysfunction.

  Q10. DIC (disseminated intravascular coagulation) in sepsis causes:
    [ ] Only clotting
    [*] Simultaneous excessive clotting AND bleeding (consumption of clotting factors)
    [ ] Only bleeding
    [ ] No hematologic problems
    Explanation: Paradox of DIC.

  Q11. The 'hour-1 bundle' for sepsis includes:
    [ ] Only antibiotics
    [*] IV fluids, blood cultures, antibiotics, and lactate measurement — all within 1 hour
    [ ] Only fluids
    [ ] Only monitoring
    Explanation: Early intervention saves lives.

  Q12. Myocardial infarction (MI) involves:
    [ ] Only the heart
    [*] Cardiovascular (coronary occlusion), nervous (pain), endocrine (stress hormones), and potentially renal and respiratory systems
    [ ] Only chest pain
    [ ] Only the immune system
    Explanation: Multi-system emergency.

  Q13. Stroke demonstrates:
    [ ] Only a brain problem
    [*] Nervous (focal deficits), cardiovascular (cause: embolism or hemorrhage), musculoskeletal (weakness), and respiratory (airway compromise) system involvement
    [ ] Only weakness
    [ ] Only speech problems
    Explanation: Multi-system implications.

  Q14. Chronic obstructive pulmonary disease (COPD) affects:
    [ ] Only the lungs
    [*] Respiratory (airflow obstruction), cardiovascular (cor pulmonale), musculoskeletal (cachexia), and mental health
    [ ] Only breathing
    [ ] Only smoking-related
    Explanation: Systemic disease.

  Q15. A patient with liver cirrhosis may present with:
    [ ] Only jaundice
    [*] Jaundice, ascites, coagulopathy, portal hypertension, encephalopathy, and renal failure (hepatorenal syndrome)
    [ ] Only abdominal pain
    [ ] No symptoms
    Explanation: Multi-system hepatic decompensation.

  Q16. Understanding pathophysiology requires:
    [ ] Memorizing diseases
    [*] Integrating knowledge of anatomy, physiology, and multiple organ system interactions
    [ ] Only reading textbooks
    [ ] Only clinical experience
    Explanation: Systems thinking.

  Q17. These case studies illustrate that:
    [ ] Diseases are simple
    [*] Most serious diseases involve multiple organ systems and require a systems-based approach
    [ ] Only one system is affected
    [ ] Treatment is straightforward
    Explanation: Clinical complexity.

  Q18. The best approach to these cases on an AP exam is:
    [ ] Guessing
    [*] Systematically identifying which organ systems are involved and explaining the pathophysiology of each
    [ ] Only naming the disease
    [ ] Only listing symptoms
    Explanation: Organized, systems-based analysis.

  Q19. Critical thinking in pathophysiology involves:
    [ ] Memorization only
    [*] Connecting cause → mechanism → effect across organ systems
    [ ] Only knowing anatomy
    [ ] Only knowing pharmacology
    Explanation: Reasoning through disease processes.

  Q20. These cases prepare students for:
    [ ] Nothing
    [*] Clinical reasoning, AP exam scenarios, and future healthcare careers
    [ ] Only exams
    [ ] Only lectures
    Explanation: Real-world application of anatomy/physiology.

--- Lesson 10.4: Review of Key Concepts (Summary) ---

  Key Concepts: Review of Key Concepts
Review of Key ConceptsBy Organ System  • Skeletal: Support, protection, movement, hematopoiesis, mineral storage.  • Muscular: Movement, posture, heat production. Sliding filament theory.  • Nervous: Rapid communication via action potentials. CNS + PNS.  • Endocrine: Hormonal regulation — feedback loops, HPG/HPA axes.  • Cardiovascular: Transport of blood, O₂, nutrients. Cardiac cycle.  • Respiratory: Gas exchange. Ventilation mechanics.  • Digestive: Mechanical/chemical digestion, absorption. Liver, pancreas.  • Urinary: Filtration, reabsorption, secretion. Fluid/electrolyte/acid-base balance.  • Reproductive: Gametogenesis, hormonal regulation (HPG axis), pregnancy.  • Immune: Innate + adaptive immunity. Memory.

--- Lesson 10.4: Review of Key Concepts (Quiz) ---

  Q1. The five functions of the skeletal system include:
    [ ] Only support
    [*] Support, protection, movement, hematopoiesis, and mineral storage
    [ ] Only movement
    [ ] Only bone production
    Explanation: Skeletal system is multifunctional.

  Q2. The sliding filament theory explains:
    [ ] Bone growth
    [*] How actin and myosin filaments interact to produce muscle contraction
    [ ] Nerve conduction
    [ ] Blood flow
    Explanation: Actin slides over myosin.

  Q3. An action potential involves:
    [ ] Only K⁺ movement
    [*] Depolarization (Na⁺ influx) followed by repolarization (K⁺ efflux)
    [ ] Only Ca²⁺ movement
    [ ] Only diffusion
    Explanation: Electrical signal propagation.

  Q4. Negative feedback in homeostasis means:
    [ ] The response amplifies the stimulus
    [*] The response opposes the stimulus to restore a set point
    [ ] There is no regulation
    [ ] The system fails
    Explanation: Most common regulatory mechanism.

  Q5. Positive feedback differs from negative in that:
    [ ] It maintains homeostasis
    [*] The response amplifies the stimulus (e.g., oxytocin in labor, LH surge at ovulation)
    [ ] It's more common
    [ ] It stops processes
    Explanation: Drives processes to completion.

  Q6. The cardiac cycle consists of:
    [ ] Only systole
    [*] Atrial systole → ventricular systole → diastole (one complete heartbeat)
    [ ] Only diastole
    [ ] Random contractions
    Explanation: Coordinated pumping sequence.

  Q7. Gas exchange occurs at the:
    [ ] Bronchi
    [ ] Trachea
    [*] Alveoli (external respiration) and tissues (internal respiration)
    [ ] Nasal cavity
    Explanation: O₂ and CO₂ exchange.

  Q8. The three processes of urine formation are:
    [ ] Only filtration
    [*] Filtration, reabsorption, and secretion
    [ ] Only reabsorption
    [ ] Only excretion
    Explanation: Complete nephron function.

  Q9. The menstrual cycle involves which key hormones?
    [ ] Only estrogen
    [*] FSH, LH, estrogen, and progesterone
    [ ] Only progesterone
    [ ] Only testosterone
    Explanation: Complete hormonal regulation of the cycle.

  Q10. Antibodies are produced by:
    [ ] T cells
    [*] Plasma cells (differentiated B cells)
    [ ] Macrophages
    [ ] NK cells
    Explanation: Humoral immunity.

  Q11. Homeostasis is maintained by all organ systems working:
    [ ] Independently
    [*] Together in a coordinated manner
    [ ] In isolation
    [ ] Only when stressed
    Explanation: Integration is key.

  Q12. The SA node is the heart's natural pacemaker located in the:
    [ ] Left ventricle
    [ ] Left atrium
    [*] Right atrium
    [ ] Right ventricle
    Explanation: Initiates each heartbeat.

  Q13. The diaphragm is essential for:
    [ ] Digestion
    [*] Breathing (contraction creates negative intrathoracic pressure → air inhalation)
    [ ] Circulation
    [ ] Urination
    Explanation: Primary muscle of respiration.

  Q14. The hypothalamus regulates:
    [ ] Only temperature
    [*] Temperature, thirst, hunger, circadian rhythms, and endocrine function via the pituitary
    [ ] Only hormones
    [ ] Only sleep
    Explanation: Master integrator.

  Q15. Hemoglobin's primary function is:
    [ ] Clotting
    [*] Oxygen transport (each molecule carries up to 4 O₂)
    [ ] Immune defense
    [ ] Nutrient transport
    Explanation: Found in RBCs.

  Q16. Insulin and glucagon have _____ effects on blood glucose.
    [ ] Similar
    [*] Opposing (insulin lowers, glucagon raises)
    [ ] No
    [ ] Identical
    Explanation: Antagonistic hormones.

  Q17. The liver's role in metabolism includes:
    [ ] Only bile production
    [*] Detoxification, bile production, protein synthesis, glycogen storage/release, and bilirubin processing
    [ ] Only storage
    [ ] Only filtration
    Explanation: Metabolic powerhouse.

  Q18. The lymphatic system's dual role is:
    [ ] Only drainage
    [*] Fluid return to blood AND immune defense (lymph nodes filter pathogens)
    [ ] Only immunity
    [ ] Only fat absorption
    Explanation: Fluid balance + immunity.

  Q19. Understanding all systems together is important because:
    [ ] They don't interact
    [*] Clinical disease almost always involves multiple systems, and effective treatment requires whole-body understanding
    [ ] Only for exams
    [ ] Only for specialists
    Explanation: Systems medicine.

  Q20. To prepare for the AP exam, students should:
    [ ] Only memorize terms
    [*] Understand mechanisms, integration, and be able to apply to clinical scenarios
    [ ] Ignore pathophysiology
    [ ] Focus on only one system
    Explanation: Application and integration are key.

--- Lesson 10.5: AP Practice: Multi-System Questions (Summary) ---

  Key Concepts: AP Practice: Multi-System Questions
AP Practice: Multi-System Questions
This lesson focuses on higher-order AP-style questions that integrate knowledge from multiple organ systems. Practice identifying connections across anatomy and physiology to strengthen exam readiness.
Strategy  • Read the clinical scenario carefully.  • Identify ALL organ systems involved.  • Think about cause → mechanism → effect.  • Eliminate obviously wrong answers first.

--- Lesson 10.5: AP Practice: Multi-System Questions (Quiz) ---

  Q1. A marathon runner collapses. Which systems are involved?
    [ ] Only muscular
    [ ] Only cardiovascular
    [*] Cardiovascular (dehydration ↓ CO), thermoregulatory (hyperthermia), renal (↓ GFR), musculoskeletal, nervous (confusion), and endocrine
    [ ] Only respiratory
    Explanation: Multi-system stress.

  Q2. In anaphylaxis, which systems are affected?
    [ ] Only skin
    [ ] Only respiratory
    [*] Immune (IgE/mast cells → histamine), cardiovascular (vasodilation → shock), respiratory (bronchospasm → airway obstruction), skin (urticaria)
    [ ] Only cardiovascular
    Explanation: Life-threatening multi-system emergency.

  Q3. A patient with chronic kidney disease develops anemia because:
    [ ] Kidneys don't filter
    [*] Kidneys produce less erythropoietin (EPO) → ↓ RBC production
    [ ] The liver fails
    [ ] Bone marrow is destroyed
    Explanation: Renal endocrine function.

  Q4. Hyperparathyroidism causes:
    [ ] Low calcium
    [*] Elevated blood calcium (increased bone resorption, renal Ca²⁺ reabsorption, and intestinal absorption via vitamin D)
    [ ] Low phosphate excretion
    [ ] No bone effects
    Explanation: PTH affects skeletal, renal, and GI systems.

  Q5. A patient with a pulmonary embolism (PE) develops:
    [ ] Only leg pain
    [*] Respiratory failure (V/Q mismatch), cardiovascular compromise (right heart strain), and potentially hemodynamic collapse
    [ ] Only cough
    [ ] Normal breathing
    Explanation: Multi-system emergency.

  Q6. Hypothyroidism affects:
    [ ] Only the thyroid
    [*] Metabolism (weight gain), cardiovascular (bradycardia), nervous (cognitive slowing), musculoskeletal (weakness), integumentary (cold intolerance, dry skin)
    [ ] Only weight
    [ ] Only mood
    Explanation: Thyroid hormones affect every organ system.

  Q7. A spinal cord injury at C3 can cause:
    [ ] Only paralysis
    [*] Loss of voluntary movement below injury, respiratory failure (phrenic nerve to diaphragm at C3-5), autonomic dysreflexia, neurogenic bladder
    [ ] Only numbness
    [ ] Only pain
    Explanation: High cervical injury = life-threatening.

  Q8. Sickle cell disease affects:
    [ ] Only blood
    [*] Cardiovascular (anemia), skeletal (bone crisis), splenic (autosplenectomy), renal, pulmonary, and neurological systems
    [ ] Only bones
    [ ] Only the spleen
    Explanation: Systemic disease from one genetic mutation.

  Q9. A patient taking corticosteroids chronically may develop:
    [ ] Only weight loss
    [*] Cushing-like features: immunosuppression, osteoporosis, hyperglycemia, muscle wasting, poor wound healing, adrenal suppression
    [ ] Only skin changes
    [ ] Only mood changes
    Explanation: Wide-ranging steroid effects.

  Q10. Pregnancy affects the cardiovascular system by:
    [ ] Decreasing blood volume
    [*] Increasing blood volume (~50%), cardiac output (~40%), and heart rate
    [ ] No changes
    [ ] Decreasing HR
    Explanation: Major cardiovascular adaptations.

  Q11. In diabetic nephropathy:
    [ ] Kidneys are normal
    [*] Chronic hyperglycemia damages glomerular capillaries → proteinuria → progressive renal failure
    [ ] Only blood sugar rises
    [ ] Only the pancreas is affected
    Explanation: Endocrine → renal system connection.

  Q12. Liver failure can cause cerebral edema because:
    [ ] The brain shrinks
    [*] Ammonia accumulates (liver can't convert to urea) → crosses BBB → hepatic encephalopathy
    [ ] The heart fails
    [ ] Kidneys stop working
    Explanation: Hepatic-neurological connection.

  Q13. A patient with COPD develops right heart failure (cor pulmonale) because:
    [ ] The left heart fails first
    [*] Chronic hypoxic pulmonary vasoconstriction → pulmonary hypertension → right ventricle overload
    [ ] The lungs collapse
    [ ] The kidneys fail

--- Lesson 10.6: Capstone Project: Human Body Systems (Summary) ---

  Key Concepts: Capstone Project: Human Body Systems
Capstone Project: Human Body SystemsProject Overview
Students select a clinical condition (e.g., heart attack, diabetes, sepsis) and analyze it from a multi-system perspective, documenting the pathophysiology across all affected organ systems.
Steps  • 1. Choose a disease or case.  • 2. Identify all organ systems affected.  • 3. Explain the pathophysiology of each system's involvement.  • 4. Describe how systems compensate or fail.  • 5. Discuss treatment strategies that target each system.  • 6. Present findings in a structured format (paper, poster, or presentation).
