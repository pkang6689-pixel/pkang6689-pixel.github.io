--- Lesson 8.1: Dot Product & Orthogonality (Summary) ---

  Key Concepts: Dot Product & Orthogonality
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the length (norm) of the vector v = (3, 4)?
||v|| = sqrt(3^2 + 4^2) = sqrt(9 + 16) = sqrt(25) = 5.
What is the unit vector in the direction of v = (6, 8)?
||v|| = sqrt(36 + 64) = 10. The unit vector is v/||v|| = (6/10, 8/10) = (3/5, 4/5). Note that (0.6, 0.8) is the same as (3/5, 4/5), but the cleanest form is (3/5, 4/5).
What is the dot product (inner product) of two vectors in R^n?
The dot product (or standard inner product) is the scalar u . v = sum of u_i * v_i. It is a scalar, not a vector or matrix.
What is an 'orthogonal set' of vectors?
An orthogonal set is a collection of vectors where every pair of distinct vectors has dot product zero. An orthonormal set additionally requires each vector to have unit length.
What is an 'orthonormal' set of vectors?
An orthonormal set consists of mutually orthogonal vectors that are each normalized to have length 1. The standard basis {e1, e2, ..., en} is the simplest example.
What is the 'orthogonal complement' of a subspace W?
The orthogonal complement W^perp consists of all vectors v such that v . w = 0 for every w in W. Together, W and W^perp span the entire space.
What is the 'Pythagorean theorem' for orthogonal vectors?
When u and v are orthogonal, ||u+v||^2 = (u+v).(u+v) = ||u||^2 + 2(u.v) + ||v||^2 = ||u||^2 + ||v||^2 since the cross term vanishes.
What is an 'orthogonal basis' for a subspace?
An orthogonal basis is a basis where all vectors are mutually orthogonal. It simplifies computations: coordinates are found by simple dot products rather than solving systems.
A data scientist normalizes feature vectors before computing cosine similarity. If x = (3, 4) and y = (5, 12), what is the cosine similarity?
cos(theta) = x.y/(||x|| ||y||) = (15 + 48)/(5 * 13) = 63/65. This high cosine similarity (approximately 0.969) indicates the feature vectors are nearly aligned.
A robotics engineer decomposes a force F = (5, 3) into components along d = (1, 0) and d_perp = (0, 1). What is the component of F along d?
The component of F along d is (F . d)/(||d||) = 5/1 = 5. Since d is already a unit vector, the scalar projection is simply the dot product F . d = 5.
An audio engineer checks if two sound signals s1 = (1, 2, -1, 0) and s2 = (2, -1, 0, 5) are orthogonal before mixing. What is s1 . s2?
s1 . s2 = 1(2) + 2(-1) + (-1)(0) + 0(5) = 2 - 2 + 0 + 0 = 0. The signals are orthogonal, so they can be separated perfectly after mixing.
A computer graphics programmer needs to find the angle of incidence of a light ray r = (1, -1) hitting a surface with normal n = (0, 1). What is the angle?
cos(theta) = |r . n|/(||r|| ||n||) = |-1|/(sqrt(2) * 1) = 1/sqrt(2). So theta = 45 degrees. The absolute value is used since we want the acute angle.
A climatologist models temperature variation using two orthogonal basis functions f1 and f2. If the temperature vector is T = 3f1 + 4f2 and {f1, f2} is orthonormal, what is the total variance ||T||^2?
By the Pythagorean theorem for orthogonal components: ||T||^2 = 3^2 + 4^2 = 9 + 16 = 25. Each coefficient squared gives the variance along that basis direction.

--- Lesson 8.2: Orthogonal Projections (Summary) ---

  Key Concepts: Orthogonal Projections
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
Project b = (3, 4) onto a = (1, 0). What is the projection?
proj_a(b) = (a.b / a.a) * a = (3/1)(1, 0) = (3, 0). Projecting onto the x-axis keeps only the x-component.
Project b = (2, 6) onto a = (1, 1). What is proj_a(b)?
a.b = 2+6=8, a.a = 1+1=2. proj = (8/2)(1,1) = 4(1,1) = (4,4).
If P is the projection matrix onto W, what is the projection onto W^perp?
The projection onto the orthogonal complement is I - P. For any vector b: b = Pb + (I-P)b, decomposing into its component in W and its component in W^perp.
Project b = (1, 2, 3) onto the xy-plane (z = 0). What is the projection?
Projecting onto the xy-plane drops the z-component: (1, 2, 3) maps to (1, 2, 0). The projection matrix is P = diag(1, 1, 0).
What is an 'orthogonal projection'?
An orthogonal projection maps a vector to the nearest point in the subspace. The key property is that the error (residual) is orthogonal to the subspace, minimizing distance.
What is the 'best approximation theorem' in the context of projections?
The best approximation theorem states that proj_W(b) is the unique vector in W that minimizes ||b - w|| over all w in W. The minimum distance is ||b - proj_W(b)||.
What is a 'least squares regression line'?
The least squares regression line minimizes the sum of squared residuals (vertical distances). It is found using the normal equation applied to the system with columns [1, x_i].
What is the 'residual' in a least squares problem?
The residual r = b - Ax-hat measures how far the prediction Ax-hat is from the actual b. It is orthogonal to the column space of A.
What is the 'orthogonal decomposition theorem'?
For any subspace W of R^n and any vector b, b = proj_W(b) + proj_{W^perp}(b) uniquely. This decomposes b into its W-component and its perpendicular component.
What is meant by 'Fourier coefficients' in the context of orthogonal projection?
When projecting v onto an orthogonal basis {u1,...,uk}, the Fourier coefficient c_i = (v.u_i)/(u_i.u_i) gives the weight of each basis vector. If the basis is orthonormal, c_i = v.u_i.
A signal processing engineer decomposes a signal s = (5, 3, 1) into components along two orthogonal directions u1 = (1, 0, 0) and u2 = (0, 1, 0). What is the projection of s onto span{u1, u2}?
proj = (s.u1)u1 + (s.u2)u2 = 5(1,0,0) + 3(0,1,0) = (5,3,0). The z-component (the part in the orthogonal complement) is discarded.
An economist wants to remove the trend from economic data by projecting onto the orthogonal complement of the trend vector t = (1, 1, 1, 1). If the data is y = (10, 12, 15, 11), what is the detrended data (I - P)y?
Mean = (10+12+15+11)/4 = 12. Projection onto t: (y.t/t.t)t = (48/4)(1,1,1,1) = (12,12,12,12). Detrended: y - proj = (10-12, 12-12, 15-12, 11-12) = (-2, 0, 3, -1).
A telecommunications engineer uses a matched filter to detect a known signal s in noisy data y. The matched filter output is the projection of y onto s. If s = (1, 1, -1, -1) and y = (2, 3, -1, 0), what is the projection scalar?
The projection scalar is s.y/s.s = (2+3+1+0)/4 = 6/4 = 1.5. This indicates the signal is present with amplitude 1.5 times the template.
A mechanical engineer models vibration data as a combination of two normal mode shapes. She projects the measured vibration onto each mode shape using orthogonal projection. If modes are orthogonal, what is the advantage?
Orthogonality allows independent computation of each mode's coefficient: c_i = (data . mode_i)/(mode_i . mode_i). No system of equations needs to be solved.
Real-World Applications
Example: A marketing analyst fits a line y = a + bx to sales data with 4 points: (1,2), (2,3), (3,5), (4,4). The system is Ax = b with A = [[1,1],[1,2],[1,3],[1,4]] and b = (2,3,5,4). What does the projection Ax-hat represent?
Ax-hat gives the predicted values (y-hat) lying on the regression line. The residual b - Ax-hat gives the vertical distances from data points to the line.
Example: A mechanical engineer models vibration data as a combination of two normal mode shapes. She projects the measured vibration onto each mode shape using orthogonal projection. If modes are orthogonal, what is the advantage?
Orthogonality allows independent computation of each mode's coefficient: c_i = (data . mode_i)/(mode_i . mode_i). No system of equations needs to be solved.
Example: An image processing engineer approximates a 1000-pixel image by projecting it onto the first 50 principal components. What percentage of the original data is in the residual if those 50 components capture 98% of the variance?
If the projection captures 98% of the variance, the residual contains the remaining 2%. This means 50 components provide a highly accurate approximation, discarding only 2% of the information.

    Explanation: The residual e = b - proj_a(b) is always orthogonal to a. This is the fundamental property of orthogonal projection: it minimizes the distance from b to the line spanned by a.

  Q7. For orthogonal projection onto a subspace W with orthonormal basis {u1, u2}, the formula is:

    Explanation: With an orthonormal basis, the projection is the sum of projections onto each basis vector: proj_W(b) = (u1.b)u1 + (u2.b)u2. Each term is a simple dot product.

    [ ] ||A||^2
    [*] ||Ax - b||^2
    [ ] det(A)
    [ ] ||x||^2
    Explanation: The least squares solution minimizes the squared error ||Ax - b||^2. It is found by solving the normal equations A^T Ax = A^T b, which projects b onto the column space of A.

    [ ] P^(-1)
    [ ] -P
    [ ] P^T
    Explanation: The projection onto the orthogonal complement is I - P. For any vector b: b = Pb + (I-P)b, decomposing into its component in W and its component in W^perp.

  Q13. What does 'idempotent' mean for a projection matrix?
    [*] P^2 = P (applying the projection twice gives the same result as once)

--- Lesson 8.3: Gram-Schmidt Process (Summary) ---

  Key Concepts: Gram-Schmidt Process
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the primary purpose of the Gram-Schmidt process?
The Gram-Schmidt process takes a set of linearly independent vectors and produces an orthonormal set that spans the same subspace.
Given v1 = (1, 0) and v2 = (1, 1), what is the first orthonormal vector e1 produced by Gram-Schmidt?
The first step of Gram-Schmidt normalizes v1. Since v1 = (1, 0) already has unit length, e1 = (1, 0).
In the Gram-Schmidt process, after obtaining the orthogonal vector u2 = v2 - proj_{u1}(v2), what is the next step?
After subtracting the projection to make u2 orthogonal to u1, we normalize u2 by dividing by its magnitude to produce the orthonormal vector e2.
Given v1 = (3, 4), what is e1 = v1 / ||v1||?
||v1|| = sqrt(9 + 16) = sqrt(25) = 5, so e1 = (3/5, 4/5).
Apply Gram-Schmidt to v1 = (1, 1, 0) and v2 = (1, 0, 1). What is the orthogonal component u2 = v2 - proj_{u1}(v2)?
proj_{u1}(v2) = (v2 . u1)/(u1 . u1) * u1 = (1)/(2) * (1,1,0) = (1/2, 1/2, 0). So u2 = (1,0,1) - (1/2,1/2,0) = (1/2, -1/2, 1).
What is the 'QR factorization' that results from the Gram-Schmidt process?
QR factorization decomposes a matrix A into Q (with orthonormal columns from Gram-Schmidt) and R (upper triangular with the projection coefficients).
What is meant by 'numerical instability' in the classical Gram-Schmidt process?
In floating-point arithmetic, the classical Gram-Schmidt process accumulates rounding errors that can cause the resulting vectors to deviate significantly from true orthogonality.
What is the 'modified Gram-Schmidt' process?
Modified Gram-Schmidt improves numerical stability by projecting each vector against previously computed orthogonal vectors sequentially, reducing accumulated rounding errors.
What is meant by 'Householder reflections' as an alternative to Gram-Schmidt?
Householder reflections are orthogonal transformations that zero out entries below the diagonal, providing a numerically superior alternative to Gram-Schmidt for QR factorization.
What is 'reorthogonalization' in the Gram-Schmidt context?
Reorthogonalization applies an additional round of orthogonal projection subtraction to correct for numerical drift, restoring near-perfect orthogonality in the computed vectors.
A structural engineer has three sensor measurements represented as vectors v1 = (1, 0, 0), v2 = (1, 1, 0), and v3 = (1, 1, 1) in a 3D stress space. She applies Gram-Schmidt to create an orthonormal basis for analysis. What is the second orthonormal vector e2?
u1 = (1,0,0), so e1 = (1,0,0). Then u2 = v2 - (v2.u1/u1.u1)u1 = (1,1,0) - 1*(1,0,0) = (0,1,0). Since ||u2|| = 1, e2 = (0,1,0).
A data scientist has feature vectors v1 = (2, 0) and v2 = (3, 1) and wants to orthonormalize them for a regression model. After applying Gram-Schmidt, what is e2?
e1 = v1/||v1|| = (1,0). proj_{e1}(v2) = (v2.e1)e1 = 3*(1,0) = (3,0). u2 = v2 - (3,0) = (0,1). Since ||u2|| = 1, e2 = (0,1).
A physicist uses Gram-Schmidt on vectors v1 = (1, 1) and v2 = (0, 1) to build an orthonormal frame for a 2D experiment. What is the R matrix in the QR factorization A = QR, where A = [v1 | v2]?
e1 = (1,1)/sqrt(2). R11 = ||v1|| = sqrt(2). R12 = v2.e1 = 1/sqrt(2). u2 = v2 - (1/sqrt(2))e1 = (0,1) - (1/2,1/2) = (-1/2,1/2), ||u2|| = 1/sqrt(2) = R22. So R = [[sqrt(2), 1/sqrt(2)], [0, 1/sqrt(2)]].
A surveyor collects three ground measurements as vectors v1 = (1, 2, 2), v2 = (0, 1, 0), v3 = (0, 0, 1). She applies Gram-Schmidt and finds ||v1|| = 3. What is the first column of Q in the QR factorization?
The first column of Q is e1 = v1/||v1|| = (1,2,2)/3 = (1/3, 2/3, 2/3). This is simply the normalization of the first vector.
A computer graphics team applies Gram-Schmidt to create an orthonormal coordinate frame from v1 = (1, 0, 1) and v2 = (0, 1, 0). What is u2 after the orthogonalization step?
proj_{u1}(v2) = (v2.u1)/(u1.u1) * u1 = (0)/(2) * (1,0,1) = (0,0,0). Since v2 is already orthogonal to u1, u2 = v2 - (0,0,0) = (0,1,0).
A quantum computing researcher applies Gram-Schmidt to states |a> = (1, 1) and |b> = (1, -1) in C^2 (using complex inner product). What is , the inner product?
= conj(1)*1 + conj(1)*(-1) = 1 - 1 = 0. The vectors are already orthogonal, so Gram-Schmidt would leave them unchanged (after normalization).
A robotics engineer has joint-angle vectors v1 = (1, 1, 0, 0) and v2 = (1, 0, 1, 0) in R^4. After Gram-Schmidt orthogonalization, what is u2?
proj_{u1}(v2) = (v2.u1)/(u1.u1) * u1 = 1/2 * (1,1,0,0) = (1/2,1/2,0,0). So u2 = (1,0,1,0) - (1/2,1/2,0,0) = (1/2, -1/2, 1, 0).
Important Concepts
An audio engineer has three frequency-domain signals represented as vectors in R^3. After applying Gram-Schmidt, she obtains orthonormal vectors {e1, e2, e3}. She wants to express a new signal s = (5, 3, 2) in this basis. If e1 = (1,0,0), e2 = (0,1,0), e3 = (0,0,1), what are the coordinates of s in the orthonormal basis
In an orthonormal basis, the coordinates are simply the dot products s.e1 = 5, s.e2 = 3, s.e3 = 2. Since the basis here is the standard basis, the coordinates are (5, 3, 2).

--- Lesson 8.4: Orthogonal Matrices (Summary) ---

  Key Concepts: Orthogonal Matrices
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the defining property of an orthogonal matrix Q?
An orthogonal matrix satisfies Q^T Q = I, meaning its transpose equals its inverse. This is equivalent to saying its columns form an orthonormal set.
What is the determinant of any orthogonal matrix?
Since Q^T Q = I, det(Q^T)det(Q) = 1, so (det Q)^2 = 1, giving det(Q) = +1 or -1.
If Q is an orthogonal matrix and x is any vector, what is ||Qx|| compared to ||x||?
Orthogonal matrices preserve vector norms: ||Qx||^2 = (Qx)^T(Qx) = x^T Q^T Q x = x^T I x = ||x||^2. This means orthogonal transformations are length-preserving (isometries).
The identity matrix I is an orthogonal matrix. What is I^T?
The identity matrix is symmetric, so I^T = I. It trivially satisfies I^T I = I*I = I, confirming it is orthogonal.
If Q is orthogonal, what is Q^T Q + Q Q^T for a square orthogonal matrix?
For a square orthogonal matrix, both Q^T Q = I and Q Q^T = I (since Q^(-1) = Q^T works on both sides). Therefore Q^T Q + Q Q^T = I + I = 2I.
What is a 'rotation matrix'?
A rotation matrix is a special orthogonal matrix (det = +1) that rotates all vectors by a fixed angle while preserving lengths and orientation.
What is a 'reflection matrix'?
A reflection matrix is an orthogonal matrix with determinant -1. It reflects vectors across a hyperplane (line in 2D, plane in 3D), reversing orientation while preserving distances.
What is the 'special orthogonal group' SO(n)?
SO(n) is the special orthogonal group consisting of all n x n orthogonal matrices with determinant +1. These represent pure rotations in n-dimensional space and form a group under matrix multiplication.
What is the 'Givens rotation'?
A Givens rotation is an orthogonal matrix that performs rotation in a single coordinate plane (i,j), modifying only two rows/columns. It is used in QR algorithms to zero out individual matrix entries.
What is a 'Householder matrix'?
A Householder matrix H = I - 2vv^T (v unit) is an orthogonal, symmetric matrix that reflects vectors across the hyperplane perpendicular to v. It has det = -1 and is its own inverse.
A computer graphics engine uses the rotation matrix Q = [[cos(30), -sin(30)], [sin(30), cos(30)]] to rotate a sprite. If the sprite's position vector is v = (4, 0), what is the y-component of the rotated position? (Use cos(30) = sqrt(3)/2, sin(30) = 1/2.)
Qv = [[sqrt(3)/2, -1/2],[1/2, sqrt(3)/2]] * (4,0) = (4*sqrt(3)/2, 4*1/2) = (2sqrt(3), 2). The y-component is 2.
A robotics engineer models a robotic arm joint using a 3x3 rotation matrix. She composes two rotations R1 and R2. If R1 rotates by 45 degrees and R2 rotates by 30 degrees (both about the same axis), what is the total rotation angle of R2*R1?
Rotations about the same axis compose by adding angles. R2*R1 rotates by 45 + 30 = 75 degrees. The product of two rotation matrices is also a rotation matrix.
A structural engineer uses the Householder reflection H = I - 2vv^T with unit vector v = (1/sqrt(2), 1/sqrt(2)) to transform a force vector. What is H * (1, 0)?
vv^T = [[1/2, 1/2],[1/2, 1/2]]. H = I - 2vv^T = [[0, -1],[-1, 0]]. H*(1,0) = (0, -1). This reflects across the line perpendicular to v, which is the line y = -x.
A machine learning engineer uses an orthogonal matrix to whiten (decorrelate) features. If the covariance matrix of the original data is C, what is the covariance after applying orthogonal transformation Q?
When data X is transformed to QX, the new covariance is Q^T C Q. If Q is chosen as the eigenvector matrix of C, this diagonalizes C, decorrelating the features.
A quantum computing researcher needs a unitary gate that maps |0> = (1,0) to (1/sqrt(2), 1/sqrt(2)) and |1> = (0,1) to (1/sqrt(2), -1/sqrt(2)). What is this gate called?
The Hadamard gate H = (1/sqrt(2))[[1,1],[1,-1]] is a fundamental quantum gate. It is both unitary and Hermitian (H = H*), and creates superposition states from basis states.

What is an 'inner product space'?
An inner product space is a vector space V together with an inner product <.,.>: V x V -> F that satisfies linearity, symmetry (or conjugate symmetry for complex), and positive definiteness.
What is a 'Hilbert space'?
A Hilbert space is a complete inner product space, meaning every Cauchy sequence of vectors converges to a vector in the space. R^n and L^2 function spaces are examples of Hilbert spaces.
What is a 'weighted inner product'?
A weighted inner product on R^n has the form  = sum of w_i * u_i * v_i where all weights w_i > 0. The positive weights allow emphasizing certain components while still satisfying all inner product axioms.

The orthogonal complement W^perp of a subspace W consists of all vectors v such that  = 0 for every w in W. The space decomposes as V = W + W^perp (direct sum).
What is a 'pre-Hilbert space'?
A pre-Hilbert space is an inner product space that is not necessarily complete. Every Hilbert space is a pre-Hilbert space, but not every pre-Hilbert space is a Hilbert space (it may have 'gaps' where Cauchy sequences don't converge).

    [*] Multiplicativity: <u, v> = <u, w><w, v>
    [ ] Symmetry: <u, v> = <v, u>
    Explanation: An inner product must satisfy symmetry, linearity in one argument, and positive definiteness. There is no multiplicativity axiom for inner products.

  Q2. The standard inner product on R^n is the dot product <u, v> = u . v. What is <(1, 2, 3), (4, 5, 6)>?
    [ ] 15
    [ ] 21
    [*] 32
    [ ] 38
    Explanation: <(1,2,3),(4,5,6)> = 1*4 + 2*5 + 3*6 = 4 + 10 + 18 = 32.

  Q3. On the vector space of continuous functions on [0, 1], which of the following defines a valid inner product?
    [ ] <f, g> = max(f(x)) * max(g(x))
    [*] <f, g> = integral from 0 to 1 of f(x)g(x) dx
    [ ] <f, g> = f(1) + g(1)
    [ ] <f, g> = f(0) * g(0)
    Explanation: The integral inner product <f,g> = integral of f(x)g(x)dx on [0,1] satisfies all inner product axioms: symmetry, linearity, and positive definiteness (integral of f^2 >= 0, with equality only for f=0 in continuous functions).

  Q4. In an inner product space, the norm induced by the inner product is defined as ||v|| = ?
    [ ] <v, v>
    [ ] <v, v>^2
    [*] sqrt(<v, v>)
    [ ] |<v, v>|
    Explanation: The norm induced by an inner product is ||v|| = sqrt(<v, v>). This generalizes the familiar Euclidean norm ||v|| = sqrt(v . v) to abstract inner product spaces.

  Q5. The Cauchy-Schwarz inequality states that for any vectors u and v in an inner product space:
    [ ] <u, v> = ||u|| + ||v||
    [ ] |<u, v>| >= ||u|| * ||v||
    [ ] <u, v>^2 = ||u||^2 + ||v||^2
    [*] |<u, v>| <= ||u|| * ||v||
    Explanation: The Cauchy-Schwarz inequality |<u,v>| <= ||u|| ||v|| is one of the most fundamental results in inner product spaces. Equality holds if and only if u and v are linearly dependent.

  Q6. Consider the weighted inner product <u, v> = 2u1*v1 + 3u2*v2 on R^2. What is <(1, 1), (1, -1)>?
    [ ] 1
    [ ] 5
    [ ] 0
    [*] -1
    Explanation: <(1,1),(1,-1)> = 2(1)(1) + 3(1)(-1) = 2 - 3 = -1.

  Q7. In an inner product space, two vectors u and v are orthogonal if and only if:
    [ ] u + v = 0
    [ ] <u, v> = 1
    [*] <u, v> = 0
    [ ] ||u|| = ||v||
    Explanation: Orthogonality in an inner product space is defined as <u, v> = 0. This generalizes the familiar notion of perpendicularity from Euclidean space to abstract settings.

  Q8. Which of the following is the triangle inequality in an inner product space?
    [*] ||u + v|| <= ||u|| + ||v||
    [ ] ||u + v|| = ||u|| * ||v||
    [ ] ||u - v|| = ||u|| + ||v||
    [ ] ||u + v|| >= ||u|| + ||v||
    Explanation: The triangle inequality ||u + v|| <= ||u|| + ||v|| holds in any inner product space and can be derived from the Cauchy-Schwarz inequality.

  Q9. In a complex inner product space, the inner product satisfies <u, v> = conj(<v, u>) instead of symmetry. This property is called:
    [*] Conjugate symmetry (Hermitian symmetry)
    [ ] Bilinearity, particularly
    [ ] Skew-symmetry in most standard
    [ ] Antisymmetry based on establish
    Explanation: In complex inner product spaces, symmetry is replaced by conjugate symmetry: <u,v> = conj(<v,u>). This ensures that <v,v> is always real and non-negative.

  Q10. The parallelogram law states that ||u + v||^2 + ||u - v||^2 = ?
    [ ] 4||u|| ||v||
    [ ] ||u||^2 + ||v||^2
    [*] 2(||u||^2 + ||v||^2)
    [ ] (||u|| + ||v||)^2
    Explanation: The parallelogram law ||u+v||^2 + ||u-v||^2 = 2(||u||^2 + ||v||^2) holds in any inner product space. A normed space is an inner product space if and only if the parallelogram law holds.

  Q11. What is an 'inner product space'?
    [*] A vector space equipped with a function that assigns a scalar to each pair of vectors, satisfying linearity, symmetry, and positive definiteness
    [ ] Any subspace of R^n, particularly regarding positive when applying fundamental principles
    [ ] A vector space with a defined cross product, particularly regarding space under the given assumptions
    [ ] A vector space where all vectors have the same length under normal circumstances
    Explanation: An inner product space is a vector space V together with an inner product <.,.>: V x V -> F that satisfies linearity, symmetry (or conjugate symmetry for complex), and positive definiteness.

  Q12. What is a 'Hilbert space'?
    [ ] A vector space with exactly two inner products
    [ ] An inner product space with only real scalars
    [*] An inner product space that is complete (every Cauchy sequence converges)
    [ ] Any finite-dimensional vector space
    Explanation: A Hilbert space is a complete inner product space, meaning every Cauchy sequence of vectors converges to a vector in the space. R^n and L^2 function spaces are examples of Hilbert spaces.

  Q13. What does 'positive definiteness' mean for an inner product?
    [*] <v, v> >= 0 for all v, and <v, v> = 0 only when v is the zero vector
    [ ] The inner product is always an integer
    [ ] The inner product of any two vectors is positive
    [ ] All eigenvalues of the inner product matrix are positive
    Explanation: Positive definiteness means <v,v> >= 0 for all vectors v, with <v,v> = 0 if and only if v = 0. This ensures the induced norm ||v|| = sqrt(<v,v>) is well-defined and only zero for the zero vector.

  Q14. What is a 'weighted inner product'?
    [ ] An inner product defined only on unit vectors
    [*] An inner product where each component is multiplied by a positive weight before summing
    [ ] An inner product where the result is always 1
    [ ] An inner product that only uses some components
    Explanation: A weighted inner product on R^n has the form <u,v> = sum of w_i * u_i * v_i where all weights w_i > 0. The positive weights allow emphasizing certain components while still satisfying all inner product axioms.

  Q15. What does the 'Riesz Representation Theorem' state in the context of Hilbert spaces?
    [ ] Every Hilbert space is finite-dimensional
    [ ] Every inner product space has a countable basis
    [ ] Every bounded operator on a Hilbert space is self-adjoint
    [*] Every continuous linear functional on a Hilbert space can be represented as an inner product with a unique vector
    Explanation: The Riesz Representation Theorem states that for every continuous linear functional f on a Hilbert space H, there exists a unique vector y in H such that f(x) = <x, y> for all x in H.

  Q16. What is an 'orthogonal complement' in an inner product space?
    [ ] The set of all unit vectors in the space
    [ ] The subspace spanned by the eigenvectors
    [*] The set of all vectors orthogonal to every vector in a given subspace
    [ ] The set of vectors with negative inner product
    Explanation: The orthogonal complement W^perp of a subspace W consists of all vectors v such that <v, w> = 0 for every w in W. The space decomposes as V = W + W^perp (direct sum).

  Q17. What does 'bilinearity' mean for an inner product on a real vector space?
    [ ] The inner product equals zero for all vector pairs
    [ ] The inner product can only be computed for pairs of vectors
    [ ] The inner product uses two different norms
    [*] The inner product is linear in both arguments simultaneously
    Explanation: Bilinearity means the inner product is linear in each argument: <au + bw, v> = a<u,v> + b<w,v> and similarly in the second argument. For complex spaces, bilinearity is replaced by sesquilinearity.

  Q18. What is a 'pre-Hilbert space'?
    [ ] A vector space without any inner product
    [*] An inner product space that may not be complete
    [ ] A normed space that satisfies the parallelogram law
    [ ] A Hilbert space with a finite basis
    Explanation: A pre-Hilbert space is an inner product space that is not necessarily complete. Every Hilbert space is a pre-Hilbert space, but not every pre-Hilbert space is a Hilbert space (it may have 'gaps' where Cauchy sequences don't converge).

  Q19. What does 'sesquilinearity' mean for a complex inner product?
    [ ] The inner product is linear in both arguments
    [*] The inner product is linear in one argument and conjugate-linear in the other
    [ ] The inner product is defined using sequences
    [ ] The inner product is always real-valued
    Explanation: Sesquilinearity (meaning 'one-and-a-half linearity') means the inner product is linear in one argument and conjugate-linear in the other: <au, v> = conj(a)<u, v>. This ensures <v, v> is real.

  Q20. What is the 'Gram matrix' associated with vectors v1, ..., vk in an inner product space?

    [*] The matrix G with entries G_ij = <v_i, v_j>
    [ ] The inverse of the matrix [v1 | ... | vk]
    [ ] The matrix whose columns are v1, ..., vk
    Explanation: The Gram matrix G has entries G_ij = <v_i, v_j>. It is always positive semi-definite, and it is positive definite if and only if the vectors are linearly independent.

  Q21. A signal processing engineer defines an inner product on the space of square-integrable functions on [0, 2pi] as <f, g> = integral of f(x)g(x)dx. She computes <sin(x), cos(x)> over [0, 2pi]. What is the result?

    [ ] 2pi
    [ ] pi
    [ ] 1

    [ ] -2

    [ ] 0

--- Lesson 8.6: Applications in Data Compression (PCA) (Summary) ---

  Key Concepts: Applications in Data Compression (PCA)
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If a 2D dataset has covariance matrix [[4, 0], [0, 1]], what is the first principal component direction?
Since the covariance matrix is diagonal, its eigenvectors are the standard basis vectors. The largest eigenvalue is 4, corresponding to eigenvector (1, 0), which is the first principal component.
In PCA, if you keep only the top k principal components from an n-dimensional dataset (k < n), what is this process called?
Keeping only the top k principal components reduces the data from n dimensions to k dimensions, a process called dimensionality reduction. This retains the most important variance while discarding less informative dimensions.
In image compression using PCA, each image is typically represented as a vector. For a 64x64 grayscale image, what is the dimension of this vector?
A 64x64 image has 64 * 64 = 4096 pixels. Each pixel value becomes one component of the vector, so the image is represented as a 4096-dimensional vector.
What is a 'principal component' in PCA?
A principal component is an eigenvector of the data's covariance matrix, pointing in a direction that captures maximum variance. Principal components are ordered by decreasing eigenvalue (variance explained).
What is a 'scree plot' in PCA?
A scree plot displays eigenvalues (or explained variance ratios) in decreasing order. The 'elbow' in the plot helps determine how many principal components to retain for adequate variance explanation.
What is 'reconstruction error' in PCA?
Reconstruction error measures how well the reduced-dimension representation approximates the original data. It equals the sum of eigenvalues of the discarded components.
What is an 'eigenface' in the context of PCA applied to face recognition?
Eigenfaces are the principal components of a face image dataset. Each eigenface represents a fundamental face pattern, and any face can be approximately reconstructed as a weighted sum of eigenfaces.
What is the 'score' of a data point in PCA?
The PCA score of a data point is its projection onto the principal components, giving its coordinates in the new basis. These scores are the reduced-dimension representation of the original data.
A music streaming service uses PCA to compress user preference vectors. If a user's original preference vector is x = (3, 1, 4, 1, 5) and the first principal component is e1 = (0, 0, 0, 0, 1), what is the user's score on the first component?
The score is the projection x . e1 = 3(0) + 1(0) + 4(0) + 1(0) + 5(1) = 5. This means the user's preference is primarily captured by the 5th genre dimension in this principal direction.
A text mining researcher represents 10,000 documents using a vocabulary of 50,000 words (TF-IDF vectors). She applies PCA to reduce to 100 dimensions. What is the approximate compression ratio?
Each document goes from 50,000 dimensions to 100 dimensions, a compression ratio of 50,000:100 = 500:1. The 100 PCA dimensions capture the most important semantic patterns in the vocabulary.
Real-World Applications
Example: What percentage of total variance is captured by a principal component with eigenvalue 6 if the total variance (sum of all eigenvalues) is 10?
The proportion of variance explained by a component is its eigenvalue divided by the sum of all eigenvalues: 6/10 = 60%.
Example: In image compression using PCA, each image is typically represented as a vector. For a 64x64 grayscale image, what is the dimension of this vector?
A 64x64 image has 64 * 64 = 4096 pixels. Each pixel value becomes one component of the vector, so the image is represented as a 4096-dimensional vector.
Example: What is the 'score' of a data point in PCA?
The PCA score of a data point is its projection onto the principal components, giving its coordinates in the new basis. These scores are the reduced-dimension representation of the original data.
Example: A genomics researcher has gene expression data with 20,000 genes across 200 patients. She applies PCA and finds that the first 3 components explain 80% of the variance. How much storage does the compressed representation require compared to the original?
The compressed representation stores 200 * 3 = 600 scores plus 3 component vectors of length 20,000 (60,000 values) = 60,600 total, versus 4,000,000 in the original. This is roughly a 66:1 compression ratio.
Example: A computer vision engineer applies PCA to a set of 1000 face images, each 100x100 pixels. The covariance matrix she computes is of what size?
Each 100x100 image is a 10,000-dimensional vector. The covariance matrix has dimensions equal to the number of features: 10,000 x 10,000. In practice, the smaller 1000x1000 matrix X X^T is often used instead.
Example: A music streaming service uses PCA to compress user preference vectors. If a user's original preference vector is x = (3, 1, 4, 1, 5) and the first principal component is e1 = (0, 0, 0, 0, 1), what is the user's score on the first component?
The score is the projection x . e1 = 3(0) + 1(0) + 4(0) + 1(0) + 5(1) = 5. This means the user's preference is primarily captured by the 5th genre dimension in this principal direction.
Example: A satellite imaging company uses PCA for image compression. An image is stored as a 500x500 matrix. Using rank-10 PCA approximation, how many values need to be stored instead of 250,000?
A rank-10 approximation of a 500x500 matrix stores 10 left singular vectors (500 each), 10 right singular vectors (500 each), and 10 singular values: 10*500 + 10*500 + 10 = 10,010 values, approximately 10,000.

--- Lesson 8.7: Applications in Machine Learning (Summary) ---

  Key Concepts: Applications in Machine Learning
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
The kernel trick in SVMs implicitly computes inner products in a higher-dimensional space. What is the polynomial kernel of degree 2 for vectors x and y in R^2?
The polynomial kernel of degree 2 is K(x,y) = (x . y + c)^2 (commonly c = 1). This computes the inner product in a feature space that includes all monomials up to degree 2 without explicitly constructing the high-dimensional features.
What is a 'feature vector' in machine learning?
A feature vector is a point in a high-dimensional space where each coordinate represents a measurable attribute of the data. Machine learning algorithms operate on these vector representations.
What is 'regularization' in the context of linear models?
Regularization adds a penalty like lambda * ||w||^2 (L2/Ridge) or lambda * ||w||_1 (L1/Lasso) to the loss function. This discourages large weights, reducing overfitting and improving generalization.
What is the 'Hessian matrix' in optimization for machine learning?
The Hessian H has entries H_ij = d^2L/dw_i dw_j. Its eigenvalues indicate curvature: positive eigenvalues mean the function curves upward (local minimum direction), negative means downward (local maximum direction), and mixed indicates a saddle point.
What is a 'latent factor' in matrix factorization for recommendation systems?
Latent factors are hidden dimensions discovered by matrix factorization. For movies, latent factors might correspond to genres, era, mood, or other abstract qualities that explain rating patterns.
What is 'spectral clustering' in machine learning?
Spectral clustering constructs a graph Laplacian from the data similarity matrix, then uses its smallest eigenvectors to embed data in a low-dimensional space where clusters are easily separable.
What is the 'Moore-Penrose pseudoinverse' used for in machine learning?
The Moore-Penrose pseudoinverse A^+ generalizes matrix inversion to non-square and singular matrices. It gives the minimum-norm least-squares solution x = A^+ b and can be computed via SVD.
A data scientist fits a linear regression model y = w1*x1 + w2*x2 + b to 100 data points. The design matrix A is 100x3 (including a column of 1s for the bias). What is the dimension of A^T A?
A is 100x3, so A^T is 3x100, and A^T A is 3x3. This small system is computationally easy to solve, giving the least-squares weights.
A recommendation engine uses matrix factorization to predict missing ratings. If user i's embedding is u_i = (2, 1) and item j's embedding is v_j = (3, 4), what is the predicted rating?
The predicted rating is u_i . v_j = 2*3 + 1*4 = 6 + 4 = 10. The dot product of user and item embeddings estimates how much the user would like the item.
A natural language processing engineer computes cosine similarity between document vectors d1 = (3, 4) and d2 = (4, 3). What is the cosine similarity?
cos(theta) = (d1 . d2) / (||d1|| ||d2||) = (12 + 12) / (5 * 5) = 24/25 = 0.96. The documents are very similar in direction.
A speech recognition system uses an orthogonal transformation to convert mel-spectrogram features. If the transformation matrix Q is 40x40 orthogonal, what is the computational benefit of using Q^T instead of Q^(-1)?
For orthogonal Q, Q^(-1) = Q^T. Computing a transpose is O(n^2) (just rearranging entries) while computing a general inverse is O(n^3). For a 40x40 matrix, this means 1,600 vs ~64,000 operations.
Real-World Applications
Example: In a convolutional neural network, a convolution operation can be expressed as multiplication by a Toeplitz matrix. This matrix has what special structure?
A Toeplitz matrix has constant values along each descending diagonal, reflecting the translation-invariant nature of convolution. The kernel weights appear shifted along the diagonals.

W1 has 100 * 50 = 5,000 parameters and W2 has 50 * 10 = 500 parameters, for a total of 5,500 trainable weight parameters.

--- Lesson 8.8: Applications in Quantum Mechanics (Summary) ---

  Key Concepts: Applications in Quantum Mechanics
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
If a qubit is in state |psi> = (1/sqrt(2))|0> + (1/sqrt(2))|1>, what is the probability of measuring |0>?
The probability of measuring |0> is |<0|psi>|^2 = |1/sqrt(2)|^2 = 1/2. This is Born's rule: the measurement probability equals the squared magnitude of the inner product with the measurement basis state.
What is a 'Hermitian matrix' in quantum mechanics?
A Hermitian matrix satisfies A = A* (conjugate transpose). Its key properties are: real eigenvalues, orthogonal eigenvectors, and spectral decomposition -- all essential for quantum observables.
What is a 'unitary matrix' in quantum computing?
A unitary matrix satisfies U*U = UU* = I (conjugate transpose equals inverse). It preserves inner products and norms, ensuring quantum state evolution is reversible and probability-preserving.
What is 'entanglement' in terms of tensor products?
An entangled state like (|00> + |11>)/sqrt(2) cannot be factored as |a> (x) |b> for any single-qubit states |a> and |b>. Entanglement is a fundamentally non-classical correlation captured by the tensor product structure.
What is the 'spectral decomposition' of a Hermitian matrix?
The spectral decomposition A = sum lambda_i |v_i>
What is a 'density matrix' in quantum mechanics?
A density matrix rho is a positive semi-definite Hermitian matrix with trace(rho) = 1. For a pure state, rho = |psi>
What is the 'commutator' of two matrices A and B?
The commutator [A, B] = AB - BA measures how much A and B fail to commute. In quantum mechanics, [A, B] != 0 means the observables A and B cannot be simultaneously measured with certainty (uncertainty principle).
What is a 'projection operator' (projector) in quantum mechanics?
A projection operator is Hermitian (P = P*) and idempotent (P^2 = P). In measurement, P_i = |v_i>/||P_i|psi>||.
What is the 'Bloch sphere' representation?
The Bloch sphere maps each qubit pure state |psi> = cos(theta/2)|0> + e^{i*phi}sin(theta/2)|1> to a point on the unit sphere using angles (theta, phi). The north pole is |0>, south is |1>, and the equator contains superposition states.
A quantum computing researcher prepares the state |psi> = (3/5)|0> + (4/5)|1>. She measures in the computational basis. What is the probability of getting outcome |1>?
By Born's rule, P(|1>) = |<1|psi>|^2 = |4/5|^2 = 16/25 = 0.64. Note that 9/25 + 16/25 = 1, confirming the state is properly normalized.
A physicist computes the expectation value of the Pauli-Z matrix Z = [[1,0],[0,-1]] in the state |psi> = (1/sqrt(2))(|0> + |1>). What is ?
=  = (1/2)[1,1][[1,0],[0,-1]][1,1]^T = (1/2)[1,1][1,-1]^T = (1/2)(1-1) = 0. The equal superposition has zero expectation for Z because +1 and -1 outcomes are equally likely.

  Q1. In quantum mechanics, the state of a qubit is represented as a vector in which space?
    [ ] R^3 (3D real vector space)
    [ ] R^2 (2D real vector space)
    [*] C^2 (2D complex vector space, a Hilbert space)
    [ ] The space of 2x2 real matrices
    Explanation: A qubit state is a unit vector in C^2, the 2-dimensional complex Hilbert space. The state |psi> = alpha|0> + beta|1> where |alpha|^2 + |beta|^2 = 1.

  Q2. The inner product <psi|phi> in quantum mechanics uses which type of inner product?
    [ ] The standard real dot product as predicted by standard models
    [*] The Hermitian (complex) inner product: <psi|phi> = sum of conj(psi_i) * phi_i
    [ ] The Frobenius inner product in accordance with classical theory
    [ ] The cross product, particularly regarding hermitian under
    Explanation: Quantum mechanics uses the Hermitian inner product where <psi|phi> = sum of conjugate(psi_i) * phi_i. This ensures <psi|psi> is real and non-negative, representing the probability normalization.

  Q3. Observable quantities in quantum mechanics are represented by which type of matrix?
    [ ] Diagonal matrices only
    [ ] Upper triangular matrices
    [*] Hermitian (self-adjoint) matrices: A = A*
    [ ] Any invertible matrix
    Explanation: Observables are Hermitian matrices (A = A*, where A* is the conjugate transpose). Hermitian matrices always have real eigenvalues, which correspond to the possible measurement outcomes.

  Q4. The Pauli-X matrix is [[0, 1], [1, 0]]. What are its eigenvalues?
    [*] +1 and -1
    [ ] i and -i
    [ ] 1 and 2
    [ ] 0 and 1
    Explanation: det(X - lambda*I) = -lambda^2 + 1 = 0 gives lambda = +1 and -1. The Pauli-X matrix is Hermitian, so its eigenvalues are real.

  Q5. If a qubit is in state |psi> = (1/sqrt(2))|0> + (1/sqrt(2))|1>, what is the probability of measuring |0>?
    [ ] 1/sqrt(2)
    [*] 1/2
    [ ] 1/4
    [ ] 1
    Explanation: The probability of measuring |0> is |<0|psi>|^2 = |1/sqrt(2)|^2 = 1/2. This is Born's rule: the measurement probability equals the squared magnitude of the inner product with the measurement basis state.

  Q6. Quantum gates are represented by which type of matrix?
    [ ] Singular matrices
    [ ] Symmetric matrices
    [ ] Positive definite matrices
    [*] Unitary matrices: U*U = I
    Explanation: Quantum gates are unitary matrices because quantum evolution must preserve the total probability (norm) of the state vector. U*U = I ensures ||U|psi>|| = ||psi|| = 1.

  Q7. The Hadamard gate H = (1/sqrt(2))[[1, 1], [1, -1]] maps |0> to which state?
    [ ] |1>
    [*] (|0> + |1>)/sqrt(2)
    [ ] (|0> - |1>)/sqrt(2)
    [ ] i|0>
    Explanation: H|0> = (1/sqrt(2))[[1,1],[1,-1]] * [1,0]^T = (1/sqrt(2))[1, 1]^T = (|0> + |1>)/sqrt(2). This creates an equal superposition state.

  Q8. The tensor product of two qubit states |a> and |b> creates a state in which space?
    [ ] R^4
    [*] C^4
    [ ] C^2
    [ ] C^3
    Explanation: The tensor product of two C^2 spaces gives C^2 (x) C^2 = C^4. A 2-qubit state is a vector in C^4 with components corresponding to |00>, |01>, |10>, |11>.

  Q9. The Pauli-Z matrix is [[1, 0], [0, -1]]. What effect does it have on the state |1> = (0, 1)^T?
    [ ] It leaves |1> unchanged
    [*] It maps |1> to -|1>
    [ ] It maps |1> to i|1>
    [ ] It maps |1> to |0>
    Explanation: Z|1> = [[1,0],[0,-1]] * [0,1]^T = [0,-1]^T = -|1>. The Pauli-Z gate applies a phase flip, multiplying the |1> component by -1.

  Q10. The expectation value of an observable A in state |psi> is given by:
    [*] <psi|A|psi>
    [ ] trace(A)
    [ ] ||A|psi>||
    [ ] det(A)
    Explanation: The expectation value <A> = <psi|A|psi> gives the average measurement outcome when the observable A is measured on many copies of the state |psi>.

  Q11. What is a 'Hermitian matrix' in quantum mechanics?
    [*] A matrix equal to its conjugate transpose: A = A*
    [ ] A matrix whose determinant is 1
    [ ] A matrix with all real entries
    [ ] A matrix that commutes with every other matrix
    Explanation: A Hermitian matrix satisfies A = A* (conjugate transpose). Its key properties are: real eigenvalues, orthogonal eigenvectors, and spectral decomposition -- all essential for quantum observables.

  Q12. What is a 'unitary matrix' in quantum computing?
    [*] A matrix that preserves inner products: <Ux|Uy> = <x|y>, satisfying U*U = I
    [ ] A matrix that maps every state to a basis state
    [ ] A matrix with unit determinant
    [ ] A matrix where every entry has absolute value 1
    Explanation: A unitary matrix satisfies U*U = UU* = I (conjugate transpose equals inverse). It preserves inner products and norms, ensuring quantum state evolution is reversible and probability-preserving.

  Q13. What does 'superposition' mean in quantum mechanics?
    [ ] A state with zero probability of being measured
    [ ] The product of two quantum states
    [*] A quantum state expressed as a linear combination of basis states
    [ ] Two particles occupying the same position
    Explanation: Superposition is the principle that a quantum state can be a linear combination of basis states: |psi> = alpha|0> + beta|1>. This is a direct consequence of quantum states living in a linear (vector) space.

  Q14. What is 'entanglement' in terms of tensor products?
    [ ] A state with maximum energy
    [*] A two-particle state that cannot be written as a tensor product of individual particle states
    [ ] The tensor product of any two states
    [ ] A state where two particles are physically connected
    Explanation: An entangled state like (|00> + |11>)/sqrt(2) cannot be factored as |a> (x) |b> for any single-qubit states |a> and |b>. Entanglement is a fundamentally non-classical correlation captured by the tensor product structure.

  Q15. What is the 'spectral decomposition' of a Hermitian matrix?
    [ ] Breaking the matrix into its real and imaginary parts
    [ ] Computing the matrix's characteristic polynomial
    [*] Writing the matrix as a sum of outer products of its eigenvectors weighted by eigenvalues: A = sum lambda_i |v_i><v_i|
    [ ] Factoring the matrix into L and U triangular matrices
    Explanation: The spectral decomposition A = sum lambda_i |v_i><v_i| expresses a Hermitian matrix as a weighted sum of projection operators onto its eigenvectors. This is fundamental to quantum measurement theory.

  Q16. What does 'Born's rule' state about quantum measurement?
    [ ] Every measurement yields the largest eigenvalue
    [ ] Repeated measurements always give the same result
    [*] The probability of measuring eigenvalue lambda_i is |<v_i|psi>|^2, the squared magnitude of the inner product
    [ ] The measurement outcome is always the expectation value
    Explanation: Born's rule connects linear algebra to physics: the probability of obtaining measurement result lambda_i is P(lambda_i) = |<v_i|psi>|^2, where |v_i> is the corresponding eigenvector.

  Q17. What is a 'density matrix' in quantum mechanics?
    [*] A positive semi-definite Hermitian matrix with trace 1 that describes a quantum state (possibly mixed)
    [ ] A matrix that measures the density of particles
    [ ] The inverse of the Hamiltonian
    [ ] A diagonal matrix of probabilities
    Explanation: A density matrix rho is a positive semi-definite Hermitian matrix with trace(rho) = 1. For a pure state, rho = |psi><psi|. For mixed states (statistical ensembles), it generalizes the state vector description.

  Q18. What is the 'commutator' of two matrices A and B?
    [*] [A, B] = AB - BA
    [ ] [A, B] = AB + BA
    [ ] [A, B] = A + B
    [ ] [A, B] = A^(-1) B
    Explanation: The commutator [A, B] = AB - BA measures how much A and B fail to commute. In quantum mechanics, [A, B] != 0 means the observables A and B cannot be simultaneously measured with certainty (uncertainty principle).

  Q19. What is a 'projection operator' (projector) in quantum mechanics?
    [ ] Any matrix that maps vectors to a lower-dimensional space
    [*] A Hermitian matrix P satisfying P^2 = P, representing a measurement outcome
    [ ] The outer product of two different eigenvectors
    [ ] A unitary matrix that rotates states
    Explanation: A projection operator is Hermitian (P = P*) and idempotent (P^2 = P). In measurement, P_i = |v_i><v_i| projects onto the eigenspace of outcome lambda_i. After measurement, the state collapses to P_i|psi>/||P_i|psi>||.

  Q20. What is the 'Bloch sphere' representation?
    [*] A geometric representation where every qubit pure state maps to a point on the surface of a unit sphere
    [ ] A 3D plot of eigenvalues
    [ ] The set of all unitary 2x2 matrices
    [ ] A spherical coordinate system for 3-qubit states
    Explanation: The Bloch sphere maps each qubit pure state |psi> = cos(theta/2)|0> + e^{i*phi}sin(theta/2)|1> to a point on the unit sphere using angles (theta, phi). The north pole is |0>, south is |1>, and the equator contains superposition states.

  Q21. A quantum computing researcher prepares the state |psi> = (3/5)|0> + (4/5)|1>. She measures in the computational basis. What is the probability of getting outcome |1>?
    [ ] 9/25
    [ ] 3/5
    [ ] 4/5
    [*] 16/25

  Q22. A physicist computes the expectation value of the Pauli-Z matrix Z = [[1,0],[0,-1]] in the state |psi> = (1/sqrt(2))(|0> + |1>). What is <Z>?
