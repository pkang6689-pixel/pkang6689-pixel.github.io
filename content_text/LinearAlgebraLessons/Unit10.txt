--- Lesson 10.1: Proofs in Linear Algebra (Summary) ---

  Key Concepts: Proofs in Linear Algebra
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is a 'direct proof' in linear algebra?
A direct proof follows a chain of logical implications from the given hypotheses to the desired conclusion, using definitions, properties of operations, and previously established theorems at each step.
What is a 'counterexample' in linear algebra proofs?
A single counterexample suffices to disprove a universal claim. For instance, the claim 'all matrices commute' is disproven by the counterexample AB != BA for A = [[1,1],[0,0]] and B = [[0,0],[1,1]].
What is the proof strategy for 'if and only if' (iff) statements?
An 'if and only if' statement P <=> Q is a biconditional requiring two proofs: the forward direction (P => Q) and the converse (Q => P). Both must be established for the equivalence to hold.
What is the 'contrapositive' of an implication, and why is it useful in proofs?
The contrapositive of 'P => Q' is 'not Q => not P', and they are logically equivalent. Sometimes the contrapositive is easier to prove. For example: 'if det(A) != 0 then A is invertible' is equivalent to 'if A is not invertible then det(A) = 0'.
What is an 'existence proof' in linear algebra?
Existence proofs come in two types: constructive (explicitly building the object, like constructing the inverse of a matrix) and non-constructive (showing that non-existence leads to contradiction).
What is a 'constructive proof'?
A constructive proof provides an explicit method to find the claimed object. For example, proving that every finite-dimensional vector space has a basis by constructively building one using the Gram-Schmidt process.
A student must prove that if A is an n x n matrix with A^2 = A (idempotent), then the eigenvalues of A are 0 or 1. She starts with Av = lambda*v. What is the next step?
From Av = lambda*v, A^2 v = lambda^2*v. Since A^2 = A, also A^2 v = Av = lambda*v. So lambda^2*v = lambda*v, giving (lambda^2 - lambda)v = 0. Since v != 0, lambda^2 - lambda = 0, so lambda(lambda - 1) = 0, meaning lambda = 0 or 1.
A student must prove that the determinant of an orthogonal matrix Q is +1 or -1. She writes: 'Q^T Q = I, so det(Q^T Q) = det(I) = 1.' What is the key algebraic step to conclude?
Using det(AB) = det(A)*det(B) and det(Q^T) = det(Q): det(Q^T Q) = det(Q^T)*det(Q) = det(Q)^2. Since det(Q^T Q) = det(I) = 1, we get det(Q)^2 = 1, so det(Q) = +/-1.
Real-World Applications
Example: What is a 'counterexample' in linear algebra proofs?
A single counterexample suffices to disprove a universal claim. For instance, the claim 'all matrices commute' is disproven by the counterexample AB != BA for A = [[1,1],[0,0]] and B = [[0,0],[1,1]].
Example: A student needs to prove: 'If A is invertible and AB = AC, then B = C.' She writes: 'Left-multiply both sides by A^{-1}: A^{-1}(AB) = A^{-1}(AC).' What property justifies the next step (A^{-1}A)B = (A^{-1}A)C?
Associativity: A^{-1}(AB) = (A^{-1}A)B = IB = B. Similarly for the right side. Then B = IB = (A^{-1}A)B = A^{-1}(AB) = A^{-1}(AC) = (A^{-1}A)C = IC = C.

  Q1. To prove that a set W is a subspace of R^n, which three conditions must be verified?

    Explanation: A subspace must contain 0, be closed under addition (u, v in W implies u+v in W), and be closed under scalar multiplication (u in W, c scalar implies cu in W). These ensure W inherits the vector space structure.

  Q2. Which proof technique is used to show that if Av = lambda*v for nonzero v, then A^k v = lambda^k v?

    Explanation: The proof first verifies det(EA) = det(E)*det(A) for each type of elementary matrix E (row swap, scaling, row addition). Since any invertible matrix is a product of elementary matrices, the general result follows by repeated application.

  Q8. A proof that the null space of a matrix is a subspace requires showing closure under addition. Given Au = 0 and Av = 0, what justifies A(u+v) = 0?
    [ ] The determinant of A is zero
    [*] The linearity of matrix multiplication: A(u+v) = Au + Av = 0 + 0 = 0

  Q9. The proof that similar matrices have the same eigenvalues uses the fact that if B = P^{-1}AP, then:

  Q10. To prove the Cauchy-Schwarz inequality |<u,v>| <= ||u|| * ||v|| by algebraic methods, a common approach starts by considering which expression?

--- Lesson 10.2: AP-Style Practice Problems (Summary) ---

  Key Concepts: AP-Style Practice Problems
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the determinant of the matrix [[2, 3], [1, 4]]?
det = 2*4 - 3*1 = 8 - 3 = 5.
If A is a 3x3 matrix with det(A) = 4, what is det(2A)?
det(cA) = c^n * det(A) for an n x n matrix. det(2A) = 2^3 * 4 = 8 * 4 = 32.
If the null space of a 5x7 matrix A has dimension 3, what is the rank of A?
By the rank-nullity theorem: rank + nullity = number of columns. rank + 3 = 7, so rank = 4.
If A has eigenvalues 2, 3, and 5, what is trace(A)?
The trace equals the sum of eigenvalues: 2 + 3 + 5 = 10.
What is the definition of 'linear independence' for vectors v1, ..., vk?
Linear independence means no nontrivial linear combination equals zero. Orthogonality implies independence, but independence does not require orthogonality.
What is the definition of the 'rank' of a matrix?
The rank is the dimension of the column space = the dimension of the row space = the number of pivot positions. It measures the 'effective dimensionality' of the transformation.
What is a 'basis' for a vector space V?
A basis must satisfy two conditions: the vectors are linearly independent (no redundancy) and they span V (every vector in V can be expressed as a linear combination). The number of vectors in a basis equals dim(V).
What is 'row echelon form' (REF)?
REF is a staircase pattern where leading entries step to the right as you go down rows, zeros appear below each pivot, and zero rows are at the bottom. Reduced REF additionally has 1s as pivots and zeros above each pivot.
What is the 'change of basis' matrix?
If B = {b1,...,bn} is a new basis, the change-of-basis matrix P = [b1|...|bn] converts coordinates: [v]_B = P^{-1} v. The coordinate vector in the new basis is obtained by applying P inverse.
What is the 'Gram-Schmidt process'?
Gram-Schmidt orthogonalizes vectors by subtracting projections onto previously computed orthogonal vectors, then normalizing. It produces an orthonormal basis and is the foundation of QR factorization.
What is the 'characteristic polynomial' of a matrix A?
The characteristic polynomial p(lambda) = det(A - lambda I) is a degree-n polynomial whose roots are exactly the eigenvalues of A. Its coefficients encode trace, determinant, and other invariants.
If A is a 4x4 matrix with eigenvalues 1, 2, 3, 4, what is det(A)?
det(A) = product of eigenvalues = 1 * 2 * 3 * 4 = 24.
If A is a 5x5 matrix with rank 3, what is the dimension of the null space?
By rank-nullity: nullity = n - rank = 5 - 3 = 2.
For the matrix A = [[1, 2], [2, 1]], the eigenvalues are 3 and -1. What is an eigenvector for lambda = 3?
(A - 3I)v = 0: [[-2, 2],[2, -2]] v = 0 gives -2v1 + 2v2 = 0, so v1 = v2. An eigenvector is (1, 1).
How to Calculate
Example: What does 'span{v1, v2, ..., vk}' mean?
Solution: The span is the set of all possible linear combinations of the given vectors with arbitrary scalar coefficients. It is the smallest subspace containing all the vectors.

    [ ] The distance between v1 and vk

--- Lesson 10.3: Applications in Computer Science (networks, algorithms) (Summary) ---

  Key Concepts: Applications in Computer Science (networks, algorithms)
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
A webpage links to 3 other pages and no pages link back to it. In the Google PageRank model, what is this page's contribution to each linked page per iteration?
In PageRank, a page distributes its rank equally among all outgoing links. With 3 outgoing links, each linked page receives 1/3 of the page's current rank.
Given the adjacency matrix A = [[0,1,0],[0,0,1],[1,0,0]], what is the trace of A?
The trace is the sum of diagonal entries: 0 + 0 + 0 = 0. A trace of 0 means no self-loops exist in the graph.
What is the computational complexity of standard matrix multiplication for two n x n matrices?
Standard matrix multiplication of two n x n matrices requires three nested loops, each iterating n times, giving O(n^3) time complexity.
What is an adjacency matrix in the context of computer science?
An adjacency matrix is a square matrix of size n x n (where n is the number of nodes) with entry (i,j) equal to 1 if there is an edge from node i to node j, and 0 otherwise.
What is PageRank in the context of linear algebra?
PageRank computes the importance of web pages by finding the dominant eigenvector (eigenvalue 1) of a column-stochastic transition matrix derived from the web's hyperlink structure.
What is a stochastic matrix?
A stochastic matrix has non-negative entries where each column (column-stochastic) or each row (row-stochastic) sums to 1, making it suitable for representing transition probabilities in Markov chains.
What is a 'steady-state vector' in the context of Markov chains?
The steady-state vector is the eigenvector of the transition matrix corresponding to eigenvalue 1, normalized so its entries sum to 1. It represents the long-run probability distribution of the Markov chain.
What is the 'power method' in numerical linear algebra?
The power method starts with a random vector and repeatedly multiplies it by the matrix, normalizing at each step. The vector converges to the eigenvector corresponding to the largest-magnitude eigenvalue.
A web consists of 3 pages. Page 1 links to pages 2 and 3, page 2 links to page 3, and page 3 links to page 1. The column-stochastic transition matrix is M = [[0, 0, 1],[1/2, 0, 0],[1/2, 1, 0]]. What is M times the vector [1/3, 1/3, 1/3]^T?
M*[1/3,1/3,1/3]^T: Row 1: 0*(1/3)+0*(1/3)+1*(1/3)=1/3. Row 2: (1/2)*(1/3)+0*(1/3)+0*(1/3)=1/6. Row 3: (1/2)*(1/3)+1*(1/3)+0*(1/3)=1/6+1/3=1/2. Result: [1/3, 1/6, 1/2]^T.
A data scientist uses the graph Laplacian L = D - A for a 4-node graph where D = diag(2,3,2,1) and A is the adjacency matrix. What is the sum of each row of L?
In the Laplacian L = D - A, row i has the degree d_i on the diagonal and -1 for each neighbor. The row sum is d_i - (sum of adjacencies in row i) = d_i - d_i = 0. Every row of a graph Laplacian sums to 0.
A computer scientist models internet traffic between 3 servers using a Markov chain with transition matrix T = [[0.5, 0.25, 0],[0.5, 0.5, 0.5],[0, 0.25, 0.5]]. What is the probability that a request currently at server 2 will be at server 1 after one step?
In the column-stochastic transition matrix, T[i][j] gives the probability of moving from state j to state i. T[1][2] = 0.25, so there is a 25% probability of transitioning from server 2 to server 1.
A game developer models a random walk on a 4-node cycle graph (1-2-3-4-1). At each step, the walker moves to an adjacent node with equal probability. The transition matrix has entries T[i][j] = 1/2 if nodes i and j are adjacent, 0 otherwise. Starting at node 1, what is the probability of being at node 3 after exactly 2 steps?
From node 1, after 1 step the walker is at node 2 or node 4 (each with prob 1/2). From node 2, prob of reaching node 3 is 1/2. From node 4, prob of reaching node 3 is 1/2. Total: (1/2)(1/2) + (1/2)(1/2) = 1/4 + 1/4 = 1/2.

    Explanation: A^2 = [[2,1,1],[1,2,1],[1,1,2]]. The diagonal entries give walks of length 2 returning to the same node. The trace is 2+2+2 = 6. Each node has 2 such walks (going to one neighbor and back).

--- Lesson 10.4: Applications in Engineering (structural analysis) (Summary) ---

  Key Concepts: Applications in Engineering (structural analysis)
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
A spring system has stiffness matrix K = [[3,-1],[-1,2]]. What is the determinant of K?
det(K) = 3*2 - (-1)*(-1) = 6 - 1 = 5. Since det(K) > 0, the stiffness matrix is positive definite (for this 2x2 case with positive diagonal entries), consistent with a stable structure.
An electrical circuit has 3 unknown currents. Kirchhoff's laws yield the system: 2I1 - I2 = 5, -I1 + 3I2 - I3 = 0, -I2 + 2I3 = 3. What is the coefficient matrix?
The coefficient matrix contains only the coefficients of the unknowns (not the constants on the right side). From the three equations, the matrix is [[2,-1,0],[-1,3,-1],[0,-1,2]].
A control engineer has the state-space model x' = Ax + Bu. If A is a 3x3 matrix, what is the dimension of the state vector x?
In the state-space model x' = Ax + Bu, if A is n x n then x must be n x 1 for the matrix multiplication Ax to be defined. With A being 3x3, x is a 3x1 column vector.
What is a 'stiffness matrix' in structural engineering?
The stiffness matrix K appears in the fundamental equation Kx = f, relating the displacement vector x to the applied force vector f. Each entry K[i][j] represents the force at degree of freedom i due to a unit displacement at degree of freedom j.
What is a 'mass matrix' in vibration analysis?
The mass matrix M appears in the equation of motion Mx'' + Kx = f. It encodes how mass is distributed among the degrees of freedom, relating accelerations to inertial forces.
What is a 'degree of freedom' in the context of structural finite element analysis?
Each degree of freedom represents an independent possible movement (translation or rotation) at a node. A 2D beam node typically has 3 DOFs: horizontal displacement, vertical displacement, and rotation.
What is 'buckling' in terms of eigenvalue problems?
Buckling occurs when a structure suddenly deforms under compression. The critical load is found from the eigenvalue problem (K - lambda*K_g)x = 0, where K is the stiffness matrix and K_g is the geometric stiffness matrix. The smallest eigenvalue gives the critical buckling load.
What is a 'load vector' in structural analysis?
The load vector f contains the external forces and moments applied at each degree of freedom. Solving Kx = f gives the resulting displacements x at each DOF.
A civil engineer designs a bridge truss with 4 joints. The assembled global stiffness matrix is K = [[4,-2,0,0],[-2,5,-2,-1],[0,-2,3,-1],[0,-1,-1,2]]. After applying boundary conditions that fix joint 1 (x1=0), the reduced system becomes a 3x3 system. What is the reduced stiffness matrix?
Fixing joint 1 means removing row 1 and column 1 from K. The remaining 3x3 submatrix is [[5,-2,-1],[-2,3,-1],[-1,-1,2]], which is the reduced stiffness matrix for the free degrees of freedom.
A mechanical engineer analyzes a 2-DOF vibration system with mass matrix M = [[2,0],[0,1]] and stiffness matrix K = [[6,-2],[-2,2]]. To find natural frequencies, they compute M^{-1}K. What is M^{-1}K?
M^{-1} = [[1/2,0],[0,1]]. M^{-1}K = [[1/2,0],[0,1]]*[[6,-2],[-2,2]] = [[(1/2)*6,(1/2)*(-2)],[1*(-2),1*2]] = [[3,-1],[-2,2]].
An electrical engineer models a resistor network with 3 nodes. Using Kirchhoff's laws, the system is Gv = I where G = [[0.5,-0.2,-0.1],[-0.2,0.4,-0.2],[-0.1,-0.2,0.5]] (conductance matrix), v is the voltage vector, and I = [1, 0, -1]^T is the current source vector. What is the sum of all entries in each row of G?
Row 1 sum: 0.5 + (-0.2) + (-0.1) = 0.2. Row 2 sum: -0.2 + 0.4 + (-0.2) = 0. Row 3 sum: -0.1 + (-0.2) + 0.5 = 0.2. The non-zero row sums indicate connections to ground (external conductances).
A robotics engineer models a 2-joint robot arm. The Jacobian matrix J = [[cos(q1)+cos(q1+q2), cos(q1+q2)],[-sin(q1)-sin(q1+q2), -sin(q1+q2)]]. At q1=0, q2=0, what is J?
At q1=0, q2=0: cos(0)+cos(0)=2, cos(0)=1, -sin(0)-sin(0)=0, -sin(0)=0. So J = [[2,1],[0,0]]. Note that det(J)=0, meaning the robot is at a singular configuration.
A thermal engineer models heat conduction in a 1D rod with 3 internal nodes. The system is Kv = q where K = [[2,-1,0],[-1,2,-1],[0,-1,2]] and q = [100, 0, 50]^T represents heat sources. What is the determinant of K?
Expanding along the first row: det(K) = 2*(2*2-(-1)*(-1)) - (-1)*((-1)*2-(-1)*0) + 0 = 2*(4-1) + 1*(-2) = 2*3 + (-2) = 6 - 2 = 4. Since det(K) > 0, the system has a unique solution.
A power systems engineer models a 3-bus electrical network. The admittance matrix Y is 3x3 and symmetric. The engineer finds that Y has eigenvalues 0, 3, and 5. What is the trace of Y?
The trace of a matrix equals the sum of its eigenvalues. trace(Y) = 0 + 3 + 5 = 8.
A materials engineer tests a composite material and measures the stress-strain relationship as a 3x3 symmetric stiffness tensor C. The engineer finds that C has eigenvalues 10, 5, and 2 (in GPa). What is the maximum strain energy factor (largest eigenvalue) and what does it represent physically?
The largest eigenvalue (10 GPa) corresponds to the principal direction along which the material is stiffest. The corresponding eigenvector gives the direction of maximum resistance to deformation.
How to Calculate
Example: A mechanical engineer analyzes a 2-DOF vibration system with mass matrix M = [[2,0],[0,1]] and stiffness matrix K = [[6,-2],[-2,2]]. To find natural frequencies, they compute M^{-1}K. What is M^{-1}K?
Solution: M^{-1} = [[1/2,0],[0,1]]. M^{-1}K = [[1/2,0],[0,1]]*[[6,-2],[-2,2]] = [[(1/2)*6,(1/2)*(-2)],[1*(-2),1*2]] = [[3,-1],[-2,2]].
Real-World Applications

Using Kirchhoff's laws, the system is Gv = I where G = [[0.5,-0.2,-0.1],[-0.2,0.4,-0.2],[-0.1,-0.2,0.5]] (conductance matrix), v is the voltage vector, and I = [1, 0, -1]^T is the current source vector. What is the sum of all entries in each row of G?
Row 1 sum: 0.5 + (-0.2) + (-0.1) = 0.2. Row 2 sum: -0.2 + 0.4 + (-0.2) = 0. Row 3 sum: -0.1 + (-0.2) + 0.5 = 0.2. The non-zero row sums indicate connections to ground (external conductances).

--- Lesson 10.6: Applications in Economics (input-output models) (Summary) ---

  Key Concepts: Applications in Economics (input-output models)
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
A 2-sector economy has consumption matrix A = [[0.2, 0.3],[0.4, 0.1]]. What is I - A?
I - A = [[1,0],[0,1]] - [[0.2,0.3],[0.4,0.1]] = [[1-0.2, 0-0.3],[0-0.4, 1-0.1]] = [[0.8, -0.3],[-0.4, 0.9]].
For the consumption matrix A = [[0.2, 0.3],[0.4, 0.1]], what is det(I - A)?
I - A = [[0.8, -0.3],[-0.4, 0.9]]. det(I - A) = 0.8*0.9 - (-0.3)*(-0.4) = 0.72 - 0.12 = 0.60. Since the determinant is positive, the system has a unique solution.
In portfolio theory, the variance of a portfolio with weight vector w and covariance matrix C is given by w^T C w. If C = [[4,1],[1,9]] and w = [0.5, 0.5]^T, what is the portfolio variance?
Cw = [[4,1],[1,9]]*[0.5,0.5]^T = [2.5, 5]^T. w^T Cw = [0.5,0.5]*[2.5,5]^T = 0.5*2.5 + 0.5*5 = 1.25 + 2.5 = 3.75. The portfolio variance is 3.75.
In game theory, a payoff matrix is P = [[3,-1],[-2,4]]. In a zero-sum game, if Player 1 chooses row 1 and Player 2 chooses column 2, what is Player 1's payoff?
The payoff matrix entry P[1][2] gives Player 1's payoff when Player 1 chooses row 1 and Player 2 chooses column 2. P[1][2] = -1, meaning Player 1 loses 1 unit.
What is the 'Leontief input-output model' in economics?
The Leontief input-output model represents an economy as interconnected sectors where each sector's output is consumed by other sectors. The linear system x = Ax + d captures how production must satisfy both inter-industry demand (Ax) and final demand (d).
What is the 'Leontief inverse' in the input-output model?
The Leontief inverse (I - A)^{-1} can be expanded as I + A + A^2 + A^3 + ... (when convergent), capturing both direct and indirect input requirements. It maps external demand d to total production x = (I-A)^{-1}d.
What is a 'covariance matrix' in financial economics?
The covariance matrix C has diagonal entries C_{ii} = Var(r_i) (the variance of asset i's return) and off-diagonal entries C_{ij} = Cov(r_i, r_j). It is always symmetric and positive semi-definite.
What is a 'transition matrix' in economic Markov models?
In economic Markov models (e.g., brand switching, employment status), the transition matrix T has entry T_{ij} giving the probability of moving from state j to state i. Each column sums to 1.
What is 'Markowitz portfolio optimization'?
Markowitz portfolio optimization minimizes portfolio variance w^T C w subject to constraints on expected return w^T mu = target and budget w^T 1 = 1. The solution uses the covariance matrix C and expected return vector mu.
An economist models a 2-sector economy with consumption matrix A = [[0.1, 0.4],[0.3, 0.2]] and final demand d = [100, 200]^T. They compute I - A = [[0.9, -0.4],[-0.3, 0.8]]. What is det(I - A)?
det(I - A) = 0.9*0.8 - (-0.4)*(-0.3) = 0.72 - 0.12 = 0.60. Since this is positive, the Leontief inverse exists and the economy can satisfy the demand.
Continuing from the previous economy with (I-A) = [[0.9,-0.4],[-0.3,0.8]] and d = [100,200]^T, the economist computes (I-A)^{-1} = (1/0.6)*[[0.8,0.4],[0.3,0.9]]. What is the total production vector x?
x = (I-A)^{-1}d = (1/0.6)*[[0.8,0.4],[0.3,0.9]]*[100,200]^T = (1/0.6)*[0.8*100+0.4*200, 0.3*100+0.9*200]^T = (1/0.6)*[160, 210]^T = [266.67, 350]^T.
A financial analyst has 3 assets with covariance matrix C = [[4,2,0],[2,9,3],[0,3,16]]. They want to invest equally in all three (w = [1/3,1/3,1/3]^T). What is the portfolio variance w^T C w?
Cw = [[4,2,0],[2,9,3],[0,3,16]]*[1/3,1/3,1/3]^T = [6/3, 14/3, 19/3]^T = [2, 14/3, 19/3]^T. w^T Cw = (1/3)*2 + (1/3)*(14/3) + (1/3)*(19/3) = 2/3 + 14/9 + 19/9 = 6/9 + 14/9 + 19/9 = 39/9 = 4.33.
An economist uses a Markov chain to model employment. The transition matrix is T = [[0.9, 0.3],[0.1, 0.7]] (columns: employed, unemployed). Starting with 80% employed and 20% unemployed, what is the employment rate after one period?
x_1 = T*x_0 = [[0.9,0.3],[0.1,0.7]]*[0.8,0.2]^T = [0.9*0.8+0.3*0.2, 0.1*0.8+0.7*0.2]^T = [0.72+0.06, 0.08+0.14]^T = [0.78, 0.22]^T. Employment rate is 78%.
A game theorist analyzes a 2-player zero-sum game with payoff matrix P = [[4, -2],[-1, 3]]. Player 1 uses mixed strategy [p, 1-p] and Player 2 uses mixed strategy [q, 1-q]. What is Player 1's expected payoff as a function of p and q?
The expected payoff is the weighted sum of all entries: E = p*q*P[1][1] + p*(1-q)*P[1][2] + (1-p)*q*P[2][1] + (1-p)*(1-q)*P[2][2] = 4pq - 2p(1-q) - (1-p)q + 3(1-p)(1-q).
A pension fund manager has a portfolio of 2 assets with expected returns mu = [8%, 12%]^T and covariance matrix C = [[4, -1],[-1, 9]]. What is the expected return of the portfolio with weights w = [0.6, 0.4]^T?
Expected return = w^T * mu = 0.6*8% + 0.4*12% = 4.8% + 4.8% = 9.6%. The portfolio expected return is a weighted average of individual asset returns.
An economist studies the long-run behavior of a brand-switching Markov chain with T = [[0.7, 0.4],[0.3, 0.6]]. The steady-state vector q satisfies Tq = q and q1 + q2 = 1. What is the steady-state distribution?
From Tq = q: 0.7*q1 + 0.4*q2 = q1 gives -0.3*q1 + 0.4*q2 = 0, so q2 = (3/4)*q1. With q1 + q2 = 1: q1 + (3/4)*q1 = 1, so (7/4)*q1 = 1, giving q1 = 4/7 and q2 = 3/7.
Real-World Applications
Example: A trade economist models 3 countries exchanging goods. The exchange matrix is E = [[0.6,0.2,0.1],[0.3,0.5,0.3],[0.1,0.3,0.6]]. What percentage of Country 1's production stays within Country 1?
In the exchange matrix, column 1 represents the distribution of Country 1's production. E[1][1] = 0.6 means 60% of Country 1's production stays in Country 1. The remaining 40% is exported (30% to Country 2, 10% to Country 3).
Example: A supply chain manager uses the Leontief model for a 3-sector economy. When you multiply a matrix M by a vector with only the j-th entry non-zero (value c), the result is c times the j-th column of M. So (I-A)^{-1}*[0,10,0]^T = 10 * (column 2 of (I-A)^{-1}), showing the ripple effect through all sectors.

Expected return = w^T * mu = 0.6*8% + 0.4*12% = 4.8% + 4.8% = 9.6%. The portfolio expected return is a weighted average of individual asset returns.

--- Lesson 10.7: Capstone Project: Real-World Linear Algebra Model (Summary) ---

  Key Concepts: Capstone Project: Real-World Linear Algebra Model
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
When building a real-world linear algebra model, what is typically the first step?
The first step in any modeling project is to clearly define the problem and identify which real-world quantities correspond to the unknowns (variables) in the mathematical model.
A capstone project on image recognition uses PCA. If the original data has 1000 features and PCA retains 50 principal components, what is the compression ratio?
The compression ratio is original features / retained components = 1000/50 = 20:1. The data is represented using 50 dimensions instead of 1000, retaining the directions of maximum variance.
A capstone project requires fitting a quadratic y = a + bx + cx^2 to 20 data points. The design matrix A has dimensions 20x3. What is the minimum number of data points needed for a unique least-squares solution?
For A^T A to be invertible, A must have rank 3, which requires at least 3 rows (data points) with distinct x-values. With fewer than 3 points, A^T A is singular and the least-squares solution is not unique.
What is a 'mathematical model' in the context of applied linear algebra?
A mathematical model translates real-world quantities and relationships into mathematical objects (vectors, matrices, equations) that can be analyzed and solved using linear algebra techniques.
What is 'sensitivity analysis' in the context of a linear algebra model?
Sensitivity analysis examines how perturbations in input data propagate through the model to affect outputs. The condition number of the coefficient matrix quantifies this sensitivity: a high condition number means small input changes cause large output changes.
What is a 'Leslie matrix' in population modeling?
A Leslie matrix models discrete-time, age-structured population dynamics. The first row contains fecundity rates (offspring per individual in each age class), and the subdiagonal contains survival probabilities (fraction surviving to the next age class).
What is 'cross-validation' in the context of model evaluation?
Cross-validation partitions data into training and validation sets, fits the model on training data, and evaluates prediction accuracy on validation data. This prevents overfitting and gives an honest estimate of model performance.
What is 'feature engineering' in the context of building a linear algebra model?
Feature engineering involves designing the columns of the design matrix A, which might include polynomial terms (x^2, x^3), interaction terms (x1*x2), logarithmic transforms, or indicator variables to capture non-linear relationships within a linear model framework.
A student's capstone project models predator-prey populations using x_{n+1} = Ax_n with A = [[1.1, -0.2],[0.1, 0.8]] and initial population x_0 = [100, 50]^T. What is x_1?
x_1 = Ax_0 = [[1.1,-0.2],[0.1,0.8]]*[100,50]^T = [1.1*100+(-0.2)*50, 0.1*100+0.8*50]^T = [110-10, 10+40]^T = [100, 50]^T. The populations remain unchanged at this point.
A climate modeling project fits a linear model temperature = a + b*CO2 + c*year to 50 years of data. The design matrix A is 50x3. After solving the normal equations, x_hat = [15.0, 0.03, -0.01]^T. What is the predicted temperature when CO2 = 400 ppm and year = 2020?
Predicted temperature = 15.0 + 0.03*400 + (-0.01)*2020 = 15.0 + 12.0 - 20.2 = 6.8.
A power grid capstone uses the admittance matrix Y (5x5) for load flow analysis. The matrix Y is symmetric with all row sums equal to zero. What is the smallest eigenvalue of Y?
If all row sums are zero, then Y * [1,1,...,1]^T = [0,0,...,0]^T, meaning the all-ones vector is an eigenvector with eigenvalue 0. For a graph Laplacian-type matrix, 0 is the smallest eigenvalue.
Real-World Applications
Example: A modeling project uses SVD to compress a 100x100 image matrix. If only the top 10 singular values are kept, how many values need to be stored (including the corresponding left and right singular vectors)?
For rank-10 SVD approximation: 10 singular values + 10 left singular vectors of size 100 + 10 right singular vectors of size 100 = 10 + 10*100 + 10*100 = 10 + 1000 + 1000 = 2010, compared to 10000 for the full matrix.
Example: In a capstone project on population dynamics, a Leslie matrix L = [[0, 2, 1],[0.5, 0, 0],[0, 0.8, 0]] models age-structured growth. What does the entry L[1][2] = 2 represent?
In a Leslie matrix, the first row contains fecundity rates. L[1][2] = 2 means that on average, each individual in age group 2 produces 2 offspring per time period.

--- Lesson 10.8: Comprehensive Review & AP Exam Practice (Summary) ---

  Key Concepts: Comprehensive Review & AP Exam Practice
Overview
This lesson covers the fundamental measures of center used to analyze and summarize data sets. You'll learn how to calculate and apply the mean, median, and mode, and understand when each is most appropriate.
Key Definitions
What is the determinant of the matrix [[1, 2, 3],[0, 4, 5],[0, 0, 6]]?
For an upper triangular matrix, the determinant is the product of the diagonal entries: 1 * 4 * 6 = 24.
If A is a 4x4 matrix with eigenvalues 1, 2, 3, and 4, what is det(A)?
The determinant of a matrix equals the product of its eigenvalues: det(A) = 1 * 2 * 3 * 4 = 24.
If A is a 3x5 matrix with rank 3, what is the dimension of the null space of A?
By the rank-nullity theorem: dim(null space) = n - rank(A) = 5 - 3 = 2. There are 2 free variables in the solution to Ax = 0.
The matrix A = [[2, 1],[0, 2]] has eigenvalue lambda = 2 with algebraic multiplicity 2. What is the geometric multiplicity?
A - 2I = [[0,1],[0,0]], which has rank 1. The eigenspace dimension (geometric multiplicity) = 2 - rank(A-2I) = 2 - 1 = 1. Since geometric multiplicity < algebraic multiplicity, A is not diagonalizable.
What is the inverse of A = [[1, 2],[3, 7]]?
det(A) = 1*7 - 2*3 = 1. A^{-1} = (1/det)*[[d,-b],[-c,a]] = (1/1)*[[7,-2],[-3,1]] = [[7,-2],[-3,1]].
If A is orthogonal (A^T A = I), what is det(A)?
From A^T A = I, taking determinants: det(A^T)*det(A) = 1, so (det(A))^2 = 1, giving det(A) = +1 or -1. Rotations have det = +1 and reflections have det = -1.
The vectors v1 = (1, 0, 1) and v2 = (0, 1, -1) span a plane in R^3. What is the dimension of this subspace?
The two vectors are linearly independent (neither is a scalar multiple of the other), so they form a basis for the subspace they span. The dimension of a 2-element basis is 2.
For the linear transformation T(x) = Ax where A = [[1, 0],[0, 0]], what is the image (range) of T?
A*[x,y]^T = [x,0]^T for any (x,y). The image is all vectors of the form (a,0), which is the x-axis. This is the column space of A, spanned by column 1 = (1,0).
What is the characteristic polynomial of A = [[3, 1],[0, 5]]?
det(A - lambda*I) = det([[3-lambda, 1],[0, 5-lambda]]) = (3-lambda)(5-lambda) - 0 = lambda^2 - 8*lambda + 15. The eigenvalues are lambda = 3 and lambda = 5 (from the diagonal of the triangular matrix).
What is the definition of 'rank' of a matrix?
The rank of a matrix is the dimension of its column space (or row space), which equals the number of linearly independent rows or columns. In row echelon form, this equals the number of pivot positions.
What is the 'Invertible Matrix Theorem' (in summary)?
The Invertible Matrix Theorem lists many equivalent conditions: A is invertible, det(A) != 0, rank = n, columns are linearly independent, Ax = b has a unique solution for all b, null space = {0}, 0 is not an eigenvalue, and more.
What is the 'Singular Value Decomposition' (SVD) of a matrix?
SVD decomposes any m x n matrix A as U*Sigma*V^T, where U (m x m) and V (n x n) are orthogonal, and Sigma (m x n) is diagonal with non-negative entries sigma_1 >= sigma_2 >= ... >= 0 (singular values). SVD exists for every matrix.
What is the 'Gram-Schmidt process'?
The Gram-Schmidt process takes linearly independent vectors {v1, ..., vk} and produces orthonormal vectors {u1, ..., uk} spanning the same subspace. Each new vector is obtained by subtracting projections onto previous vectors and normalizing.
A student is reviewing for the AP exam. They encounter the matrix A = [[2, -1, 0],[-1, 2, -1],[0, -1, 2]]. What is the trace of A, and how does it relate to the eigenvalues?
trace(A) = 2 + 2 + 2 = 6. The trace of any square matrix equals the sum of its eigenvalues (counted with multiplicity). The eigenvalues of this tridiagonal matrix are 2-sqrt(2), 2, and 2+sqrt(2), which indeed sum to 6.
An AP practice problem asks: given that A is a 3x3 matrix with det(A) = 5, what is det(2A)?
For an n x n matrix, det(cA) = c^n * det(A). Here n=3 and c=2, so det(2A) = 2^3 * 5 = 8 * 5 = 40.
A comprehensive practice problem: if T: R^3 -> R^2 is a linear transformation with matrix A (2x3), what is the maximum possible rank of A?
The rank of an m x n matrix is at most min(m, n). For a 2x3 matrix, rank <= min(2, 3) = 2. Rank 2 is achievable when both rows are linearly independent.
An AP review problem asks: if A is a symmetric matrix with eigenvalues 2, 5, and 7, what is det(A^2)?
det(A) = product of eigenvalues = 2*5*7 = 70. det(A^2) = (det(A))^2 = 70^2 = 4900. Alternatively, eigenvalues of A^2 are 4, 25, 49, and their product is 4*25*49 = 4900.

    Explanation: A*[x,y]^T = [x,0]^T for any (x,y). The image is all vectors of the form (a,0), which is the x-axis. This is the column space of A, spanned by column 1 = (1,0).

  Q10. What is the characteristic polynomial of A = [[3, 1],[0, 5]]?
    [ ] (3 + lambda)(5 + lambda)
    [ ] lambda^2 - 8*lambda + 14
    [*] (3 - lambda)(5 - lambda) = lambda^2 - 8*lambda + 15
    [ ] lambda^2 + 8*lambda + 15
    Explanation: det(A - lambda*I) = det([[3-lambda, 1],[0, 5-lambda]]) = (3-lambda)(5-lambda) - 0 = lambda^2 - 8*lambda + 15. The eigenvalues are lambda = 3 and lambda = 5 (from the diagonal of the triangular matrix).

  Q11. What is the definition of 'rank' of a matrix?
    [ ] The number of rows minus the number of columns in accordance with classical theory
    [ ] The number of non-zero entries in the matrix under equilibrium conditions
    [*] The number of linearly independent rows (or equivalently, columns) of the matrix, equal to the number of pivots in row echelon form
    [ ] The largest entry in the matrix, particularly regarding form in the context of this analysis
    Explanation: The rank of a matrix is the dimension of its column space (or row space), which equals the number of linearly independent rows or columns. In row echelon form, this equals the number of pivot positions.

  Q12. What does the term 'diagonalizable' mean for a square matrix A?
    [ ] A has distinct entries on its diagonal according to conventional analysis
    [ ] The diagonal entries of A are all non-zero under equilibrium conditions
    [ ] A is already a diagonal matrix, particularly regarding then as observed in standard experiments
    [*] A can be written as PDP^{-1} where D is a diagonal matrix of eigenvalues and P's columns are the corresponding eigenvectors
    Explanation: A matrix is diagonalizable if it has n linearly independent eigenvectors, forming the columns of P. Then A = PDP^{-1} where D = diag(lambda_1, ..., lambda_n). This simplifies computations like A^k = PD^kP^{-1}.

  Q13. What is the 'Invertible Matrix Theorem' (in summary)?
    [ ] A theorem stating that all matrices are invertible
    [ ] A theorem that only applies to 2x2 matrices
    [ ] A theorem about how to compute matrix inverses using cofactors only
    [*] A collection of equivalent conditions for an n x n matrix, including: A is invertible iff det(A) != 0 iff rank(A) = n iff Ax = 0 has only the trivial solution
    Explanation: The Invertible Matrix Theorem lists many equivalent conditions: A is invertible, det(A) != 0, rank = n, columns are linearly independent, Ax = b has a unique solution for all b, null space = {0}, 0 is not an eigenvalue, and more.

  Q14. What does 'orthogonal projection' of a vector b onto a subspace W mean?
    [ ] Scaling b so that it lies in W
    [ ] Reflecting b across the subspace W
    [*] The closest vector in W to b, obtained by decomposing b = proj_W(b) + b_perp where b_perp is orthogonal to W
    [ ] The component of b that is perpendicular to W
    Explanation: The orthogonal projection proj_W(b) is the unique vector in W closest to b (minimizing ||b - w|| for w in W). The residual b - proj_W(b) is perpendicular to every vector in W.

  Q15. What is the 'Singular Value Decomposition' (SVD) of a matrix?
    [*] A factorization A = U*Sigma*V^T where U and V are orthogonal and Sigma is diagonal with non-negative singular values
    [ ] The factorization of A into its eigenvalues and eigenvectors
    [ ] A way to split A into symmetric and skew-symmetric parts
    [ ] A decomposition that only works for square matrices
    Explanation: SVD decomposes any m x n matrix A as U*Sigma*V^T, where U (m x m) and V (n x n) are orthogonal, and Sigma (m x n) is diagonal with non-negative entries sigma_1 >= sigma_2 >= ... >= 0 (singular values). SVD exists for every matrix.

  Q16. What does it mean for a set of vectors to be a 'basis' for a vector space V?
    [ ] The set contains all vectors in V, particularly regarding does when measured under controlled conditions
    [ ] The set is orthogonal, particularly regarding basis following conventional understanding
    [*] The set is linearly independent and spans V, providing a unique coordinate representation for every vector in V
    [ ] The set has exactly n vectors for any vector space
    Explanation: A basis for V is a linearly independent set that spans V. Every vector in V can be expressed uniquely as a linear combination of the basis vectors. The number of vectors in a basis defines the dimension of V.

  Q17. What is the 'Gram-Schmidt process'?
    [*] An algorithm that converts a set of linearly independent vectors into an orthonormal set spanning the same subspace
    [ ] An algorithm for solving systems of linear equations
    [ ] A technique for computing the determinant of large matrices
    [ ] A method for finding eigenvalues of symmetric matrices
    Explanation: The Gram-Schmidt process takes linearly independent vectors {v1, ..., vk} and produces orthonormal vectors {u1, ..., uk} spanning the same subspace. Each new vector is obtained by subtracting projections onto previous vectors and normalizing.

  Q18. What does the 'Cayley-Hamilton Theorem' state?
    [ ] The trace equals the sum of eigenvalues only for symmetric matrices
    [ ] A matrix and its transpose always have different eigenvalues
    [ ] Every matrix has real eigenvalues
    [*] Every square matrix satisfies its own characteristic equation: if p(lambda) = det(A - lambda*I), then p(A) = 0
    Explanation: The Cayley-Hamilton theorem states that substituting a matrix A into its own characteristic polynomial yields the zero matrix. For a 2x2 matrix with p(lambda) = lambda^2 - tr(A)*lambda + det(A), we get A^2 - tr(A)*A + det(A)*I = 0.

  Q19. What does 'change of basis' mean in linear algebra?
    [*] Expressing the same vector or linear transformation using a different set of basis vectors, related by a transition matrix
    [ ] Changing the number of dimensions in the vector space
    [ ] Replacing all basis vectors with the standard basis
    [ ] Multiplying all basis vectors by the same scalar
    Explanation: Change of basis converts coordinates from one basis B to another basis B' using the transition matrix P. If [v]_B is the coordinate vector in basis B, then [v]_{B'} = P^{-1}[v]_B. For a transformation A, the representation in the new basis is P^{-1}AP.

  Q20. What does it mean for a matrix to be 'positive definite'?
    [ ] A matrix that is both invertible and symmetric
    [ ] A matrix with all positive entries
    [*] A symmetric matrix A such that x^T A x > 0 for all non-zero vectors x, equivalently all eigenvalues are positive
    [ ] A matrix whose determinant is positive
    Explanation: A symmetric matrix A is positive definite if x^T A x > 0 for every non-zero vector x. Equivalent conditions: all eigenvalues are positive, all leading principal minors are positive, and A has a Cholesky factorization A = LL^T.

  Q21. A student is reviewing for the AP exam. They encounter the matrix A = [[2, -1, 0],[-1, 2, -1],[0, -1, 2]]. What is the trace of A, and how does it relate to the eigenvalues?
    [ ] Trace = 6, which equals the determinant
    [ ] Trace = 3, which equals the product of the eigenvalues
    [ ] Trace = 0, which means A is singular
    [*] Trace = 6, which equals the sum of the eigenvalues
    Explanation: trace(A) = 2 + 2 + 2 = 6. The trace of any square matrix equals the sum of its eigenvalues (counted with multiplicity). The eigenvalues of this tridiagonal matrix are 2-sqrt(2), 2, and 2+sqrt(2), which indeed sum to 6.

  Q22. An AP practice problem asks: given that A is a 3x3 matrix with det(A) = 5, what is det(2A)?
    [ ] 80
    [*] 40
    [ ] 10
    [ ] 20
    Explanation: For an n x n matrix, det(cA) = c^n * det(A). Here n=3 and c=2, so det(2A) = 2^3 * 5 = 8 * 5 = 40.

  Q23. A comprehensive review question asks: the system Ax = b has infinitely many solutions. Which of the following must be true?
    [ ] A must be a square matrix
    [*] The null space of A is non-trivial (contains non-zero vectors)
    [ ] b must be the zero vector
    [ ] det(A) is positive
    Explanation: If Ax = b has infinitely many solutions, then there exist at least two solutions x1 and x2 with x1 != x2. Then A(x1-x2) = 0 with x1-x2 != 0, meaning the null space contains a non-zero vector.

  Q24. A review problem states: if A is a 4x4 matrix and the columns of A are linearly dependent, which of the following is definitely true?
    [ ] The row space of A is trivial
    [ ] A has no eigenvalues
    [*] det(A) = 0
    [ ] All entries of A are zero
    Explanation: If the columns are linearly dependent, then rank(A) < 4, so A is not invertible, and by the Invertible Matrix Theorem, det(A) = 0. Also, 0 is an eigenvalue of A.

  Q25. An exam question asks: if A and B are similar matrices (B = P^{-1}AP for some invertible P), which quantities must A and B share?
    [*] Eigenvalues, determinant, trace, and rank
    [ ] Eigenvalues and eigenvectors
    [ ] Only eigenvalues
    [ ] All entries of the matrix
    Explanation: Similar matrices represent the same linear transformation in different bases. They share eigenvalues (same characteristic polynomial), determinant (det(P^{-1}AP) = det(A)), trace (tr(P^{-1}AP) = tr(A)), and rank. They do NOT generally share eigenvectors.

  Q26. A student encounters this AP problem: the projection of b = [3, 4]^T onto the line spanned by a = [1, 0]^T is what vector?
    [*] [3, 0]^T
    [ ] [0, 4]^T
    [ ] [1, 0]^T
    [ ] [3, 4]^T
    Explanation: proj_a(b) = ((a . b)/(a . a)) * a = ((1*3+0*4)/(1*1+0*0)) * [1,0]^T = (3/1)*[1,0]^T = [3,0]^T. The projection drops the y-component, keeping only the x-component.

  Q27. An exam review asks: which of the following matrices is NOT invertible?
    [ ] [[1, 2],[3, 4]]
    [*] [[1, 2],[2, 4]]
    [ ] [[1, 0],[0, 1]]
    [ ] [[5, 0],[0, 3]]
    Explanation: det([[1,2],[2,4]]) = 1*4 - 2*2 = 0, so it is not invertible. The columns [1,2] and [2,4] are linearly dependent (column 2 = 2 * column 1). The other matrices all have non-zero determinants.

  Q28. A comprehensive practice problem: if T: R^3 -> R^2 is a linear transformation with matrix A (2x3), what is the maximum possible rank of A?
    [ ] 1
    [*] 2
    [ ] 5
    [ ] 3
    Explanation: The rank of an m x n matrix is at most min(m, n). For a 2x3 matrix, rank <= min(2, 3) = 2. Rank 2 is achievable when both rows are linearly independent.

  Q29. An AP review problem asks: if A is a symmetric matrix with eigenvalues 2, 5, and 7, what is det(A^2)?
    [ ] 2450
    [ ] 70
    [ ] 14
    [*] 4900
    Explanation: det(A) = product of eigenvalues = 2*5*7 = 70. det(A^2) = (det(A))^2 = 70^2 = 4900. Alternatively, eigenvalues of A^2 are 4, 25, 49, and their product is 4*25*49 = 4900.

  Q30. A final review question: a 5x5 matrix A has eigenvalues 0, 1, 2, 3, and 4. What can be said about the system Ax = b?
    [ ] Ax = b always has infinitely many solutions when evaluated systematically
    [*] Ax = b may have no solution or infinitely many solutions depending on b, because 0 is an eigenvalue so A is singular
    [ ] Ax = b always has a unique solution as observed in standard experiments
    [ ] Ax = b never has a solution, particularly regarding solution based on the prevailing evidence
    Explanation: Since 0 is an eigenvalue, det(A) = 0*1*2*3*4 = 0, so A is singular (not invertible). By the Invertible Matrix Theorem, Ax = b does not have a unique solution for all b. For some b (those in the column space), infinitely many solutions exist; for other b, no solution exists.
